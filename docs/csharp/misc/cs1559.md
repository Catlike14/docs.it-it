# <a name="compiler-error-cs1559"></a>Errore del compilatore CS1559
Non è possibile usare 'object' per il metodo Main perché è importato  
  
 Una classe non valida specificata per il [/principale](../../csharp/language-reference/compiler-options/main-compiler-option.md) l'opzione del compilatore; la classe non può essere utilizzata come un percorso per il [Main](../../csharp/programming-guide/main-and-command-args/index.md) metodo.
