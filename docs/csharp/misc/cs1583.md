# Errore del compilatore CS1583
'file' non è un file di risorse Win32 valido  
  
 Questo errore si verifica quando si specifica un nome file con l'opzione del compilatore [\/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) che non è un file di risorse Win32 valido o formattato correttamente. In Visual Studio viene specificato il nome del file nel riquadro Applicazione di Creazione progetti.
