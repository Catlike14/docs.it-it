## <a name="example"></a><span data-ttu-id="a71d0-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="a71d0-106">Example</span></span>  
 <span data-ttu-id="a71d0-107">L'esempio seguente genera l'errore C3015.</span><span class="sxs-lookup"><span data-stu-id="a71d0-107">The following sample generates C3015.</span></span>  
  
```csharp  
// CS3015.cs  
// compile with: /target:library  
using System;  
  
[assembly:CLSCompliant(true)]  
public class MyAttribute : Attribute  
{  
   public MyAttribute(int[] ai) {}   // CS3015  
   // try the following line instead  
   // public MyAttribute(int ai) {}  
}  
```
