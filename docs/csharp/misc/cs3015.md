## <a name="example"></a><span data-ttu-id="33f79-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="33f79-106">Example</span></span>  
 <span data-ttu-id="33f79-107">L'esempio seguente genera l'errore C3015.</span><span class="sxs-lookup"><span data-stu-id="33f79-107">The following sample generates C3015.</span></span>  
  
```csharp

[assembly:CLSCompliant(true)]  
public class MyAttribute : Attribute  
{  
   public MyAttribute(int[] ai) {}   // CS3015  
   // try the following line instead  
   // public MyAttribute(int ai) {}  
}  
```
