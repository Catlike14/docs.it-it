## <a name="example"></a><span data-ttu-id="89e82-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="89e82-106">Example</span></span>  
 <span data-ttu-id="89e82-107">L'esempio seguente genera l'errore C3015.</span><span class="sxs-lookup"><span data-stu-id="89e82-107">The following sample generates C3015.</span></span>

```  
[assembly:CLSCompliant(true)]

}  
```
