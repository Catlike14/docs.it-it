# <a name="compiler-error-cs0754"></a><span data-ttu-id="2f285-102">Errore del compilatore CS0754</span><span class="sxs-lookup"><span data-stu-id="2f285-102">Compiler Error CS0754</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="2f285-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="2f285-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="2f285-106">Rimuovere la qualificazione di interfaccia esplicita dalla dichiarazione di metodo.</span><span class="sxs-lookup"><span data-stu-id="2f285-106">Remove the explicit interface qualification from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2f285-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="2f285-107">Example</span></span>  
 <span data-ttu-id="2f285-108">Il codice seguente genera l'errore CS0754:</span><span class="sxs-lookup"><span data-stu-id="2f285-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
