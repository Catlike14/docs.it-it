# <a name="compiler-error-cs1917"></a><span data-ttu-id="58eb3-102">Errore del compilatore CS1917</span><span class="sxs-lookup"><span data-stu-id="58eb3-102">Compiler Error CS1917</span></span>
<span data-ttu-id="58eb3-103">Non è possibile assegnare i membri del campo di sola lettura 'name' di tipo 'struct name' con un inizializzatore di oggetto perché è di un tipo valore.</span><span class="sxs-lookup"><span data-stu-id="58eb3-103">Members of read-only field 'name' of type 'struct name' cannot be assigned with an object initializer because it is of a value type.</span></span>

-   <span data-ttu-id="58eb3-106">Modificare lo struct in un tipo di classe.</span><span class="sxs-lookup"><span data-stu-id="58eb3-106">Change the struct to a class type.</span></span>  
  
-   <span data-ttu-id="58eb3-107">Inizializzare lo struct con un costruttore.</span><span class="sxs-lookup"><span data-stu-id="58eb3-107">Initialize the struct with a constructor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="58eb3-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="58eb3-108">Example</span></span>  
 <span data-ttu-id="58eb3-109">Il codice seguente genera l'errore CS1917:</span><span class="sxs-lookup"><span data-stu-id="58eb3-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```
