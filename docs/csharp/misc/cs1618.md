# <a name="compiler-error-cs1618"></a>Errore del compilatore CS1618
Impossibile creare delegato con 'method'. Dispone di un attributo Conditional  
  
 Non è possibile creare un delegato con un metodo condizionale perché il metodo potrebbe non essere disponibile in alcune build.  
  
 L'esempio seguente genera l'errore CS1618:  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,   
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()   
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
