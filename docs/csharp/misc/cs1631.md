# <a name="compiler-error-cs1631"></a>Errore del compilatore CS1631
Impossibile produrre un valore nel corpo di una clausola catch  
  
 L'istruzione yield non Ã¨ consentita dall'interno del corpo di una clausola catch. Per evitare questo errore, spostare l'istruzione yield al di fuori del corpo della clausola catch.  
  
 L'esempio seguente genera l'errore CS1631:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
