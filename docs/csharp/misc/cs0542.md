# <a name="compiler-error-cs0542"></a><span data-ttu-id="43cab-102">Errore del compilatore CS0542</span><span class="sxs-lookup"><span data-stu-id="43cab-102">Compiler Error CS0542</span></span>
<span data-ttu-id="43cab-103">'user-defined type': i nomi dei membri non possono essere uguali a quelli del tipo di inclusione</span><span class="sxs-lookup"><span data-stu-id="43cab-103">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="43cab-104">I membri di una classe o struct non possono avere lo stesso nome della classe o dello struct, a meno che il membro sia un costruttore.</span><span class="sxs-lookup"><span data-stu-id="43cab-104">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="43cab-105">L'esempio seguente genera l'errore CS0542:</span><span class="sxs-lookup"><span data-stu-id="43cab-105">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="43cab-106">Questo errore può essere generato se si inserisce accidentalmente un tipo restituito in un costruttore, che in effetti lo rende un metodo ordinario.</span><span class="sxs-lookup"><span data-stu-id="43cab-106">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="43cab-107">L'esempio seguente genera l'errore CS0542 perché `F` è un metodo, non un costruttore, perché ha un tipo restituito:</span><span class="sxs-lookup"><span data-stu-id="43cab-107">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="43cab-108">Se una classe è denominata 'Item' e ha un indicizzatore dichiarato come `this`, potrebbe essere visualizzato questo errore.</span><span class="sxs-lookup"><span data-stu-id="43cab-108">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="43cab-109">Nel codice creato il nome 'Item' viene assegnato a un indicizzatore predefinito, provocando il conflitto.</span><span class="sxs-lookup"><span data-stu-id="43cab-109">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>
