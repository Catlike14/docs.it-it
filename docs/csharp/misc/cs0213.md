 <span data-ttu-id="995b8-104">Una variabile locale in un metodo [unsafe](../../csharp/language-reference/keywords/unsafe.md) o in un parametro è già di tipo fixed (nello stack), pertanto non è possibile accettare l'indirizzo di una di queste due variabili in un'espressione di tipo [fixed](../../csharp/language-reference/keywords/fixed-statement.md) .</span><span class="sxs-lookup"><span data-stu-id="995b8-104">A local variable in an [unsafe](../../csharp/language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="995b8-105">Per altre informazioni, vedere [Codice unsafe e puntatori](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="995b8-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="995b8-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="995b8-106">Example</span></span>  
 <span data-ttu-id="995b8-107">L'esempio seguente genera l'errore CS0213.</span><span class="sxs-lookup"><span data-stu-id="995b8-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
