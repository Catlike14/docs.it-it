# <a name="compiler-error-cs0744"></a><span data-ttu-id="bc646-102">Errore del compilatore CS0744</span><span class="sxs-lookup"><span data-stu-id="bc646-102">Compiler Error CS0744</span></span>
<span data-ttu-id="bc646-103">È prevista la parola chiave contestuale 'equals'</span><span class="sxs-lookup"><span data-stu-id="bc646-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="bc646-104">Il percorso per una clausola `join` è `join`...`in`...`on`...`equals`, come mostrato nell'esempio:</span><span class="sxs-lookup"><span data-stu-id="bc646-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y

## <a name="example"></a><span data-ttu-id="bc646-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="bc646-107">Example</span></span>  
 <span data-ttu-id="bc646-108">Il codice seguente genera l'errore CS0744:</span><span class="sxs-lookup"><span data-stu-id="bc646-108">The following code generates CS0744:</span></span>  
  
```  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
