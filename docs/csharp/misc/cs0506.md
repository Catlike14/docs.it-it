# <a name="compiler-error-cs0506"></a><span data-ttu-id="26bd0-102">Errore del compilatore CS0506</span><span class="sxs-lookup"><span data-stu-id="26bd0-102">Compiler Error CS0506</span></span>
<span data-ttu-id="26bd0-103">function1': non è possibile eseguire l'override del membro ereditato 'function2' perché non è contrassegnato come "virtual", "abstract" o "override"</span><span class="sxs-lookup"><span data-stu-id="26bd0-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="26bd0-104">È stato eseguito l'override di un metodo che non era contrassegnato esplicitamente come **virtual**, **abstract**o `override`.</span><span class="sxs-lookup"><span data-stu-id="26bd0-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="26bd0-105">L'esempio seguente genera l'errore CS0506:</span><span class="sxs-lookup"><span data-stu-id="26bd0-105">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
