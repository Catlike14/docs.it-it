<span data-ttu-id="645ca-103">'type': gli elementi di matrice non possono essere di tipo statico</span><span class="sxs-lookup"><span data-stu-id="645ca-103">'type': array elements cannot be of static type</span></span>

 <span data-ttu-id="645ca-105">L'esempio seguente genera l'errore CS0719:</span><span class="sxs-lookup"><span data-stu-id="645ca-105">The following sample generates CS0719:</span></span>  
  
```csharp

{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
