 <span data-ttu-id="24e20-106">L'esempio seguente genera l'errore CS0205:</span><span class="sxs-lookup"><span data-stu-id="24e20-106">The following sample generates CS0205:</span></span>

```  
   abstract public void M();  
}

      base.M();   // CS0205, delete this line

   {  
   }  
}  
```
