 <span data-ttu-id="203df-104">I vincoli sul parametro di tipo di un tipo o metodo generico devono essere disposti in un ordine specifico: prima `class` o `struct` , se presente, quindi i vincoli di interfaccia e infine tutti i vincoli del costruttore.</span><span class="sxs-lookup"><span data-stu-id="203df-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="203df-105">Questo errore deriva dal fatto che il vincolo `class` o `struct` non viene visualizzato per primo.</span><span class="sxs-lookup"><span data-stu-id="203df-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="203df-106">Per risolvere questo errore, riordinare le clausole di vincolo.</span><span class="sxs-lookup"><span data-stu-id="203df-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="203df-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="203df-107">Example</span></span>  
 <span data-ttu-id="203df-108">L'esempio seguente genera l'errore CS0449.</span><span class="sxs-lookup"><span data-stu-id="203df-108">The following sample generates CS0449.</span></span>  
  
```  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4   
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
