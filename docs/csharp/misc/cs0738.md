## <a name="example"></a><span data-ttu-id="b5053-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="b5053-108">Example</span></span>  
 <span data-ttu-id="b5053-109">Il codice seguente genera l'errore CS0738 perché il metodo di classe restituisce `void` e il membro di interfaccia con lo stesso nome restituisce `int`:</span><span class="sxs-lookup"><span data-stu-id="b5053-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```  
{  
    int TestMethod();  
}

{  
    public void TestMethod() { } // CS0738

    // public int TestMethod();  
}  
```
