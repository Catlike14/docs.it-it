## <a name="example"></a><span data-ttu-id="ce2e8-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="ce2e8-108">Example</span></span>  
 <span data-ttu-id="ce2e8-109">Il codice seguente genera l'errore CS0738 perché il metodo di classe restituisce `void` e il membro di interfaccia con lo stesso nome restituisce `int`:</span><span class="sxs-lookup"><span data-stu-id="ce2e8-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

    int TestMethod();
