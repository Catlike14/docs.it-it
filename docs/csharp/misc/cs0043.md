# <a name="compiler-error-cs0043"></a>Errore del compilatore CS0043
Il file PDB 'file' ha un formato non corretto o non aggiornato. Eliminarlo e ricompilare.  
  
 Eliminare il file PDB per questa compilazione e ricompilare.
