## <a name="example"></a><span data-ttu-id="2772b-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="2772b-107">Example</span></span>  
 <span data-ttu-id="2772b-108">Il codice seguente genera l'errore CS0763:</span><span class="sxs-lookup"><span data-stu-id="2772b-108">The following code generates CS0763:</span></span>

    public partial class C  
    {  
        static partial void Part();  
        partial void Part() // CS0763

            return 1;  
        }  
  
    }
