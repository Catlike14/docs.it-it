## <a name="example"></a><span data-ttu-id="b9635-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="b9635-107">Example</span></span>  
 <span data-ttu-id="b9635-108">L'esempio seguente genera l'errore CS0241.</span><span class="sxs-lookup"><span data-stu-id="b9635-108">The following sample generates CS0241.</span></span> <span data-ttu-id="b9635-109">L'esempio seguente mostra anche come simulare, tramite overload, un metodo con argomenti predefiniti.</span><span class="sxs-lookup"><span data-stu-id="b9635-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
