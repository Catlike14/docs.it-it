 <span data-ttu-id="c6188-104">La specifica del linguaggio C# non consente il codice di tipo unsafe negli iteratori.</span><span class="sxs-lookup"><span data-stu-id="c6188-104">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="c6188-105">L'esempio seguente genera l'errore CS1629:</span><span class="sxs-lookup"><span data-stu-id="c6188-105">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs

class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629

         int *p = &i;

      }  
   }  
}
```
