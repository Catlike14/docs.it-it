# <a name="compiler-error-cs2035"></a>Errore del compilatore CS2035
Errore di sintassi della riga di comando: manca ':\<numero >' per l'opzione 'compiler_option'  
  
 Alcune opzioni del compilatore richiedono un valore.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS2035.  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
