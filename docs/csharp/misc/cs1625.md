# <a name="compiler-error-cs1625"></a>Errore del compilatore CS1625
Impossibile eseguire la produzione nel corpo di una clausola finally  
  
 L'istruzione yield non Ã¨ consentita nel corpo di una clausola finally. Per evitare questo errore, spostare l'istruzione yield fuori dalla clausola finally.  
  
 L'esempio seguente genera l'errore CS1625:  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
