 <span data-ttu-id="f98aa-108">L'esempio seguente genera l'errore CS0307:</span><span class="sxs-lookup"><span data-stu-id="f98aa-108">The following sample generates CS0307:</span></span>  
  
```csharp  
// CS0307.cs  
class C  
{  
   public int P { get { return 1; } }  
   public static void Main()  
   {  
      C c = new C();  
      int p = c.P<int>();  // CS0307 – C.P is a property  
      // Try this instead  
      // int p = c.P;  
   }  
}  
```
