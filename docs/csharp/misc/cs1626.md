<span data-ttu-id="7da5e-103">Impossibile produrre un valore nel corpo di un blocco try con una clausola catch</span><span class="sxs-lookup"><span data-stu-id="7da5e-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="7da5e-104">L'istruzione yield non è consentita in un blocco try se è presente una clausola catch associata al blocco try.</span><span class="sxs-lookup"><span data-stu-id="7da5e-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="7da5e-105">Per evitare questo errore, spostare l'istruzione yield fuori dalla clausola catch.</span><span class="sxs-lookup"><span data-stu-id="7da5e-105">To avoid this error, move the yield statement out of the catch clause.</span></span>  
  
 <span data-ttu-id="7da5e-106">L'esempio seguente genera l'errore CS1626:</span><span class="sxs-lookup"><span data-stu-id="7da5e-106">The following sample generates CS1626:</span></span>  
  
```  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
