# <a name="compiler-error-cs1722"></a>Errore del compilatore CS1722
La classe base 'class' deve precedere le interfacce  
  
 Quando si specifica una classe da cui ereditare e le interfacce da implementare, Ã¨ necessario specificare prima il nome della classe.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1722.  
  
```  
// CS1722.cs  
// compile with: /target:library  
public class A {}  
interface I {}  
  
public class MyClass : I, A {}   // CS1722  
public class MyClass2 : A, I {}   // OK  
```
