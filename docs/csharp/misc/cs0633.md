# <a name="compiler-error-cs0633"></a><span data-ttu-id="530aa-102">Errore del compilatore CS0633</span><span class="sxs-lookup"><span data-stu-id="530aa-102">Compiler Error CS0633</span></span>
<span data-ttu-id="530aa-103">L'argomento dell'attributo 'attribute' deve essere un identificatore valido</span><span class="sxs-lookup"><span data-stu-id="530aa-103">The argument to the 'attribute' attribute must be a valid identifier</span></span>  
  
 <span data-ttu-id="530aa-104">Qualsiasi argomento passato agli attributi <xref:System.Diagnostics.ConditionalAttribute> o <xref:System.Runtime.CompilerServices.IndexerNameAttribute> deve essere un identificatore valido.</span><span class="sxs-lookup"><span data-stu-id="530aa-104">Any argument that you pass to the <xref:System.Diagnostics.ConditionalAttribute> or <xref:System.Runtime.CompilerServices.IndexerNameAttribute> attributes must be a valid identifier.</span></span> <span data-ttu-id="530aa-105">Ciò significa che non può contenere caratteri come "+" che non sono validi all'interno degli identificatori.</span><span class="sxs-lookup"><span data-stu-id="530aa-105">This means that it may not contain characters such as "+" that are illegal when they occur in identifiers.</span></span>  
  
## <a name="example"></a><span data-ttu-id="530aa-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="530aa-106">Example</span></span>  
 <span data-ttu-id="530aa-107">L'esempio seguente illustra l'errore CS0633 quando si usa l'attributo <xref:System.Diagnostics.ConditionalAttribute>.</span><span class="sxs-lookup"><span data-stu-id="530aa-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="530aa-108">L'esempio seguente genera l'errore CS0633.</span><span class="sxs-lookup"><span data-stu-id="530aa-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="530aa-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="530aa-109">Example</span></span>  
 <span data-ttu-id="530aa-110">L'esempio seguente illustra l'errore CS0633 quando si usa l'attributo <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span><span class="sxs-lookup"><span data-stu-id="530aa-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
