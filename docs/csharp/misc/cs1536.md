# <a name="compiler-error-cs1536"></a>Errore del compilatore CS1536
Tipo parametro void non è valido  
  
 Non è necessario o non è valido specificare un parametro [void](../../csharp/language-reference/keywords/void.md) diverso da un puntatore void.  
  
 L'esempio seguente genera l'errore CS1536:  
  
```  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
