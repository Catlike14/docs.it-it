# <a name="compiler-error-cs0220"></a><span data-ttu-id="f1640-102">Errore del compilatore CS0220</span><span class="sxs-lookup"><span data-stu-id="f1640-102">Compiler Error CS0220</span></span>
<span data-ttu-id="f1640-103">Operazione in overflow in fase di compilazione in modalità checked</span><span class="sxs-lookup"><span data-stu-id="f1640-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="f1640-104">In modalità [checked](../../csharp/language-reference/keywords/checked.md), ovvero l'impostazione predefinita, è stata rilevata un'operazione che ha provocato una perdita di dati.</span><span class="sxs-lookup"><span data-stu-id="f1640-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="f1640-105">Correggere gli input dell'assegnazione oppure usare [unchecked](../../csharp/language-reference/keywords/unchecked.md) per risolvere l'errore.</span><span class="sxs-lookup"><span data-stu-id="f1640-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="f1640-106">Per ulteriori informazioni, vedere [Checked e Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="f1640-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="f1640-107">L'esempio seguente genera l'errore CS0220:</span><span class="sxs-lookup"><span data-stu-id="f1640-107">The following sample generates CS0220:</span></span>  
  
```  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
