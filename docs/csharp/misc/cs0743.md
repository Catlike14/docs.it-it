# <a name="compiler-error-cs0743"></a><span data-ttu-id="2c406-102">Errore del compilatore CS0743</span><span class="sxs-lookup"><span data-stu-id="2c406-102">Compiler Error CS0743</span></span>

 <span data-ttu-id="2c406-104">Il percorso per una clausola `join` è `join`...`in`...`on`...`equals`, come mostrato nell'esempio:</span><span class="sxs-lookup"><span data-stu-id="2c406-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y

## <a name="example"></a><span data-ttu-id="2c406-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="2c406-107">Example</span></span>  
 <span data-ttu-id="2c406-108">Il codice seguente genera l'errore CS0743:</span><span class="sxs-lookup"><span data-stu-id="2c406-108">The following code generates CS0743:</span></span>  
  
```  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
