# <a name="compiler-error-cs0743"></a><span data-ttu-id="8389a-102">Errore del compilatore CS0743</span><span class="sxs-lookup"><span data-stu-id="8389a-102">Compiler Error CS0743</span></span>
<span data-ttu-id="8389a-103">È prevista la parola chiave contestuale 'on'</span><span class="sxs-lookup"><span data-stu-id="8389a-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="8389a-104">Il percorso per una clausola `join` è `join`...`in`...`on`...`equals`, come mostrato nell'esempio:</span><span class="sxs-lookup"><span data-stu-id="8389a-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8389a-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="8389a-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="8389a-106">Aggiungere la parola chiave `on` alla clausola `join` .</span><span class="sxs-lookup"><span data-stu-id="8389a-106">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8389a-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="8389a-107">Example</span></span>  
 <span data-ttu-id="8389a-108">Il codice seguente genera l'errore CS0743:</span><span class="sxs-lookup"><span data-stu-id="8389a-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
