# <a name="compiler-error-cs0520"></a>Errore del compilatore CS0520
Il tipo predefinito 'name' è dichiarato in modo non corretto  
  
 Uno o più file richiesti dal compilatore non sono stati trovati. Reinstallare Common Language Runtime di .NET Framework.
