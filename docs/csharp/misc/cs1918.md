# <a name="compiler-error-cs1918"></a><span data-ttu-id="97311-102">Errore del compilatore CS1918</span><span class="sxs-lookup"><span data-stu-id="97311-102">Compiler Error CS1918</span></span>
<span data-ttu-id="97311-103">Non è possibile assegnare i membri della proprietà 'name' di tipo 'type' con un inizializzatore di oggetto perché è di un tipo valore.</span><span class="sxs-lookup"><span data-stu-id="97311-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="97311-104">Questo errore si verifica quando si tenta di usare un inizializzatore di oggetto per inizializzare le proprietà di un tipo struct che è una proprietà della classe in fase di inizializzazione.</span><span class="sxs-lookup"><span data-stu-id="97311-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="97311-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="97311-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="97311-106">Se è necessario inizializzare completamente i campi della proprietà nell'inizializzatore di oggetto, impostare lo struct su un tipo classe.</span><span class="sxs-lookup"><span data-stu-id="97311-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="97311-107">In caso contrario, inizializzare i membri dello struct in una chiamata a metodo separata dopo avere creato l'oggetto mediante l'inizializzatore di oggetto.</span><span class="sxs-lookup"><span data-stu-id="97311-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="97311-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="97311-108">Example</span></span>  
 <span data-ttu-id="97311-109">L'esempio seguente genera l'errore CS1918:</span><span class="sxs-lookup"><span data-stu-id="97311-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
