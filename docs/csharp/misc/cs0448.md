# <a name="compiler-error-cs0448"></a><span data-ttu-id="cf024-102">Errore del compilatore CS0448</span><span class="sxs-lookup"><span data-stu-id="cf024-102">Compiler Error CS0448</span></span>
<span data-ttu-id="cf024-103">Il tipo restituito per l'operatore ++ o -- deve essere o derivare dal tipo che lo contiene</span><span class="sxs-lookup"><span data-stu-id="cf024-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>  
  
 <span data-ttu-id="cf024-104">Quando si esegue l'override dell'operatore `++` o `--` , deve essere restituito un tipo identico a quello del tipo contenitore oppure un tipo derivato da quest'ultimo.</span><span class="sxs-lookup"><span data-stu-id="cf024-104">When you override the `++` or `--` operators, they must return the same type as the containing type, or return a type that is derived from the containing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cf024-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="cf024-105">Example</span></span>  
 <span data-ttu-id="cf024-106">L'esempio seguente genera l'errore CS0448.</span><span class="sxs-lookup"><span data-stu-id="cf024-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="cf024-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="cf024-107">Example</span></span>  
 <span data-ttu-id="cf024-108">L'esempio seguente genera l'errore CS0448.</span><span class="sxs-lookup"><span data-stu-id="cf024-108">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
