<span data-ttu-id="fce57-103">Utilizzo della parola chiave 'base' non valido in questo contesto</span><span class="sxs-lookup"><span data-stu-id="fce57-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="fce57-104">La parola chiave [base](../../csharp/language-reference/keywords/base.md) deve essere usata per specificare un particolare membro della classe base.</span><span class="sxs-lookup"><span data-stu-id="fce57-104">The [base](../../csharp/language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="fce57-105">Per altre informazioni, vedere [Costruttori](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="fce57-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="fce57-106">L'esempio seguente genera l'errore CS0175:</span><span class="sxs-lookup"><span data-stu-id="fce57-106">The following sample generates CS0175:</span></span>  
  
```  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
