# <a name="compiler-error-cs0432"></a><span data-ttu-id="d679a-102">Errore del compilatore CS0432</span><span class="sxs-lookup"><span data-stu-id="d679a-102">Compiler Error CS0432</span></span>

 <span data-ttu-id="d679a-104">Questo errore si verifica quando si usa "::" a destra di un identificatore che non è un alias.</span><span class="sxs-lookup"><span data-stu-id="d679a-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="d679a-105">Per risolvere l'errore, usare ".".</span><span class="sxs-lookup"><span data-stu-id="d679a-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="d679a-106">L'esempio seguente genera l'errore CS0432:</span><span class="sxs-lookup"><span data-stu-id="d679a-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
