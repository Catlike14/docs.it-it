 <span data-ttu-id="4be4e-106">L'esempio seguente genera l'errore CS0432:</span><span class="sxs-lookup"><span data-stu-id="4be4e-106">The following example generates CS0432:</span></span>  
  
```  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
