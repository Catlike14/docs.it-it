# <a name="compiler-error-cs0568"></a><span data-ttu-id="e21f4-102">Errore del compilatore CS0568</span><span class="sxs-lookup"><span data-stu-id="e21f4-102">Compiler Error CS0568</span></span>
<span data-ttu-id="e21f4-103">Gli struct non possono contenere costruttori espliciti senza parametri</span><span class="sxs-lookup"><span data-stu-id="e21f4-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="e21f4-104">Ogni [struct](../../csharp/language-reference/keywords/struct.md) include già un costruttore predefinito che inizializza l'oggetto sul valore zero.</span><span class="sxs-lookup"><span data-stu-id="e21f4-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="e21f4-105">I costruttori creati per uno struct devono quindi accettare uno o più parametri.</span><span class="sxs-lookup"><span data-stu-id="e21f4-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="e21f4-106">L'esempio seguente genera l'errore CS0568:</span><span class="sxs-lookup"><span data-stu-id="e21f4-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
