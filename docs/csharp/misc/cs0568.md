# <a name="compiler-error-cs0568"></a><span data-ttu-id="b37f7-102">Errore del compilatore CS0568</span><span class="sxs-lookup"><span data-stu-id="b37f7-102">Compiler Error CS0568</span></span>
<span data-ttu-id="b37f7-103">Gli struct non possono contenere costruttori espliciti senza parametri</span><span class="sxs-lookup"><span data-stu-id="b37f7-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="b37f7-104">Ogni [struct](../../csharp/language-reference/keywords/struct.md) include già un costruttore predefinito che inizializza l'oggetto sul valore zero.</span><span class="sxs-lookup"><span data-stu-id="b37f7-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="b37f7-105">I costruttori creati per uno struct devono quindi accettare uno o più parametri.</span><span class="sxs-lookup"><span data-stu-id="b37f7-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="b37f7-106">L'esempio seguente genera l'errore CS0568:</span><span class="sxs-lookup"><span data-stu-id="b37f7-106">The following sample generates CS0568:</span></span>  
  
```  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
