# <a name="compiler-error-cs0035"></a><span data-ttu-id="90592-102">Errore del compilatore CS0035</span><span class="sxs-lookup"><span data-stu-id="90592-102">Compiler Error CS0035</span></span>
<span data-ttu-id="90592-103">L'operatore 'operator' è ambiguo su un operando di tipo 'type'</span><span class="sxs-lookup"><span data-stu-id="90592-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="90592-104">l compilatore ha a disposizione più conversioni e non è in grado di sceglierne una prima di applicare l'operatore.</span><span class="sxs-lookup"><span data-stu-id="90592-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="90592-105">Per ulteriori informazioni, vedere [le conversioni definite dall'utente basati su modelli](/cpp/misc/templated-user-defined-conversions) e [gli operatori di conversione](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="90592-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="90592-106">L'esempio seguente genera l'errore CS0035:</span><span class="sxs-lookup"><span data-stu-id="90592-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
