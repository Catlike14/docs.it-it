# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="03af2-102">Avviso del compilatore (livello 2) CS0162</span><span class="sxs-lookup"><span data-stu-id="03af2-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="03af2-103">È stato rilevato codice non raggiungibile</span><span class="sxs-lookup"><span data-stu-id="03af2-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="03af2-104">Il compilatore ha rilevato un codice che non verrà mai eseguito.</span><span class="sxs-lookup"><span data-stu-id="03af2-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="03af2-105">L'esempio seguente genera l'errore CS0162:</span><span class="sxs-lookup"><span data-stu-id="03af2-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
