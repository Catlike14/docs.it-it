 <span data-ttu-id="adf4c-105">Un metodo contrassegnato `extern` deve anche essere contrassegnato con un attributo, ad esempio, il [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attributo.</span><span class="sxs-lookup"><span data-stu-id="adf4c-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>

   }
