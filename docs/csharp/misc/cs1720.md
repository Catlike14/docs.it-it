# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="cb919-102">Avviso del compilatore (livello 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="cb919-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="cb919-103">L'espressione determinerà sempre un'eccezione System.NullReferenceException perché il valore predefinito di 'generic type' è null</span><span class="sxs-lookup"><span data-stu-id="cb919-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="cb919-104">Se si scrive un'espressione che include il valore predefinito di una variabile di tipo generico che è un tipo riferimento, ad esempio una classe, si verificherà questo errore.</span><span class="sxs-lookup"><span data-stu-id="cb919-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="cb919-105">Si consideri la seguente espressione:</span><span class="sxs-lookup"><span data-stu-id="cb919-105">Consider the following expression:</span></span>  
  
```  
default(T).ToString()  
```  
  
 <span data-ttu-id="cb919-106">Poiché `T` è un tipo riferimento, il suo valore predefinito è null e se si tenta di applicarvi il metodo <xref:System.Object.ToString%2A> viene generata un'eccezione <xref:System.NullReferenceException>.</span><span class="sxs-lookup"><span data-stu-id="cb919-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cb919-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="cb919-107">Example</span></span>  
 <span data-ttu-id="cb919-108">Il vincolo del riferimento di classe sul tipo `T` assicura che `T` sia un tipo riferimento.</span><span class="sxs-lookup"><span data-stu-id="cb919-108">The class reference constraint on type `T` ensures that `T` is a reference type.</span></span>  
  
 <span data-ttu-id="cb919-109">L'esempio seguente genera l'errore CS1720.</span><span class="sxs-lookup"><span data-stu-id="cb919-109">The following sample generates CS1720.</span></span>  
  
```  
// CS1720.cs  
using System;  
public class Tester   
{  
    public static void GenericClass<T>(T t1) where T : class   
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
