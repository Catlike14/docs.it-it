# <a name="compiler-error-cs1101"></a><span data-ttu-id="e079c-102">Errore del compilatore CS1101</span><span class="sxs-lookup"><span data-stu-id="e079c-102">Compiler Error CS1101</span></span>
<span data-ttu-id="e079c-103">Non è possibile usare il modificatore di parametro 'ref' con 'this'.</span><span class="sxs-lookup"><span data-stu-id="e079c-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="e079c-104">Quando la parola chiave `this` modifica il primo parametro di un metodo statico, indica al compilatore che il metodo è un metodo di estensione.</span><span class="sxs-lookup"><span data-stu-id="e079c-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="e079c-105">Non sono necessari o consentiti altri modificatori sul primo parametro di un metodo di estensione.</span><span class="sxs-lookup"><span data-stu-id="e079c-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e079c-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="e079c-106">Example</span></span>  
 <span data-ttu-id="e079c-107">L'esempio seguente genera l'errore CS1101:</span><span class="sxs-lookup"><span data-stu-id="e079c-107">The following example generates CS1101:</span></span>  
  
```  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
