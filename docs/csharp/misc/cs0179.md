# <a name="compiler-error-cs0179"></a><span data-ttu-id="039a2-102">Errore del compilatore CS0179</span><span class="sxs-lookup"><span data-stu-id="039a2-102">Compiler Error CS0179</span></span>
<span data-ttu-id="039a2-103">'member' non può essere di tipo extern e dichiarare un corpo</span><span class="sxs-lookup"><span data-stu-id="039a2-103">'member' cannot be extern and declare a body</span></span>  
  
 <span data-ttu-id="039a2-104">Quando un membro della classe è contrassegnato come [extern](../../csharp/language-reference/keywords/extern.md), significa che la definizione si trova in un altro file di definizione del membro.</span><span class="sxs-lookup"><span data-stu-id="039a2-104">When a class member is marked [extern](../../csharp/language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="039a2-105">Quindi, un membro di classe contrassegnato come **extern** non può essere definito nella classe.</span><span class="sxs-lookup"><span data-stu-id="039a2-105">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="039a2-106">Rimuovere la parola chiave `extern` o eliminare la definizione.</span><span class="sxs-lookup"><span data-stu-id="039a2-106">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="039a2-107">Per altre informazioni, vedere [Metodi](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="039a2-107">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="039a2-108">L'esempio seguente genera l'errore CS0179:</span><span class="sxs-lookup"><span data-stu-id="039a2-108">The following sample generates CS0179:</span></span>

```csharp  
      return 0;

   // public extern int ExternMethod(int aa);

   {  
   }

```
