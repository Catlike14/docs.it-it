# <a name="compiler-error-cs1577"></a>Errore del compilatore CS1577
Generazione assembly non riuscita - motivo  
  
 La parte della compilazione relativa alla generazione dell'assembly non è riuscita dopo che il compilatore C# ha compilato il codice sorgente. Per altre informazioni, vedere la documentazione sugli errori relativi all'utilità ALink ([Errori e avvisi di Al.exe](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)).
