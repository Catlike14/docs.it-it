# <a name="compiler-error-cs1577"></a>Errore del compilatore CS1577
Generazione assembly non riuscita - motivo  
  
 La parte della compilazione relativa alla generazione dell'assembly non è riuscita dopo che il compilatore C# ha compilato il codice sorgente. Vedere la documentazione sugli errori relativi all'utilità ALink ([Al.exe errori e avvisi](http://msdn.microsoft.com/library/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) per ulteriori informazioni.
