# <a name="compiler-error-cs1605"></a>Errore del compilatore CS1605
Non è possibile passare 'var' come argomento ref o out perché è di sola lettura  
  
 Una variabile passata come un [ref](../../csharp/language-reference/keywords/ref.md) oppure [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) il parametro deve essere modificata nel metodo chiamato. Non è quindi possibile passare un parametro di sola lettura come `ref` o `out`.
