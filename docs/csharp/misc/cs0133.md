# <a name="compiler-error-cs0133"></a>Errore del compilatore CS0133
L'espressione assegnata a 'variable' deve essere costante  
  
 Una variabile [const](../../csharp/language-reference/keywords/const.md) non pu√≤ accettare come valore un'espressione non costante. Per altre informazioni, vedere [Costanti](../../csharp/programming-guide/classes-and-structs/constants.md).  
  
 L'esempio seguente genera l'errore CS0133:  
  
```  
// CS0133.cs  
public class MyClass  
{  
   public const int i = c;   // CS0133, c is not constant  
   public static int c = i;  
   // try the following line instead  
   // public const int i = 6;  
  
   public static void Main()  
   {  
   }  
}  
```
