<span data-ttu-id="b3ed7-103">'member' non può essere sealed perché non è un override</span><span class="sxs-lookup"><span data-stu-id="b3ed7-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="b3ed7-104">[sealed](../../csharp/language-reference/keywords/sealed.md) è stato usato in un membro non contrassegnato anche con [override](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="b3ed7-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="b3ed7-105">Per altre informazioni, vedere [Ereditarietà](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="b3ed7-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="b3ed7-106">L'esempio seguente genera l'errore CS0238:</span><span class="sxs-lookup"><span data-stu-id="b3ed7-106">The following sample generates CS0238:</span></span>  
  
```  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
