<span data-ttu-id="ae4da-103">L'etichetta 'label' è già presente in questa istruzione switch</span><span class="sxs-lookup"><span data-stu-id="ae4da-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="ae4da-104">In un'istruzione [switch](../../csharp/language-reference/keywords/switch.md) è stata ripetuta un'etichetta.</span><span class="sxs-lookup"><span data-stu-id="ae4da-104">A label was repeated in a [switch](../../csharp/language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="ae4da-105">Per altre informazioni, vedere [switch](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="ae4da-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="ae4da-106">L'esempio seguente genera l'errore CS0152:</span><span class="sxs-lookup"><span data-stu-id="ae4da-106">The following sample generates CS0152:</span></span>  
  
```  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
