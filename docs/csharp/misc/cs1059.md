## <a name="example"></a><span data-ttu-id="ce183-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="ce183-110">Example</span></span>  
 <span data-ttu-id="ce183-111">L'esempio seguente genera l'errore CS1059 perché `i` è una costante anziché una variabile e `E` è un tipo `Enum` , i cui elementi sono anch'essi valori costanti.</span><span class="sxs-lookup"><span data-stu-id="ce183-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
