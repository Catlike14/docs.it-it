 <span data-ttu-id="c7d4a-106">L'esempio seguente genera l'errore CS0440:</span><span class="sxs-lookup"><span data-stu-id="c7d4a-106">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs

using global = MyClass;   // CS0440

{  
    static void Main()

        // Note how global refers to the global namespace

    }  
}
```
