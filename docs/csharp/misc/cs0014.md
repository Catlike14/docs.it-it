# Errore del compilatore CS0014
Impossibile trovare il file richiesto 'file'  
  
 Un file è richiesto dal compilatore ma non è presente nel sistema. Verificare che il percorso sia corretto. Se il file è un file system di [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)], potrebbe essere necessario ripristinare l'installazione oppure potrebbe essere necessario rimuovere e reinstallare [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] completamente.
