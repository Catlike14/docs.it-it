# <a name="compiler-error-cs0014"></a>Errore del compilatore CS0014
Impossibile trovare il file richiesto 'file'  
  
 Un file è richiesto dal compilatore ma non è presente nel sistema. Verificare che il percorso sia corretto. Se il file è un [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] file system, quindi si potrebbe essere necessario ripristinare l'installazione o l'altro rimuovere e reinstallare [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] completamente.
