# <a name="compiler-error-cs0014"></a>Errore del compilatore CS0014
Impossibile trovare il file richiesto 'file'  
  
 Un file è richiesto dal compilatore ma non è presente nel sistema. Verificare che il percorso sia corretto. Se il file è un file di sistema di Visual Studio, potrebbe essere necessario ripristinare l'installazione oppure rimuovere e reinstallare completamente Visual Studio.
