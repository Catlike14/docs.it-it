# <a name="compiler-error-cs0216"></a><span data-ttu-id="8bfbc-102">Errore del compilatore CS0216</span><span class="sxs-lookup"><span data-stu-id="8bfbc-102">Compiler Error CS0216</span></span>
<span data-ttu-id="8bfbc-103">L'operatore 'operator' richiede che sia definito anche un operatore 'missing_operator' corrispondente</span><span class="sxs-lookup"><span data-stu-id="8bfbc-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="8bfbc-104">Un operatore [true](../../csharp/language-reference/keywords/true.md) definito dall'utente richiede un operatore [false](../../csharp/language-reference/keywords/false.md) definito dall'utente e viceversa.</span><span class="sxs-lookup"><span data-stu-id="8bfbc-104">A user-defined [true](../../csharp/language-reference/keywords/true.md) operator requires a user-defined [false](../../csharp/language-reference/keywords/false.md) operator, and vice versa.</span></span> <span data-ttu-id="8bfbc-105">Per altre informazioni, vedere [Operatori](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span><span class="sxs-lookup"><span data-stu-id="8bfbc-105">For more information, see [Operators](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span></span>  
  
 <span data-ttu-id="8bfbc-106">L'esempio seguente genera l'errore CS0216:</span><span class="sxs-lookup"><span data-stu-id="8bfbc-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
