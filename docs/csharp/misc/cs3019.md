## <a name="example"></a><span data-ttu-id="4cf8a-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="4cf8a-107">Example</span></span>  
 <span data-ttu-id="4cf8a-108">L'esempio seguente genera l'errore CS3019:</span><span class="sxs-lookup"><span data-stu-id="4cf8a-108">The following sample generates CS3019:</span></span>

```  
[assembly: CLSCompliant(true)]

    }

    }  
}  
```
