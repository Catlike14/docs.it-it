# <a name="compiler-error-cs1575"></a><span data-ttu-id="412d0-102">Errore del compilatore CS1575</span><span class="sxs-lookup"><span data-stu-id="412d0-102">Compiler Error CS1575</span></span>
<span data-ttu-id="412d0-103">In un'espressione stackalloc occorre specificare [] dopo il tipo</span><span class="sxs-lookup"><span data-stu-id="412d0-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="412d0-104">Le dimensioni dell'allocazione richiesta, con [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), devono essere specificate tra parentesi quadre.</span><span class="sxs-lookup"><span data-stu-id="412d0-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="412d0-105">L'esempio seguente genera l'errore CS1575:</span><span class="sxs-lookup"><span data-stu-id="412d0-105">The following sample generates CS1575:</span></span>  
  
```  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
