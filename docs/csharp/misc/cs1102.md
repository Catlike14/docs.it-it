## <a name="example"></a><span data-ttu-id="8a3ea-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="8a3ea-108">Example</span></span>  
 <span data-ttu-id="8a3ea-109">L'esempio seguente genera l'errore CS1102:</span><span class="sxs-lookup"><span data-stu-id="8a3ea-109">The following example generates CS1102:</span></span>

// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102

}
