 <span data-ttu-id="e2da6-104">Si è provato a eseguire l'override di un tipo di dati non di proprietà come [proprietà](../../csharp/programming-guide/classes-and-structs/properties.md), ma questa operazione non è consentita.</span><span class="sxs-lookup"><span data-stu-id="e2da6-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>
