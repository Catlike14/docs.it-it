# <a name="compiler-error-cs0544"></a><span data-ttu-id="88f27-102">Errore del compilatore CS0544</span><span class="sxs-lookup"><span data-stu-id="88f27-102">Compiler Error CS0544</span></span>
<span data-ttu-id="88f27-103">'property override': non è possibile eseguire l'override. 'non-property' non è una proprietà</span><span class="sxs-lookup"><span data-stu-id="88f27-103">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="88f27-104">Si è provato a eseguire l'override di un tipo di dati non di proprietà come [proprietà](../../csharp/programming-guide/classes-and-structs/properties.md), ma questa operazione non è consentita.</span><span class="sxs-lookup"><span data-stu-id="88f27-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="88f27-105">L'esempio seguente genera l'errore CS0544:</span><span class="sxs-lookup"><span data-stu-id="88f27-105">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
