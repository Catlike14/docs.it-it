 <span data-ttu-id="d014a-104">Si è provato a eseguire l'override di un tipo di dati non di proprietà come [proprietà](../../csharp/programming-guide/classes-and-structs/properties.md), ma questa operazione non è consentita.</span><span class="sxs-lookup"><span data-stu-id="d014a-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

```csharp

{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
