# <a name="compiler-error-cs0572"></a>Errore del compilatore CS0572
'type': non è possibile fare riferimento a un tipo con un'espressione. Provare con 'path_to_type'.  
  
 Si è provato ad accedere a un membro di una classe tramite un identificatore, ma non è consentito in questa situazione.  
  
 L'esempio seguente genera l'errore CS0572:  
  
```  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
