# <a name="compiler-error-cs0572"></a><span data-ttu-id="fec5a-102">Errore del compilatore CS0572</span><span class="sxs-lookup"><span data-stu-id="fec5a-102">Compiler Error CS0572</span></span>
<span data-ttu-id="fec5a-103">'type': non è possibile fare riferimento a un tipo con un'espressione. Provare con 'path_to_type'.</span><span class="sxs-lookup"><span data-stu-id="fec5a-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="fec5a-104">Si è provato ad accedere a un membro di una classe tramite un identificatore, ma non è consentito in questa situazione.</span><span class="sxs-lookup"><span data-stu-id="fec5a-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="fec5a-105">L'esempio seguente genera l'errore CS0572:</span><span class="sxs-lookup"><span data-stu-id="fec5a-105">The following sample generates CS0572:</span></span>  
  
```  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
