# <a name="compiler-error-cs0588"></a>Errore del compilatore CS0588
Errore interno del compilatore: fase 'LEX'  
  
 Provare a determinare se il compilatore non è riuscito perché non è in grado di analizzare una sintassi imprevista. Se non è questo il caso, provare a [comunicazioni con Microsoft](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
