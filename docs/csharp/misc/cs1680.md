# <a name="compiler-error-cs1680"></a>Errore del compilatore CS1680
Opzione dell'alias di riferimento non valida: 'alias='. Nome file mancante.  
  
 Questo errore di verifica quando si usa la funzionalit√† `alias` con l'opzione del compilatore **/reference** senza specificare un nome file valido.  
  
 L'esempio seguente genera l'errore CS1680.  
  
```  
// CS1680.cs  
// compile with: /reference:alias=  
// CS1680 expected  
// To resolve, specify the name of a file with an assembly manifest  
class MyClass {}  
```
