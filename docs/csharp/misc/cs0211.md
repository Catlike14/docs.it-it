<span data-ttu-id="f4068-103">Non è possibile accettare l'indirizzo dell'espressione data</span><span class="sxs-lookup"><span data-stu-id="f4068-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="f4068-104">È possibile accettare l'indirizzo dei campi, delle variabili locali e il riferimento indiretto dei puntatori, ma non è possibile accettare, ad esempio, l'indirizzo della somma di due variabili locali.</span><span class="sxs-lookup"><span data-stu-id="f4068-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="f4068-105">Per altre informazioni, vedere [Codice unsafe e puntatori](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="f4068-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="f4068-106">L'esempio seguente genera l'errore CS0211:</span><span class="sxs-lookup"><span data-stu-id="f4068-106">The following sample generates CS0211:</span></span>  
  
```  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
