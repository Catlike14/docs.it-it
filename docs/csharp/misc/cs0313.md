## <a name="to-correct-this-error"></a><span data-ttu-id="86052-108">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="86052-108">To correct this error</span></span>

    ```  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="86052-111">Esempio</span><span class="sxs-lookup"><span data-stu-id="86052-111">Example</span></span>  
 <span data-ttu-id="86052-112">Il codice seguente genera l'errore CS0313:</span><span class="sxs-lookup"><span data-stu-id="86052-112">The following code generates CS0313:</span></span>  
  
```  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
