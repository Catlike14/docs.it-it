# <a name="compiler-error-cs2001"></a>Errore del compilatore CS2001
Il file di origine 'file' non è stato trovato  
  
 Non è possibile individuare un file di origine passato al compilatore. Controllare che il nome file non contenga errori di digitazione e che il percorso sia stato specificato correttamente.
