## <a name="example"></a><span data-ttu-id="54424-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="54424-108">Example</span></span>  
 <span data-ttu-id="54424-109">L'esempio seguente genera l'errore CS0403.</span><span class="sxs-lookup"><span data-stu-id="54424-109">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
