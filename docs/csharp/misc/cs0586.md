# <a name="compiler-error-cs0586"></a>Errore del compilatore CS0586
Errore interno del compilatore: fase 'stage'  
  
 Provare a determinare se il compilatore non è riuscito perché non è in grado di analizzare una sintassi imprevista. Se questo non è il caso, provare a [comunicazioni con Microsoft](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
