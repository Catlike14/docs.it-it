# Errore del compilatore CS1647
Espressione troppo lunga o complessa per essere compilata accanto a 'code'  
  
 Si è verificato un overflow dello stack nel compilatore che sta elaborando il codice. Per risolvere questo errore, semplificare il codice. Se il codice è valido, contattare il [Supporto tecnico](http://msdn.microsoft.com/it-it/77e75b8b-817d-45bf-9c38-458930d873b4).
