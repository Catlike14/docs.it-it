## <a name="example"></a><span data-ttu-id="89727-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="89727-106">Example</span></span>  
 <span data-ttu-id="89727-107">L'esempio seguente genera l'errore CS0704:</span><span class="sxs-lookup"><span data-stu-id="89727-107">The following sample generates CS0704:</span></span>  
  
```  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
