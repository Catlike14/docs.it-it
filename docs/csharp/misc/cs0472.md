## <a name="example"></a><span data-ttu-id="80981-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="80981-105">Example</span></span>  
 <span data-ttu-id="80981-106">L'esempio seguente genera l'errore CS0472.</span><span class="sxs-lookup"><span data-stu-id="80981-106">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
