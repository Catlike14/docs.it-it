# <a name="compiler-error-cs0174"></a>Errore del compilatore CS0174
È necessaria una classe base per il riferimento 'base'  
  
 Questo errore si verifica solo quando .NET Framework Common Language Runtime compila il codice sorgente per la classe `System.Object` , che è l'unica classe che non contiene classi base.
