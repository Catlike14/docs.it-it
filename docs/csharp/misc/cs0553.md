# <a name="compiler-error-cs0553"></a>Errore del compilatore CS0553
'conversion routine': conversione definita dall'utente verso o dalla classe base  
  
 Le conversioni definite dall'utente verso i valori di una classe base non sono consentite. Non Ã¨ necessario usare questo operatore.  
  
 L'esempio seguente genera l'errore CS0553:  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553

         return new ii();

      }  
   }
```
