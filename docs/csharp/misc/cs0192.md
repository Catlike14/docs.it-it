 <span data-ttu-id="147b6-104">Un campo (variabile) contrassegnato con il [readonly](../../csharp/language-reference/keywords/readonly.md) parola chiave non può essere passato a un [ref](../../csharp/language-reference/keywords/ref.md) o [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametro ad eccezione all'interno di un costruttore.</span><span class="sxs-lookup"><span data-stu-id="147b6-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="147b6-105">Per altre informazioni, vedere [Campi](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="147b6-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>

## <a name="example"></a><span data-ttu-id="147b6-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="147b6-107">Example</span></span>  
 <span data-ttu-id="147b6-108">L'esempio seguente genera l'errore CS0192.</span><span class="sxs-lookup"><span data-stu-id="147b6-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
