<span data-ttu-id="16aaa-103">La parola chiave 'base' non è disponibile nel contesto corrente</span><span class="sxs-lookup"><span data-stu-id="16aaa-103">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="16aaa-104">La parola chiave [base](../../csharp/language-reference/keywords/base.md) è stata usata al di fuori di un metodo, di una proprietà o di un costruttore.</span><span class="sxs-lookup"><span data-stu-id="16aaa-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="16aaa-105">L'esempio seguente genera l'errore CS1512:</span><span class="sxs-lookup"><span data-stu-id="16aaa-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
