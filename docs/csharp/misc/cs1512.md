 <span data-ttu-id="7dfc1-104">La parola chiave [base](../../csharp/language-reference/keywords/base.md) è stata usata al di fuori di un metodo, di una proprietà o di un costruttore.</span><span class="sxs-lookup"><span data-stu-id="7dfc1-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>

```  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:

    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
