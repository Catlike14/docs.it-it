# <a name="compiler-error-cs0828"></a><span data-ttu-id="fa03a-102">Errore del compilatore CS0828</span><span class="sxs-lookup"><span data-stu-id="fa03a-102">Compiler Error CS0828</span></span>
<span data-ttu-id="fa03a-103">Non è possibile assegnare 'expression' a una proprietà di tipo anonimo.</span><span class="sxs-lookup"><span data-stu-id="fa03a-103">Cannot assign 'expression' to anonymous type property.</span></span>

1.  <span data-ttu-id="fa03a-106">Aggiungere una dichiarazione di tipo sul lato sinistro dell'assegnazione o modificare l'espressione sul lato destro in modo che abbia un tipo accettabile.</span><span class="sxs-lookup"><span data-stu-id="fa03a-106">Either add a type declaration to the left side of the assignment, or change the expression on the right side so that it has an acceptable type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fa03a-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="fa03a-107">Example</span></span>

    public static int Main()
