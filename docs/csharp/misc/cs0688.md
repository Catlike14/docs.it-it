## <a name="example"></a><span data-ttu-id="2d426-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="2d426-108">Example</span></span>  
 <span data-ttu-id="2d426-109">L'esempio seguente genera l'errore CS0688.</span><span class="sxs-lookup"><span data-stu-id="2d426-109">The following sample generates CS0688.</span></span> <span data-ttu-id="2d426-110">Per risolvere l'errore senza modificare la classe base, rimuovere l'attributo di sicurezza dal metodo che esegue l'override.</span><span class="sxs-lookup"><span data-stu-id="2d426-110">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="2d426-111">Questa operazione non risolverà il problema di sicurezza.</span><span class="sxs-lookup"><span data-stu-id="2d426-111">This will not solve the security problem.</span></span>  
  
```csharp  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base   
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
