# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="5021f-102">Avviso del compilatore (livello 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="5021f-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="5021f-103">Indicizzazione di una matrice con indice negativo. Gli indici di matrice iniziano sempre da zero</span><span class="sxs-lookup"><span data-stu-id="5021f-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="5021f-104">Non usare un numero negativo per l'indicizzazione in una matrice.</span><span class="sxs-lookup"><span data-stu-id="5021f-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="5021f-105">L'esempio seguente genera l'errore CS0251:</span><span class="sxs-lookup"><span data-stu-id="5021f-105">The following sample generates CS0251:</span></span>  
  
```  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
