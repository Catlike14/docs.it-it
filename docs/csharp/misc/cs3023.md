## <a name="example"></a><span data-ttu-id="200dc-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="200dc-106">Example</span></span>  
 <span data-ttu-id="200dc-107">L'esempio seguente genera l'avviso CS3023:</span><span class="sxs-lookup"><span data-stu-id="200dc-107">The following example generates warning CS3023:</span></span>

{  
    [return:System.CLSCompliant(true)]  // CS3023
