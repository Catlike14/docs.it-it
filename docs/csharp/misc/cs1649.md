# <a name="compiler-error-cs1649"></a><span data-ttu-id="368dd-102">Errore del compilatore CS1649</span><span class="sxs-lookup"><span data-stu-id="368dd-102">Compiler Error CS1649</span></span>
<span data-ttu-id="368dd-103">I membri del campo readonly 'identifier' non possono essere passati a un parametro ref o out (tranne che in un costruttore)</span><span class="sxs-lookup"><span data-stu-id="368dd-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="368dd-104">Questo errore si verifica quando si passa a una funzione una variabile membro di un campo `readonly` come argomento `ref` o `out` .</span><span class="sxs-lookup"><span data-stu-id="368dd-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="368dd-105">Questa operazione non è consentita perché i parametri `ref` e `out` possono venire modificati dalla funzione.</span><span class="sxs-lookup"><span data-stu-id="368dd-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="368dd-106">Per correggere l'errore, rimuovere la parola chiave `readonly` dal campo oppure non passare i membri del campo `readonly` alla funzione.</span><span class="sxs-lookup"><span data-stu-id="368dd-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="368dd-107">Ad esempio, è possibile provare a creare una variabile temporanea modificabile e passarla come argomento `ref` , come mostrato nell'esempio seguente.</span><span class="sxs-lookup"><span data-stu-id="368dd-107">For example, you might try creating a temporary variable which can be modifed and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="368dd-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="368dd-108">Example</span></span>  
 <span data-ttu-id="368dd-109">L'esempio seguente genera l'errore CS1649:</span><span class="sxs-lookup"><span data-stu-id="368dd-109">The following sample generates CS1649:</span></span>  
  
```  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();   
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
