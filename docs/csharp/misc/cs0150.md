# <a name="compiler-error-cs0150"></a><span data-ttu-id="5625c-102">Errore del compilatore CS0150</span><span class="sxs-lookup"><span data-stu-id="5625c-102">Compiler Error CS0150</span></span>
<span data-ttu-id="5625c-103">È previsto un valore costante</span><span class="sxs-lookup"><span data-stu-id="5625c-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="5625c-104">È stata trovata una variabile in cui è prevista una costante.</span><span class="sxs-lookup"><span data-stu-id="5625c-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="5625c-105">Per altre informazioni, vedere [switch](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="5625c-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="5625c-106">L'esempio seguente genera l'errore CS0150:</span><span class="sxs-lookup"><span data-stu-id="5625c-106">The following sample generates CS0150:</span></span>  
  
```  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="5625c-107">Questo errore viene generato anche quando una dimensione di matrice viene specificata con un valore della variabile e inizializzata con un inizializzatore di matrice.</span><span class="sxs-lookup"><span data-stu-id="5625c-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="5625c-108">Per rimuovere l'errore, inizializzare la matrice in una o più istruzioni separate.</span><span class="sxs-lookup"><span data-stu-id="5625c-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
