## <a name="example"></a><span data-ttu-id="9a41a-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="9a41a-106">Example</span></span>  
 <span data-ttu-id="9a41a-107">L'esempio seguente genera l'errore CS0082:</span><span class="sxs-lookup"><span data-stu-id="9a41a-107">The following example generates CS0082:</span></span>

    //property  
    public int MyProp  
    {  
        get //CS0082

            return 1;  
        }

    }  
}
