# <a name="compiler-error-cs1641"></a><span data-ttu-id="404f4-102">Errore del compilatore CS1641</span><span class="sxs-lookup"><span data-stu-id="404f4-102">Compiler Error CS1641</span></span>
<span data-ttu-id="404f4-103">In un campo buffer a dimensione fissa, l'identificatore della dimensione della matrice deve trovarsi dopo il nome del campo</span><span class="sxs-lookup"><span data-stu-id="404f4-103">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="404f4-104">A differenza delle matrici normali, i buffer a dimensione fissa richiedono una dimensione costante da specificare nel punto di dichiarazione.</span><span class="sxs-lookup"><span data-stu-id="404f4-104">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="404f4-105">Per risolvere questo errore, aggiungere un valore letterale intero positivo o un numero intero positivo costante e inserire le parentesi quadre dopo l'identificatore.</span><span class="sxs-lookup"><span data-stu-id="404f4-105">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="404f4-106">L'esempio seguente genera l'errore CS1641:</span><span class="sxs-lookup"><span data-stu-id="404f4-106">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
