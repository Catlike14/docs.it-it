# Errore del compilatore CS0569
'method2': non è possibile eseguire l'override di 'method1' perché non è supportato dal linguaggio  
  
 Questo errore si verifica quando si deriva da una classe base che è stata scritta in un altro linguaggio e quando il compilatore non supporta il metodo di cui si sta tentando di eseguire l'override.
