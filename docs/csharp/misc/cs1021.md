# <a name="compiler-error-cs1021"></a>Errore del compilatore CS1021
La costante integrale è troppo grande  
  
 Un valore assegnato a un [tipo integrale](../../csharp/language-reference/keywords/integral-types-table.md) è maggiore del valore intero senza segno massimo.  
  
 L'esempio seguente genera l'errore CS1021:  
  
```  
// CS1021.cs  
enum F : int  
{  
   a=(int)2590000000000000000000   // CS1021  
}  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
```
