# <a name="compiler-error-cs0202"></a><span data-ttu-id="bad92-102">Errore del compilatore CS0202</span><span class="sxs-lookup"><span data-stu-id="bad92-102">Compiler Error CS0202</span></span>
<span data-ttu-id="bad92-103">Con foreach il tipo restituito ''type' di 'type.GetEnumerator()' deve essere associato a un metodo MoveNext pubblico e a una proprietà Current pubblica appropriati</span><span class="sxs-lookup"><span data-stu-id="bad92-103">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="bad92-104">Una funzione <xref:System.Collections.IEnumerable.GetEnumerator%2A> , usata per abilitare l'uso delle istruzioni foreach, non può restituire un puntatore o una matrice, ma un'istanza di una classe che può fungere da enumeratore.</span><span class="sxs-lookup"><span data-stu-id="bad92-104">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="bad92-105">I requisiti da soddisfare per fungere da enumeratore includono la proprietà public Current e il metodo public MoveNext.</span><span class="sxs-lookup"><span data-stu-id="bad92-105">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="bad92-106">In C# 2.0 il metodo MoveNext e la proprietà Current vengono generati automaticamente.</span><span class="sxs-lookup"><span data-stu-id="bad92-106">In C# 2.0, the compiler will automatically generate Current and MoveNext for you.</span></span> <span data-ttu-id="bad92-107">Per ulteriori informazioni, vedere l'esempio di codice in [interfacce generiche](../../csharp/programming-guide/generics/generic-interfaces.md).</span><span class="sxs-lookup"><span data-stu-id="bad92-107">For more information, see the code example in [Generic Interfaces](../../csharp/programming-guide/generics/generic-interfaces.md).</span></span>  
  
 <span data-ttu-id="bad92-108">L'esempio seguente genera l'errore CS0202:</span><span class="sxs-lookup"><span data-stu-id="bad92-108">The following sample generates CS0202:</span></span>  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
