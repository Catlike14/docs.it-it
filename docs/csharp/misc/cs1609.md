# <a name="compiler-error-cs1609"></a>Errore del compilatore CS1609
Impossibile inserire modificatori nelle dichiarazioni delle funzioni di accesso agli eventi  
  
 I modificatori possono essere inseriti solo nelle dichiarazioni di evento e non nelle dichiarazioni della funzione di accesso eventi. Per altre informazioni, vedere [Uso delle propriet√†](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1609.  
  
```  
// CS1609.cs  
// compile with: /target:library  
delegate int Del();  
class A  
{  
   public event Del MyEvent   
   {  
      private add {}   // CS1609  
      // try the following line instead  
      // add {}  
      remove {}  
   }  
}  
```
