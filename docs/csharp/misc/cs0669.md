# <a name="compiler-error-cs0669"></a>Errore del compilatore CS0669

Una classe con l'attributo ComImport non può avere un costruttore definito dall'utente.

Il livello di interoperabilità COM in Common Language Runtime fornisce il costruttore per le classi [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) . Di conseguenza, è possibile usare un oggetto COM come oggetto gestito nel runtime.

L'esempio seguente genera l'errore CS0669:

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
