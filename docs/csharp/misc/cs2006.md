# <a name="compiler-error-cs2006"></a>Errore del compilatore CS2006
Errore nella sintassi della riga di comando: manca 'text' per l'opzione 'option'  
  
 La sintassi per *option* richiede testo aggiuntivo. Per informazioni, vedere [Opzioni del compilatore](../../csharp/language-reference/compiler-options/index.md).
