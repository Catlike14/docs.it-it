# <a name="compiler-error-cs1655"></a><span data-ttu-id="614c8-102">Errore del compilatore CS1655</span><span class="sxs-lookup"><span data-stu-id="614c8-102">Compiler Error CS1655</span></span>
<span data-ttu-id="614c8-103">Non è possibile passare i campi di 'variable' come un argomento ref o out perché è 'readonly variable type'</span><span class="sxs-lookup"><span data-stu-id="614c8-103">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="614c8-104">Questo errore si verifica se si tenta di passare un membro di una variabile [foreach](../../csharp/language-reference/keywords/foreach-in.md) , una variabile [using](../../csharp/language-reference/keywords/using-statement.md) o una variabile [fixed](../../csharp/language-reference/keywords/fixed-statement.md) a una funzione come argomento ref o out.</span><span class="sxs-lookup"><span data-stu-id="614c8-104">This error occurs if you are attempting to pass a member of a [foreach](../../csharp/language-reference/keywords/foreach-in.md) variable, a [using](../../csharp/language-reference/keywords/using-statement.md) variable, or a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="614c8-105">Questo non è consentito perché queste variabili sono considerate di sola lettura in questi contesti.</span><span class="sxs-lookup"><span data-stu-id="614c8-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="614c8-106">L'esempio seguente genera l'errore CS1655:</span><span class="sxs-lookup"><span data-stu-id="614c8-106">The following sample generates CS1655:</span></span>  
  
```  
// CS1655.cs  
struct S   
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
