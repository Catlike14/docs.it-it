## <a name="example"></a><span data-ttu-id="e146c-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="e146c-105">Example</span></span>  
 <span data-ttu-id="e146c-106">L'esempio seguente genera l'errore CS1529:</span><span class="sxs-lookup"><span data-stu-id="e146c-106">The following sample generates CS1529:</span></span>

```csharp  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition

```
