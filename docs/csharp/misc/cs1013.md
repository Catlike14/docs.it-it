# <a name="compiler-error-cs1013"></a>Errore del compilatore CS1013
Numero non valido  
  
 Il compilatore ha rilevato un numero in formato non valido. Per altre informazioni sui tipi integrali, vedere [Integral Types Table](../../csharp/language-reference/keywords/integral-types-table.md).  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1013:  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
