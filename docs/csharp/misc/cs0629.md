<span data-ttu-id="4786c-103">Il membro condizionale 'member' non può implementare il membro di interfaccia 'base class member' nel tipo 'Type Name'</span><span class="sxs-lookup"><span data-stu-id="4786c-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="4786c-104">Il [condizionale](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attributo non può essere utilizzato dall'implementazione di un'interfaccia.</span><span class="sxs-lookup"><span data-stu-id="4786c-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="4786c-105">L'esempio seguente genera l'errore CS0629:</span><span class="sxs-lookup"><span data-stu-id="4786c-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
