## <a name="example"></a><span data-ttu-id="63921-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="63921-106">Example</span></span>  
 <span data-ttu-id="63921-107">L'esempio seguente genera l'errore CS0136:</span><span class="sxs-lookup"><span data-stu-id="63921-107">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
