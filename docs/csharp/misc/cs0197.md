# <a name="compiler-warning-level-1-cs0197"></a><span data-ttu-id="b2c5e-102">Avviso del compilatore (livello 1) CS0197</span><span class="sxs-lookup"><span data-stu-id="b2c5e-102">Compiler Warning (level 1) CS0197</span></span>
<span data-ttu-id="b2c5e-103">Passare 'argument' come parametro ref o out oppure accettarne l'indirizzo potrebbe provocare un'eccezione in fase di esecuzione perché è un campo di una classe con marshalling per riferimento</span><span class="sxs-lookup"><span data-stu-id="b2c5e-103">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="b2c5e-104">Tutte le classi che derivano direttamente o indirettamente da <xref:System.MarshalByRefObject> sono classi con marshalling per riferimento.</span><span class="sxs-lookup"><span data-stu-id="b2c5e-104">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="b2c5e-105">Tali classi possono essere sottoposte a marshalling per riferimento oltre i limiti del processo e del computer.</span><span class="sxs-lookup"><span data-stu-id="b2c5e-105">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="b2c5e-106">Le istanze di queste classi possono quindi essere proxy per gli oggetti remoti.</span><span class="sxs-lookup"><span data-stu-id="b2c5e-106">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="b2c5e-107">Non è possibile passare un campo di un oggetto proxy come [ref](../../csharp/language-reference/keywords/ref.md) o [out](../../csharp/language-reference/keywords/out.md). In tal caso, non è possibile passare i campi di tale classe come `ref` o `out`, a meno che l'istanza è [questo](../../csharp/language-reference/keywords/this.md), che non può essere un oggetto proxy.</span><span class="sxs-lookup"><span data-stu-id="b2c5e-107">You cannot pass a field of a proxy object as [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../../csharp/language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b2c5e-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="b2c5e-108">Example</span></span>  
 <span data-ttu-id="b2c5e-109">L'esempio seguente genera l'errore CS0197.</span><span class="sxs-lookup"><span data-stu-id="b2c5e-109">The following sample generates CS0197.</span></span>  
  
```  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
