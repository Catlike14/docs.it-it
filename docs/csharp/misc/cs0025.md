# Errore del compilatore CS0025
Impossibile trovare il file di libreria standard 'file'  
  
 Non è stato trovato un file che è richiesto dal compilatore. Verificare che il percorso sia corretto e che il file esista.  
  
 Se il file è un file system di [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)], potrebbe essere necessario ripristinare l'installazione di [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] o reinstallare completamente l'applicazione.
