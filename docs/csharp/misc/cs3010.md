## <a name="example"></a><span data-ttu-id="6ad28-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="6ad28-107">Example</span></span>

```  
// CS3010.cs

public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010  
}

{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
