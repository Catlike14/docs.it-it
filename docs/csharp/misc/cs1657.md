 <span data-ttu-id="d85a7-104">Questo errore si verifica quando una variabile viene passata come argomento [ref](../../csharp/language-reference/keywords/ref.md) o [out](../../csharp/language-reference/keywords/out.md) in un contesto in cui la variabile è di sola lettura.</span><span class="sxs-lookup"><span data-stu-id="d85a7-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="d85a7-105">I contesti di sola lettura includono le variabili di iterazione [foreach](../../csharp/language-reference/keywords/foreach-in.md) , le variabili [using](../../csharp/language-reference/keywords/using-statement.md) e le variabili `fixed` .</span><span class="sxs-lookup"><span data-stu-id="d85a7-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="d85a7-106">Per risolvere questo errore, non chiamare le funzioni che accettano la variabile `foreach`, `using` o `fixed` come parametro `ref` o `out` nei blocchi `using` , nelle istruzioni `foreach` e nelle istruzioni `fixed` .</span><span class="sxs-lookup"><span data-stu-id="d85a7-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d85a7-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="d85a7-107">Example</span></span>  
 <span data-ttu-id="d85a7-108">L'esempio seguente genera l'errore CS1657:</span><span class="sxs-lookup"><span data-stu-id="d85a7-108">The following sample generates CS1657:</span></span>  
  
```  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
