# <a name="compiler-error-cs0587"></a>Errore del compilatore CS0587
Errore interno del compilatore: fase 'stage'  
  
 Provare a determinare se il compilatore non è riuscito perché non è in grado di analizzare una sintassi imprevista. Se questo non è il caso, provare a [comunicazioni con Microsoft](/visualstudio/ide/talk-to-us).
