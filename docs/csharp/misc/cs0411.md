## <a name="example"></a><span data-ttu-id="84167-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="84167-107">Example</span></span>  
 <span data-ttu-id="84167-108">L'esempio seguente genera l'errore CS0411:</span><span class="sxs-lookup"><span data-stu-id="84167-108">The following sample generates CS0411:</span></span>  
  
```  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
