# <a name="compiler-error-cs0009"></a>Errore del compilatore CS0009
Non è possibile aprire il file di metadati 'file' — 'description'  
  
 Il file specificato con l'opzione del compilatore [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) non contiene metadati validi.
