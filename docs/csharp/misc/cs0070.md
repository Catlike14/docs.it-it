# <a name="compiler-error-cs0070"></a><span data-ttu-id="44f16-102">Errore del compilatore CS0070</span><span class="sxs-lookup"><span data-stu-id="44f16-102">Compiler Error CS0070</span></span>
<span data-ttu-id="44f16-103">L'evento 'event' può trovarsi soltanto sul lato sinistro di += o di -= (tranne quando è utilizzato dall'interno del tipo 'type')</span><span class="sxs-lookup"><span data-stu-id="44f16-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="44f16-104">All'esterno della classe in cui è definito, un [evento](../../csharp/language-reference/keywords/event.md) può solo aggiungere o sottrarre riferimenti.</span><span class="sxs-lookup"><span data-stu-id="44f16-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="44f16-105">Per altre informazioni, vedere [Eventi](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="44f16-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="44f16-106">L'esempio seguente genera l'errore CS0070:</span><span class="sxs-lookup"><span data-stu-id="44f16-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
