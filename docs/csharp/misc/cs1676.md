# <a name="compiler-error-cs1676"></a><span data-ttu-id="89dda-102">Errore del compilatore CS1676</span><span class="sxs-lookup"><span data-stu-id="89dda-102">Compiler Error CS1676</span></span>
<span data-ttu-id="89dda-103">Il parametro 'number' non deve essere dichiarato con la parola chiave 'keyword'</span><span class="sxs-lookup"><span data-stu-id="89dda-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="89dda-104">Questo errore si verifica quando il modificatore di tipo di parametro in un metodo anonimo non corrisponde a quello usato nella dichiarazione del delegato, a cui si esegue il cast del metodo.</span><span class="sxs-lookup"><span data-stu-id="89dda-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="89dda-105">L'esempio seguente genera l'errore CS1676:</span><span class="sxs-lookup"><span data-stu-id="89dda-105">The following sample generates CS1676:</span></span>  
  
```  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
