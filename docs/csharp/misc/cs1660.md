# <a name="compiler-error-cs1660"></a><span data-ttu-id="c5c5a-102">Errore del compilatore CS1660</span><span class="sxs-lookup"><span data-stu-id="c5c5a-102">Compiler Error CS1660</span></span>
<span data-ttu-id="c5c5a-103">Impossibile convertire il blocco di metodi anonimi nel tipo 'type' perché non è un tipo delegato</span><span class="sxs-lookup"><span data-stu-id="c5c5a-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="c5c5a-104">Questo errore si verifica se si tenta di assegnare o convertire un blocco di metodo anonimo a un tipo che non è un tipo delegato.</span><span class="sxs-lookup"><span data-stu-id="c5c5a-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="c5c5a-105">L'esempio seguente genera l'errore CS1660:</span><span class="sxs-lookup"><span data-stu-id="c5c5a-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
