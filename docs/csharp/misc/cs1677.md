 <span data-ttu-id="b9143-104">Questo errore si verifica quando il modificatore di tipo di parametro in un metodo anonimo non corrisponde a quello usato nella dichiarazione del delegato, a cui si esegue il cast del metodo.</span><span class="sxs-lookup"><span data-stu-id="b9143-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b9143-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="b9143-105">Example</span></span>  
 <span data-ttu-id="b9143-106">L'esempio seguente genera l'errore CS1677:</span><span class="sxs-lookup"><span data-stu-id="b9143-106">The following sample generates CS1677:</span></span>  
  
```  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
