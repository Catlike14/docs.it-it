# <a name="compiler-error-cs1525"></a><span data-ttu-id="ea145-102">Errore del compilatore CS1525</span><span class="sxs-lookup"><span data-stu-id="ea145-102">Compiler Error CS1525</span></span>
<span data-ttu-id="ea145-103">'character' non è un termine valido nell'espressione</span><span class="sxs-lookup"><span data-stu-id="ea145-103">Invalid expression term 'character'</span></span>  
  
 <span data-ttu-id="ea145-104">Il compilatore ha rilevato un carattere non valido in un'espressione.</span><span class="sxs-lookup"><span data-stu-id="ea145-104">The compiler detected an invalid character in an expression.</span></span>  
  
 <span data-ttu-id="ea145-105">L'esempio seguente genera l'errore CS1525:</span><span class="sxs-lookup"><span data-stu-id="ea145-105">The following sample generates CS1525:</span></span>  
  
```csharp  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 <span data-ttu-id="ea145-106">Un'etichetta vuota può generare anche l'errore CS1525, come nell'esempio seguente:</span><span class="sxs-lookup"><span data-stu-id="ea145-106">An empty label can also generate CS1525, as in the following sample:</span></span>  
  
```csharp  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
