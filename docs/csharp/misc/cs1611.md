 <span data-ttu-id="d8bc3-105">L'esempio seguente genera l'errore CS1611:</span><span class="sxs-lookup"><span data-stu-id="d8bc3-105">The following sample generates CS1611:</span></span>

```csharp  
   public static void Test(params ref int[] a)   // CS1611, remove ref

   }

```
