# <a name="compiler-error-cs0668"></a><span data-ttu-id="c6227-102">Errore del compilatore CS0668</span><span class="sxs-lookup"><span data-stu-id="c6227-102">Compiler Error CS0668</span></span>
<span data-ttu-id="c6227-103">Due indicizzatori hanno nomi diversi. L'attributo IndexerName deve essere usato con lo stesso nome in ogni indicizzatore all'interno di un tipo.</span><span class="sxs-lookup"><span data-stu-id="c6227-103">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="c6227-104">I valori passati all'attributo **IndexerName** devono essere identici per tutti gli indicizzatori di un tipo.</span><span class="sxs-lookup"><span data-stu-id="c6227-104">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="c6227-105">Per altre informazioni sull'attributo **IndexerName** , vedere [Classe IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="c6227-105">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="c6227-106">L'esempio seguente genera l'errore CS0668:</span><span class="sxs-lookup"><span data-stu-id="c6227-106">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
