# <a name="compiler-error-cs0745"></a><span data-ttu-id="cb86e-102">Errore del compilatore CS0745</span><span class="sxs-lookup"><span data-stu-id="cb86e-102">Compiler Error CS0745</span></span>
<span data-ttu-id="cb86e-103">È prevista la parola chiave contestuale 'by'</span><span class="sxs-lookup"><span data-stu-id="cb86e-103">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="cb86e-104">Il percorso per la clausola `group` è `group...by` seguito da un oggetto `into`facoltativo, come mostrato nell'esempio seguente:</span><span class="sxs-lookup"><span data-stu-id="cb86e-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>  
  
```csharp  
string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
var query = from name in names  
            group name by name[0];  
```  
  
 <span data-ttu-id="cb86e-105">oppure</span><span class="sxs-lookup"><span data-stu-id="cb86e-105">or</span></span>  
  
```csharp  
var query2 = from name in names  
             group name by name[0] into g  
             //...additional query clauses  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="cb86e-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="cb86e-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="cb86e-107">Aggiungere la parola chiave `by` alla clausola `group` .</span><span class="sxs-lookup"><span data-stu-id="cb86e-107">Add the `by` keyword to the `group` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cb86e-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="cb86e-108">Example</span></span>  
 <span data-ttu-id="cb86e-109">Il codice seguente genera l'errore CS0745:</span><span class="sxs-lookup"><span data-stu-id="cb86e-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
