# <a name="compiler-error-cs0745"></a><span data-ttu-id="356c0-102">Errore del compilatore CS0745</span><span class="sxs-lookup"><span data-stu-id="356c0-102">Compiler Error CS0745</span></span>
<span data-ttu-id="356c0-103">È prevista la parola chiave contestuale 'by'</span><span class="sxs-lookup"><span data-stu-id="356c0-103">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="356c0-104">Il percorso per la clausola `group` è `group...by` seguito da un oggetto `into`facoltativo, come mostrato nell'esempio seguente:</span><span class="sxs-lookup"><span data-stu-id="356c0-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>  
  
```  
string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
var query = from name in names  
            group name by name[0];  
```  
  
 <span data-ttu-id="356c0-105">oppure</span><span class="sxs-lookup"><span data-stu-id="356c0-105">or</span></span>  
  
```  
var query2 = from name in names  
             group name by name[0] into g  
             //...additional query clauses  
```

## <a name="example"></a><span data-ttu-id="356c0-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="356c0-108">Example</span></span>  
 <span data-ttu-id="356c0-109">Il codice seguente genera l'errore CS0745:</span><span class="sxs-lookup"><span data-stu-id="356c0-109">The following code generates CS0745:</span></span>  
  
```  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
