# <a name="compiler-error-cs0042"></a>Errore del compilatore CS0042
Errore imprevisto durante la creazione del file di informazioni di debug 'file' — 'reason'  
  
 Il compilatore non è riuscito a creare le informazioni di debug; *reason* contiene informazioni aggiuntive sulla condizione di errore.
