 <span data-ttu-id="99cb9-104">Questo errore si verifica se l'istruzione restituita del blocco di metodi anonimi ha un tipo che non è convertibile in modo implicito nel tipo restituito del delegato.</span><span class="sxs-lookup"><span data-stu-id="99cb9-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

{  
  
  public static void Main()  
  {  
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662

     // MyDelegate d = dekegate(int i) { return (int)1.0; };  
  }  
}
