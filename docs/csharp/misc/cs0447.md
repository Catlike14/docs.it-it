<span data-ttu-id="6f2af-103">Gli attributi non possono essere utilizzati su argomenti di tipo, solo su parametri di tipo</span><span class="sxs-lookup"><span data-stu-id="6f2af-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="6f2af-104">Questo errore si verifica quando si applica un attributo a un argomento di tipo presente in un'istruzione di chiamata.</span><span class="sxs-lookup"><span data-stu-id="6f2af-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="6f2af-105">È possibile applicare un attributo a un parametro di tipo in un'istruzione di dichiarazione di classe o metodo simile alla seguente:</span><span class="sxs-lookup"><span data-stu-id="6f2af-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```  
  
 <span data-ttu-id="6f2af-106">La riga di codice seguente genera questo errore.</span><span class="sxs-lookup"><span data-stu-id="6f2af-106">The following line of code will generate this error.</span></span> <span data-ttu-id="6f2af-107">Si presuppone che la classe `C`, definita nella riga di codice precedente, sia un metodo statico denominato `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="6f2af-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```
