# <a name="compiler-error-cs0447"></a><span data-ttu-id="584b6-102">Errore del compilatore CS0447</span><span class="sxs-lookup"><span data-stu-id="584b6-102">Compiler Error CS0447</span></span>
<span data-ttu-id="584b6-103">Gli attributi non possono essere utilizzati su argomenti di tipo, solo su parametri di tipo</span><span class="sxs-lookup"><span data-stu-id="584b6-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="584b6-104">Questo errore si verifica quando si applica un attributo a un argomento di tipo presente in un'istruzione di chiamata.</span><span class="sxs-lookup"><span data-stu-id="584b6-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="584b6-105">È possibile applicare un attributo a un parametro di tipo in un'istruzione di dichiarazione di classe o metodo simile alla seguente:</span><span class="sxs-lookup"><span data-stu-id="584b6-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>  
  
```csharp

```  
  
 <span data-ttu-id="584b6-106">La riga di codice seguente genera questo errore.</span><span class="sxs-lookup"><span data-stu-id="584b6-106">The following line of code will generate this error.</span></span> <span data-ttu-id="584b6-107">Si presuppone che la classe `C`, definita nella riga di codice precedente, sia un metodo statico denominato `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="584b6-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>  
  
```csharp  
C<[some attribute] T>.MyStaticMethod();  
```  
  
## <a name="example"></a><span data-ttu-id="584b6-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="584b6-108">Example</span></span>

```csharp  
{  
    public interface I<A>   
    {  
        void Meth<B>();  
    }  
    public class B : I<int>   
    {  
        void I<[Test] int>.Meth<X>() { }  // CS0447  
    }

```
