# <a name="compiler-error-cs0005"></a>Errore del compilatore CS0005
L'opzione 'compiler_option' del compilatore deve essere seguita da un argomento  
  
 Alcune opzioni del compilatore richiedono parametri. Se gli argomenti richiesti non vengono passati, viene generato l'errore CS0005.  
  
 Per altre informazioni, vedere [Opzioni del compilatore C#](../../csharp/language-reference/compiler-options/index.md).
