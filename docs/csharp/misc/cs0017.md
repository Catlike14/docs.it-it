# <a name="compiler-error-cs0017"></a><span data-ttu-id="29b8b-102">Errore del compilatore CS0017</span><span class="sxs-lookup"><span data-stu-id="29b8b-102">Compiler Error CS0017</span></span>
<span data-ttu-id="29b8b-103">Nel programma 'output file name' è definito più di un punto di ingresso.</span><span class="sxs-lookup"><span data-stu-id="29b8b-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="29b8b-104">Compilare con /main per specificare il tipo contenente il punto di ingresso.</span><span class="sxs-lookup"><span data-stu-id="29b8b-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="29b8b-105">Un programma può avere solo uno [Main](../../csharp/programming-guide/main-and-command-args/index.md) metodo.</span><span class="sxs-lookup"><span data-stu-id="29b8b-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="29b8b-106">Per risolvere questo errore, è possibile eliminare tutti i metodi Main nel codice, tranne uno, oppure è possibile utilizzare il [/principale](../../csharp/language-reference/compiler-options/main-compiler-option.md) opzione del compilatore per specificare il metodo Main si desidera utilizzare.</span><span class="sxs-lookup"><span data-stu-id="29b8b-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="29b8b-107">L'esempio seguente genera l'errore CS0017:</span><span class="sxs-lookup"><span data-stu-id="29b8b-107">The following sample generates CS0017:</span></span>  
  
```  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
