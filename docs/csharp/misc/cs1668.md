# <a name="compiler-warning-level-2-cs1668"></a>Avviso del compilatore (livello 2) CS1668
Il percorso di ricerca 'path' specificato in 'path string' non è valido - 'system error message'  
  
 Il percorso specificato per [/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) nella riga di comando non è valido oppure un percorso nella variabile di ambiente LIB non è valido. Controllare il percorso usato per verificare che esista e che sia accessibile. Il messaggio di errore tra virgolette singole è l'errore restituito dal sistema operativo.
