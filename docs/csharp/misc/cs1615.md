# <a name="compiler-error-cs1615"></a><span data-ttu-id="fac74-102">Errore del compilatore CS1615</span><span class="sxs-lookup"><span data-stu-id="fac74-102">Compiler Error CS1615</span></span>
<span data-ttu-id="fac74-103">L'argomento 'number' non deve essere passato con la parola chiave 'keyword'</span><span class="sxs-lookup"><span data-stu-id="fac74-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="fac74-104">Una delle parole chiave `ref` o **out** usata quando la funzione non accettava un parametro `ref` o **out** per l'argomento.</span><span class="sxs-lookup"><span data-stu-id="fac74-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="fac74-105">Per risolvere l'errore, rimuovere la parola chiave non corretta e usare la parola chiave appropriata che corrisponde alla dichiarazione di funzione, se presente.</span><span class="sxs-lookup"><span data-stu-id="fac74-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="fac74-106">L'esempio seguente genera l'errore CS1615:</span><span class="sxs-lookup"><span data-stu-id="fac74-106">The following sample generates CS1615:</span></span>  
  
```  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
