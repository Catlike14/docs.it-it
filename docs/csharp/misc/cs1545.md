## <a name="example"></a><span data-ttu-id="33cec-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="33cec-106">Example</span></span>  
  
```  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="33cec-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="33cec-107">Example</span></span>  
 <span data-ttu-id="33cec-108">L'esempio seguente genera l'errore CS1545.</span><span class="sxs-lookup"><span data-stu-id="33cec-108">The following sample generates CS1545.</span></span>

```  
   public static void Main() {  
      Manager Ed = new Manager();

   }  
}  
```
