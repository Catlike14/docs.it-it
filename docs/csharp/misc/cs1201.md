# Avviso del compilatore (livello 1) CS1201
La funzionalità 'invalid feature' è deprecata. Usare 'valid feature'.  
  
 La funzionalità che si sta tentando di usare ora è deprecata. Aggiornare il codice per usare la funzionalità valida.
