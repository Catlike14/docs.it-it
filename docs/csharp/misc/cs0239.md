 <span data-ttu-id="b3198-104">Un membro non può eseguire l' [override](../../csharp/language-reference/keywords/override.md) di un membro ereditato di tipo [sealed](../../csharp/language-reference/keywords/sealed.md) .</span><span class="sxs-lookup"><span data-stu-id="b3198-104">A member cannot [override](../../csharp/language-reference/keywords/override.md) a [sealed](../../csharp/language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="b3198-105">Per ulteriori informazioni, vedere [Checked e Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="b3198-105">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="b3198-106">L'esempio seguente genera l'errore CS0239:</span><span class="sxs-lookup"><span data-stu-id="b3198-106">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
