## <a name="example"></a><span data-ttu-id="ff3d0-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="ff3d0-105">Example</span></span>  
 <span data-ttu-id="ff3d0-106">L'esempio seguente genera l'errore CS1621:</span><span class="sxs-lookup"><span data-stu-id="ff3d0-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };

    }

    {  
    }  
}
```  
