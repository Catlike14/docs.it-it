 <span data-ttu-id="4089a-104">Una [classe](../../csharp/language-reference/keywords/class.md) è derivata da un' [interfaccia](../../csharp/language-reference/keywords/interface.md), ma non ha implementato uno o più membri dell'interfaccia.</span><span class="sxs-lookup"><span data-stu-id="4089a-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="4089a-105">Una classe deve implementare tutti i membri delle interfacce da cui deriva oppure essere dichiarata `abstract`.</span><span class="sxs-lookup"><span data-stu-id="4089a-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4089a-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="4089a-106">Example</span></span>  
 <span data-ttu-id="4089a-107">L'esempio seguente genera l'errore CS0535.</span><span class="sxs-lookup"><span data-stu-id="4089a-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
