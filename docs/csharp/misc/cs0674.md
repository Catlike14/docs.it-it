 <span data-ttu-id="15064-105">Il compilatore c# non consente l'uso di <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; utilizzare [params](../../csharp/language-reference/keywords/params.md) invece.</span><span class="sxs-lookup"><span data-stu-id="15064-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="15064-106">L'esempio seguente genera l'errore CS0674:</span><span class="sxs-lookup"><span data-stu-id="15064-106">The following sample generates CS0674:</span></span>  
  
```csharp

{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674

   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
