# <a name="compiler-error-cs0156"></a><span data-ttu-id="f7996-102">Errore del compilatore CS0156</span><span class="sxs-lookup"><span data-stu-id="f7996-102">Compiler Error CS0156</span></span>
<span data-ttu-id="f7996-103">L'utilizzo dell'istruzione throw senza argomenti non è consentito in una clausola finally annidata all'interno della clausola catch di inclusione più vicina</span><span class="sxs-lookup"><span data-stu-id="f7996-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="f7996-104">Oggetto [generare](../../csharp/language-reference/keywords/throw.md) istruzione senza parametri può essere utilizzata solo un **catch** clausola che non accetta parametri.</span><span class="sxs-lookup"><span data-stu-id="f7996-104">A [throw](../../csharp/language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="f7996-105">Per ulteriori informazioni, vedere [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) e [le eccezioni e gestione delle eccezioni](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="f7996-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="f7996-106">L'esempio seguente genera l'errore CS0156:</span><span class="sxs-lookup"><span data-stu-id="f7996-106">The following sample generates CS0156:</span></span>  
  
```  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
