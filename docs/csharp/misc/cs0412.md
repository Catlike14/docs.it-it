## <a name="example"></a><span data-ttu-id="afa4c-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="afa4c-106">Example</span></span>  
 <span data-ttu-id="afa4c-107">L'esempio seguente genera l'errore CS0412:</span><span class="sxs-lookup"><span data-stu-id="afa4c-107">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
