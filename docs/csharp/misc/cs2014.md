# <a name="compiler-warning-level-1-cs2014"></a>Avviso del compilatore (livello 1) CS2014
L'opzione del compilatore 'old option' è obsoleta. Usare 'new option'  
  
 Il formato dell'opzione del compilatore è deprecato. Per altre informazioni, vedere [Opzioni del compilatore C#](../../csharp/language-reference/compiler-options/index.md) .
