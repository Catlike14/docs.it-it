# <a name="compiler-error-cs1623"></a><span data-ttu-id="18808-102">Errore del compilatore CS1623</span><span class="sxs-lookup"><span data-stu-id="18808-102">Compiler Error CS1623</span></span>
<span data-ttu-id="18808-103">Gli iteratori non possono avere parametri ref o out</span><span class="sxs-lookup"><span data-stu-id="18808-103">Iterators cannot have ref or out parameters</span></span>  
  
 <span data-ttu-id="18808-104">Questo errore si verifica se un metodo iteratore accetta un parametro `ref` o `out` .</span><span class="sxs-lookup"><span data-stu-id="18808-104">This error occurs if an iterator method takes a `ref` or `out` parameter.</span></span> <span data-ttu-id="18808-105">Per evitare questo errore, rimuovere la parola chiave `ref` o `out` dalla firma del metodo.</span><span class="sxs-lookup"><span data-stu-id="18808-105">To avoid this error, remove the `ref` or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="18808-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="18808-106">Example</span></span>  
 <span data-ttu-id="18808-107">L'esempio seguente genera l'errore CS1623:</span><span class="sxs-lookup"><span data-stu-id="18808-107">The following sample generates CS1623:</span></span>  
  
```  
// CS1623.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {  
        yield return i;  
    }  
  
    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {  
        f = 0.0F;  
        yield return f;  
    }  
}  
```
