# <a name="compiler-error-cs0502"></a><span data-ttu-id="cd57d-102">Errore del compilatore CS0502</span><span class="sxs-lookup"><span data-stu-id="cd57d-102">Compiler Error CS0502</span></span>
<span data-ttu-id="cd57d-103">'member' non può essere contemporaneamente abstract e sealed</span><span class="sxs-lookup"><span data-stu-id="cd57d-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="cd57d-104">Un membro di classe non può essere contemporaneamente [abstract](../../csharp/language-reference/keywords/abstract.md) e [sealed](../../csharp/language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="cd57d-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="cd57d-105">L'esempio seguente genera l'errore CS0502:</span><span class="sxs-lookup"><span data-stu-id="cd57d-105">The following sample generates CS0502:</span></span>  
  
```  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
