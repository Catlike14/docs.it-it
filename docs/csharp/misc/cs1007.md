 <span data-ttu-id="36849-104">Quando si dichiara una [proprietà](../../csharp/programming-guide/classes-and-structs/using-properties.md), è necessario dichiarare anche i metodi per le sue funzioni di accesso.</span><span class="sxs-lookup"><span data-stu-id="36849-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="36849-105">Una proprietà, tuttavia, non può avere più di un metodo per la funzione di accesso `get` o più di un metodo per la funzione di accesso `set` .</span><span class="sxs-lookup"><span data-stu-id="36849-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="36849-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="36849-106">Example</span></span>  
 <span data-ttu-id="36849-107">L'esempio seguente genera l'errore CS1007:</span><span class="sxs-lookup"><span data-stu-id="36849-107">The following sample generates CS1007:</span></span>

```  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
