# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="60167-102">Avviso del compilatore (livello 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="60167-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="60167-103">L'espressione specificata è sempre del tipo fornito ('type')</span><span class="sxs-lookup"><span data-stu-id="60167-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="60167-104">Se un'istruzione condizionale restituisce sempre **true**, non è necessaria un'istruzione condizionale.</span><span class="sxs-lookup"><span data-stu-id="60167-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="60167-105">Questo avviso viene visualizzato quando si tenta di valutare un tipo usando l'operatore **is** .</span><span class="sxs-lookup"><span data-stu-id="60167-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="60167-106">Se la valutazione è un tipo valore, il controllo non è necessario.</span><span class="sxs-lookup"><span data-stu-id="60167-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="60167-107">L'esempio seguente genera l'errore CS0183:</span><span class="sxs-lookup"><span data-stu-id="60167-107">The following sample generates CS0183:</span></span>  
  
```  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```
