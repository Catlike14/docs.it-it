# <a name="compiler-error-cs4009"></a><span data-ttu-id="0fbd2-102">CS4009 errore del compilatore</span><span class="sxs-lookup"><span data-stu-id="0fbd2-102">Compiler Error CS4009</span></span>

<span data-ttu-id="0fbd2-103">'Type.Method': un punto di ingresso non può essere contrassegnato con il `async` modificatore.</span><span class="sxs-lookup"><span data-stu-id="0fbd2-103">'Type.Method': an entry point cannot be marked with the `async` modifier.</span></span>

<span data-ttu-id="0fbd2-104">Non è possibile utilizzare il `async` (parola chiave) nel punto di ingresso dell'applicazione (in genere il `Main` (metodo)).</span><span class="sxs-lookup"><span data-stu-id="0fbd2-104">You cannot use the `async` keyword in the application entry point (typically the `Main` method).</span></span>  

<span data-ttu-id="0fbd2-105">Nell'esempio seguente produce CS4009:</span><span class="sxs-lookup"><span data-stu-id="0fbd2-105">The following example produces CS4009:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```
