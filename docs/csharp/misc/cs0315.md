## <a name="example"></a><span data-ttu-id="7a369-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="7a369-108">Example</span></span>  
 <span data-ttu-id="7a369-109">L'esempio seguente genera l'errore CS0315:</span><span class="sxs-lookup"><span data-stu-id="7a369-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}

    }
```  
