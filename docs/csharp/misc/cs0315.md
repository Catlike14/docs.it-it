# <a name="compiler-error-cs0315"></a><span data-ttu-id="43937-102">Errore del compilatore CS0315</span><span class="sxs-lookup"><span data-stu-id="43937-102">Compiler Error CS0315</span></span>
<span data-ttu-id="43937-103">Il tipo 'valueType' non può essere utilizzato come parametro di tipo ' nel metodo o nel tipo generico ' TypeorMethod\<T >'.</span><span class="sxs-lookup"><span data-stu-id="43937-103">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="43937-104">Non esistono conversioni boxing da 'valueType' a 'referenceType'.</span><span class="sxs-lookup"><span data-stu-id="43937-104">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="43937-105">Questo errore si verifica quando si vincola un tipo generico a una determinata classe e si tenta di costruire un'istanza di quella classe usando un tipo che non può essere convertito in esso tramite boxing.</span><span class="sxs-lookup"><span data-stu-id="43937-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>

## <a name="example"></a><span data-ttu-id="43937-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="43937-108">Example</span></span>  
 <span data-ttu-id="43937-109">L'esempio seguente genera l'errore CS0315:</span><span class="sxs-lookup"><span data-stu-id="43937-109">The following example generates CS0315:</span></span>  
  
```  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
