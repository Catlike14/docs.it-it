## <a name="example"></a><span data-ttu-id="8779d-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="8779d-109">Example</span></span>  
 <span data-ttu-id="8779d-110">Il codice seguente genera l'errore CS1947:</span><span class="sxs-lookup"><span data-stu-id="8779d-110">The following code generates CS1947:</span></span>  
  
```  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
