# <a name="c-60-draft-language-specification"></a>Bozza di specifica del linguaggio C# 6.0
La specifica del linguaggio C# è la fonte ufficiale per la sintassi e l'utilizzo di C#. Queste specifiche contengono informazioni dettagliate su tutti gli aspetti del linguaggio, inclusi molti argomenti non trattati nella documentazione di Visual C#.

È possibile scaricare la versione 5.0 di queste specifiche dall'[Area download Microsoft](http://www.microsoft.com/download/details.aspx?id=7029). Se Visual Studio 2015 è installato nel computer in uso, le specifiche sono disponibili anche nel computer, nella cartella Programmi (x86)/Microsoft Visual Studio 14.0/VC#/Specifications/1033. Se è installata un'altra versione di Visual Studio o se Visual Studio è installato in una lingua diversa dall'inglese, modificare il percorso come appropriato.

La versione 6.0 della specifica non è stata approvata come standard. Questo sito contiene la [*bozza* della specifica di C# 6.0](../../../../_csharplang/spec/lexical-structure.md). Viene creata dai file markdown contenuti [nell'archivio GitHub dotnet/csharplang](https://github.com/dotnet/csharplang/blob/master/spec/README.md).

Le segnalazioni dei problemi relativi alla bozza di specifica devono essere create nell'archivio [dotnet/csharplang](https://github.com/dotnet/csharplang/issues). Oppure, se si è interessati a correggere eventuali errori riscontrati, è possibile inviare una [richiesta pull](https://github.com/dotnet/csharplang/pulls) allo stesso archivio.
