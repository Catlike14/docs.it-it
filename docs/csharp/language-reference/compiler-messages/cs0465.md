# <a name="compiler-warning-level-1-cs0465"></a><span data-ttu-id="5caa1-102">Avviso del compilatore (livello 1) CS0465</span><span class="sxs-lookup"><span data-stu-id="5caa1-102">Compiler Warning (level 1) CS0465</span></span>
<span data-ttu-id="5caa1-103">L'introduzione di un metodo 'Finalize' può interferire con la chiamata di un distruttore.</span><span class="sxs-lookup"><span data-stu-id="5caa1-103">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="5caa1-104">Si desiderava dichiarare un distruttore?</span><span class="sxs-lookup"><span data-stu-id="5caa1-104">Did you intend to declare a destructor?</span></span>  
  
 <span data-ttu-id="5caa1-105">Questo avviso viene generato quando si crea una classe con un metodo la cui firma è `public virtual void Finalize`.</span><span class="sxs-lookup"><span data-stu-id="5caa1-105">This warning occurs when you create a class with a method whose signature is `public virtual void Finalize`.</span></span>  
  
 <span data-ttu-id="5caa1-106">Se tale classe viene usata come classe di base e se la classe di derivazione definisce un distruttore, il distruttore eseguirà l'override del metodo `Finalize` della classe di base e non di <xref:System.Object.Finalize%2A>.</span><span class="sxs-lookup"><span data-stu-id="5caa1-106">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5caa1-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="5caa1-107">Example</span></span>  
 <span data-ttu-id="5caa1-108">L'esempio seguente genera l'errore CS0465.</span><span class="sxs-lookup"><span data-stu-id="5caa1-108">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
