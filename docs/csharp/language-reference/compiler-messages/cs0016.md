# <a name="compiler-error-cs0016"></a>Errore del compilatore CS0016
Non è possibile scrivere nel file di output 'file' - 'motivo'  
  
 Il compilatore non è riuscito a scrivere in un file di output. Controllare il percorso del file per verificare che esista. Se un file compilato in precedenza è già presente nel percorso, assicurarsi che sia scrivibile e che nessun processo abbia un blocco attivo sul file. Assicurarsi, ad esempio, che il file eseguibile non sia in esecuzione quando si tenta la compilazione.
