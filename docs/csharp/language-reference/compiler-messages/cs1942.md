# <a name="compiler-error-cs1942"></a><span data-ttu-id="106c2-102">Errore del compilatore CS1942</span><span class="sxs-lookup"><span data-stu-id="106c2-102">Compiler Error CS1942</span></span>
<span data-ttu-id="106c2-103">Il tipo dell'espressione nella clausola 'clausola' non è corretto.</span><span class="sxs-lookup"><span data-stu-id="106c2-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="106c2-104">L'inferenza del tipo non è riuscita nella chiamata a 'metodo'.</span><span class="sxs-lookup"><span data-stu-id="106c2-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="106c2-105">Questo errore viene generato in genere quando alla variabile di intervallo viene assegnato un tipo esplicito non corretto.</span><span class="sxs-lookup"><span data-stu-id="106c2-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="106c2-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="106c2-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="106c2-107">Se la variabile di intervallo è tipizzata in modo esplicito, assicurarsi che il tipo sia uguale a, o convertibile in modo implicito da, il tipo degli elementi nella raccolta di cui viene eseguita l'iterazione.</span><span class="sxs-lookup"><span data-stu-id="106c2-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="106c2-108">Se la variabile di intervallo è preceduta dalla parola chiave `var`, rimuovere `var`.</span><span class="sxs-lookup"><span data-stu-id="106c2-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="106c2-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="106c2-109">Example</span></span>  
 <span data-ttu-id="106c2-110">Il codice seguente genera l'errore CS1942:</span><span class="sxs-lookup"><span data-stu-id="106c2-110">The following code generates CS1942:</span></span>  
  
```csharp  
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```  
  
 <span data-ttu-id="106c2-111">L'errore CS1942 è correlato a CS1949 perché l'uso di `var` con una variabile di intervallo causa l'esito negativo dell'operazione `Cast<T>` sottostante perché `var` non è un tipo.</span><span class="sxs-lookup"><span data-stu-id="106c2-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
