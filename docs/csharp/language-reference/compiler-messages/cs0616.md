# <a name="compiler-error-cs0616"></a>Errore del compilatore CS0616
'classe' non è una classe Attribute  
  
 Si è provato a usare una classe non Attribute in un blocco di attributi. Tutti i tipi di attributo devono essere ereditati da <xref:System.Attribute?displayProperty=fullName>.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS0616.  
  
```csharp  
// CS0616.cs  
// compile with: /target:library  
[CMyClass(i = 5)]   // CS0616  
public class CMyClass {}  
```  
  
## <a name="example"></a>Esempio  
 L'esempio seguente mostra come potrebbe essere definito un attributo:  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
