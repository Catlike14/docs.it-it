# <a name="compiler-error-cs1946"></a><span data-ttu-id="956e6-102">Errore del compilatore CS1946</span><span class="sxs-lookup"><span data-stu-id="956e6-102">Compiler Error CS1946</span></span>
<span data-ttu-id="956e6-103">Non è possibile convertire un'espressione di metodo anonimo in un albero delle espressioni.</span><span class="sxs-lookup"><span data-stu-id="956e6-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
 <span data-ttu-id="956e6-104">Un metodo anonimo rappresenta un set di istruzioni, ma un albero delle espressioni non può contenere un'istruzione.</span><span class="sxs-lookup"><span data-stu-id="956e6-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="956e6-105">Pertanto, un metodo anonimo non può essere rappresentato da un albero delle espressioni.</span><span class="sxs-lookup"><span data-stu-id="956e6-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

## <a name="example"></a><span data-ttu-id="956e6-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="956e6-108">Example</span></span>  
 <span data-ttu-id="956e6-109">L'esempio seguente genera l'errore CS1946:</span><span class="sxs-lookup"><span data-stu-id="956e6-109">The following example generates CS1946:</span></span>

// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.

    }
