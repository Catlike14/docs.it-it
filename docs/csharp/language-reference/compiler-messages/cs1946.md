# <a name="compiler-error-cs1946"></a><span data-ttu-id="ae489-102">Errore del compilatore CS1946</span><span class="sxs-lookup"><span data-stu-id="ae489-102">Compiler Error CS1946</span></span>
<span data-ttu-id="ae489-103">Non è possibile convertire un'espressione di metodo anonimo in un albero delle espressioni.</span><span class="sxs-lookup"><span data-stu-id="ae489-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
 <span data-ttu-id="ae489-104">Un metodo anonimo rappresenta un set di istruzioni, ma un albero delle espressioni non può contenere un'istruzione.</span><span class="sxs-lookup"><span data-stu-id="ae489-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="ae489-105">Pertanto, un metodo anonimo non può essere rappresentato da un albero delle espressioni.</span><span class="sxs-lookup"><span data-stu-id="ae489-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ae489-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="ae489-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="ae489-107">Sostituire il metodo anonimo con un'espressione lambda.</span><span class="sxs-lookup"><span data-stu-id="ae489-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ae489-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="ae489-108">Example</span></span>  
 <span data-ttu-id="ae489-109">L'esempio seguente genera l'errore CS1946:</span><span class="sxs-lookup"><span data-stu-id="ae489-109">The following example generates CS1946:</span></span>  
  
```csharp  
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
