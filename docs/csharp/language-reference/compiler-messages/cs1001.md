# <a name="compiler-error-cs1001"></a><span data-ttu-id="117bf-102">Errore del compilatore CS1001</span><span class="sxs-lookup"><span data-stu-id="117bf-102">Compiler Error CS1001</span></span>
<span data-ttu-id="117bf-103">È previsto un identificatore</span><span class="sxs-lookup"><span data-stu-id="117bf-103">Identifier expected</span></span>  
  
 <span data-ttu-id="117bf-104">Non è stato fornito un identificatore.</span><span class="sxs-lookup"><span data-stu-id="117bf-104">You did not supply an identifier.</span></span> <span data-ttu-id="117bf-105">Un identificatore è il nome di un elemento specificato, ad esempio una classe, uno struct, uno spazio dei nomi, un metodo, una variabile e così via.</span><span class="sxs-lookup"><span data-stu-id="117bf-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="117bf-106">L'esempio seguente dichiara una classe semplice ma non fornisce un nome per la classe:</span><span class="sxs-lookup"><span data-stu-id="117bf-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="117bf-107">L'esempio seguente genera l'errore CS1001 perché, quando si dichiara un'enumerazione, è necessario specificarne i membri:</span><span class="sxs-lookup"><span data-stu-id="117bf-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="117bf-108">I nomi di parametro sono obbligatori anche se il compilatore non li usa, ad esempio, in una definizione di interfaccia.</span><span class="sxs-lookup"><span data-stu-id="117bf-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="117bf-109">Questi parametri sono obbligatori, in modo che i programmatori che usano l'interfaccia possano ricavare informazioni sul significato dei parametri.</span><span class="sxs-lookup"><span data-stu-id="117bf-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
