<span data-ttu-id="728ce-103">'Nome proprietà' deve dichiarare un corpo perché non è contrassegnato come abstract o extern.</span><span class="sxs-lookup"><span data-stu-id="728ce-103">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="728ce-104">Le proprietà implementate automaticamente devono definire entrambe le funzioni di accesso get e set.</span><span class="sxs-lookup"><span data-stu-id="728ce-104">Automatically implemented properties must define both get and set accessors.</span></span>

## <a name="example"></a><span data-ttu-id="728ce-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="728ce-110">Example</span></span>
