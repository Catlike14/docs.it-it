# Compiler Error CS0523
Il membro struct 'campo struct2' di tipo 'struct1' provoca un ciclo nel layout della struttura.  
  
 Le definizioni di due strutture includono riferimenti ricorsivi.  Modificare le definizioni [struct](../../../csharp/language-reference/keywords/struct.md) in modo che ciascuna di esse non definisca se stessa sulla base dell'altra.  Queste limitazioni sono valide solo per le strutture perch√© le strutture sono tipi di valore.  Se si utilizzano riferimenti ricorsivi, dichiarare i tipi come classi.  
  
 Il seguente codice di esempio genera l'errore CS0523:  
  
```  
// CS0523.cs  
// compile with: /target:library  
struct RecursiveLayoutStruct1  
{  
   public RecursiveLayoutStruct2 field;  
}  
  
struct RecursiveLayoutStruct2  
{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
