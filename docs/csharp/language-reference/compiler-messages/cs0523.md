# <a name="compiler-error-cs0523"></a>Errore del compilatore CS0523
Il membro struct 'campo struct2' di tipo 'struct1' causa un ciclo nel layout dello struct  
  
 Le definizioni dei due struct includono riferimenti ricorsivi. Modificare le definizioni degli [struct](../../../csharp/language-reference/keywords/struct.md) in modo che ognuno non includa la definizione dell'altro. Questa limitazione si applica solo agli struct, perch√© sono tipi valore. Per usare riferimenti ricorsivi, dichiarare i tipi come classi.  
  
 L'esempio seguente genera l'errore CS0523:  
  
```csharp  
// CS0523.cs  
// compile with: /target:library  
struct RecursiveLayoutStruct1  
{  
   public RecursiveLayoutStruct2 field;  
}  
  
struct RecursiveLayoutStruct2  
{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
