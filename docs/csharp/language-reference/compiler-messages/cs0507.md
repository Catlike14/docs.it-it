# <a name="compiler-error-cs0507"></a><span data-ttu-id="c2cce-102">Errore del compilatore CS0507</span><span class="sxs-lookup"><span data-stu-id="c2cce-102">Compiler Error CS0507</span></span>
<span data-ttu-id="c2cce-103">'funzione1': non è possibile cambiare i modificatori di accesso quando viene eseguito l'override di 'accesso' del membro ereditato 'funzione2'</span><span class="sxs-lookup"><span data-stu-id="c2cce-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="c2cce-104">È stato eseguito un tentativo di modificare la specifica di accesso nell'override di un metodo.</span><span class="sxs-lookup"><span data-stu-id="c2cce-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c2cce-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="c2cce-105">Example</span></span>  
 <span data-ttu-id="c2cce-106">L'esempio seguente genera l'errore CS0507.</span><span class="sxs-lookup"><span data-stu-id="c2cce-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="c2cce-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="c2cce-107">Example</span></span>  
 <span data-ttu-id="c2cce-108">L'errore CS0507 può verificarsi anche se una classe tenta di eseguire l'override di un metodo contrassegnato come `protected internal` definito nei metadati di riferimento.</span><span class="sxs-lookup"><span data-stu-id="c2cce-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="c2cce-109">In questa situazione, il metodo sottoposto a override deve essere contrassegnato come `protected`.</span><span class="sxs-lookup"><span data-stu-id="c2cce-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="c2cce-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="c2cce-110">Example</span></span>  
 <span data-ttu-id="c2cce-111">L'esempio seguente genera l'errore CS0507.</span><span class="sxs-lookup"><span data-stu-id="c2cce-111">The following sample generates CS0507.</span></span>
