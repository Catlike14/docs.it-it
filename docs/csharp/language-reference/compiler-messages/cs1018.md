# <a name="compiler-error-cs1018"></a>Errore del compilatore CS1018
Ãˆ prevista la parola chiave 'this' o 'base'  
  
 Il compilatore ha rilevato una dichiarazione di costruttore incompleta.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1018 e suggerisce vari modi per risolvere l'errore:  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's default constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
