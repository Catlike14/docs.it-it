## <a name="example"></a><span data-ttu-id="371ef-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="371ef-110">Example</span></span>  
 <span data-ttu-id="371ef-111">L'esempio seguente genera l'errore CS0433.</span><span class="sxs-lookup"><span data-stu-id="371ef-111">The following example generates CS0433.</span></span>  
  
```csharp

using TypeBindConflicts;  
public class Test   
{  
   public static void Main()

      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }

```  
  
## <a name="example"></a><span data-ttu-id="371ef-112">Esempio</span><span class="sxs-lookup"><span data-stu-id="371ef-112">Example</span></span>

```csharp  
   }  
}  
```
