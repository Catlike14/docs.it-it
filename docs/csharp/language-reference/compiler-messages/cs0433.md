# <a name="compiler-error-cs0433"></a><span data-ttu-id="f81de-102">Errore del compilatore CS0433</span><span class="sxs-lookup"><span data-stu-id="f81de-102">Compiler Error CS0433</span></span>

## <a name="example"></a><span data-ttu-id="f81de-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="f81de-110">Example</span></span>  
 <span data-ttu-id="f81de-111">L'esempio seguente genera l'errore CS0433.</span><span class="sxs-lookup"><span data-stu-id="f81de-111">The following example generates CS0433.</span></span>  
  
```csharp  
// CS0433_3.cs

using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="f81de-112">Esempio</span><span class="sxs-lookup"><span data-stu-id="f81de-112">Example</span></span>

