# <a name="compiler-error-cs1943"></a><span data-ttu-id="5c726-102">Errore del compilatore CS1943</span><span class="sxs-lookup"><span data-stu-id="5c726-102">Compiler Error CS1943</span></span>
<span data-ttu-id="5c726-103">Un'espressione di tipo 'tipo' non è consentita in una clausola from successiva in un'espressione di query con tipo di origine 'tipo'.</span><span class="sxs-lookup"><span data-stu-id="5c726-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="5c726-104">L'inferenza del tipo non è riuscita nella chiamata a 'metodo'.</span><span class="sxs-lookup"><span data-stu-id="5c726-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="5c726-105">Tutte le variabili di intervallo devono rappresentare tipi queryable.</span><span class="sxs-lookup"><span data-stu-id="5c726-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5c726-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="5c726-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="5c726-107">Assicurarsi che il tipo sia un tipo queryable che implementa `IEnumerable`, `IEnumerable<T>` o un'interfaccia derivata o qualsiasi altro tipo per cui è definito un modello di query.</span><span class="sxs-lookup"><span data-stu-id="5c726-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2.  <span data-ttu-id="5c726-108">Se il tipo è un `IEnumerable` non generico, specificare un tipo esplicito nella variabile di intervallo.</span><span class="sxs-lookup"><span data-stu-id="5c726-108">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5c726-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="5c726-109">Example</span></span>  
 <span data-ttu-id="5c726-110">Il codice seguente genera l'errore CS1943:</span><span class="sxs-lookup"><span data-stu-id="5c726-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
