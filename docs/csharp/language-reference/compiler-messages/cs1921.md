# <a name="compiler-error-cs1921"></a><span data-ttu-id="b34e2-102">Errore del compilatore CS1921</span><span class="sxs-lookup"><span data-stu-id="b34e2-102">Compiler Error CS1921</span></span>
<span data-ttu-id="b34e2-103">Firma per l'elemento inizializzatore non corretta nella corrispondenza migliore del metodo di overload per 'metodo'.</span><span class="sxs-lookup"><span data-stu-id="b34e2-103">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="b34e2-104">Il metodo Add inizializzabile deve essere un metodo di istanza accessibile.</span><span class="sxs-lookup"><span data-stu-id="b34e2-104">The initializable Add must be an accessible instance method.</span></span>  
  
 <span data-ttu-id="b34e2-105">Questo errore viene generato quando si tenta di usare un inizializzatore di insieme con una classe senza un metodo `Add` non statico pubblico.</span><span class="sxs-lookup"><span data-stu-id="b34e2-105">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="b34e2-106">Se il metodo `Add` non è accessibile a causa del livello di protezione (`private`, `protected`, `internal`), si verificherà l'errore CS0122, quindi questo errore indica probabilmente che il metodo è definito come `static`.</span><span class="sxs-lookup"><span data-stu-id="b34e2-106">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>
