# <a name="compiler-error-cs0122"></a>Errore del compilatore CS0122
'membro' Ã¨ inaccessibile a causa del livello di protezione  
  
 Il [modificatore di accesso](../../../csharp/language-reference/keywords/modifiers.md) per una classe membro impedisce l'accesso al membro. Per altre informazioni, vedere [Modificatori di accesso](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Una delle cause di questo errore (non illustrato nell'esempio riportato di seguito) potrebbe essere l'omissione del flag del compilatore **/out** nella destinazione di un assembly Friend. Per altre informazioni, vedere [Assembly Friend](http://msdn.microsoft.com/library/df0c70ea-2c2a-4bdc-9526-df951ad2d055) e [/out (Opzioni del compilatore C#)](../../../csharp/language-reference/compiler-options/out-compiler-option.md).  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS0122:  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
