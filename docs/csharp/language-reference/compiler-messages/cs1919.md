# <a name="compiler-error-cs1919"></a><span data-ttu-id="213a2-102">Errore del compilatore CS1919</span><span class="sxs-lookup"><span data-stu-id="213a2-102">Compiler Error CS1919</span></span>
<span data-ttu-id="213a2-103">Impossibile usare il tipo unsafe "type name" nella creazione di un oggetto.</span><span class="sxs-lookup"><span data-stu-id="213a2-103">Unsafe type 'type name' cannot be used in object creation.</span></span>  
  
 <span data-ttu-id="213a2-104">L'operatore `new` consente di creare oggetti solo nell'heap gestito.</span><span class="sxs-lookup"><span data-stu-id="213a2-104">The `new` operator creates objects only on the managed heap.</span></span> <span data-ttu-id="213a2-105">Tuttavia, è possibile creare oggetti nella memoria non gestita in modo indiretto usando le funzionalità di interoperabilità del linguaggio per chiamare i metodi nativi che restituiscono puntatori.</span><span class="sxs-lookup"><span data-stu-id="213a2-105">However, you can create objects in unmanaged memory indirectly by using the interoperability capabilities of the language to call native methods that return pointers.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="213a2-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="213a2-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="213a2-107">Usare un tipo sicuro nell'espressione di creazione dell'oggetto.</span><span class="sxs-lookup"><span data-stu-id="213a2-107">Use a safe type in the new object creation expression.</span></span> <span data-ttu-id="213a2-108">Ad esempio, usare `char` o `int` anziché `char*` o `int*`.</span><span class="sxs-lookup"><span data-stu-id="213a2-108">For example, use `char` or `int` instead of `char*` or `int*`.</span></span>  
  
2.  <span data-ttu-id="213a2-109">Se è necessario creare gli oggetti nella memoria non gestita, usare un metodo Win32 o COM oppure scrivere una funzione in C o C++ e chiamarla da C#.</span><span class="sxs-lookup"><span data-stu-id="213a2-109">If you must create objects in unmanaged memory, use a Win32 or COM method or else write your own function in C or C++ and call it from C#.</span></span>  
  
## <a name="example"></a><span data-ttu-id="213a2-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="213a2-110">Example</span></span>  
 <span data-ttu-id="213a2-111">Nell'esempio seguente viene generato l'errore CS1919 perché un tipo di puntatore è unsafe:</span><span class="sxs-lookup"><span data-stu-id="213a2-111">The following example generates CS1919 because a pointer type is unsafe:</span></span>  
  
```csharp  
// cs1919.cs  
// Compile with: /unsafe  
unsafe public class C  
{  
    public static int Main()  
    {  
        var col1 = new int* { }; // CS1919  
        var col2 = new char* { }; // CS1919  
        return 1;  
    }  
}  
```
