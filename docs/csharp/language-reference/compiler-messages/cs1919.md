<span data-ttu-id="67adc-103">Impossibile usare il tipo unsafe "type name" nella creazione di un oggetto.</span><span class="sxs-lookup"><span data-stu-id="67adc-103">Unsafe type 'type name' cannot be used in object creation.</span></span>

## <a name="example"></a><span data-ttu-id="67adc-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="67adc-110">Example</span></span>

 <span data-ttu-id="67adc-111">Nell'esempio seguente viene generato l'errore CS1919 perché un tipo di puntatore è unsafe:</span><span class="sxs-lookup"><span data-stu-id="67adc-111">The following example generates CS1919 because a pointer type is unsafe:</span></span>

```csharp
{  
    public static int Main()

        var col1 = new int* { }; // CS1919

    }  
}  
```
