# <a name="compiler-error-cs0563"></a>Errore del compilatore CS0563
Uno dei parametri di un operatore binario deve essere il tipo che lo contiene  
  
 La dichiarazione di metodo per un [overload degli operatori](../../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md) deve seguire determinate linee guida.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS0563:  
  
```csharp  
// CS0563.cs  
public class iii  
{  
    public static implicit operator int(iii x)  
    {  
        return 0;  
    }  
    public static implicit operator iii(int x)  
    {  
        return null;  
    }  
    public static int operator +(int aa, int bb)   // CS0563   
    // Use the following line instead:  
    // public static int operator +(int aa, iii bb)      
    {  
        return 0;  
    }  
    public static void Main()  
    {  
    }  
}  
```
