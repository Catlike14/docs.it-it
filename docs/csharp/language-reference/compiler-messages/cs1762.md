 <span data-ttu-id="92bae-105">È stato aggiunto un riferimento a un assembly (assembly1) con la proprietà `Embed Interop Types` impostata su `True`.</span><span class="sxs-lookup"><span data-stu-id="92bae-105">You have added a reference to an assembly (assembly1) that has the `Embed Interop Types` property set to `True`.</span></span> <span data-ttu-id="92bae-106">Si dà così istruzione al compilatore di incorporare le informazioni sui tipi di interoperabilità da tale assembly.</span><span class="sxs-lookup"><span data-stu-id="92bae-106">This instructs the compiler to embed interop type information from that assembly.</span></span> <span data-ttu-id="92bae-107">Il compilatore non può tuttavia incorporare le informazioni sui tipi di interoperabilità da tale assembly perché anche un altro assembly a cui si fa riferimento (assembly2) fa riferimento a tale assembly (assembly1) e ha la proprietà `Embed Interop Types` impostata su `False`.</span><span class="sxs-lookup"><span data-stu-id="92bae-107">However, the compiler cannot embed interop type information from that assembly because another assembly that you have referenced (assembly2) also references that assembly (assembly1) and has the `Embed Interop Types` property set to `False`.</span></span>
