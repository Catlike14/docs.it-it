# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="df795-102">Avviso del compilatore (livello 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="df795-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="df795-103">L'accesso a un membro di 'membro' potrebbe causare un'eccezione in fase di esecuzione perché è un campo di una classe con marshaling per riferimento</span><span class="sxs-lookup"><span data-stu-id="df795-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="df795-104">Questo avviso viene generato quando si tenta di chiamare un metodo, una proprietà o un indicizzatore su un membro di una classe che deriva da <xref:System.MarshalByRefObject> e il membro è un tipo valore.</span><span class="sxs-lookup"><span data-stu-id="df795-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="df795-105">Gli oggetti che ereditano da `MarshalByRefObject` sono generalmente destinati al marshalling per riferimento in un dominio applicazione.</span><span class="sxs-lookup"><span data-stu-id="df795-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="df795-106">In presenza di codice che tenta di accedere direttamente al membro tipo valore di uno di questi oggetti in un dominio applicazione, si verificherà un'eccezione di runtime.</span><span class="sxs-lookup"><span data-stu-id="df795-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="df795-107">Per risolvere l'avviso, copiare prima di tutto il membro in una variabile locale e chiamare il metodo su tale variabile.</span><span class="sxs-lookup"><span data-stu-id="df795-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="df795-108">L'esempio seguente genera l'errore CS1690:</span><span class="sxs-lookup"><span data-stu-id="df795-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
