# <a name="compiler-error-cs1019"></a><span data-ttu-id="d484e-102">Errore del compilatore CS1019</span><span class="sxs-lookup"><span data-stu-id="d484e-102">Compiler Error CS1019</span></span>
<span data-ttu-id="d484e-103">È previsto un operatore unario che supporti l'overload</span><span class="sxs-lookup"><span data-stu-id="d484e-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="d484e-104">È stato dichiarato un oggetto simile a un operatore unario di overload, ma l'operatore è assente o in una posizione non valida nella firma.</span><span class="sxs-lookup"><span data-stu-id="d484e-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="d484e-105">Un *operatore unario* è un operatore che agisce su un singolo operando.</span><span class="sxs-lookup"><span data-stu-id="d484e-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="d484e-106">Ad esempio, `++` è un operatore unario.</span><span class="sxs-lookup"><span data-stu-id="d484e-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="d484e-107">È possibile eseguire l'overload di alcuni operatori unari usando la parola chiave `operator` e specificando un singolo parametro del tipo su cui agisce l'operatore.</span><span class="sxs-lookup"><span data-stu-id="d484e-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="d484e-108">Ad esempio, per eseguire l'overload dell'operatore `++` per una classe `Temperature` definita dall'utente in modo da poter scrivere `Temperature++`, è possibile definirlo in questo modo:</span><span class="sxs-lookup"><span data-stu-id="d484e-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="d484e-109">Se si riceve questo errore, è stato dichiarato un oggetto simile a un operatore unario di overload, ma l'operatore stesso è assente o si trova in una posizione non valida nella firma.</span><span class="sxs-lookup"><span data-stu-id="d484e-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="d484e-110">Se si rimuove `++` dalla firma dell'esempio precedente, verrà generato l'errore CS1019.</span><span class="sxs-lookup"><span data-stu-id="d484e-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="d484e-111">Il codice seguente genera l'errore CS1019:</span><span class="sxs-lookup"><span data-stu-id="d484e-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
