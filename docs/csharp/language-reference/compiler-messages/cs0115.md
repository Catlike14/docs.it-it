## <a name="example"></a><span data-ttu-id="eb010-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="eb010-106">Example</span></span>  
 <span data-ttu-id="eb010-107">L'esempio seguente genera l'errore CS0115.</span><span class="sxs-lookup"><span data-stu-id="eb010-107">The following sample generates CS0115.</span></span> <span data-ttu-id="eb010-108">È possibile risolvere l'errore CS0115 in uno dei due modi seguenti:</span><span class="sxs-lookup"><span data-stu-id="eb010-108">You can resolve CS0115 in one of two ways:</span></span>  
  
-   <span data-ttu-id="eb010-109">Rimuovere la parola chiave `override` dal metodo in `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="eb010-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
-   <span data-ttu-id="eb010-110">Usare `MyClass1` come classe di base per `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="eb010-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
