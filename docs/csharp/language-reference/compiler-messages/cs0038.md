<span data-ttu-id="f87b3-103">Impossibile accedere a un membro non statico di tipo outer 'tipo1' tramite il tipo annidato 'tipo2'</span><span class="sxs-lookup"><span data-stu-id="f87b3-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="f87b3-104">Un campo in una classe non è automaticamente disponibile in una classe annidata.</span><span class="sxs-lookup"><span data-stu-id="f87b3-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="f87b3-105">Affinché sia disponibile per una classe annidata, il campo deve essere [statico](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="f87b3-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="f87b3-106">In caso contrario, è necessario creare un'istanza della classe outer.</span><span class="sxs-lookup"><span data-stu-id="f87b3-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="f87b3-107">Per altre informazioni, vedere [Tipi annidati](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="f87b3-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="f87b3-108">L'esempio seguente genera l'errore CS0038:</span><span class="sxs-lookup"><span data-stu-id="f87b3-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
