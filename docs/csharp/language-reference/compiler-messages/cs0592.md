## <a name="example"></a><span data-ttu-id="0d772-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="0d772-108">Example</span></span>  
 <span data-ttu-id="0d772-109">L'esempio seguente genera l'errore CS0592:</span><span class="sxs-lookup"><span data-stu-id="0d772-109">The following sample generates CS0592:</span></span>  
  
```csharp  
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute   
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.   
public class A    
{  
    public static void Main()  
    {  
    }  
}  
```
