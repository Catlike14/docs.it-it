# <a name="compiler-error-cs1936"></a><span data-ttu-id="b897b-102">Errore del compilatore CS1936</span><span class="sxs-lookup"><span data-stu-id="b897b-102">Compiler Error CS1936</span></span>

<span data-ttu-id="b897b-103">Non è stata trovata un'implementazione del modello di query per il tipo di origine 'type'.</span><span class="sxs-lookup"><span data-stu-id="b897b-103">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="b897b-104">'method' non è presente.</span><span class="sxs-lookup"><span data-stu-id="b897b-104">'method' not found.</span></span>  
  
<span data-ttu-id="b897b-105">Per eseguire una query su un tipo di origine, è necessario che il tipo usi i metodi operatori di query standard che vengono chiamati nella query.</span><span class="sxs-lookup"><span data-stu-id="b897b-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="b897b-106">L'implementazione può essere effettuata sotto forma di membri di classi o metodi di estensione inseriti all'ambito con la direttiva `using` appropriata.</span><span class="sxs-lookup"><span data-stu-id="b897b-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="b897b-107">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="b897b-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="b897b-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="b897b-110">Example</span></span>

<span data-ttu-id="b897b-111">L'esempio seguente produce l'errore CS1936:</span><span class="sxs-lookup"><span data-stu-id="b897b-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="b897b-112">In genere, questo errore si verifica quando si esegue accidentalmente una query su un oggetto di un tipo anziché su una raccolta di tali oggetti.</span><span class="sxs-lookup"><span data-stu-id="b897b-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
