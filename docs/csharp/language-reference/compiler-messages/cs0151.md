# <a name="compiler-error-cs0151"></a><span data-ttu-id="1600a-102">Errore del compilatore CS0151</span><span class="sxs-lookup"><span data-stu-id="1600a-102">Compiler Error CS0151</span></span>
<span data-ttu-id="1600a-103">È previsto un valore di tipo integrale</span><span class="sxs-lookup"><span data-stu-id="1600a-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="1600a-104">È stata usata una variabile in una situazione in cui era richiesto un tipo di dati integrale.</span><span class="sxs-lookup"><span data-stu-id="1600a-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="1600a-105">Per altre informazioni, vedere [Tipi](../../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="1600a-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1600a-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="1600a-106">Example</span></span>  
 <span data-ttu-id="1600a-107">Questo errore può verificarsi quando non viene eseguita alcuna conversione o se le conversioni implicite disponibili generano una situazione ambigua.</span><span class="sxs-lookup"><span data-stu-id="1600a-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="1600a-108">L'esempio seguente genera l'errore CS0151.</span><span class="sxs-lookup"><span data-stu-id="1600a-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="1600a-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="1600a-109">Example</span></span>

```csharp  
class C  
{  
    static void Main()  
    {  
  
        switch (M()) // CS0151  
        {  
            default:  
                break;  
        }  
    }  
  
    static void M()  
    {  
    }  
}  
```
