## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="989a6-106">Esempio di conversione ambigua</span><span class="sxs-lookup"><span data-stu-id="989a6-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="989a6-107">Questo errore può verificarsi quando non viene eseguita alcuna conversione o se le conversioni implicite disponibili generano una situazione ambigua.</span><span class="sxs-lookup"><span data-stu-id="989a6-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="989a6-108">L'esempio seguente genera l'errore CS0151:</span><span class="sxs-lookup"><span data-stu-id="989a6-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="989a6-109">Esempio di un metodo void</span><span class="sxs-lookup"><span data-stu-id="989a6-109">Example of void method</span></span>

<span data-ttu-id="989a6-110">La chiamata a un metodo [void](../../../csharp/language-reference/keywords/void.md) in un'espressione di ricerca [switch](../keywords/switch.md) genera l'errore CS0151.</span><span class="sxs-lookup"><span data-stu-id="989a6-110">A [void](../../../csharp/language-reference/keywords/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="989a6-111">È possibile risolvere l'errore chiamando un metodo che restituisce un tipo integrale, ad esempio [int](../../../csharp/language-reference/keywords/int.md) o [long](../../../csharp/language-reference/keywords/long.md).</span><span class="sxs-lookup"><span data-stu-id="989a6-111">You can fix the error by calling a method that returns an integral type such as [int](../../../csharp/language-reference/keywords/int.md) or [long](../../../csharp/language-reference/keywords/long.md) instead.</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
