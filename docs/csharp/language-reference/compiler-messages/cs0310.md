<span data-ttu-id="b8d03-103">'nometipo' deve essere un tipo non astratto con un costruttore pubblico senza parametri per essere utilizzato come parametro 'parametro' nel tipo o nel metodo 'generico'</span><span class="sxs-lookup"><span data-stu-id="b8d03-103">The type 'typename' must be a non-abstract type with a public parameterless constructor in order to use it as parameter 'parameter' in the generic type or method 'generic'</span></span>  
  
 <span data-ttu-id="b8d03-104">Il tipo o metodo generico definisce un nuovo vincolo nella relativa clausola where, pertanto qualsiasi tipo deve avere un costruttore pubblico senza parametri per poter essere usato come argomento di tipo per quel tipo o metodo generico.</span><span class="sxs-lookup"><span data-stu-id="b8d03-104">The generic type or method defines a new constraint in its where clause, so any type must have a public parameterless constructor in order to be used as a type argument for that generic type or method.</span></span> <span data-ttu-id="b8d03-105">Per evitare questo errore, assicurarsi che il tipo abbia il costruttore corretto oppure modificare la clausola di vincolo del tipo o del metodo generico.</span><span class="sxs-lookup"><span data-stu-id="b8d03-105">To avoid this error, make sure that the type has the correct constructor, or modify the constraint clause of the generic type or method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b8d03-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="b8d03-106">Example</span></span>  
 <span data-ttu-id="b8d03-107">L'esempio seguente genera l'errore CS0310:</span><span class="sxs-lookup"><span data-stu-id="b8d03-107">The following sample generates CS0310:</span></span>  
  
```csharp  
// CS0310.cs  
using System;  
  
class G<T> where T : new()  
{  
    T t;  
  
    public G()  
    {  
        t = new T();  
        Console.WriteLine(t);  
    }  
}  
  
class B  
{  
    private B() { }  
    // Try this instead:  
    // public B() { }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        G<B> g = new G<B>();   // CS0310  
        Console.WriteLine(g.ToString());  
    }  
}  
```
