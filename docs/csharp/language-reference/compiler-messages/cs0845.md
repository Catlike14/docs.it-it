# <a name="compiler-error-cs0845"></a><span data-ttu-id="52996-102">Errore del compilatore CS0845</span><span class="sxs-lookup"><span data-stu-id="52996-102">Compiler Error CS0845</span></span>
<span data-ttu-id="52996-103">Un'espressione lambda dell'albero delle espressioni non può contenere un operatore coalescing con un valore letterale null nella parte sinistra.</span><span class="sxs-lookup"><span data-stu-id="52996-103">An expression tree lambda may not contain a coalescing operator with a null literal left-hand side.</span></span>  
  
 <span data-ttu-id="52996-104">Dato che il valore null non ha un tipo, l'operatore coalescing per null non può operare su di esso.</span><span class="sxs-lookup"><span data-stu-id="52996-104">Because null by itself does not have a type, the null coalescing operator cannot operate on it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="52996-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="52996-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="52996-106">Eseguire il cast del valore letterale null a un oggetto.</span><span class="sxs-lookup"><span data-stu-id="52996-106">Cast the null literal to an object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="52996-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="52996-107">Example</span></span>  
 <span data-ttu-id="52996-108">Il codice seguente genera l'errore CS0845:</span><span class="sxs-lookup"><span data-stu-id="52996-108">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
