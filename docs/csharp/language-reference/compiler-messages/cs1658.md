# <a name="compiler-warning-level-1-cs1658"></a>Avviso del compilatore (livello 1) CS1658
'testo avviso'. Vedere anche l'errore 'codice di errore'  
  
 Il compilatore genera questo avviso quando viene eseguito l'override di un errore con un avviso. Per informazioni sul problema, fare riferimento all'errore indicato. Per trovare l'errore appropriato dall'IDE di Visual Studio, usare l'indice. Ad esempio, se il testo precedente Ã¨ "Vedere anche l'errore 'CS1037'", cercare CS1037 nell'indice.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1658.  
  
```csharp  
// CS1658.cs  
// compile with: /doc:x.xml  
// CS1584 expected  
/// <summary>  
/// </summary>  
public class C  
{  
    /// <see cref="C.F(params object[])"/>  // CS1658  
    public static void M()  
    {  
    }  
  
    /// <summary>  
    /// </summary>  
    public void F(params object[] o)  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
