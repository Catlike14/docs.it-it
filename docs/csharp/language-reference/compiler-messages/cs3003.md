# <a name="compiler-warning-level-1-cs3003"></a>Avviso del compilatore (livello 1) CS3003
Il tipo di parametro "variable" non è conforme a CLS  
  
 Una variabile [public](../../../csharp/language-reference/keywords/public.md), [protected](../../../csharp/language-reference/keywords/protected.md) o `protected internal` deve essere di un tipo conforme a Common Language Specification (CLS). Per altre informazioni sulla conformità a CLS, vedere [Indipendenza del linguaggio e componenti indipendenti dal linguaggio](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Esempio  
 Il seguente codice di esempio genera l'avviso CS3003:  
  
```csharp  
// CS3003.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort a1;   // CS3003, public variable  
    public static void Main()  
    {  
    }  
}  
```
