# <a name="compiler-error-cs0051"></a>Errore del compilatore CS0051
Accessibilità incoerente: il tipo di parametro 'tipo' è meno accessibile del metodo 'metodo'  
  
 Il tipo restituito e ciascuno dei tipi a cui viene fatto riferimento nell'elenco di parametri formali di un metodo deve essere accessibile almeno quanto il metodo stesso. Assicurarsi che i tipi usati nelle firme dei metodi non siano accidentalmente privati a causa dell'omissione del modificatore `public`. Per altre informazioni, vedere [Modificatori di accesso](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS0051:  
  
```csharp  
// CS0051.cs  
public class A  
{  
    // Try making B public since F is public  
    // B is implicitly private here  
    class B  
    {  
    }  
  
    public static void F(B b)  // CS0051  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
