# <a name="compiler-error-cs1009"></a><span data-ttu-id="8e59f-102">Errore del compilatore CS1009</span><span class="sxs-lookup"><span data-stu-id="8e59f-102">Compiler Error CS1009</span></span>
<span data-ttu-id="8e59f-103">Sequenza di escape non riconosciuta</span><span class="sxs-lookup"><span data-stu-id="8e59f-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="8e59f-104">È stato rilevato un carattere imprevisto subito dopo una barra rovesciata (\\) in una [stringa](../../../csharp/language-reference/keywords/string.md).</span><span class="sxs-lookup"><span data-stu-id="8e59f-104">An unexpected character follows a backslash (\\) in a [string](../../../csharp/language-reference/keywords/string.md).</span></span> <span data-ttu-id="8e59f-105">Nel compilatore è previsto uno dei caratteri di escape validi.</span><span class="sxs-lookup"><span data-stu-id="8e59f-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="8e59f-106">Per altre informazioni, vedere [Caratteri di escape](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="8e59f-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="8e59f-107">L'esempio seguente genera l'errore CS1009.</span><span class="sxs-lookup"><span data-stu-id="8e59f-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="8e59f-108">Questo errore si verifica in genere quando si utilizza la barra rovesciata in un nome di file, come mostrato nel seguente esempio.</span><span class="sxs-lookup"><span data-stu-id="8e59f-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="8e59f-109">Per correggere l'errore, usare "\\\\" o la stringa letterale @-quoted, come illustrato nell'esempio riportato di seguito.</span><span class="sxs-lookup"><span data-stu-id="8e59f-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
