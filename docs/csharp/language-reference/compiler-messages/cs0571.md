# <a name="compiler-error-cs0571"></a><span data-ttu-id="a9180-102">Errore del compilatore CS0571</span><span class="sxs-lookup"><span data-stu-id="a9180-102">Compiler Error CS0571</span></span>
<span data-ttu-id="a9180-103">'funzione': impossibile chiamare in modo esplicito l'operatore o la funzione di accesso</span><span class="sxs-lookup"><span data-stu-id="a9180-103">'function' : cannot explicitly call operator or accessor</span></span>  
  
 <span data-ttu-id="a9180-104">Alcuni operatori hanno nomi interni.</span><span class="sxs-lookup"><span data-stu-id="a9180-104">Certain operators have internal names.</span></span> <span data-ttu-id="a9180-105">Ad esempio, **op_Increment** è il nome interno dell'operatore ++.</span><span class="sxs-lookup"><span data-stu-id="a9180-105">For example, **op_Increment** is the internal name of the ++ operator.</span></span> <span data-ttu-id="a9180-106">Non è possibile usare o chiamare in modo esplicito questi nomi di metodi.</span><span class="sxs-lookup"><span data-stu-id="a9180-106">You should not use or explicitly call such method names.</span></span>  
  
 <span data-ttu-id="a9180-107">L'esempio seguente genera l'errore CS0571:</span><span class="sxs-lookup"><span data-stu-id="a9180-107">The following sample generates CS0571:</span></span>  
  
```csharp  
// CS0571.cs  
public class MyClass  
{  
   public static MyClass operator ++ (MyClass c)  
   {  
      return null;  
   }  
  
   public static int prop  
   {  
      get  
      {  
         return 1;  
      }  
      set  
      {  
      }  
   }  
  
   public static void Main()  
   {  
      op_Increment(null);   // CS0571  
      // use the increment operator as follows  
      // MyClass x = new MyClass();  
      // x++;  
  
      set_prop(1);      // CS0571  
      // try the following line instead  
      // prop = 1;  
   }  
}  
```
