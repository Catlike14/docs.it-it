# <a name="compiler-error-cs1674"></a><span data-ttu-id="9e1f5-102">Errore del compilatore CS1674</span><span class="sxs-lookup"><span data-stu-id="9e1f5-102">Compiler Error CS1674</span></span>
<span data-ttu-id="9e1f5-103">'T': il tipo usato in un'istruzione using deve essere convertibile in modo implicito in 'System.IDisposable'</span><span class="sxs-lookup"><span data-stu-id="9e1f5-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="9e1f5-104">L'[istruzione using](../../../csharp/language-reference/keywords/using-statement.md) è progettata per garantire l'eliminazione di un oggetto alla fine del blocco `using`, di conseguenza, solo i tipi eliminabili possono essere usati in un'istruzione di questo tipo.</span><span class="sxs-lookup"><span data-stu-id="9e1f5-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="9e1f5-105">Ad esempio, i tipi valore non sono eliminabili e i parametri di tipo non vincolati come classi non possono essere considerati eliminabili.</span><span class="sxs-lookup"><span data-stu-id="9e1f5-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9e1f5-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="9e1f5-106">Example</span></span>  
 <span data-ttu-id="9e1f5-107">L'esempio seguente genera l'errore CS1674.</span><span class="sxs-lookup"><span data-stu-id="9e1f5-107">The following sample generates CS1674.</span></span>

```csharp  
// CS1674.cs

   {  
      int a = 0;

```  
  
## <a name="example"></a><span data-ttu-id="9e1f5-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="9e1f5-108">Example</span></span>  
 <span data-ttu-id="9e1f5-109">L'esempio seguente genera l'errore CS1674.</span><span class="sxs-lookup"><span data-stu-id="9e1f5-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
