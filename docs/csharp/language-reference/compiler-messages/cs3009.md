# Compiler Warning (level 1) CS3009
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

'tipo': il tipo base 'tipo' non è conforme a CLS.  
  
 In un assembly contrassegnato come conforme a CLS \(Common Language Specification\) è stato specificato un tipo base non conforme a CLS.  Rimuovere l'attributo che contrassegna l'assembly come conforme a CLS oppure l'attributo che indica il tipo come non conforme a CLS.  Per ulteriori informazioni sulla conformità a CLS, vedere [Scrittura di codice conforme a CLS](http://msdn.microsoft.com/it-it/4c705105-69a2-4e5e-b24e-0633bc32c7f3) e [Indipendenza del linguaggio e componenti indipendenti dal linguaggio](../Topic/Language%20Independence%20and%20Language-Independent%20Components.md).  
  
## Esempio  
 Il seguente codice di esempio genera l'avviso CS3009:  
  
```  
// CS3009.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
[CLSCompliant(false)]  
public class B  
{  
}  
  
public class C : B   // CS3009  
{  
    public static void Main () {}  
}  
```
