## <a name="example"></a><span data-ttu-id="c170d-115">Esempio</span><span class="sxs-lookup"><span data-stu-id="c170d-115">Example</span></span>  
 <span data-ttu-id="c170d-116">L'esempio seguente genera l'errore CS0106.</span><span class="sxs-lookup"><span data-stu-id="c170d-116">The following sample generates CS0106.</span></span>  
  
```csharp

{  
   interface I  
   {  
      void m();  
      static public void f();   // CS0106

   {  
      public void I.m() {}   // CS0106  
      public static void Main() {}  
   }  
}
```
