## <a name="example"></a><span data-ttu-id="37299-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="37299-107">Example</span></span>  
 <span data-ttu-id="37299-108">L'esempio seguente genera l'errore CS0071.</span><span class="sxs-lookup"><span data-stu-id="37299-108">The following sample generates CS0071.</span></span>  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
