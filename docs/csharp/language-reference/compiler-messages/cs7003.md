# <a name="compiler-error-cs7003"></a>CS7003 errore del compilatore
Uso imprevisto di un nome generico non associato

Questo errore si verifica se si utilizza un tipo generico che necessitano di un parametro di tipo senza passare per il nome di un tipo di parametro generico tra parentesi angolari. Questo utilizzo pu√≤ essere una dichiarazione di variabile o riportato un oggetto.

## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
Quando si utilizza un tipo generico, fornire un nome del tipo di parametro tra parentesi quadre.  

 ## <a name="example"></a>Esempio  
 Nell'esempio seguente genera l'errore CS7003:  
  
```csharp  
// CS7003.cs  
    class Program
    {
        static void Main(string[] args)
        {
            var myVar1 = new MyGenericClass<>();  //CS7003

            MyGenericClass<> var2;                //CS7003
        }
    }

    public class MyGenericClass<T> { }
```
