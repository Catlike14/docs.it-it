## <a name="example"></a><span data-ttu-id="23583-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="23583-106">Example</span></span>  
 <span data-ttu-id="23583-107">L'esempio seguente genera l'errore CS0050 perché non è specificato alcun modificatore di accessibilità per `MyClass` e viene quindi usata l'impostazione predefinita per l'accessibilità `private`.</span><span class="sxs-lookup"><span data-stu-id="23583-107">The following sample generates CS0050 because no accessiblity modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>

```csharp  
// CS0050.cs  
class MyClass //accessibility defaults to private  
// try the following line instead  
// public class MyClass   
{  
}  
  
public class MyClass2  
{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
