# Compiler Error CS0001
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Errore interno del compilatore  
  
 Provare a stabilire se il compilatore non funziona correttamente perché non è in grado di analizzare una sintassi imprevista.  Se si riceve ripetutamente questo errore, contattare Microsoft.
