# Istruzioni di selezione (Riferimenti per C#)
Le istruzioni di selezione trasferiscono il controllo del programma a un flusso specifico a seconda che una determinata condizione sia o meno `true`.  
  
 Nelle istruzioni di selezione vengono utilizzate le seguenti parole chiave:  
  
-   [if](../../../csharp/language-reference/keywords/if-else.md)  
  
-   [else](../../../csharp/language-reference/keywords/if-else.md)  
  
-   [switch](../../../csharp/language-reference/keywords/switch.md)  
  
-   [case](../../../csharp/language-reference/keywords/switch.md)  
  
-   [default](../../../csharp/language-reference/keywords/switch.md)
