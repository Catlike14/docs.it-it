# <a name="using-c-reference"></a>using (Riferimenti per C#)
La parola chiave `using` ha due usi principali:  
  
-   Come direttiva, quando viene usata per creare un alias per uno spazio dei nomi o per importare tipi definiti in altri spazi dei nomi. Vedere [Direttiva using](../../../csharp/language-reference/keywords/using-directive.md).  
  
-   Come istruzione, quando definisce un ambito alla fine del quale viene eliminato un oggetto. Vedere [Istruzione using](../../../csharp/language-reference/keywords/using-statement.md).  
  
La direttiva [using static](using-static.md) consente inoltre di definire un tipo i cui membri statici sono accessibili senza specificare un nome di tipo.
