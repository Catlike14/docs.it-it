# <a name="integral-types-table-c-reference"></a>Tabella dei tipi integrali (Riferimenti per C#)
La tabella seguente elenca le dimensioni e gli intervalli dei tipi integrali, che costituiscono un sottoinsieme dei tipi semplici.  
  
|Tipo|Intervallo|Dimensioni|  
|----------|-----------|----------|  
|[sbyte](../../../csharp/language-reference/keywords/sbyte.md)|Da -128 a 127|Valore intero con segno a 8 bit|  
|[byte](../../../csharp/language-reference/keywords/byte.md)|Da 0 a 255|Intero senza segno a 8 bit|  
|[char](../../../csharp/language-reference/keywords/char.md)|Da U+0000 a U+ffff|Carattere Unicode a 16 bit|  
|[short](../../../csharp/language-reference/keywords/short.md)|Da –32,768 a 32,767|Valore intero a 16 bit con segno|  
|[ushort](../../../csharp/language-reference/keywords/ushort.md)|Da 0 a 65.535|Intero senza segno a 16 bit|  
|[int](../../../csharp/language-reference/keywords/int.md)|da -2.147.483.648 a 2.147.483.647|Valore intero a 32 bit con segno|  
|[uint](../../../csharp/language-reference/keywords/uint.md)|Da 0 a 4.294.967.295|Intero senza segno a 32 bit|  
|[long](../../../csharp/language-reference/keywords/long.md)|Da -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807|Numero intero con segno a 64 bit|  
|[ulong](../../../csharp/language-reference/keywords/ulong.md)|Da 0 a 18.446.744.073.709.551.615|Intero senza segno a 64 bit|  
  
 Se il valore rappresentato da un valore letterale Integer è esterno all'intervallo consentito per `ulong`, si verifica un errore di compilazione.
