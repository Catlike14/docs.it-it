# /help, /? (C# Compiler Options)
Questa opzione consente di inviare a stdout un elenco di opzioni del compilatore con una breve descrizione per ciascuna opzione.  
  
## Sintassi  
  
```  
/help  
/?  
```  
  
## Note  
 Se l'opzione viene inclusa in una compilazione, non verrà creato alcun file di output e non avrà luogo alcuna compilazione.  
  
 Questa opzione del compilatore non è disponibile in Visual Studio e non può essere modificata a livello di codice.
