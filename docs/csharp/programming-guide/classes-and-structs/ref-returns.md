# <a name="ref-returns-and-ref-locals"></a><span data-ttu-id="1c042-103">Valori restituiti e variabili locali ref</span><span class="sxs-lookup"><span data-stu-id="1c042-103">Ref returns and ref locals</span></span>

<span data-ttu-id="1c042-104">A partire da C# 7, C# supporta i valori restituiti di riferimento (valori restituiti ref).</span><span class="sxs-lookup"><span data-stu-id="1c042-104">Starting with C# 7, C# supports reference return values (ref returns).</span></span> <span data-ttu-id="1c042-105">Un valore restituito di riferimento consente a un metodo di restituire a un chiamante un riferimento a una variabile, invece di un valore.</span><span class="sxs-lookup"><span data-stu-id="1c042-105">A reference return value allows a method to return a reference to a variable, rather than a value, back to a caller.</span></span> <span data-ttu-id="1c042-106">Il chiamante può quindi scegliere di trattare la variabile come se fosse stata restituita da un valore o un riferimento.</span><span class="sxs-lookup"><span data-stu-id="1c042-106">The caller can then choose to treat the returned variable as if it were returned by value or by reference.</span></span> <span data-ttu-id="1c042-107">Il chiamante può creare una nuova variabile che è di per sé un riferimento al valore restituito, chiamata variabile locale ref.</span><span class="sxs-lookup"><span data-stu-id="1c042-107">The caller can create a new variable that is itself a reference to the returned value, called a ref local.</span></span>

## <a name="what-is-a-reference-return-value"></a><span data-ttu-id="1c042-108">Che cos'è un valore restituito di riferimento?</span><span class="sxs-lookup"><span data-stu-id="1c042-108">What is a reference return value?</span></span>

<span data-ttu-id="1c042-109">La maggior parte degli sviluppatori ha familiarità con il passaggio di un argomento a un metodo chiamato *tramite un riferimento*.</span><span class="sxs-lookup"><span data-stu-id="1c042-109">Most developers are familiar with passing an argument to a called method *by reference*.</span></span> <span data-ttu-id="1c042-110">Un elenco di argomenti del metodo chiamato include una variabile passata da un riferimento e tutte le modifiche apportate al relativo valore dal metodo chiamato vengono osservate dal chiamante.</span><span class="sxs-lookup"><span data-stu-id="1c042-110">A called method's argument list includes a variable passed by reference, and any changes made to its value by the called method are observed by the caller.</span></span> <span data-ttu-id="1c042-111">Un *valore restituito di riferimento* indica che un metodo restituisce un *riferimento* (o un alias) a una variabile il cui ambito include il metodo e la cui durata deve estendersi oltre la restituzione del metodo.</span><span class="sxs-lookup"><span data-stu-id="1c042-111">A *reference return value* means that a method returns a *reference* (or an alias) to some variable whose scope includes the method and whose lifetime must extend beyond the return of the method.</span></span> <span data-ttu-id="1c042-112">Le modifiche effettuate dal chiamante al valore restituito del metodo vengono apportate alla variabile restituita dal metodo.</span><span class="sxs-lookup"><span data-stu-id="1c042-112">Modifications to the method's return value by the caller are made to the variable that is returned by the method.</span></span>

<span data-ttu-id="1c042-113">La dichiarazione che un metodo restituisce un *valore restituito di riferimento* indica che il metodo restituisce un alias a una variabile.</span><span class="sxs-lookup"><span data-stu-id="1c042-113">Declaring that a method returns a *reference return value* indicates that the method returns an alias to a variable.</span></span> <span data-ttu-id="1c042-114">La finalità della progettazione è spesso quella di fare in modo che il codice chiamante abbia accesso a tale variabile tramite l'alias, inclusa la possibilità di modificarla.</span><span class="sxs-lookup"><span data-stu-id="1c042-114">The design intent is often that the calling code should have access to that variable through the alias, including to modify it.</span></span> <span data-ttu-id="1c042-115">Ne consegue che i metodi restituiti per riferimento non possono avere il tipo restituito `void`.</span><span class="sxs-lookup"><span data-stu-id="1c042-115">It follows that methods returning by reference cannot have the return type `void`.</span></span>

<span data-ttu-id="1c042-116">Esistono alcune restrizioni per l'espressione che un metodo può restituire come valore restituito di riferimento.</span><span class="sxs-lookup"><span data-stu-id="1c042-116">There are some restrictions on the expression that a method can return as a reference return value.</span></span> <span data-ttu-id="1c042-117">Sono inclusi:</span><span class="sxs-lookup"><span data-stu-id="1c042-117">These include:</span></span>

- <span data-ttu-id="1c042-118">Il valore restituito deve avere una durata che si estende oltre l'esecuzione del metodo.</span><span class="sxs-lookup"><span data-stu-id="1c042-118">The return value must have a lifetime that extends beyond the execution of the method.</span></span> <span data-ttu-id="1c042-119">In altre parole, non può essere una variabile locale nel metodo che la restituisce.</span><span class="sxs-lookup"><span data-stu-id="1c042-119">In other words, it cannot be a local variable in the method that returns it.</span></span> <span data-ttu-id="1c042-120">Può essere un'istanza o un campo statico di una classe oppure un argomento passato al metodo.</span><span class="sxs-lookup"><span data-stu-id="1c042-120">It can be an instance or static field of a class, or it can be an argument passed to the method.</span></span> <span data-ttu-id="1c042-121">Se si tenta di restituire una variabile locale, viene generato l'errore del compilatore CS8168, "Non è possibile restituire la variabile locale 'obj' per riferimento perché non è una variabile locale ref".</span><span class="sxs-lookup"><span data-stu-id="1c042-121">Attempting to return a local variable generates compiler error CS8168, "Cannot return local 'obj' by reference because it is not a ref local."</span></span>

- <span data-ttu-id="1c042-122">Il valore restituito non può essere il valore letterale `null`.</span><span class="sxs-lookup"><span data-stu-id="1c042-122">The return value cannot be the literal `null`.</span></span> <span data-ttu-id="1c042-123">Se si tenta di restituire `null`, viene generato l'errore del compilatore CS8156, "Non è possibile usare un'espressione in questo contesto perché non può essere restituita per riferimento".</span><span class="sxs-lookup"><span data-stu-id="1c042-123">Attempting to return `null` generates compiler error CS8156, "An expression cannot be used in this context because it may not be returned by reference."</span></span>

   <span data-ttu-id="1c042-124">Un metodo con un valore restituito ref può restituire un alias a una variabile il cui valore è attualmente il valore null (senza istanza) o un [tipo nullable](../nullable-types/index.md) per un tipo valore.</span><span class="sxs-lookup"><span data-stu-id="1c042-124">A method with a ref return can return an alias to a variable whose value is currently the null (uninstantiated) value or a [nullable type](../nullable-types/index.md) for a value type.</span></span>
 
- <span data-ttu-id="1c042-125">Il valore restituito non può essere una costante, un membro di enumerazione, il valore restituito per valore da una proprietà o un metodo di un oggetto `class` o `struct`.</span><span class="sxs-lookup"><span data-stu-id="1c042-125">The return value cannot be a constant, an enumeration member, the by-value return value from a property, or a method of a `class` or `struct`.</span></span> <span data-ttu-id="1c042-126">Se si tenta di restituire questi valori, viene generato l'errore del compilatore CS8156, "Non è possibile usare un'espressione in questo contesto perché non può essere restituita per riferimento".</span><span class="sxs-lookup"><span data-stu-id="1c042-126">Attempting to return these generates compiler error CS8156, "An expression cannot be used in this context because it may not be returned by reference."</span></span>

<span data-ttu-id="1c042-127">Inoltre, poiché un metodo asincrono può eseguire la restituzione prima di aver terminato l'esecuzione mentre il relativo valore restituito non è ancora noto, i valori restituiti di riferimento non sono consentiti nei metodi asincroni.</span><span class="sxs-lookup"><span data-stu-id="1c042-127">In addition, because an asynchronous method may return before it has finished execution, while its return value is still unknown, reference return values are not allowed on async methods.</span></span>
 
## <a name="defining-a-ref-return-value"></a><span data-ttu-id="1c042-128">Definizione di un valore restituito ref</span><span class="sxs-lookup"><span data-stu-id="1c042-128">Defining a ref return value</span></span>

<span data-ttu-id="1c042-129">È possibile definire un valore restituito ref aggiungendo la parola chiave [ref](../../language-reference/keywords/ref.md) al tipo restituito della firma del metodo.</span><span class="sxs-lookup"><span data-stu-id="1c042-129">You define a ref return value by adding the [ref](../../language-reference/keywords/ref.md) keyword to the return type of the method signature.</span></span> <span data-ttu-id="1c042-130">Ad esempio, la firma seguente indica che la proprietà `GetContactInformation` restituisce al chiamante un riferimento a un oggetto `Person`:</span><span class="sxs-lookup"><span data-stu-id="1c042-130">For example, the following signature indicates that the `GetContactInformation` property returns a reference to a `Person` object to the caller:</span></span>

```csharp
public ref Person GetContactInformation(string fname, string lname);
```

<span data-ttu-id="1c042-131">Inoltre, il nome dell'oggetto restituito da ciascuna istruzione [return](../../language-reference/keywords/return.md) nel corpo del metodo deve essere preceduto dalla parola chiave [ref](../../language-reference/keywords/ref.md).</span><span class="sxs-lookup"><span data-stu-id="1c042-131">In addition, the name of the object returned by each [return](../../language-reference/keywords/return.md) statement in the method body must be preceded by the [ref](../../language-reference/keywords/ref.md) keyword.</span></span> <span data-ttu-id="1c042-132">Ad esempio, l'istruzione `return` seguente restituisce un riferimento a un oggetto `Person` denominato `p`:</span><span class="sxs-lookup"><span data-stu-id="1c042-132">For example, the following `return` statement returns a reference to a `Person` object named `p`:</span></span>

```csharp
return ref p;
```

<span data-ttu-id="1c042-134">Il valore restituito ref è un alias per un'altra variabile nell'ambito del metodo chiamato.</span><span class="sxs-lookup"><span data-stu-id="1c042-134">The ref return value is an alias to another variable in the called method's scope.</span></span> <span data-ttu-id="1c042-135">È possibile interpretare qualsiasi uso del valore restituito ref come uso della variabile di cui effettua l'aliasing:</span><span class="sxs-lookup"><span data-stu-id="1c042-135">You can interpret any use of the ref return as using the variable it aliases:</span></span>

- <span data-ttu-id="1c042-138">Se lo si restituisce *per riferimento*, si restituisce un alias alla stessa variabile.</span><span class="sxs-lookup"><span data-stu-id="1c042-138">If you return it *by reference* you are returning an alias to that same variable.</span></span>
- <span data-ttu-id="1c042-139">Se lo si passa a un altro metodo *per riferimento*, si passa un riferimento alla variabile di cui effettua l'aliasing.</span><span class="sxs-lookup"><span data-stu-id="1c042-139">If you pass it to another method *by reference* you are passing a reference to the variable it aliases.</span></span>
- <span data-ttu-id="1c042-140">Quando si crea un alias [locale ref](#ref-local), si crea un nuovo alias per la stessa variabile.</span><span class="sxs-lookup"><span data-stu-id="1c042-140">When you make a [ref local](#ref-local) alias, you make a new alias to the same variable.</span></span>


## <a name="ref-locals"></a><span data-ttu-id="1c042-141">Variabili locali ref</span><span class="sxs-lookup"><span data-stu-id="1c042-141">Ref locals</span></span>

<span data-ttu-id="1c042-142">Si supponga che il metodo `GetContactInformation` venga dichiarato come valore restituito ref:</span><span class="sxs-lookup"><span data-stu-id="1c042-142">Assume the `GetContactInformation` method is declared as a ref return:</span></span>

```csharp
public ref Person GetContactInformation(string fname, string lname)
```

<span data-ttu-id="1c042-143">Un'assegnazione per valore legge il valore di una variabile e la assegna a una nuova variabile:</span><span class="sxs-lookup"><span data-stu-id="1c042-143">A by-value assignment reads the value of a variable and assigns it to a new variable:</span></span>

Person p = contacts.GetContactInformation("Brandie", "Best");
```

<span data-ttu-id="1c042-144">L'assegnazione precedente dichiara `p` come variabile locale.</span><span class="sxs-lookup"><span data-stu-id="1c042-144">The preceding assignment declares `p` as a local variable.</span></span> <span data-ttu-id="1c042-145">Il valore iniziale viene copiato dalla lettura del valore restituito da `GetContactInformation`.</span><span class="sxs-lookup"><span data-stu-id="1c042-145">Its initial value is copied from reading the value returned by `GetContactInformation`.</span></span> <span data-ttu-id="1c042-146">Eventuali assegnazioni future a `p` non modificheranno il valore della variabile restituita da `GetContactInformation`.</span><span class="sxs-lookup"><span data-stu-id="1c042-146">Any future assignments to `p` will not change the value of the variable returned by `GetContactInformation`.</span></span> <span data-ttu-id="1c042-147">La variabile `p` non è più un alias per la variabile restituita.</span><span class="sxs-lookup"><span data-stu-id="1c042-147">The variable `p` is no longer an alias to the variable returned.</span></span>

<span data-ttu-id="1c042-148">Si dichiara una variabile *locale ref* per copiare l'alias nel valore originale.</span><span class="sxs-lookup"><span data-stu-id="1c042-148">You declare a *ref local* variable to copy the alias to the original value.</span></span> <span data-ttu-id="1c042-149">Nell'assegnazione seguente `p` è un alias per la variabile restituita da `GetContactInformation`.</span><span class="sxs-lookup"><span data-stu-id="1c042-149">In the following assignment, `p` is an alias to the variable returned from `GetContactInformation`.</span></span>

```

<span data-ttu-id="1c042-150">L'utilizzo successivo di `p` è uguale all'utilizzo della variabile restituita da `GetContactInformation` perché `p` è un alias per tale variabile.</span><span class="sxs-lookup"><span data-stu-id="1c042-150">Subsequent usage of `p` is the same as using the variable returned by `GetContactInformation` because `p` is an alias for that variable.</span></span> <span data-ttu-id="1c042-151">Le modifiche apportate a `p` modificano anche la variabile restituita da `GetContactInformation`.</span><span class="sxs-lookup"><span data-stu-id="1c042-151">Changes to `p` also change the variable returned from `GetContactInformation`.</span></span>

<span data-ttu-id="1c042-152">Si noti che la parola chiave `ref` viene usata sia prima della dichiarazione di variabile locale *che* prima della chiamata al metodo.</span><span class="sxs-lookup"><span data-stu-id="1c042-152">Note that the `ref` keyword is used both before the local variable declaration *and* before the method call.</span></span> 

<span data-ttu-id="1c042-153">È possibile accedere a un valore per riferimento nello stesso modo.</span><span class="sxs-lookup"><span data-stu-id="1c042-153">You can access a value by reference in the same way.</span></span> <span data-ttu-id="1c042-154">In alcuni casi, l'accesso a un valore per riferimento migliora le prestazioni evitando un'operazione di copia potenzialmente dispendiosa.</span><span class="sxs-lookup"><span data-stu-id="1c042-154">In some cases, accessing a value by reference increases performance by avoiding a potentially expensive copy operation.</span></span> <span data-ttu-id="1c042-155">L'istruzione seguente, ad esempio, spiega come sia possibile definire un valore locale di riferimento usato per fare riferimento a un valore.</span><span class="sxs-lookup"><span data-stu-id="1c042-155">For example, the following statement shows how one can define a ref local value that is used to reference a value.</span></span>

```csharp
ref VeryLargeStruct reflocal = ref veryLargeStruct;
```

<span data-ttu-id="1c042-156">Si noti che la parola chiave `ref` viene usata sia prima della dichiarazione di variabile locale *che* prima del valore nel secondo esempio.</span><span class="sxs-lookup"><span data-stu-id="1c042-156">Note that the `ref` keyword is used both before the local variable declaration *and* before the value in the second example.</span></span> <span data-ttu-id="1c042-157">Se non si includono entrambe le parole chiave `ref` nella dichiarazione di variabile e nell'assegnazione nei due esempi, viene generato l'errore del compilatore CS8172, "Non è possibile inizializzare una variabile per riferimento con un valore".</span><span class="sxs-lookup"><span data-stu-id="1c042-157">Failure to include both `ref` keywords in the variable declaration and assignment in both examples results in compiler error CS8172, "Cannot initialize a by-reference variable with a value."</span></span>
