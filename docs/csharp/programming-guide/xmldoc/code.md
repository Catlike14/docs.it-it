# <a name="ltcodegt-c-programming-guide"></a>&lt;code&gt; (Guida per programmatori C#)
## <a name="syntax"></a>Sintassi  
  
```xml  
<code>content</code>  
```  
  
#### <a name="parameters"></a>Parametri  
 `content`  
 Testo da contrassegnare come codice.
