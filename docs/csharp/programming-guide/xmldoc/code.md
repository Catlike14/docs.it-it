# &lt;code&gt; (Guida per programmatori C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

## Sintassi  
  
```  
<code>content</code>  
```  
  
#### Parametri  
 `content`  
 Testo che si desidera contrassegnare come codice.
