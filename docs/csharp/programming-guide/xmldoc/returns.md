# &lt;returns&gt; (Guida per programmatori C#)
## Sintassi  
  
```  
<returns>description</returns>  
```  
  
#### Parametri  
 `description`  
 Descrizione del valore restituito.  
  
## Note  
 Il tag \<returns\> viene utilizzato nel commento di una dichiarazione di metodo per descrivere il valore restituito.  
  
 Eseguire la compilazione con [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) per elaborare in un file i commenti per la creazione della documentazione.
