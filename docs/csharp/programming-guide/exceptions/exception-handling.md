## <a name="finally-blocks"></a><span data-ttu-id="518c0-124">Blocchi finally</span><span class="sxs-lookup"><span data-stu-id="518c0-124">Finally Blocks</span></span>  
 <span data-ttu-id="518c0-125">Un blocco `finally` consente la pulizia delle azioni eseguite in un blocco `try`.</span><span class="sxs-lookup"><span data-stu-id="518c0-125">A `finally` block enables you to clean up actions that are performed in a `try` block.</span></span> <span data-ttu-id="518c0-126">Se presente, il blocco `finally` viene eseguito per ultimo, dopo il blocco `try` e i blocchi `catch` corrispondenti.</span><span class="sxs-lookup"><span data-stu-id="518c0-126">If present, the `finally` block executes last, after the `try` block and any matched `catch` block.</span></span> <span data-ttu-id="518c0-127">Un blocco `finally` viene sempre eseguito, sia che venga o meno generata un'eccezione o che venga o meno individuato un blocco `catch` corrispondente al tipo di eccezione.</span><span class="sxs-lookup"><span data-stu-id="518c0-127">A `finally` block always runs, regardless of whether an exception is thrown or a `catch` block matching the exception type is found.</span></span>
