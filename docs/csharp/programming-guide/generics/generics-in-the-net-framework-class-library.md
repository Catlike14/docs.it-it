# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Generics nella libreria di classi .NET Framework (Guida per programmatori C#)
La versione 2.0 della libreria di classi .NET Framework offre il nuovo spazio dei nomi <xref:System.Collections.Generic>, che include diverse classi di raccolta generiche pronte per l'uso, insieme alle interfacce associate. Anche altri spazi dei nomi, come <xref:System>, offrono nuove interfacce generiche, ad esempio <xref:System.IComparable%601>. Tali classi e interfacce risultano più efficaci e indipendenti dai tipi rispetto alle classi di raccolte non generiche disponibili nelle versioni precedenti di .NET Framework. Prima di progettare e implementare classi di raccolte personalizzate, valutare la possibilità di usare o derivare una classe da una delle classi incluse nella libreria di classi .NET Framework.
