# Linee guida di progettazione per le eccezioni
Gestione delle eccezioni ha molti vantaggi rispetto a segnalazione errori basato sul valore restituito. Framework buona struttura consente lo sviluppatore di applicazioni di sfruttare i vantaggi delle eccezioni. In questa sezione vengono presentati i vantaggi delle eccezioni e vengono fornite linee guida per un uso efficace.
