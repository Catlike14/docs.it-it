# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernizzare le app .NET esistenti per le applicazioni ottimizzato su Cloud

> Visione: Modernizzare le applicazioni .NET Framework esistenti per le applicazioni ottimizzato su Cloud per migliorare notevolmente la flessibilità di distribuzione, pertanto è possibile distribuire più velocemente e ridurre i costi per il recapito dell'applicazione.

Per sfruttare i vantaggi del cloud e le nuove tecnologie come i contenitori, è consigliabile modernizzare almeno parzialmente le applicazioni .NET esistenti. In definitiva, la modernizzazione delle applicazioni aziendali consentirà una riduzione del costo totale di proprietà.

Parzialmente modernizzazione un'app non significa necessariamente una migrazione completa e rielaborazione. È possibile inizialmente modernizzare le applicazioni esistenti con importanti ma facile modernizzazione. È possibile mantenere il codice corrente, scritto in versioni di .NET Framework esistenti, con tutte le dipendenze di Windows e IIS. Figura 4-1 vengono evidenziate le app come ottimizzato su Cloud vengono posizionate nei modelli a livello di maturità modernizzazione applicazione Azure.
