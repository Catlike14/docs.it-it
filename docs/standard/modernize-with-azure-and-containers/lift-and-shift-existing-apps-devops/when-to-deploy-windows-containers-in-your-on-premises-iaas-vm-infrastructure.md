# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Quando per la distribuzione di contenitori di Windows in locale infrastruttura IaaS VM

-   L'organizzazione potrebbe non essere possibile passare al cloud o potrebbe semplicemente non essere in grado di spostare nel cloud per un motivo di business. Tuttavia, è comunque possibile ottenere i vantaggi dell'utilizzo di contenitori di Windows nel proprio Data Center.

-   Potrebbe essere altri elementi che vengono utilizzati in locale e che potrebbe rallentare il quando si tenta di spostare nel cloud. Dipendenze, ad esempio, sicurezza o di autenticazione con Active Directory locale Windows Server o qualsiasi altra risorsa locale.

-   Se si inizia a utilizzare subito i contenitori di Windows, è possibile apportare una migrazione graduale al cloud domani da una posizione migliore. Contenitori di Windows sta diventando un'unità di distribuzione per qualsiasi cloud con alcun blocco aggiuntivo.
