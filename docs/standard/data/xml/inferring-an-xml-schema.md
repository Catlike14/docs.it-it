# Inferenza di uno schema XML
Viene descritto come usare la classe <xref:System.Xml.Schema.XmlSchemaInference> SOM \(Schema Object Model\) per inferire uno schema XSD \(XML Schema Definition Language\) dalla struttura di un documento XML.  
  
 La classe <xref:System.Xml.Schema.XmlSchemaInference> SOM nello spazio dei nomi <xref:System.Xml.Schema?displayProperty=fullName> consente di inferire uno schema XSD \(XML Schema Definition Language\) dalla struttura di un documento XML.  La classe <xref:System.Xml.Schema.XmlSchemaInference> genera uno schema XML in grado di convalidare il documento XML.
