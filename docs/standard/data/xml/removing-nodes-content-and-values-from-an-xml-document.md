# <a name="removing-nodes-content-and-values-from-an-xml-document"></a>Rimozione di nodi, contenuto e valori da un documento XML
Una volta che un DOM XML si trova in memoria, è possibile rimuovere i nodi dall'albero oppure rimuovere il contenuto e i valori da alcuni tipi di nodi. Per informazioni su come rimuovere un nodo foglia o un intero sottoalbero di nodi da un documento, vedere [rimozione di nodi dal DOM](../../../../docs/standard/data/xml/removing-nodes-from-the-dom.md). Per informazioni su come rimuovere gli attributi di un elemento, vedere [rimozione di attributi da un nodo di elemento nel DOM](../../../../docs/standard/data/xml/removing-attributes-from-an-element-node-in-the-dom.md). Per informazioni sulla rimozione del contenuto di un nodo lasciando però il nodo dell'albero, vedere [la rimozione di contenuto di un nodo nel DOM](../../../../docs/standard/data/xml/removing-node-content-in-the-dom.md).
