# <a name="net-architecture-guidance"></a>Indicazioni sull'architettura di .NET

## <a name="containerized-docker-application-lifecycle-with-the-microsoft-platform-and-toolscontainerized-lifecycle-architectureindexmd"></a>[Ciclo di vita dell'applicazione nei contenitori Docker con la piattaforma Microsoft e strumenti](./containerized-lifecycle-architecture/index.md)

Questa guida viene fornita un'introduzione ai processi del ciclo di vita end-to-end consigliata che si userà per sviluppare, convalidare e distribuire applicazioni nei contenitori di Docker con Visual Studio e Microsoft Azure.

## <a name="modernize-existing-net-applications-with-azure-cloud-and-windows-containersmodernize-with-azure-and-containersindexmd"></a>[Modernizzare le applicazioni .NET esistenti con cloud di Azure e i contenitori di Windows](./modernize-with-azure-and-containers/index.md)

Questa guida viene fornita un'introduzione alle strategie di cui che è necessario eseguire la migrazione di applicazioni web esistenti per il cloud di Azure e i contenitori di Windows. Si apprenderà sulle strategie di codice, la migrazione dei dati, orchestrators ed elabora CI/CD-ROM.

## <a name="architect-modern-web-applications-with-aspnet-core-and-azuremodern-web-apps-azure-architectureindexmd"></a>[Architetto moderne applicazioni web con ASP.NET Core e Azure](modern-web-apps-azure-architecture/index.md)

Questa guida viene fornita un'introduzione all'architettura consigliata, progettazione e i processi di distribuzione che si userà per applicazioni ASP.NET e ASP.NET Core e ospitare le applicazioni in Azure.

## <a name="architecting-container-and-microservice-based-applicationsmicroservices-architectureindexmd"></a>[Architettura di applicazioni basate su contenitori e microservizi](microservices-architecture/index.md)

Questa guida offre un'introduzione allo sviluppo di applicazioni basate su microservizi e alla relativa gestione tramite i contenitori. Vengono descritti gli approcci alla progettazione e all'implementazione dell'architettura mediante i contenitori di .NET Core e Docker.
