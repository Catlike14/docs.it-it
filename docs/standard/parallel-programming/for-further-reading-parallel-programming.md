# For Further Reading (Parallel Programming)
Le risorse seguenti contengono informazioni aggiuntive sulla programmazione in parallelo in .NET Framework:  
  
-   Nel [Centro per sviluppatori dedicato all'elaborazione in parallelo](http://go.microsoft.com/fwlink/?LinkID=160570) del sito Web MSDN \(la pagina potrebbe essere in inglese\) sono disponibili collegamenti al contenuto e ai post sul forum più recenti relativi alla programmazione in parallelo in .NET Framework e in C\+\+ nativo.  
  
-   Nel [blog sulla programmazione in parallelo con .NET](http://go.microsoft.com/fwlink/?LinkID=169627) del sito dei blog di MSDN \(la pagina potrebbe essere in inglese\) sono disponibili molti articoli dettagliati sulla programmazione in parallelo in .NET Framework.  
  
-   Nel [blog sul Visualizzatore di concorrenze](http://go.microsoft.com/fwlink/?LinkID=169630) del sito dei blog di MSDN \(la pagina potrebbe essere in inglese\) viene descritto il nuovo strumento di profilatura delle prestazioni incluso in alcune edizioni di [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)].  
  
-   Nel [forum sulle estensioni parallele](http://go.microsoft.com/fwlink/?LinkID=169628) del sito dei forum di MSDN \(la pagina potrebbe essere in inglese\) è possibile chiedere e fornire informazioni sulla programmazione in parallelo.  
  
-   La pagina relativa ad [esempi di estensione parallela](http://go.microsoft.com/fwlink/?LinkID=165717) nel sito Web MSDN Code Gallery contiene molti esempi che illustrano tecniche di programmazione parallele intermedie e avanzate \(la pagina potrebbe essere in inglese\).
