# <a name="nowin32manifest-visual-basic"></a>/nowin32manifest (Visual Basic)
Indica al compilatore di non incorporare un manifesto dell'applicazione nel file eseguibile.  
  
## <a name="syntax"></a>Sintassi  
  
```  
/nowin32manifest  
```  
  
## <a name="remarks"></a>Note

Per ulteriori informazioni sulla virtualizzazione, vedere [distribuzione ClickOnce in Windows Vista](https://docs.microsoft.com/visualstudio/deployment/clickonce-deployment-on-windows-vista).
