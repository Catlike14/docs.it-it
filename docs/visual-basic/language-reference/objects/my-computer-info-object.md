# My.Computer.Info Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Fornisce le proprietà che consentono di ottenere informazioni sulla memoria, sugli assembly caricati, sul nome e sul sistema operativo del computer.

 Per informazioni sulle proprietà dell'oggetto di `My.Computer.Info` , vedere <xref:Microsoft.VisualBasic.Devices.ComputerInfo>.

 **Assembly:** Visual Basic Runtime Library \(in Microsoft.VisualBasic.dll\)
