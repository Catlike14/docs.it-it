# <a name="myapplicationlog-object"></a>Oggetto My.Application.Log
Fornisce una proprietà e i metodi per scrivere le informazioni relative a eventi ed eccezioni nei listener di log dell'applicazione.  
  
## <a name="remarks"></a>Note  
 Per informazioni sui metodi e proprietà del `My.Application.Log` dell'oggetto, vedere <xref:Microsoft.VisualBasic.Logging.Log>.</xref:Microsoft.VisualBasic.Logging.Log>  
  
 Per ulteriori informazioni, vedere [registrazione di informazioni dall'applicazione](../../../visual-basic/developing-apps/programming/log-info/logging-information-from-the-application.md).  
  
> [!NOTE]
>  È anche possibile usare le classi in .NET Framework per registrare informazioni dall'applicazione. Per ulteriori informazioni, vedere [traccia e strumentazione di applicazioni](http://msdn.microsoft.com/library/773b6fc4-9013-4322-b728-5dec7a72e743).
