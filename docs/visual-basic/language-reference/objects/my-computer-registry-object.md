# <a name="mycomputerregistry-object"></a>Oggetto My.Computer.Registry
Specifica proprietà e metodi per la modifica del Registro di sistema.  
  
## <a name="remarks"></a>Note  
 Per informazioni sui metodi e proprietà del `My.Computer.Registry` , vedere <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.  
  
 Per ulteriori informazioni, vedere [lettura e scrittura al Registro di sistema](../../../visual-basic/developing-apps/programming/computer-resources/reading-from-and-writing-to-the-registry.md).  
  
> [!NOTE]
>  È inoltre possibile modificare il Registro di sistema utilizzando i metodi di <xref:Microsoft.Win32.Registry?displayProperty=nameWithType> classe.
