# <a name="mycomputerregistry-object"></a>Oggetto My.Computer.Registry
Fornisce proprietà e metodi per la modifica del Registro di sistema.  
  
## <a name="remarks"></a>Note  
 Per informazioni sui metodi e proprietà del `My.Computer.Registry` dell'oggetto, vedere <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.</xref:Microsoft.VisualBasic.MyServices.RegistryProxy>  
  
 Per ulteriori informazioni, vedere [di lettura e scrittura al Registro di sistema](../../../visual-basic/developing-apps/programming/computer-resources/reading-from-and-writing-to-the-registry.md).  
  
> [!NOTE]
>  È inoltre possibile modificare il Registro di sistema utilizzando metodi della <xref:Microsoft.Win32.Registry?displayProperty=fullName>classe.</xref:Microsoft.Win32.Registry?displayProperty=fullName>
