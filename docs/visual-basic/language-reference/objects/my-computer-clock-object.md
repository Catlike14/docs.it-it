# My.Computer.Clock Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Fornisce le proprietà per accedere all'ora locale corrente e al tempo universale coordinato \(UTC\), equivalente all'ora di Greenwich, dall'orologio di sistema.  
  
## Note  
 Per informazioni sui metodi e le proprietà di `My.Computer.Clock` oggetto, vedere <xref:Microsoft.VisualBasic.Devices.Clock>.  
  
## Requisiti  
 **Spazio dei nomi:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Classe:** <xref:Microsoft.VisualBasic.Devices.Clock>  
  
 **Assembly:** Visual Basic Runtime Library \(in Microsoft.VisualBasic.dll\)
