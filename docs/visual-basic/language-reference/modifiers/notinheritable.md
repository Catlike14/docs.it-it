# NotInheritable (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Specifica che una classe non può essere utilizzata come classe base.  
  
## Note  
 **Termini alternativi**.  Una classe che non può essere ereditata viene talvolta definita *sealed*.  
  
 È possibile utilizzare il modificatore `NotInheritable` nel seguente contesto:
