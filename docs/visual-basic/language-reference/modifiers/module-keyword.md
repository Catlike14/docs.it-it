# <a name="module-ltkeywordgt-visual-basic"></a>Modulo &lt;(parola chiave)&gt; (Visual Basic)
Specifica che un attributo all'inizio di un file di origine si applica al modulo in assembly corrente.  
  
## <a name="remarks"></a>Note  
 Numero di attributi relativi a un singolo elemento di programmazione, ad esempio una classe o proprietà. Per applicare tale attributo, collegare il blocco di attributi tra parentesi angolari (`< >`), direttamente all'istruzione di dichiarazione.  
  
 Se un attributo relativo non solo per l'elemento seguente, ma al modulo in assembly corrente, inserire il blocco di attributi all'inizio del file di origine e di identificare l'attributo con il `Module` (parola chiave). Se si applica all'intero assembly, utilizzare il [Assembly](../../../visual-basic/language-reference/modifiers/assembly.md) (parola chiave).  
  
 Il `Module` modificatore non è lo stesso come il [istruzione Module](../../../visual-basic/language-reference/statements/module-statement.md).
