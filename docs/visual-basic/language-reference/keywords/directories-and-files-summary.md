# <a name="directories-and-files-summary-visual-basic"></a>Riepilogo di directory e file (Visual Basic)

 Utilizzando il `My` funzionalità garantisce una maggiore produttività e prestazioni nelle operazioni dei / o file rispetto all'utilizzo di questi membri. Per ulteriori informazioni, vedere <xref:Microsoft.VisualBasic.FileIO.FileSystem>.</xref:Microsoft.VisualBasic.FileIO.FileSystem>
