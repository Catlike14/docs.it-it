# <a name="property-not-found"></a>Proprietà non trovata
Questo oggetto non supporta la proprietà specificata.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare l'ortografia del nome della proprietà.  
  
2.  Controllare la documentazione dell'oggetto per assicurarsi che non si sta tentando di accedere a un risultato simile a una proprietà "text" quando l'oggetto supporta in realtà una "didascalia" o una proprietà denominata in modo analogo.
