# Can&#39;t open &#39;&lt;filename&gt;&#39; for writing
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Non è possibile aprire il file specificato per la scrittura, forse perché è stato già aperto.  
  
 **ID errore:** BC2012  
  
### Per correggere l'errore  
  
1.  Chiudere il file e riaprirlo.  
  
2.  Controllare le autorizzazioni del file.
