# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>Può &#39; aprire t &#39; &lt;filename&gt;&#39; per la scrittura
Impossibile aprire il file specificato per la scrittura, probabilmente perché è già stata aperta.  
  
 **ID errore:** BC2012  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Chiudere il file e riaprirlo.  
  
2.  Controllare le autorizzazioni del file.
