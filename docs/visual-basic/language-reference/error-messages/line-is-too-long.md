# <a name="line-is-too-long"></a>Riga troppo lunga
Le righe di testo di origine non possono superare 65535 caratteri.  
  
 **ID errore:** BC30494  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Ridurre la lunghezza della riga a 65535 caratteri o meno.
