# <a name="type-mismatch-visual-basic"></a>Tipo non corrispondente (Visual Basic)
Si è tentato di convertire un valore di un altro tipo in modo che non è valido.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare l'assegnazione per assicurarsi che sia valido.  
  
2.  Assicurarsi che non si ha superato un oggetto a una procedura che prevede un valore o una singola proprietà.  
  
3.  Assicurarsi che non è stato utilizzato un nome di modulo o un progetto in cui è prevista un'espressione.
