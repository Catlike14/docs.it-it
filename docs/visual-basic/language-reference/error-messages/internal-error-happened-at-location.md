# <a name="internal-error-happened-at-ltlocationgt"></a>Si è verificato un errore interno &lt;percorso&gt;
Si è verificato un errore interno. La riga in cui si è verificato è contenuta nel messaggio di errore.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che questo errore non è stato generato dal `Error` istruzione o `Raise` metodo; se non è stato, contattare il servizio supporto tecnico clienti Microsoft per segnalare le condizioni in cui è visualizzato il messaggio.
