## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Per aprire un file protetto da scrittura, modificare l'attributo di protezione da scrittura del file.  
  
2.  Assicurarsi che un altro processo non ha bloccato il file e attendere per aprire il file fino a quando non lo rilascia altro processo.  
  
3.  Per visualizzare il Registro di sistema, verificare che le autorizzazioni utente includano questo tipo di accesso del Registro di sistema.
