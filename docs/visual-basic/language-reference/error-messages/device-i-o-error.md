# <a name="device-io-error"></a>Errore di I/O del dispositivo
Si è verificato un errore di input o output mentre il programma stava utilizzando un dispositivo, ad esempio una stampante o un'unità disco.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Assicurarsi che il dispositivo funzioni correttamente e quindi ripetere l'operazione.
