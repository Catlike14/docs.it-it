# <a name="39module39-statements-can-occur-only-at-file-or-namespace-level"></a>&#39; Modulo &#39; le istruzioni possono trovarsi solo a livello di file o spazio dei nomi
`Module`le istruzioni devono essere visualizzate nella parte superiore del file di origine immediatamente dopo `Option` e `Imports` istruzioni, gli attributi globali e le dichiarazioni dello spazio dei nomi, ma prima di tutte le altre dichiarazioni.  
  
 **ID errore:** BC30617  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare il `Module` all'inizio del file di origine o di dichiarazione dello spazio dei nomi.
