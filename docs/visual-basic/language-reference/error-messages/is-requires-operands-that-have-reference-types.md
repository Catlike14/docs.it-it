# <a name="39is39-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-39lttypenamegt39"></a>&#39; è &#39; gli operandi devono che dispongono di tipi di riferimento, ma questo operando ha tipo di valore &#39; &lt;typename&gt;&#39;
Il `Is` operatore di confronto determina se due variabili oggetto fanno riferimento alla stessa istanza. Questo confronto non è definito per i tipi di valore.  
  
 **ID errore:** BC30020  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare l'operatore di confronto aritmetico appropriato o `Like` l'operatore per confrontare due tipi di valore.
