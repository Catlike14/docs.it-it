# <a name="property-or-method-not-found"></a>Proprietà o metodo non trovato
Il metodo di oggetto di riferimento o una proprietà dell'oggetto non è definito.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Può avere digitato correttamente il nome dell'oggetto. Per visualizzare le proprietà e metodi definiti per un oggetto, visualizzare il Visualizzatore oggetti. Selezionare la libreria di oggetto appropriato per visualizzare un elenco di metodi e le proprietà disponibili.
