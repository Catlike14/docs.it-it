# Default property &#39;&lt;propertyname1&gt;&#39; conflicts with default property &#39;&lt;propertyname2&gt;&#39; in &#39;&lt;classname&gt;&#39; and so should be declared &#39;Shadows&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Una proprietà è dichiarata con lo stesso nome di una proprietà definita nella classe base.  In questo caso la proprietà della classe deve nascondere la proprietà della classe base.  
  
 Si tratta di un messaggio di avvertimento.  Per impostazione predefinita viene utilizzato l'oggetto `Shadows`.  Per ulteriori informazioni su come nascondere gli avvisi o considerarli come errori, vedere [Configurazione degli avvisi in Visual Basic](/visual-studio/ide/configuring-warnings-in-visual-basic).  
  
 **ID errore:** BC40007  
  
### Per correggere l'errore  
  
-   Aggiungere la parola chiave `Shadows` alla dichiarazione o cambiare il nome della proprietà che si intende dichiarare.
