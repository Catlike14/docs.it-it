# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>&#39;&lt;nome1&gt; &#39; è ambiguo, importato dagli spazi dei nomi o tipi di &#39; &lt;name2&gt;&#39;
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il compilatore Visual Basic non dispone di regole di risoluzione dei conflitti. è necessario risolvere le ambiguità nei nomi manualmente.  
  
 **ID errore:** BC30561  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Tale ambiguità rimuovendo le importazioni dello spazio dei nomi.  
  
2.  Specificare un nome completo.
