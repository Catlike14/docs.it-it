# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>'&lt;nome1&gt;'è ambiguo, importato dagli spazi dei nomi o tipi'&lt;nome2&gt;'
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilatore non dispone di regole di risoluzione dei conflitti, è necessario evitare ambiguità nei nomi manualmente.  
  
 **ID errore:** BC30561  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Evitare ambiguità nel nome rimuovendo le importazioni dello spazio dei nomi.  
  
2.  Specificare un nome completo.
