# <a name="can39t-create-necessary-temporary-file"></a>Possono&#39;t creare file temporaneo necessario
L'unità è piena, che contiene la directory specificata dalla variabile di ambiente TEMP o variabile di ambiente TEMP specifica un'unità non valida o di sola lettura o la directory.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Eliminare file dal disco, se completo.  
  
2.  Specificare un'unità diversa nella variabile di ambiente TEMP.  
  
3.  Specificare un'unità valida per la variabile di ambiente TEMP.  
  
4.  Rimuovere la restrizione di sola lettura dall'unità attualmente specificati o directory.
