# <a name="39ltfunctionnamegt39-is-not-declared-smart-devicevisual-basic-compiler-error"></a>&#39; &lt;functionname&gt;&#39; non è dichiarato (Smart Device o Visual Basic errore del compilatore)
<`functionname`> non è dichiarato. La funzionalità di I/O dei file è in genere disponibile nello spazio dei nomi `Microsoft.VisualBasic` , ma non è supportata dalla versione di destinazione di .NET Compact Framework.  
  
 **ID errore:** BC30766  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Eseguire operazioni sui file con funzioni definite nel `System.IO` dello spazio dei nomi.
