# <a name="some-subkeys-cannot-be-deleted"></a>Impossibile eliminare alcune sottochiavi
È stato effettuato un tentativo di eliminare una chiave del Registro di sistema, ma l'operazione non riuscita perché non è possibile eliminare alcune sottochiavi. In genere si tratta di una mancanza di autorizzazioni.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Assicurarsi di che disporre di autorizzazioni sufficienti per eliminare le sottochiavi.
