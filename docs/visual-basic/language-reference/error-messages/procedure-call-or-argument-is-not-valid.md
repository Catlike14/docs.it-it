# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>Chiamata di routine o argomento non valido (Visual Basic)
Impossibile completare una parte della chiamata.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare gli intervalli consentiti per gli argomenti per verificare che nessuna impostazione superi i valori consentiti.
