# <a name="unable-to-find-required-file-39ltfilenamegt39"></a>Impossibile trovare il file richiesto '&lt;filename&gt;'

 **ID errore:** BC30655  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Reinstallare Visual Studio.
