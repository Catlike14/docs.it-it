# <a name="xml-namespace-uri-39httpwwww3orgxml1998namespace39-can-be-bound-only-to-39xmlns39"></a>Spazio dei nomi XML URI &#39; http://www.w3.org/XML/1998/namespace &#39; può essere associato solo a &#39; xmlns &#39;
All'URI http://www.w3.org/XML/1998/namespace viene utilizzato in una dichiarazione dello spazio dei nomi XML. Questo URI è uno spazio dei nomi riservato e non può essere inclusa in una dichiarazione dello spazio dei nomi XML.  
  
 **ID errore:** BC31183  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la dichiarazione dello spazio dei nomi XML o sostituire all'URI http://www.w3.org/XML/1998/namespace con un URI dello spazio dei nomi valido.
