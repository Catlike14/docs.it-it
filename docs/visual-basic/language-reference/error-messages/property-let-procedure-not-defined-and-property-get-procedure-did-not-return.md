# <a name="property-let-procedure-not-defined-and-property-get-procedure-did-not-return-an-object"></a>La routine Property Let non è stata definita e la routine Property Get non ha restituito un oggetto
Determinate proprietà, metodi e le operazioni sono applicabili solo a `Collection` oggetti. Specificata un'operazione o proprietà esclusiva per le raccolte, ma l'oggetto non è una raccolta.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare l'ortografia del nome oggetto o una proprietà o verificare che l'oggetto è un `Collection` oggetto.  
  
2.  Esaminare il `Add` metodo utilizzato per aggiungere l'oggetto alla raccolta per assicurarsi che la sintassi sia corretta e che gli identificatori sono stati digitati correttamente.
