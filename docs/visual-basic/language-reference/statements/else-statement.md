# <a name="else-statement-visual-basic"></a>Istruzione Else (Visual Basic)
Introduce un gruppo di istruzioni da eseguire o compilare se nessun altro gruppo di istruzioni condizionali è stato eseguito o compilato.  
  
## <a name="remarks"></a>Note  
 È possibile usare la parola chiave `Else` nei contesti seguenti:

 [Istruzione Select...Case](../../../visual-basic/language-reference/statements/select-case-statement.md)
