 <span data-ttu-id="906bd-105">I nomi XML sono spesso causa di complessità nella programmazione XML.</span><span class="sxs-lookup"><span data-stu-id="906bd-105">XML names are often a source of complexity in XML programming.</span></span> <span data-ttu-id="906bd-106">Un nome XML è composto da uno spazio dei nomi XML (detto anche URI dello spazio dei nomi XML) e da un nome locale.</span><span class="sxs-lookup"><span data-stu-id="906bd-106">An XML name consists of an XML namespace (also called an XML namespace URI) and a local name.</span></span> <span data-ttu-id="906bd-107">Uno spazio dei nomi XML è simile a uno spazio di nomi in un'applicazione basata su [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)].</span><span class="sxs-lookup"><span data-stu-id="906bd-107">An XML namespace is similar to a namespace in a [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)]-based program.</span></span> <span data-ttu-id="906bd-108">Consente infatti di qualificare in modo univoco i nomi di elementi e attributi</span><span class="sxs-lookup"><span data-stu-id="906bd-108">It enables you to uniquely qualify the names of elements and attributes.</span></span> <span data-ttu-id="906bd-109">e quindi di evitare conflitti di nomi tra le diverse parti di un documento XML.</span><span class="sxs-lookup"><span data-stu-id="906bd-109">This helps avoid name conflicts between various parts of an XML document.</span></span> <span data-ttu-id="906bd-110">Dopo aver dichiarato uno spazio dei nomi XML, è possibile selezionare un nome locale che deve essere univoco all'interno di tale spazio dei nomi.</span><span class="sxs-lookup"><span data-stu-id="906bd-110">When you have declared an XML namespace, you can select a local name that only has to be unique within that namespace.</span></span>  
  
 <span data-ttu-id="906bd-111">Un'altra peculiarità dei nomi XML è costituita dai *prefissi di spazio dei nomi* XML.</span><span class="sxs-lookup"><span data-stu-id="906bd-111">Another aspect of XML names is XML *namespace prefixes*.</span></span> <span data-ttu-id="906bd-112">Sono i prefissi XML a costituire la causa di maggior complessità dei nomi XML.</span><span class="sxs-lookup"><span data-stu-id="906bd-112">XML prefixes cause most of the complexity of XML names.</span></span> <span data-ttu-id="906bd-113">I prefissi consentono di creare un collegamento per uno spazio dei nomi XML al fine di rendere il documento XML più conciso e comprensibile.</span><span class="sxs-lookup"><span data-stu-id="906bd-113">These prefixes enable you to create a shortcut for an XML namespace, which makes the XML document more concise and understandable.</span></span> <span data-ttu-id="906bd-114">Tuttavia, il significato dei prefissi XML dipende dal contesto ed è proprio questo aspetto a renderli complessi.</span><span class="sxs-lookup"><span data-stu-id="906bd-114">However, XML prefixes depend on their context to have meaning, which adds complexity.</span></span> <span data-ttu-id="906bd-115">Ad esempio, è possibile associare il prefisso XML `aw` a un unico spazio dei nomi XML in un'unica parte di un albero XML e a uno spazio dei nomi XML diverso in una parte diversa dell'albero XML.</span><span class="sxs-lookup"><span data-stu-id="906bd-115">For example, the XML prefix `aw` could be associated with one XML namespace in one part of an XML tree, and with a different XML namespace in a different part of the XML tree.</span></span>
