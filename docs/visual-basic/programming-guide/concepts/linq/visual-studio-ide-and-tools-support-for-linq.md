# <a name="visual-studio-ide-and-tools-support-for-linq-visual-basic"></a>IDE di Visual Studio e strumenti di supporto per LINQ (Visual Basic)
Ambiente di sviluppo integrato (IDE) di Visual Studio fornisce le seguenti funzionalità che supportano lo sviluppo di applicazioni LINQ:  
  
## <a name="object-relational-designer"></a>Object Relational Designer  
 Progettazione relazionale oggetti è uno strumento di progettazione visiva che è possibile utilizzare in [LINQ to SQL](https://msdn.microsoft.com/library/bb386976) applicazioni per generare le classi in Visual Basic che rappresentano i dati relazionali in un database sottostante. Per ulteriori informazioni, vedere [LINQ to SQL Tools in Visual Studio](https://docs.microsoft.com/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2).  
  
## <a name="sqlmetal-command-line-tool"></a>Strumento da riga di comando SQLMetal  
 SQLMetal è uno strumento da riga di comando che può essere utilizzato nei processi di compilazione per generare classi da un database esistente per l'utilizzo in LINQ alle applicazioni di SQL. Per ulteriori informazioni, vedere [SqlMetal.exe (strumento di generazione di codice)](http://msdn.microsoft.com/library/819e5a96-7646-4fdb-b14b-fe31221b0614).  
  
## <a name="linq-aware-code-editor"></a>Editor di codice compatibile con LINQ  
 L'editor di codice Visual Basic supporta LINQ ampiamente con funzionalità di formattazione e IntelliSense.  
  
## <a name="visual-studio-debugger-support"></a>Supporto del Debugger di Visual Studio  
 Il debugger di Visual Studio supporta il debug delle espressioni di query. Per ulteriori informazioni, vedere [debug LINQ](https://docs.microsoft.com/visualstudio/debugger/debugging-linq).
