# <a name="generation-operations-visual-basic"></a>Operazioni di generazione (Visual Basic)
La generazione si riferisce alla creazione di una nuova sequenza di valori.  
  
 Nella sezione seguente sono elencati i metodi dell'operatore query standard che eseguono la generazione.  
  
## <a name="methods"></a>Metodi  
  
|Nome metodo|Descrizione|Sintassi delle espressioni di Query Visual Basic|Altre informazioni|  
|-----------------|-----------------|------------------------------------------|----------------------|  
|DefaultIfEmpty|Sostituisce una raccolta vuota con una raccolta singleton per impostazione predefinita.|Non applicabile.|<xref:System.Linq.Enumerable.DefaultIfEmpty%2A?displayProperty=fullName></xref:System.Linq.Enumerable.DefaultIfEmpty%2A?displayProperty=fullName><br /><br /> <xref:System.Linq.Queryable.DefaultIfEmpty%2A?displayProperty=fullName></xref:System.Linq.Queryable.DefaultIfEmpty%2A?displayProperty=fullName>|  
|Empty|Restituisce una raccolta vuota.|Non applicabile.|<xref:System.Linq.Enumerable.Empty%2A?displayProperty=fullName></xref:System.Linq.Enumerable.Empty%2A?displayProperty=fullName>|  
|Intervallo|Genera una raccolta che contiene una sequenza di numeri.|Non applicabile.|<xref:System.Linq.Enumerable.Range%2A?displayProperty=fullName></xref:System.Linq.Enumerable.Range%2A?displayProperty=fullName>|  
|Repeat|Genera una raccolta che contiene un valore ripetuto.|Non applicabile.|<xref:System.Linq.Enumerable.Repeat%2A?displayProperty=fullName></xref:System.Linq.Enumerable.Repeat%2A?displayProperty=fullName>|
