# <a name="linq-and-reflection-visual-basic"></a>LINQ e Reflection (Visual Basic)
L'API di reflection della libreria di classi .NET Framework consente di esaminare i metadati in un assembly .NET e creare raccolte di tipi, membri dei tipi, parametri e così via che si trovano in tale assembly. Poiché queste raccolte supportano generica `IEnumerable` interfaccia, è possibile eseguire query utilizzando LINQ.
