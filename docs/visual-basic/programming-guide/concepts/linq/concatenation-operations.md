# <a name="concatenation-operations-visual-basic"></a>Operazioni di concatenazione (Visual Basic)
Concatenazione si riferisce all'operazione di aggiunta di una sequenza a un altro.  
  
 Nella figura seguente viene illustrata un'operazione di concatenazione di due sequenze di caratteri.  
  
 ![Grafico che mostra la concatenazione di due sequenze. ](../../../../csharp/programming-guide/concepts/linq/media/concat.png "Concat")  
  
 Nella sezione seguente sono elencati i metodi dell'operatore query standard che eseguono la concatenazione.
