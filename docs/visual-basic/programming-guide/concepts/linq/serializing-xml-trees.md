# <a name="serializing-xml-trees-visual-basic"></a>La serializzazione di strutture ad albero XML (Visual Basic)
Per serializzazione di un albero XML si intende la generazione di codice XML dall'albero XML. È possibile serializzare in un file, a un'implementazione concreta della <xref:System.IO.TextWriter>classe, o a un'implementazione concreta di un <xref:System.Xml.XmlWriter>.</xref:System.Xml.XmlWriter> </xref:System.IO.TextWriter>  
  
 È possibile controllare vari aspetti della serializzazione, ad esempio se impostare i rientri per il codice XML serializzato e se scrivere una dichiarazione XML.
