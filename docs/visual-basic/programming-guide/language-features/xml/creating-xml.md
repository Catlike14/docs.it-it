# <a name="creating-xml-in-visual-basic"></a>Creazione di XML in Visual Basic
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]Consente di utilizzare *valori letterali XML* direttamente nel codice. Sintassi del valore letterale XML rappresenta [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] oggetti ed è simile alla sintassi XML 1.0. Questo rende più facile creare elementi, documenti e frammenti XML a livello di programmazione perché il codice ha la stessa struttura del XML finale.
