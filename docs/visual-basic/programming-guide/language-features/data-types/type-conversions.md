# <a name="type-conversions-in-visual-basic"></a>Conversioni di tipi in Visual Basic
Il processo di modifica di un valore da un tipo di dati a un altro tipo è denominato *conversione*. Le conversioni sono *ampliamento* o *narrowing*, a seconda della capacità di dati dei tipi coinvolti. Sono inoltre *implicita* o *esplicita*, a seconda della sintassi nel codice sorgente.
