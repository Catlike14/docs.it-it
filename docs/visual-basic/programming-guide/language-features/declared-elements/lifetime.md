 <span data-ttu-id="519da-109">Oggetto *variabile membro* (dichiarato a livello di modulo, all'esterno di qualsiasi routine) è in genere uguale alla durata dell'elemento in cui è dichiarata.</span><span class="sxs-lookup"><span data-stu-id="519da-109">A *member variable* (declared at module level, outside any procedure) typically has the same lifetime as the element in which it is declared.</span></span> <span data-ttu-id="519da-110">Una variabile non condivisa dichiarata in una classe o struttura esiste come una copia separata per ogni istanza della classe o struttura in cui viene dichiarato.</span><span class="sxs-lookup"><span data-stu-id="519da-110">A nonshared variable declared in a class or structure exists as a separate copy for each instance of the class or structure in which it is declared.</span></span> <span data-ttu-id="519da-111">Ognuna di queste variabili è uguale alla durata della relativa istanza.</span><span class="sxs-lookup"><span data-stu-id="519da-111">Each such variable has the same lifetime as its instance.</span></span> <span data-ttu-id="519da-112">Tuttavia, un `Shared` variabile ha un'unica durata, rimane attiva per il tempo di esecuzione dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="519da-112">However, a `Shared` variable has only a single lifetime, which lasts for the entire time your application is running.</span></span>
