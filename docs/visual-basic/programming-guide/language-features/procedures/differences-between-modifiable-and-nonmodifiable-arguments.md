## <a name="modifiable-and-nonmodifiable-arguments"></a><span data-ttu-id="65031-119">Argomenti modificabili e</span><span class="sxs-lookup"><span data-stu-id="65031-119">Modifiable and Nonmodifiable Arguments</span></span>  
 <span data-ttu-id="65031-120">Oggetto *argomento modificabile* è un attributo con un elemento sottostante modificabile.</span><span class="sxs-lookup"><span data-stu-id="65031-120">A *modifiable argument* is one with a modifiable underlying element.</span></span> <span data-ttu-id="65031-121">Il codice chiamante può archiviare un nuovo valore in qualsiasi momento, e se si passa l'argomento [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), il codice nella procedura possa inoltre modificare l'elemento sottostante nel codice chiamante.</span><span class="sxs-lookup"><span data-stu-id="65031-121">The calling code can store a new value at any time, and if you pass the argument [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), the code in the procedure can also modify the underlying element in the calling code.</span></span>  
  
 <span data-ttu-id="65031-122">Oggetto *argomento non modificabile* non contiene un elemento sottostante non modificabile o è passato [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span><span class="sxs-lookup"><span data-stu-id="65031-122">A *nonmodifiable argument* either has a nonmodifiable underlying element or is passed [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span></span> <span data-ttu-id="65031-123">La procedura non è possibile modificare l'elemento sottostante nel codice chiamante, anche se si tratta di un elemento modificabile.</span><span class="sxs-lookup"><span data-stu-id="65031-123">The procedure cannot modify the underlying element in the calling code, even if it is a modifiable element.</span></span> <span data-ttu-id="65031-124">Se si tratta di un elemento non è modificabile, il codice non è possibile modificarlo.</span><span class="sxs-lookup"><span data-stu-id="65031-124">If it is a nonmodifiable element, the calling code itself cannot modify it.</span></span>  
  
 <span data-ttu-id="65031-125">La routine chiamata potrebbe modificare la copia locale di un argomento non è modificabile, ma tale modifica non ha effetto sull'elemento sottostante nel codice chiamante.</span><span class="sxs-lookup"><span data-stu-id="65031-125">The called procedure might modify its local copy of a nonmodifiable argument, but that modification does not affect the underlying element in the calling code.</span></span>
