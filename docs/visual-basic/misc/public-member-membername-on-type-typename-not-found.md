# <a name="public-member-39ltmembernamegt39-on-type-39lttypenamegt39-not-found"></a>Membro pubblico '&lt;membername&gt;'nel tipo'&lt;typename&gt;' non trovato
Il membro a cui si sta tentando di accedere non è stato trovato. Questo errore può verificarsi quando si accede a membri non pubblici di oggetti ad associazione tardiva.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Se il membro esiste, verificare che sia dichiarato come `Public`.
