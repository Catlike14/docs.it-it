# <a name="39sub39-or-39function39-expected-after-39delegate39"></a>&#39;Sub&#39; o &#39;funzione&#39; previsto dopo &#39;delegato&#39;
Un'istruzione `Delegate` non specifica una routine `Sub` o `Function` . La parola chiave `Sub` o `Function` deve seguire immediatamente la parola chiave `Delegate` .  
  
 **ID errore:** BC30278  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Aggiungere la parola chiave `Sub` o `Function` immediatamente dopo `Delegate`.  
  
2.  Specificare un nome di routine, un elenco di argomenti e un tipo restituito in base alle necessit√†.
