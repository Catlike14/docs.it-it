# <a name="xml-namespace-prefix-39prefix39-is-already-declared"></a>Prefisso dello spazio dei nomi XML &#39;prefisso&#39; è già dichiarato
Un prefisso dello spazio dei nomi XML definito da un'istruzione `Imports` corrisponde a un prefisso dello spazio dei nomi XML definito in precedenza.  
  
 **ID errore:** BC30573  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare un nome univoco per il prefisso dello spazio dei nomi XML.
