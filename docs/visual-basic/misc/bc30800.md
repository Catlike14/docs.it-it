# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Gli argomenti del metodo devono essere racchiusi tra parentesi
Le regole relative alle chiamate di stored procedure sono semplici nelle versioni pi√π recenti di Visual Basic. Tutti gli argomenti devono essere racchiusi tra parentesi.  
  
 **ID errore:** BC30800  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Racchiudere l'elenco di argomenti tra parentesi, ad esempio:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
