# &#39;ByVal&#39; e &#39;ByRef&#39; non possono essere combinati
Le parole chiave `ByVal` e `ByRef` si escludono reciprocamente.  
  
 **ID errore:** BC30641  
  
### Per correggere l'errore  
  
-   Specificare `ByVal` o `ByRef`, ma non entrambi.
