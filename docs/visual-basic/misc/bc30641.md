# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39; ByVal &#39; e &#39; ByRef &#39; non possono essere combinati
Le parole chiave `ByVal` e `ByRef` si escludono reciprocamente.  
  
 **ID errore:** BC30641  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare `ByVal` o `ByRef`, ma non entrambi.
