# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39;&lt;NomeVariabile&gt; &#39; non è una variabile locale o un parametro e pertanto non può essere utilizzata come un &#39;genera&#39; variabile
Le variabili nelle istruzioni `Try...Catch...Finally` devono essere dichiarate localmente oppure devono essere parametri per la routine corrente.  
  
 **ID errore:** BC31082  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Dichiarare le variabili locali o i parametri per le istruzioni `Try...Catch...Finally` .
