# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>Espressioni 'AddressOf' non sono valide nelle finestre di debug
`AddressOf`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30731  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere le istruzioni `AddressOf` dal codice di debug.
