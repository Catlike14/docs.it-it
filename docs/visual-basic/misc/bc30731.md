# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39; AddressOf &#39; espressioni non sono valide nelle finestre di debug
Le istruzioni`AddressOf` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30731  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere le istruzioni `AddressOf` dal codice di debug.
