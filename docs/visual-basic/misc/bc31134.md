# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>&#39;AddHandler&#39; e &#39;RemoveHandler&#39; non possono essere dichiarati come parametri del metodo &#39;ByRef&#39;
I parametri dei metodi `AddHandler` e `RemoveHandler` non possono essere dichiarati con il modificatore `ByRef` .  
  
 **ID errore:** BC31134  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la parola chiave `ByRef` dal parametro.
