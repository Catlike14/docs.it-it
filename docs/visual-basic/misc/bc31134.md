# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>Parametri del metodo 'AddHandler' e 'RemoveHandler' non possono essere dichiarati 'ByRef'
I parametri dei metodi `AddHandler` e `RemoveHandler` non possono essere dichiarati con il modificatore `ByRef` .  
  
 **ID errore:** BC31134  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la parola chiave `ByRef` dal parametro.
