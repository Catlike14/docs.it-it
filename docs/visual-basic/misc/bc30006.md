# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Necessario riferimento al modulo '&lt;modulename&gt;'contenente la definizione dell'evento'&lt;eventname&gt;'
Necessario riferimento al modulo '`modulename`>' contenente la definizione dell'evento '`eventname`>'. Aggiungerne uno al progetto.  
  
 L'evento è definito in un modulo a cui non si fa riferimento direttamente nel progetto. Il [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilatore richiede un riferimento per evitare ambiguità nel caso in cui l'evento è definito in più di un modulo.  
  
 **ID errore:** BC30006  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Includere il nome del modulo senza riferimento nei riferimenti del progetto.
