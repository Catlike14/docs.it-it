# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Necessario riferimento al modulo &#39; &lt;modulename&gt; &#39; che contiene la definizione dell'evento &#39; &lt;eventname&gt;&#39;
Necessario riferimento al modulo ' <`modulename`>' contenente la definizione dell'evento ' <`eventname`>'. Aggiungerne uno al progetto.  
  
 L'evento è definito in un modulo a cui non si fa riferimento direttamente nel progetto. Il compilatore Visual Basic richiede un riferimento per evitare ambiguità nel caso in cui l'evento è definito in più di un modulo.  
  
 **ID errore:** BC30006  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Includere il nome del modulo senza riferimento nei riferimenti del progetto.
