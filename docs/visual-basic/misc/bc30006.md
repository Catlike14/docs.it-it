# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Necessario un riferimento al modulo &#39; &lt;modulename&gt;&#39; contenente la definizione dell'evento &#39;&lt; EventName&gt;&#39;
Necessario riferimento al modulo ' <`modulename`>' contenente la definizione dell'evento ' <`eventname`>'. Aggiungerne uno al progetto.  
  
 L'evento è definito in un modulo a cui non si fa riferimento direttamente nel progetto. Il compilatore [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] richiede un riferimento per evitare ambiguità nel caso in cui l'evento venga definito in più moduli.  
  
 **ID errore:** BC30006  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Includere il nome del modulo senza riferimento nei riferimenti del progetto.
