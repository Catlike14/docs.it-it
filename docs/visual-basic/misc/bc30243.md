# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;modificatore&gt; &#39; non è valido in una dichiarazione di evento
Un'istruzione `Event` contiene una parola chiave non valida, ad esempio `ReadOnly`.  
  
 **ID errore:** BC30243  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere la parola chiave non valida dall'istruzione `Event` .
