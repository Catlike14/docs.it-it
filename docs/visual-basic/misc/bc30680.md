# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39;Area #End&#39; deve essere preceduto da un oggetto corrispondente &#39;#Region&#39;
Con `#Region` è possibile specificare un blocco di codice da espandere o comprimere durante l'uso della funzionalità di struttura dell'Editor di codice di Visual Studio. L'inizio e la fine delle istruzioni `#Region` devono essere nello stesso blocco di codice.  
  
 **ID errore:** BC30680  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Inserire `#Region` nella posizione appropriata prima della corrispondente `#End Region` istruzione.
