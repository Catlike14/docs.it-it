# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'Using' non sono valide nella finestra controllo immediato
`Using`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30186  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione `Using` dalla finestra di controllo **immediato** .
