# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39; MyBase &#39; non può essere utilizzato con metodo &#39; &lt;NomeMetodo&gt;&#39; perché è dichiarato &#39; MustOverride &#39;
Si è provato a usare `MyBase` con un metodo dichiarato come `MustOverride`.  
  
 **ID errore:** BC30399  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la dichiarazione `MustOverride` .
