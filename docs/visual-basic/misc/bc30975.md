# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;Per&#39; variabile di controllo ciclo già in uso da un tipo di inclusione &#39;per&#39; ciclo
Tutti i cicli annidati `For` devono usare variabili di controllo univoche.  
  
 **ID errore:** BC30975  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare uno dei nomi della variabile di controllo.
