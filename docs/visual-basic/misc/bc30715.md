# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Catch &#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Catch` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30715  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Catch` dal codice di debug.
