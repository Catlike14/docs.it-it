# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>Metodo '&lt;methodname&gt;'non può gestire l'evento'&lt;eventname&gt;' perché non hanno la stessa firma
I parametri o tipi di parametro per il gestore eventi specificato non corrispondono a quelli definiti per l'evento specificato.  
  
 **ID errore:** BC31029  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Ridefinire l'evento o il gestore eventi in modo che gli elenchi di parametri corrispondano.
