# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Prefisso dello spazio dei nomi XML &#39;prefisso&#39; è riservato per l'utilizzo da parte di XML e lo spazio dei nomi URI non può essere modificato
Un prefisso dello spazio dei nomi XML riservato, ad esempio `xmlns`, è stato usato in un valore letterale XML.  
  
 **ID errore:** BC31152  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il prefisso dello spazio dei nomi XML in un valore non riservato.
