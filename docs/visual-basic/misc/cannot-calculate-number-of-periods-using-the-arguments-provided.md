# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>Impossibile calcolare il numero di periodi utilizzando gli argomenti forniti
Una chiamata al `NPer` funzione non contiene tutti gli argomenti obbligatori.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che i valori `Rate`, `Prnt` e `PV` siano inclusi nella chiamata di funzione.
