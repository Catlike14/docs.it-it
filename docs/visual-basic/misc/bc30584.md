# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39;&lt;tipo di&gt; &#39; non può essere ereditato più di una volta
Un'istruzione `Inherits` è stata duplicata.  
  
 **ID errore:** BC30584  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione `Inherits` ridondante.
