# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39; &lt;NomeArgomento&gt;&#39; non è un parametro di &#39;&lt; MethodDeclaration&gt;&#39;
Una chiamata di routine fornisce un argomento aggiuntivo non definito dalla routine.  
  
 **ID errore:** BC30272  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'argomento superfluo dall'elenco di argomenti.
