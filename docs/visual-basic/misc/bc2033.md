# Si &#232; verificato un errore durante l&#39;incorporamento del manifesto Win32: l&#39;opzione /win32manifest &#232; in conflitto con /nowin32manifest
Al compilatore Visual Basic è stata passata un'opzione `/win32manifest` del compilatore e un'opzione `/nowin32manifest` del compilatore. Solo una di queste opzioni può essere passata al compilatore di Visual Basic.  
  
 **ID errore:** BC2033  
  
### Per correggere l'errore  
  
1.  Rimuovere l'opzione `/win32manifest` del compilatore o l'opzione `/nowin32manifest` del compilatore.
