# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="fc071-102">Parametro di tipo con un &#39;struttura&#39; vincolo non può essere usato come vincolo</span><span class="sxs-lookup"><span data-stu-id="fc071-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="fc071-103">Un parametro di tipo con un vincolo `Structure` viene usato come vincolo per un altro parametro di tipo.</span><span class="sxs-lookup"><span data-stu-id="fc071-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="fc071-104">Il vincolo `Structure` richiede che l'argomento di tipo passato al parametro di tipo sia un tipo valore.</span><span class="sxs-lookup"><span data-stu-id="fc071-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="fc071-105">Tuttavia, un tipo valore non può essere implementato o ereditato, quindi non è significativo usarlo come vincolo, che richiederebbe all'altro parametro di tipo di implementarlo o ereditarlo da esso.</span><span class="sxs-lookup"><span data-stu-id="fc071-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="fc071-106">L'unica interpretazione significativa di questa situazione è che entrambi gli argomenti di tipo devono essere esattamente dello stesso tipo.</span><span class="sxs-lookup"><span data-stu-id="fc071-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="fc071-107">In tal caso, il tipo generico richiede un solo parametro di tipo.</span><span class="sxs-lookup"><span data-stu-id="fc071-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="fc071-108">L'istruzione seguente può generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="fc071-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="fc071-109">Il tipo passato a `t2` non può implementare o ereditare il tipo passato a `t1`, perché il tipo passato a `t1` deve essere un tipo valore.</span><span class="sxs-lookup"><span data-stu-id="fc071-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="fc071-110">**ID errore:** BC32114</span><span class="sxs-lookup"><span data-stu-id="fc071-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fc071-111">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="fc071-111">To correct this error</span></span>  
  
-   <span data-ttu-id="fc071-112">Rimuovere il parametro di tipo vincolato a `Structure` dall'elenco di vincoli sull'altro parametro di tipo.</span><span class="sxs-lookup"><span data-stu-id="fc071-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="fc071-113">Se entrambi i parametri di tipo richiedono lo stesso tipo valore, definire il tipo generico con un solo parametro di tipo.</span><span class="sxs-lookup"><span data-stu-id="fc071-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
