# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>&#39; &lt;nomemembro1&gt;&#39; e &#39;&lt; nomemembro2&gt;&#39; non è in rapporto di overload perché si differenziano solo per i tipi di parametri facoltativi
I membri di overload devono presentare tipi di dati diversi per uno o più parametri obbligatori.  
  
 **ID errore:** BC30696  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Definire almeno un parametro obbligatorio come tipo di dati diverso.
