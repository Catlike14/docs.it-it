# <a name="39end39-statement-not-valid"></a>&#39;Fine&#39; istruzione non valido
La parola chiave `End` è usata in modo errato oppure non è appropriata nel contesto corrente.  
  
 **ID errore:** BC30678  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Usare `End` da sola per fare in modo che una routine perda l'ambito.  
  
2.  Usare `End` con altre parole chiave per terminare un blocco di codice, ad esempio una struttura o una proprietà.
