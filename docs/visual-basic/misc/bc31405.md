# <a name="optional-parameters-cannot-have-structure-types"></a>I parametri facoltativi non possono avere tipi di struttura
I parametri facoltativi non possono essere tipi di struttura.  
  
 **ID errore:** BC31405  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Dichiarare i parametri del tipo di struttura in base alle esigenze.  
  
2.  Dichiarare un membro di classe statico al posto di una struttura.
