# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'On Error' non valide nella finestra controllo immediato
`On Error`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30720  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `On Error` dal codice di debug.
