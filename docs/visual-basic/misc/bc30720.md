# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; errore &#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`On Error` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30720  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `On Error` dal codice di debug.
