# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>Progetto '&lt;NomeProgetto&gt;'contiene un riferimento indiretto all'assembly'&lt;assemblyname&gt;'che contiene'&lt;classname&gt;'
Progetto '\<NomeProgetto >' contiene un riferimento indiretto all'assembly '\<assemblyname >' che contiene '\<classname >'. Aggiungere un riferimento a '\<assemblyname >' al progetto.  
  
 Il progetto contiene membri che dipendono da un assembly non incluso nei riferimenti del progetto.  
  
 **ID errore:** BC31515  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere un riferimento all'assembly facendo clic con il pulsante destro del mouse su **Riferimenti** in **Esplora soluzioni** e scegliendo **Aggiungi riferimento**.
