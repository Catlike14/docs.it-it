# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>Metodo non può contenere sia un &#39;Try&#39; istruzione e un &#39;in caso di errore&#39; o &#39;Riprendi&#39; istruzione
È stata combinata un'istruzione `Try` con `On Error` o `Resume`.  
  
 **ID errore:** BC30544  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere `On Error` o `Resume`.
