# L&#39;attributo XML &#39;version&#39; deve essere il primo attributo nella dichiarazione XML
L'attributo `version` di un valore letterale documento XML non Ã¨ il primo attributo specificato. L'attributo `version` deve essere il primo attributo specificato per un valore letterale documento XML, in base alla specifica XML 1.0.  
  
 **ID errore:** BC31156  
  
### Per correggere l'errore  
  
-   Riposizionare l'attributo `version` del valore letterale documento XML in modo che sia il primo attributo.
