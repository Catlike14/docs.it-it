# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-the-interface-39ltinterfacename2gt39-from-which-it-inherits-could-be-identical-to-interface-39ltinterfacename3gt39-for-some-type-arguments"></a><span data-ttu-id="95e04-102">Impossibile ereditare l'interfaccia &#39; &lt;nomeinterfaccia1&gt;&#39; perché l'interfaccia &#39;&lt; nomeinterfaccia2&gt;&#39; da cui eredita potrebbe essere identica all'interfaccia &#39;&lt; nomeinterfaccia3&gt;&#39; per alcuni argomenti di tipo</span><span class="sxs-lookup"><span data-stu-id="95e04-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because the interface &#39;&lt;interfacename2&gt;&#39; from which it inherits could be identical to interface &#39;&lt;interfacename3&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="95e04-103">Un'interfaccia generica eredita da due o più interfacce generiche e due delle ereditarietà potrebbero essere in conflitto per determinati valori di argomenti di tipo.</span><span class="sxs-lookup"><span data-stu-id="95e04-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="95e04-104">Le istruzioni seguenti possono generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="95e04-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="95e04-105">Se l'interfaccia `derivedInterface` è costruita o implementata specificando lo stesso tipo sia in `t1` che in `t2`, deve ereditare due versioni di `interfaceX` con argomenti di tipo identici.</span><span class="sxs-lookup"><span data-stu-id="95e04-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="95e04-106">In questo modo si produrrebbe un'ambiguità sulla versione a cui accedere.</span><span class="sxs-lookup"><span data-stu-id="95e04-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="95e04-107">**ID errore:** BC32121</span><span class="sxs-lookup"><span data-stu-id="95e04-107">**Error ID:** BC32121</span></span>
