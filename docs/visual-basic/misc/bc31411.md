# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39;&lt;TypeName&gt; &#39; deve essere dichiarato come &#39;MustInherit&#39; perché contiene metodi dichiarati come &#39;MustOverride&#39;
I tipi che contengono i membri `MustOverride` devono essere dichiarati come `MustInherit`.  
  
 **ID errore:** BC31411  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere il modificatore `MustInherit` al tipo.
