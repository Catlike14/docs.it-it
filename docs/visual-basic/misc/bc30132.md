# <a name="label-39ltlabelnamegt39-is-not-defined"></a>Etichetta &#39; &lt;NomeEtichetta&gt;&#39; non è definito
Viene fatto riferimento a un'etichetta di riga o un numero di riga, che tuttavia non è definito dell'ambito del riferimento. L'etichetta deve essere all'interno della routine che contiene il riferimento.  
  
 **ID errore:** BC30132  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Ristrutturare il codice in modo che l'etichetta di riga e il riferimento siano nella stessa routine.
