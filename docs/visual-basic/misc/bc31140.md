# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>&#39;Personalizzato&#39; modificatore può essere utilizzato solo immediatamente prima di un &#39;evento&#39; dichiarazione
La parola chiave `Custom` deve precedere solo la parola chiave `Event` in una dichiarazione di evento personalizzata.  
  
 **ID errore:** BC31140  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare la parola chiave `Custom` in modo che preceda la parola chiave `Event` .  
  
     -oppure-  
  
-   Rimuovere la parola chiave `Custom` .
