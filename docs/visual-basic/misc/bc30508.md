# <a name="39ltnamegt39-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-39lttypenamegt39-through-ltnamegt"></a>&#39;&lt;nome&gt; &#39; non può esporre il tipo &lt;type1&gt; in &lt;identificatore&gt; &lt;type2&gt; &#39; &lt;typename&gt; &#39; tramite &lt;nome&gt;
Si è provato a esporre un tipo privato al di fuori della classe pubblica `MyClass`.  
  
 **ID errore:** BC30508  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare il tipo `Public`.
