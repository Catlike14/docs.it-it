# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-declared-39mustinherit39"></a>&#39;&lt;TypeName&gt; &#39; non può essere utilizzato come attributo perché è dichiarato come &#39;MustInherit&#39;
Le classi Attribute personalizzate non possono essere dichiarate come `MustInherit`.  
  
 **ID errore:** BC31506  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il modificatore `MustInherit` dagli attributi personalizzati.
