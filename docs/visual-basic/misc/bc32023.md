# <a name="expression-is-of-type-39lttypenamegt39-which-is-not-a-collection-type"></a>Espressione è di tipo '&lt;typename&gt;', che non è un tipo di raccolta
La variabile di gruppo specificata in un `For Each` istruzione non è un oggetto raccolta o una matrice e il relativo tipo non implementa il <xref:System.Collections.IEnumerable>interfaccia.</xref:System.Collections.IEnumerable> Il tipo è necessario un supporto di [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] il modello di progettazione di raccolta o implementare <xref:System.Collections.IEnumerable>.</xref:System.Collections.IEnumerable>  
  
 **ID errore:** BC32023  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare la variabile di gruppo è un tipo di classe che supporta la la [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] progettazione insieme o implementa <xref:System.Collections.IEnumerable>.</xref:System.Collections.IEnumerable>
