# L&#39;espressione &#232; di tipo &#39;&lt;nometipo&gt;&#39;, che non &#232; un tipo raccolta
La variabile di gruppo specificata in un'istruzione `For Each` non Ã¨ un oggetto raccolta o una matrice e il relativo tipo non implementa l'interfaccia <xref:System.Collections.IEnumerable>. Il tipo deve supportare lo schema progettuale della raccolta di [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] o implementare <xref:System.Collections.IEnumerable>.  
  
 **ID errore:** BC32023  
  
### Per correggere l'errore  
  
-   Dichiarare la variabile di gruppo come tipo classe che supporta la progettazione raccolta di [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] o implementa <xref:System.Collections.IEnumerable>.
