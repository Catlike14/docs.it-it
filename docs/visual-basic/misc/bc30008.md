# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>Necessario un riferimento al modulo &#39; &lt;modulename&gt;&#39; contenente la classe di base &#39;&lt; ClassName&gt;&#39;
Necessario riferimento al modulo '\<NomeModulo >' contenente la classe base\<NomeClasse >'. Aggiungerne uno al progetto.  
  
 La classe è definita in un modulo a cui non si fa riferimento direttamente nel progetto. Il compilatore [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] richiede un riferimento per evitare ambiguità nel caso in cui la classe venga definita in più moduli.  
  
 **ID errore:** BC30008  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Includere il nome del modulo senza riferimento nei riferimenti del progetto.
