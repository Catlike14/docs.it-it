# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>Necessario riferimento al modulo &#39; &lt;modulename&gt; &#39; che contiene la classe di base &#39; &lt;classname&gt;&#39;
Necessario riferimento al modulo '\<NomeModulo >' contenente la classe base\<NomeClasse >'. Aggiungerne uno al progetto.  
  
 La classe è definita in un modulo a cui non si fa riferimento direttamente nel progetto. Il compilatore Visual Basic richiede un riferimento per evitare ambiguità nel caso in cui la classe è definita in più di un modulo.  
  
 **ID errore:** BC30008  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Includere il nome del modulo senza riferimento nei riferimenti del progetto.
