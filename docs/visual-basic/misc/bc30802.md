# <a name="39type39-statements-are-no-longer-supported-use-39structure39-statements-instead"></a>&#39;Tipo&#39; le istruzioni non sono più supportate; Utilizzare &#39;struttura&#39; istruzioni invece
La parola chiave `Type` non può più essere usata per definire i tipi definiti dall'utente.  
  
 **ID errore:** BC30802  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare l'istruzione `Structure` per definire un nuovo tipo.
