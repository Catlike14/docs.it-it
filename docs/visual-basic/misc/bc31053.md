# <a name="39implements39-statement-must-follow-any-39inherits39-statement-and-precede-all-declarations-in-a-class"></a>&#39; implementa &#39; istruzione deve seguire uno &#39; Inherits &#39; istruzione e precedere tutte le dichiarazioni in una classe
Ãˆ stata rilevata un'istruzione `Implements` in una posizione non valida.  
  
 **ID errore:** BC31053  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Inserire le istruzioni `Implements` subito dopo le istruzioni `Inherits` , ma prima di qualsiasi dichiarazione. Ad esempio:  
  
    ```  
    Class Derived  
       Inherits Base  
       Implements One  
       Sub SubOne() Implements One.Method1  
          ' Add code to implement the method.  
       End Sub  
    End Class  
    ```
