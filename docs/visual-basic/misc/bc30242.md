# <a name="39ltmodifiergt39-is-not-valid-on-a-method-declaration"></a>&#39; &lt;modificatore&gt;&#39; non è valido in una dichiarazione di metodo
Un'istruzione `Function` o `Sub` contiene una parola chiave non valida.  
  
 **ID errore:** BC30242  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la parola chiave non valida dall'istruzione `Function` o `Sub` .
