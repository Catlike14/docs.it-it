# <a name="39set39-parameter-must-have-the-same-type-as-the-containing-property"></a>&#39;Impostare&#39; il parametro deve avere lo stesso tipo di proprietà che lo contiene
Il parametro per la routine di proprietà `Set` presenta un tipo diverso dalla proprietà a cui appartiene.  
  
 **ID errore:** BC31064  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il tipo di dati del parametro su `Set` in modo che corrisponda al tipo di dati per la proprietà. Ad esempio:  
  
    ```  
    Class Class1  
       ' Declare a local variable to hold the property value.  
       Private Fval As Integer  
  
       Property F() As Integer  
          Get  
             Return Fval  
          End Get  
          Set(ByVal Value As Integer)  
             Fval = Value  
          End Set  
       End Property  
    End Class  
    ```
