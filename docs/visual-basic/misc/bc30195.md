# <a name="expected-one-of-39dim39-39const39-39public39-39private39-39protected39-39friend39-39shadows39-39readonly39-or-39shared39"></a>Era previsto uno di &#39;Dim&#39;, &#39;Const&#39;, &#39;pubblica&#39;, &#39;privato&#39;, &#39;Protected&#39;, &#39;Friend&#39;, &#39;Shadows&#39;, &#39;ReadOnly&#39;o &#39;condiviso&#39;
Un'istruzione di dichiarazione manca in una parola chiave di dichiarazione. L'errore potrebbe essere causato dal fatto che una dichiarazione di attributo chiama un metodo.  
  
 **ID errore:** BC30195  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare se un metodo viene dichiarato all'interno di una dichiarazione di attributo.  
  
2.  Avviare l'istruzione con la parola chiave di dichiarazione appropriata.
