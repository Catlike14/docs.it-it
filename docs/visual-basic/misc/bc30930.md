# <a name="39ltnamegt39-necessary-for-compiling-this-construct-cannot-be-found"></a>&#39;&lt;nome&gt;&#39;, necessario per compilare il costrutto non è stato trovato
Un elemento necessario per la compilazione non è stato trovato.  
  
 **ID errore:** BC30930  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Specificare l'elemento helper necessario.  
  
2.  Verificare che il codice sia codice gestito valido.
