# <a name="integer-constant-expected"></a>È prevista la costante Integer
È presente una direttiva `#ExternalSource` in cui il secondo argomento non è un valore letterale Integer. In questo contesto è valido solo un valore letterale Integer. Un membro costante o di enumerazione denominato non è valido.  
  
 **ID errore:** BC30204  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Usare un membro costante o di enumerazione denominato anziché un valore letterale.  
  
2.  Fornire un valore letterale Integer come secondo argomento per la direttiva `#ExternalSource` .
