# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Compilatore Visual Basic è in grado di correggere l'errore seguente: &lt;errore&gt;
Il compilatore Visual Basic non è in grado di risolvere l'errore seguente: <`error`>.  
  
 Alcune funzionalità IDE sono state disabilitate.  
  
 Si è verificato un errore interno nel compilatore Visual Basic.  
  
 **ID errore:** BC31021  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Salvare il lavoro e riavviare [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
2.  Se l'errore si ripete, reinstallare [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Se l'errore persiste dopo la reinstallazione, inviare una notifica al Servizio Supporto Tecnico Clienti Microsoft.
