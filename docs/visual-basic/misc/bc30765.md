# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Tutti i progetti di una soluzione Visual Basic devono avere la stessa piattaforma di destinazione. Il progetto che si tenta di aggiungere ha come destinazione una piattaforma diversa da quella specificata per la soluzione
Tutti i progetti di una soluzione [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] devono avere la stessa piattaforma di destinazione, ma il progetto che si intende aggiungere ha come destinazione una piattaforma diversa da quella specificata per la soluzione.  
  
 **ID errore:** BC30765  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere solo progetti destinati alla stessa piattaforma specificata dalla soluzione.
