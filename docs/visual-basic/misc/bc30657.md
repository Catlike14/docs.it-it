# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;nomeroutine&gt; &#39; contiene un tipo restituito non supportato o tipi di parametro che non sono supportati
Una routine è dichiarata con parametri o un tipo restituito non può essere rappresentato da Visual Studio.  
  
 **ID errore:** BC30657  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Definire i tipi restituiti o parametri come un tipo supportato.  
  
-   Sviluppare un nuovo tipo che descrive i dati usati da questa routine.
