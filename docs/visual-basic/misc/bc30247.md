# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>&#39; &lt;modificatore&gt;&#39; non è valido in una dichiarazione di variabile locale
Un'istruzione `Dim` all'interno di una routine contiene una parola chiave non valida, ad esempio `Overloads`.  
  
 **ID errore:** BC30247  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la parola chiave non valida dall'istruzione `Dim` .
