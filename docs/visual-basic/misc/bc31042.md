# <a name="39sub-new39-cannot-implement-interface-members"></a>&#39;Sub New&#39; non può implementare i membri di interfaccia
`Sub New` è un costruttore e non può implementare membri.  
  
 **ID errore:** BC31042  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Implements` dalle routine `Sub New` .
