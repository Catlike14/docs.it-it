# <a name="expected-closing-3939-for-xml-entity"></a>Previsto di chiusura &#39;; &#39; per l'entità XML
In un riferimento a entità XML (ad esempio, `>`) manca un punto e virgola di chiusura (;).  
  
 **ID errore:** BC31178  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Includere il punto e virgola alla fine del riferimento a entità XML.
