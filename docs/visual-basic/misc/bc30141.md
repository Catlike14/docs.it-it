# <a name="unable-to-create-assembly-linker-object-lterror-messagegt"></a>Impossibile creare l'oggetto Assembly Linker: &lt;messaggio di errore&gt;
Il compilatore Visual Basic chiama Assembly Linker (Al.exe, definito anche Alink) per generare un assembly con un manifesto. Il linker ha segnalato un errore di creazione del proprio oggetto di lavoro. Potrebbe trattarsi di un problema di generazione di un riferimento a un file di metadati o di un errore di chiamata di un file di importazione.  
  
 **ID errore:** BC30141  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Esaminare il messaggio di errore tra virgolette e consultare l'argomento [Al.exe](../../framework/tools/al-exe-assembly-linker.md) per spiegazioni e suggerimenti aggiuntivi.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
