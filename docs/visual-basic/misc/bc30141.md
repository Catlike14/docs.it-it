# <a name="unable-to-create-assembly-linker-object-lterror-messagegt"></a>Impossibile creare l'oggetto Assembly Linker: &lt;messaggio di errore&gt;
Il compilatore [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] chiama Assembly Linker (Al.exe, definito anche Alink) per generare un assembly con un manifesto. Il linker ha segnalato un errore di creazione del proprio oggetto di lavoro. Potrebbe trattarsi di un problema di generazione di un riferimento a un file di metadati o di un errore di chiamata di un file di importazione.  
  
 **ID errore:** BC30141  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Esaminare il messaggio di errore riportato e vedere l'argomento [Errori e avvisi di Al.exe](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) per spiegazioni e suggerimenti aggiuntivi.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
