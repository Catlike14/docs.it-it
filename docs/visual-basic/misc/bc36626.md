# <a name="lambda-expression-cannot-be-converted-to-39lttypenamegt39-because-type-39lttypenamegt39-is-declared-39mustinherit39-and-cannot-be-created"></a>Espressione lambda non può essere convertita in &#39; &lt;typename&gt;&#39; perché tipo &#39;&lt; TypeName&gt;&#39; è dichiarato &#39; MustInherit &#39; e può essere creato
Un'istruzione tenta di convertire un'espressione lambda in un tipo che può essere solo una classe base e non può essere usato per creare un'istanza.
