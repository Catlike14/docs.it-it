# Non &#232; possibile convertire l&#39;espressione lambda in &#39;&lt;nometipo&gt;&#39; perch&#233; il tipo &#39;&lt;nometipo&gt;&#39; &#232; dichiarato &#39;MustInherit&#39; e non pu&#242; essere creato
Un'istruzione tenta di convertire un'espressione lambda in un tipo che può essere solo una classe base e non può essere usato per creare un'istanza.
