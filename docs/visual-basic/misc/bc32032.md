# <a name="39end-function39-must-be-the-first-statement-on-a-line"></a>&#39; End Function &#39; deve essere la prima istruzione in una riga
Un'istruzione `End Function` segue un separatore di istruzioni due punti (:). `End Function` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC32032
