# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>I nomi completi dei tipi negli spazi dei nomi non possono essere più &lt;numero&gt; caratteri
Il nome completo del tipo non può superare il numero specificato di caratteri.  
  
 **ID errore:** BC30031  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Abbreviare il nome del tipo.
