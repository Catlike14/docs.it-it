 <span data-ttu-id="32b34-105">Il codice seguente illustra le conversioni non riuscite.</span><span class="sxs-lookup"><span data-stu-id="32b34-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="32b34-106">Il delegato è `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="32b34-106">The delegate is `FunDel`.</span></span>

 <span data-ttu-id="32b34-107">Ognuna delle funzioni seguenti differisce da `FunDel` in modo tale da generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="32b34-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date

 <span data-ttu-id="32b34-108">Ognuna delle istruzioni di assegnazione seguenti genera l'errore.</span><span class="sxs-lookup"><span data-stu-id="32b34-108">Each of the following assignment statements causes the error.</span></span>

    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2
