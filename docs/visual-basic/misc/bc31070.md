# <a name="class-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Classe in un'interfaccia non può essere dichiarati come &#39; &lt;identificatore&gt;&#39;
Il modificatore specificato con una classe dichiarata in un'interfaccia non è valido.  
  
 **ID errore:** BC31070  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere i modificatori come `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`o `Overridable`.
