# <a name="required-attribute-39version39-missing-from-xml-declaration"></a>Attributo obbligatorio &#39;versione&#39; mancante nella dichiarazione XML
L'attributo `version` per un valore letterale del documento XML Ã¨ mancante.  
  
 **ID errore:** BC31153  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere l'attributo `version` alla dichiarazione del documento XML, come nell'esempio seguente: `<?xml version="1.0"?>`
