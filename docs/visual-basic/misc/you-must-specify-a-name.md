# <a name="you-must-specify-a-name"></a>È necessario specificare un nome
Quando si eseguono operazioni sui file, ad esempio la ridenominazione di un file, è necessario specificare il nome.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare il nome.
