# Non &#232; possibile trovare la directory .NET Framework: &lt;errore&gt;
La directory in cui sono archiviati gli assembly [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] non è stata trovata.  
  
 **ID errore:** BC31508  
  
### Per correggere l'errore  
  
1.  Ripetere la compilazione del programma, per controllare se l'errore si verifica di nuovo.  
  
2.  Se l'errore si ripresenta, salvare il lavoro e riavviare [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].  
  
3.  Se l'errore si ripete, reinstallare [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Se l'errore persiste dopo la reinstallazione, inviare una notifica al Servizio Supporto Tecnico Clienti Microsoft.
