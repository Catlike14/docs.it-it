# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39;&lt;TypeName&gt; &#39; è un tipo non supportato
Il tipo specificato non può essere rappresentato da Visual Studio.  
  
 **ID errore:** BC30649  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare un tipo supportato.  
  
-   Sviluppare un nuovo tipo.
