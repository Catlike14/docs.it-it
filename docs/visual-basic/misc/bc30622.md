# <a name="39end-module39-must-be-preceded-by-a-matching-39module39"></a>&#39;End Module&#39; deve essere preceduto da un oggetto corrispondente &#39;modulo&#39;
I blocchi del modulo iniziano con un'istruzione `Module` e terminano con un costrutto `End Module` .  
  
 **ID errore:** BC30622  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il blocco del modulo inizi con un'istruzione `Module` .
