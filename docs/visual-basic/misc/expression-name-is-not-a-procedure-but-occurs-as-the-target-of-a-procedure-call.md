# <a name="expression-39ltnamegt39-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>Espressioni &#39; &lt;nome&gt;&#39; non è una procedura, ma si presenta come destinazione di una chiamata di routine
È stata eseguita una chiamata di routine a un'espressione che non è una routine.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che il nome della routine sia stato digitato correttamente.  
  
2.  Fornire la routine che <`name`> chiamate.
