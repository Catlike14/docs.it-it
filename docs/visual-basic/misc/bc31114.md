# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39; Evento personalizzato &#39; deve terminare con un corrispondente &#39; Evento di fine &#39;
Una dichiarazione `Custom Event` deve terminare con un'istruzione `End Event` .  
  
 **ID errore:** BC31114  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare che la dichiarazione `Custom Event` termini con un'istruzione `End Event` .
