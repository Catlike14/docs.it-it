# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39; SyncLock &#39; l'istruzione deve terminare con un corrispondente &#39; End SyncLock &#39;
I blocchi`SyncLock` iniziano con la parola chiave `SyncLock` e terminano con il costrutto `End SyncLock` .  
  
 **ID errore:** BC30675  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il blocco `SyncLock` termini con un costrutto `End SyncLock` .
