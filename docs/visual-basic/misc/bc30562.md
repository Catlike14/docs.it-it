# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39;&lt;nome&gt; &#39; è ambiguo tra le dichiarazioni nei moduli &#39; &lt;modulename1&gt; &#39; e &#39; &lt;modulename2&gt;&#39;
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il compilatore Visual Basic non dispone di regole di risoluzione dei conflitti. è necessario risolvere le ambiguità nei nomi manualmente.  
  
 **ID errore:** BC30562  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un nome completo.
