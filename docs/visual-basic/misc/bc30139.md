# <a name="error-setting-assembly-manifest-option-lterror-messagegt"></a>Errore durante l'impostazione di opzione del manifesto dell'assembly: &lt;messaggio di errore&gt;
Il compilatore Visual Basic chiama Assembly Linker (Al.exe, definito anche Alink) per generare un assembly con un manifesto. Il linker ha segnalato un errore durante la ricerca di un file di chiave o di un nome di chiave con cui firmare l'assembly.  
  
 **ID errore:** BC30139  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Esaminare il messaggio di errore tra virgolette e vedere gli errori indicati in [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) per spiegazioni e suggerimenti aggiuntivi.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
