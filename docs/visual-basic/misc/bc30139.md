# <a name="error-setting-assembly-manifest-option-lterror-messagegt"></a>Errore durante l'impostazione di opzione del manifesto dell'assembly: &lt;messaggio di errore&gt;
Il compilatore [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] chiama Assembly Linker (Al.exe, definito anche Alink) per generare un assembly con un manifesto. Il linker ha segnalato un errore durante la ricerca di un file di chiave o di un nome di chiave con cui firmare l'assembly.  
  
 **ID errore:** BC30139  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Esaminare il messaggio di errore riportato e vedere la pagina dal titolo [Errori e avvisi di Al.exe](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) per spiegazioni e suggerimenti aggiuntivi.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
