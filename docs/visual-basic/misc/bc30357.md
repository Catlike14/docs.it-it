# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39;Eredita&#39; le istruzioni devono precedere tutte le dichiarazioni in un'interfaccia
Un'istruzione `Inherits` deve essere la prima riga non vuota non di commento in una definizione di classe.  
  
 **ID errore:** BC30357  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare l'istruzione `Inherits` in modo che preceda tutte le dichiarazioni.
