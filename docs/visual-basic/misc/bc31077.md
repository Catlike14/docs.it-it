# <a name="39ltassemblynamegt39-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>&#39; &lt;assemblyname&gt;&#39; è un assembly e non può fare riferimento come modulo
Si è provato a usare un assembly come modulo.  
  
 **ID errore:** BC31077  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il riferimento a un modulo.
