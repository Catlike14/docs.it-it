## <a name="to-correct-this-error"></a><span data-ttu-id="c4685-111">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="c4685-111">To correct this error</span></span>  
  
-   <span data-ttu-id="c4685-112">Se la matrice di parametri non specifica il tipo di dati che si vuole estendere, aggiungere un nuovo primo parametro che specifichi il tipo.</span><span class="sxs-lookup"><span data-stu-id="c4685-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="c4685-113">Se la matrice di parametri specifica il tipo di dati che si vuole estendere, modificarla in una matrice standard, che richiede un argomento, anziché una matrice di parametri.</span><span class="sxs-lookup"><span data-stu-id="c4685-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="c4685-114">Le matrici normali possono essere estese.</span><span class="sxs-lookup"><span data-stu-id="c4685-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```
