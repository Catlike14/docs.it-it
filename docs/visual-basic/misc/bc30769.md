# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>La versione di destinazione di .NET Compact Framework non supporta la &#39;fine&#39; istruzione
La versione di .NET Compact Framework in uso non supporta l'istruzione `End` .  
  
 **ID errore:** BC30769  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione.
