# <a name="39ltnamespacenamegt39-is-ambiguous"></a>'&lt;NomeSpazioDeiNomi&gt;' è ambiguo
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilatore non dispone di tutte le regole di risoluzione dei conflitti, è necessario evitare ambiguità nei nomi manualmente.  
  
 **ID errore:** BC30554  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un nome completo.
