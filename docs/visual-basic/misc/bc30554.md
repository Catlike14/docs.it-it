# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39;&lt;NomeSpazioDeiNomi&gt; &#39; è ambiguo
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il compilatore Visual Basic non dispone di eventuali regole di risoluzione dei conflitti; è necessario risolvere le ambiguità nei nomi manualmente.  
  
 **ID errore:** BC30554  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un nome completo.
