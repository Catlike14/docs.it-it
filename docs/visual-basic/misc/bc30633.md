# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;Impostare&#39; l'istruzione deve terminare con un corrispondente &#39;fine Set&#39;
Le routine della proprietà`Set` devono terminare con un'istruzione `End Set` .  
  
 **ID errore:** BC30633  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che la routine della proprietà `Set` termini con un costrutto `End Set` .
