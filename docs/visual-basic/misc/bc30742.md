# <a name="value-39ltvaluename1gt39-cannot-be-converted-to-39ltvaluename2gt39"></a>Valore &#39; &lt;valuename1&gt; &#39; non può essere convertito in &#39; &lt;valuename2&gt;&#39;
È stata tentata una conversione del tipo tra due tipi incompatibili.  
  
 **ID errore:** BC30742  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare l'esecuzione del cast dei tipi.
