# Valori letterali XML e propriet&#224; axis XML non disponibili
Valori letterali XML e proprietà axis XML non disponibili. Aggiungere i riferimenti a System.Xml, System.Xml.Linq e System.Core.  
  
 Il codice in fase di compilazione include valori letterali XML o proprietà axis XML, ma non fa riferimento agli assembly necessari per compilare i valori letterali XML o le proprietà axis XML.  
  
 **ID errore:** BC31190  
  
### Per correggere l'errore  
  
-   Aggiungere riferimenti agli assembly System.Xml.dll, System.Xml.Linq.dll e System.Core.dll.
