# <a name="39if39-operands-cannot-be-named-arguments"></a><span data-ttu-id="77c0a-102">&#39;Se&#39; gli operandi non possono essere argomenti denominati</span><span class="sxs-lookup"><span data-stu-id="77c0a-102">&#39;If&#39; operands cannot be named arguments</span></span>
<span data-ttu-id="77c0a-103">L'uso degli argomenti denominati negli operandi dell'operatore `If` non è valido.</span><span class="sxs-lookup"><span data-stu-id="77c0a-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="77c0a-104">L'esempio seguente genera questo errore:</span><span class="sxs-lookup"><span data-stu-id="77c0a-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="77c0a-105">Questo comportamento è diverso da quello della funzione `IIf` , che consente argomenti denominati, come mostrato nel codice seguente:</span><span class="sxs-lookup"><span data-stu-id="77c0a-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="77c0a-106">**ID errore:** BC33105</span><span class="sxs-lookup"><span data-stu-id="77c0a-106">**Error ID:** BC33105</span></span>
