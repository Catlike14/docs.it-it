# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="3579d-102">Il valore letterale XML non può trovarsi in questa posizione a meno che non sia racchiuso tra parentesi</span><span class="sxs-lookup"><span data-stu-id="3579d-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="3579d-103">Una dichiarazione di valore letterale XML viene usata in un'espressione in una posizione ambigua per il compilatore [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .</span><span class="sxs-lookup"><span data-stu-id="3579d-103">An XML literal declaration is used in an expression in a location that is ambiguous to the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler.</span></span> <span data-ttu-id="3579d-104">Vale a dire il [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] non è possibile determinare se minore di-rispetto a (<) carattere è da intendersi come operatore di confronto o inizio di un valore letterale XML.</span><span class="sxs-lookup"><span data-stu-id="3579d-104">That is, the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="3579d-105">Nel codice seguente ne viene illustrato un esempio:</span><span class="sxs-lookup"><span data-stu-id="3579d-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="3579d-106">**ID errore:** BC31198</span><span class="sxs-lookup"><span data-stu-id="3579d-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3579d-107">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="3579d-107">To correct this error</span></span>

                  Select element

                  Select element
