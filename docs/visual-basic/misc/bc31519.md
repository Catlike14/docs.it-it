# <a name="39ltfilenamegt39-cannot-be-referenced-because-it-is-not-an-assembly"></a>&#39;&lt;filename&gt; &#39; non è possibile farvi riferimento perché non è un assembly
Si è fatto riferimento a un file che non è un assembly .NET.  
  
 **ID errore:** BC31519  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare il nome e il percorso dell'assembly.  
  
2.  Verificare che l'assembly non sia danneggiato.
