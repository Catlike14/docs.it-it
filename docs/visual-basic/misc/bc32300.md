# <a name="object-no-longer-exists-due-to-compile-error-or-deletion"></a>L'oggetto non esiste più a causa di un errore di compilazione o di un'eliminazione
Si è verificato un errore di accesso di elemento di codice all'interno di [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilatore.  
  
 **ID errore:** BC32300  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Ripetere la compilazione del programma, per controllare se l'errore si verifica di nuovo.  
  
2.  Se l'errore si ripresenta, reinstallare il compilatore [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
3.  Se l'errore persiste dopo la reinstallazione, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
