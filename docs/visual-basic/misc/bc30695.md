# <a name="lttypenamegt-39ltmembernamegt39-must-be-declared-39shadows39-because-another-member-with-this-name-is-declared-39shadows39"></a>&lt;TypeName&gt; &#39;&lt; MemberName&gt;&#39; deve essere dichiarato &#39; Shadows &#39; perché un altro membro con questo nome è dichiarato come &#39; Shadows &#39;
La proprietà o la routine ha lo stesso nome di un'altra proprietà o una routine contrassegnata con il modificatore `Shadows` .  
  
 **ID errore:** BC30695  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare il modificatore `Shadows` .
