# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>Impossibile creare un'interfaccia di Runtime .NET: &lt;errore&gt;
Si è verificato un errore del compilatore interno durante la creazione di un'interfaccia, oppure una chiamata al metodo su un'interfaccia è fallita.  
  
 **ID errore:** BC31024  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Salvare il lavoro e riavviare [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
2.  Se l'errore si ripete, reinstallare [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Se l'errore persiste dopo la reinstallazione, inviare una notifica al Servizio Supporto Tecnico Clienti Microsoft.
