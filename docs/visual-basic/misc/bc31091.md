# <a name="import-of-type-39lttypenamegt39-from-assembly-or-module-39ltnamegt39-failed"></a>Importazione di tipo &#39; &lt;typename&gt;&#39; dall'assembly o modulo &#39;&lt; nome&gt;&#39; non riuscita
Verificare che tutte le librerie a cui viene fatto riferimento siano compatibili tra loro.  
  
 **ID errore:** BC31091  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che i file che contengono il tipo a cui si fa riferimento non siano danneggiati.
