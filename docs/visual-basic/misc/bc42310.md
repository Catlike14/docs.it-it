# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>XML commento tag &#39; includere &#39; deve avere un &#39; &lt;attributo&gt;&#39; attributo
Tag di commento XML 'include' deve avere un \<attributo > attributi. Il commento XML verrà ignorato.  
  
 Un attributo `file` o `path` , passato al tag `<`include`>` , è assente o incompleto.  
  
 **ID errore:** BC42310  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere o completare l'attributo richiesto il \<includono > tag.
