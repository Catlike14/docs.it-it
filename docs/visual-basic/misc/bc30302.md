# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Carattere di tipo &#39; &lt;carattere&gt; &#39; non può essere utilizzato in una dichiarazione con un tipo esplicito
È stato usato un carattere tipo, ad esempio `$`, quando si è specificato in modo esplicito un tipo.  
  
 **ID errore:** BC30302  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il carattere tipo oppure rimuovere la clausola `As <Type>` dalla dichiarazione.
