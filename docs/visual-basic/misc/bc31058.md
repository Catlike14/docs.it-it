# <a name="conflicts-with-39ltname1gt39-which-is-implicitly-created-for-39ltname2gt39-in-lttypegt-39ltname3gt39"></a>È in conflitto con &#39; &lt;nome1&gt;&#39;, che viene creato in modo implicito per &#39; &lt;name2&gt; &#39; nella &lt;tipo&gt; &#39; &lt;name3&gt;&#39;
Il nome di un membro di tipo è in conflitto con un altro membro creato in modo implicito.  
  
 **ID errore:** BC31058  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rinominare il membro dichiarato in modo esplicito per rimuovere il conflitto di denominazione.
