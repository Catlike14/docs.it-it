# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39; &lt;typename&gt;&#39; è un tipo in &#39;&lt; ContainerName&gt;&#39; e non può essere utilizzato come espressione
Uno dei nomi in uso è già stato dichiarato come nome di un tipo e non può essere usato in un'espressione.  
  
 **ID errore:** BC30691  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Assegnare un nome univoco al membro.  
  
-   Racchiudere il nome del membro tra parentesi quadre.
