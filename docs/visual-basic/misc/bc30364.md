# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39; Sub nuovo &#39; non può essere dichiarato &#39; &lt;modificatore&gt;&#39;
Si è provato a dichiarare un oggetto `Sub New` con un modificatore, ad esempio `Overrides`, che non è valido.  
  
 **ID errore:** BC30364  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il modificatore.
