# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>Tipo '&lt;typename&gt;'viene importato da diverse versioni dell'assembly'&lt;assemblyname&gt;'
Tipo '\<typename >' viene importato da diverse versioni dell'assembly '\<assemblyname >'. Impossibile utilizzare diverse versioni dello stesso assembly con nome sicuro nello stesso progetto.  
  
 Un tipo è stato importato da versioni diverse dello stesso assembly, creando ambiguità.  
  
 **ID errore:** BC31525  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere i riferimenti a più versioni dello stesso assembly.
