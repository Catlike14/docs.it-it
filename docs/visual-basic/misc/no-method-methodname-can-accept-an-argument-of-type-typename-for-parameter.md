# <a name="no-method-39ltmethodnamegt39-can-accept-an-argument-of-type-39lttypenamegt39-for-parameter-39ltparameternamegt39"></a>Nessun metodo &#39; &lt;NomeMetodo&gt;&#39; può accettare un argomento di tipo &#39;&lt; TypeName&gt;&#39; per il parametro &#39;&lt; ParameterName&gt;&#39;
Una chiamata al metodo contiene un argomento di un tipo che non è valido.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare i tipi degli argomenti.  
  
2.  Verificare la corrispondenza tra i tipi degli argomenti e i tipi di dei parametri del metodo. Modificare i tipi in base alle esigenze.
