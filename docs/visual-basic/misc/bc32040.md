# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>Il &#39; &lt;(parola chiave)&gt;&#39; parola chiave viene utilizzata per l'overload dei membri ereditati; non utilizzare il &#39;&lt; parola chiave&gt;&#39; parola chiave per l'overload di &#39; Sub nuovo &#39;
Un costruttore è dichiarato con la parola chiave `Overloads` .  
  
 Visual Basic non supporta costruttori con funzione di eredità o di overload.  
  
 **ID errore:** BC32040  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la parola chiave `Overloads` da tutte le dichiarazioni di costruttore.
