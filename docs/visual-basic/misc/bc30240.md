# <a name="39exit39-must-be-followed-by-39sub39-39function39-39property39-39do39-39for39-39while39-39select39-or-39try39"></a>&#39; Uscita &#39; deve essere seguita da &#39; Sub &#39; &#39; Funzione &#39; &#39; Proprietà &#39; &#39; Do &#39; &#39; Per &#39; &#39; Mentre &#39; &#39; Selezionare &#39; o &#39; Provare a &#39;
Un'istruzione `Exit` contiene una parola chiave non valida. `Exit` deve specificare il blocco da cui il controllo viene trasferito all'istruzione che segue il blocco; ad esempio, `End While`.  
  
 **ID errore:** BC30240  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere una parola chiave valida che segue `Exit`o rimuovere l'istruzione `Exit` .
