# <a name="39ltmembernamegt-has-the-same-name-as-a-type-parameter"></a>&#39; &lt;membername&gt; ha lo stesso nome di un parametro di tipo
Un membro di una classe o struttura generica ha lo stesso nome di un parametro di tipo di quella classe o struttura.  
  
 Ogni variabile, costante, enumerazione, routine, propriet√†, evento e delegato deve avere un nome diverso da qualsiasi parametro di tipo.  
  
 **ID errore:** BC32054  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il nome del membro della classe o struttura in modo che sia diverso dal nome di ogni parametro di tipo.
