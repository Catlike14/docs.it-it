# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'RaiseEvent' non sono valide nella finestra controllo immediato
`RaiseEvent`Impossibile utilizzare le istruzioni nel **immediato** finestra.  
  
 **ID errore:** BC30739  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Testare le istruzioni `RaiseEvent` dal codice nell' **Editor di codice**.
