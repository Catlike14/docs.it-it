# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39;RaiseEvent&#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`RaiseEvent` non possono essere usate nella finestra di controllo **immediato** .  
  
 **ID errore:** BC30739  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Testare le istruzioni `RaiseEvent` dal codice nell' **Editor di codice**.
