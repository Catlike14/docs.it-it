# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39; &lt;nome&gt;&#39; è dichiarato nel progetto &#39;&lt; nomeprogetto1&gt;&#39; che non viene fatto riferimento da progetto &#39;&lt; nomeprogetto2&gt;&#39;
Un elemento di programmazione è definito in un progetto al quale il progetto corrente non fa riferimento.  
  
 **ID errore:** BC32004  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Nel progetto corrente aggiungere un riferimento al progetto che contiene l'elemento desiderato.
