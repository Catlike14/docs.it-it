# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>Metodo in una struttura non può essere dichiarato &#39; Protetto &#39; o &#39; Protected Friend &#39;
Un metodo in una struttura è stato dichiarato come `Protected` o `Protected Friend`.  
  
 **ID errore:** BC31067  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare i metodi nelle strutture come `Public` o `Private`.
