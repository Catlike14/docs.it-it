# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-type-39date39"></a>Argomento &#39; &lt;NomeArgomento&gt; &#39; non può essere convertito nel tipo &#39;Data&#39;
Una conversione ha tentato di modificare una variabile nel tipo `Date`, ma il valore non è riconosciuto come data valida.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Trovare una conversione significativa.
