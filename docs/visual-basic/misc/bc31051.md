# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace o tipo &#39; &lt;nome&gt;&#39; è già stato importato
Si è provato a importare più volte uno spazio dei nomi.  
  
 **ID errore:** BC31051  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Imports` ridondanti.
