# <a name="method-39ltmethodname1gt39-cannot-implement-partial-method-39ltmethodname2gt39-because-39ltmethodname3gt39-already-implements-it"></a>Metodo &#39; &lt;NomeMetodo1&gt;&#39; non può implementare il metodo parziale &#39;&lt; NomeMetodo2&gt;&#39; perché &#39;&lt; nomemetodo3&gt;&#39; è già implementato da
Metodo '\<NomeMetodo1 >' non può implementare il metodo parziale '\<NomeMetodo2 >' perché '\<nomemetodo3 >' è già implementato. Un solo metodo può implementare un metodo parziale.  
  
 Non è possibile avere due metodi parziali che implementano la stessa dichiarazione di metodo parziale. Il codice seguente causa questo errore.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID errore:** BC31434
