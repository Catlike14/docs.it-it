# <a name="39wend39-statements-are-no-longer-supported-use-39end-while39-statements-instead"></a>&#39;Wend&#39; le istruzioni non sono più supportate; Utilizzare &#39;End While&#39; istruzioni invece
Il costrutto `End While` sostituisce la parola chiave `Wend` .  
  
 **ID errore:** BC30809  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare `End While` per terminare un blocco `While` .
