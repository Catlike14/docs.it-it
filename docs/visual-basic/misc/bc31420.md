# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>'&lt;membro&gt;' è in conflitto con il membro riservato con questo nome è dichiarato in modo implicito in tutte le enumerazioni
Il nome di un membro di tipo è in conflitto con il nome di un membro dichiarato in modo implicito in tutte le enumerazioni.  
  
 Il [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilatore crea membri impliciti corrispondenti a determinati elementi di programmazione dichiarati. Le enumerazioni dichiarano implicitamente il membro `value__member`.  
  
 **ID errore:** BC31420  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il nome del membro.
