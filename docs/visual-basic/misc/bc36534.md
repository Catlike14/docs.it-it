# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>Non è possibile convertire l'espressione in un albero delle espressioni
È possibile convertire in alberi delle espressioni la maggior parte delle espressioni, ma non tutte. Questo errore indica che si è provato a convertire un'espressione che non può essere convertita.  
  
 **ID errore:** BC36534
