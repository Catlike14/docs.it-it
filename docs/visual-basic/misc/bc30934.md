<span data-ttu-id="1787c-103">Valuta un'espressione utilizzata per un argomento dell'attributo a un tipo di dati diverso da quello del parametro di attributo corrispondente, e [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] non consente la conversione del tipo richiesto per gli argomenti degli attributi.</span><span class="sxs-lookup"><span data-stu-id="1787c-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="1787c-104">Un attributo fornisce metadati per l'elemento a cui è applicato e il compilatore deve essere in grado di costruire tutti i metadati in fase di compilazione.</span><span class="sxs-lookup"><span data-stu-id="1787c-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="1787c-105">Per questo motivo, ogni attributo deve usare valori costanti in fase di compilazione e, di conseguenza, anche ogni argomento di attributo deve restituire un valore costante.</span><span class="sxs-lookup"><span data-stu-id="1787c-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>  
  
 <span data-ttu-id="1787c-106">Alcune conversioni di tipo non possono produrre valori costanti in fase di compilazione.</span><span class="sxs-lookup"><span data-stu-id="1787c-106">Certain type conversions cannot produce values that are constant at compile time.</span></span> <span data-ttu-id="1787c-107">Ad esempio, la conversione di `String` in `Double` o in `Date` dipende dalle impostazioni locali in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="1787c-107">For example, converting a `String` to a `Double` or a `Date` depends on the locale setting at run time.</span></span> <span data-ttu-id="1787c-108">Altre conversioni, ad esempio la conversione di una matrice di un tipo derivato in una matrice di `Object`, presentano diversi problemi che impediscono al compilatore di accettarle negli argomenti di attributo.</span><span class="sxs-lookup"><span data-stu-id="1787c-108">Other conversions, such as an array of a derived type to an array of `Object`, present a variety of problems that do not permit the compiler to allow them on attribute arguments.</span></span>
