# <a name="39option-strict39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Opzione Strict&#39; può essere seguita solo da &#39;in&#39; o &#39;disattivato&#39;
Le istruzioni`Option Strict` devono specificare `On` o `Off`. Tutti gli altri valori non sono validi.  
  
 **ID errore:** BC30620  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare `On` o `Off` per l'istruzione `Option Strict` .
