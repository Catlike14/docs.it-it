# <a name="argument-39access39-is-not-valid-valid-values-for-output-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argomento &#39;accesso&#39; non è valido. I valori validi per la modalità Output sono &#39;OpenAccess. Write&#39; e &#39;OpenAccess. default&#39;
Un valore access non è valido in modalità Output.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Modificare il valore in `OpenAccess.Write` o `OpenAccess.Default`.
