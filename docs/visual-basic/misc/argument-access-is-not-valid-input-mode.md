# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Argomento &#39;accesso&#39; non è valido. I valori validi per la modalità Input sono &#39;Read&#39; e &#39;OpenAccess. default&#39;
Un valore access non è valido in modalità `Input` .  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il valore in `OpenAccess.Read` o `OpenAccess.Default`.
