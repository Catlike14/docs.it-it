# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Opzione Infer&#39; può essere seguita solo da &#39;in&#39; o &#39;disattivato&#39;
Le istruzioni`Option Infer` devono specificare `On` o `Off`. Gli altri valori non sono validi.  
  
 **ID errore:** BC30979  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare `Option Infer On` o `Option Infer Off`.
