# <a name="39join39-expected"></a>&#39; Join &#39; previsto
Una clausola `Group Join` Ã¨ stata specificata senza una parola chiave `Join` .  
  
 **ID errore:** BC36631  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Aggiungere la parola chiave `Join` alla clausola `Group Join` , come illustrato nell'esempio seguente:  
  
```vb  
Dim query = From var1 In collection1 _  
            Join var2 In collection2 _  
            Group Join var3 In collection3 _  
            On var2.ID Equals var3 Into Matches = Group _  
            On var1 Equals var2.ID _  
            Select JoinID = var1, var2.Name, Matches  
```
