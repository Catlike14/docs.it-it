# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>L'accesso alla proprietà deve assegnare un valore alla proprietà o usare quello corrente
Si è provato ad accedere a una proprietà senza assegnarle un valore o usare quello corrente.
  
 **ID errore:** BC30545  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Assegnare un valore alla proprietà.  
  
     \- oppure -  
  
-   Utilizzare il valore della proprietà.
