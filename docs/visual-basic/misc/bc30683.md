# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39; eredita &#39; l'istruzione deve precedere tutte le dichiarazioni in una classe
Le istruzioni `Inherits` devono trovarsi all'inizio delle definizioni di classe.  
  
 **ID errore:** BC30683  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare l'istruzione `Inherits` all'inizio della classe.
