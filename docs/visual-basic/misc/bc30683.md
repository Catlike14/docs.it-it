# L&#39;istruzione &#39;Inherits&#39; deve precedere tutte le dichiarazioni in una classe
Le istruzioni `Inherits` devono trovarsi all'inizio delle definizioni di classe.  
  
 **ID errore:** BC30683  
  
### Per correggere l'errore  
  
-   Spostare l'istruzione `Inherits` all'inizio della classe.
