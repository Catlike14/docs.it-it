# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parametro &#39; &lt;parametername&gt; &#39; nel metodo di estensione &#39; &lt;NomeMetodo&gt; &#39; definita &#39; &lt;typename&gt; &#39; già ha un argomento corrispondente
Un argomento viene passato in base al nome più di una volta nella stessa chiamata di routine.  
  
 **ID errore:** BC36584  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare se uno dei nomi degli argomenti sia stato digitato in modo errato.  
  
-   Rimuovere l'argomento duplicato dall'elenco di argomenti.
