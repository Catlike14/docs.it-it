# &#39;&lt;modificatore&gt;&#39; non &#232; valido in una dichiarazione di costante locale
Un'istruzione `Const` all'interno di una routine contiene una parola chiave non valida, ad esempio `Overloads`.  
  
 **ID errore:** BC30246  
  
### Per correggere l'errore  
  
1.  Rimuovere la parola chiave non valida dall'istruzione `Const`.
