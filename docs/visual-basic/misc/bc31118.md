# <a name="39end-addhandler39-must-be-the-first-statement-on-a-line"></a>&#39;End AddHandler&#39; deve essere la prima istruzione in una riga
Un'istruzione `End AddHandler` segue un separatore di istruzioni due punti (:). `End AddHandler` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC31118
