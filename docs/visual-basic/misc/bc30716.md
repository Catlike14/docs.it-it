# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Infine&#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Finally` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30716  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Finally` dal codice di debug.
