# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'Finally' non valide nella finestra controllo immediato
`Finally`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30716  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Finally` dal codice di debug.
