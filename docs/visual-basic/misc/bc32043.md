# <a name="too-many-type-arguments-to-39ltgenericmethodnamegt39"></a>Troppi argomenti di tipo per &#39; &lt;nomemetodogenerico è&gt;&#39;
È stato richiamato un metodo generico con più argomenti di tipo rispetto ai parametri di tipo.  
  
 Quando si richiama un metodo generico, è necessario fornire un argomento di tipo per ogni parametro di tipo definito dal metodo.  
  
 **ID errore:** BC32043  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere gli argomenti di tipo dall'elenco di argomenti di tipo in modo che ne esista uno per ogni parametro di tipo nel metodo generico richiamato.
