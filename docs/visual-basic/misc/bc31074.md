# Impossibile trovare il costruttore di delegato per il tipo &#39;&lt;nometipo&gt;&#39;, necessario per l&#39;implementazione dei delegati
L'assembly che contiene il tipo potrebbe essere danneggiato o non valido.  
  
 **ID errore:** BC31074  
  
### Per correggere l'errore  
  
1.  Verificare che l'assembly contenente il tipo sia valido.  
  
2.  Ripetere la compilazione del programma, per controllare se l'errore si verifica di nuovo.  
  
3.  Se l'errore si ripresenta, salvare il lavoro e riavviare [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].
