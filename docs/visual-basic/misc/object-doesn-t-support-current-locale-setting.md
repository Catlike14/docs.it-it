# <a name="object-doesn39t-support-current-locale-setting-visual-basic"></a>Oggetto &#39; t supporto corrente internazionali (Visual Basic)
Si è provato ad accedere a un oggetto che non supporta le impostazioni locali per il progetto corrente.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare quali siano le impostazioni locali supportate dall'oggetto.
