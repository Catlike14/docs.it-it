# <a name="date-constant-is-not-valid"></a>La costante di data non è valida
Una costante di data è dichiarata con una data non valida.  
  
 **ID errore:** BC31085  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Correggere la data usata nella costante.
