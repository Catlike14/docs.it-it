# <a name="structure-39ltstructurenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Struttura &#39; &lt;nomestruttura&gt;&#39; non può essere indicizzata perché non contiene proprietà predefinite
Si è provato a usare una sintassi di proprietà predefinita con una struttura che non contiene alcuna proprietà predefinita.  
  
 **ID errore:** BC30690  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare la sintassi di proprietà standard per accedere alle proprietà della struttura.
