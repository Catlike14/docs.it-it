# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39; Condiviso &#39; può essere combinato con &#39; &lt;identificatore&gt;&#39; in una dichiarazione di proprietà
Si è provato a combinare il modificatore `Shared` con un identificatore che non è valido in tale combinazione in una dichiarazione di proprietà.  
  
 **ID errore:** BC30502  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'identificatore.
