# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion"></a>Di overload non accessibile &#39; &lt;NomeMetodo&gt; &#39; può essere chiamato con questi argomenti senza una conversione di narrowing
È stato chiamato un metodo di overload, ma non è possibile associare un metodo all'elenco di argomenti forniti senza una conversione verso un tipo di dati più piccolo.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Specificare `Option Strict Off`.  
  
2.  Modificare gli argomenti in modo che corrispondano a una delle firme del metodo di overload.
