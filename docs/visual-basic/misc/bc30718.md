# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Selezionare&#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Select` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30718  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Select` dal codice di debug.
