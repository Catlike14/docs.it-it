# <a name="enum-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Enumeratore in un'interfaccia non può essere dichiarato &#39; &lt;identificatore&gt;&#39;
Il modificatore specificato con un'enumerazione dichiarata in un'interfaccia non è valido.  
  
 **ID errore:** BC31069  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere i modificatori come `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`o `Overridable`.
