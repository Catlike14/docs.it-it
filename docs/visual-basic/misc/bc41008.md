# <a name="use-command-line-option-39ltoptiongt39-or-appropriate-project-settings-instead-of-39ltparametergt39"></a>Utilizzare l'opzione della riga di comando '&lt;opzione&gt;'o le impostazioni di progetto anziché appropriate'&lt;parametro&gt;'
La modalità preferita per specificare un file che contiene una chiave pubblica per un assembly, un contenitore di chiavi pubbliche per un assembly o un assembly con firma parziale è utilizzare il [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] opzioni del compilatore. Si sconsiglia di utilizzare il <xref:System.Reflection.AssemblyKeyFileAttribute>, <xref:System.Reflection.AssemblyKeyNameAttribute>, o <xref:System.Reflection.AssemblyDelaySignAttribute>gli attributi nel codice.</xref:System.Reflection.AssemblyDelaySignAttribute> </xref:System.Reflection.AssemblyKeyNameAttribute> </xref:System.Reflection.AssemblyKeyFileAttribute>  
  
 **ID errore:** BC41008  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Utilizzare il [/keyfile](../../visual-basic/reference/command-line-compiler/keyfile.md), [/keycontainer](../../visual-basic/reference/command-line-compiler/keycontainer.md), o [/delaysign](../../visual-basic/reference/command-line-compiler/delaysign.md) [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] opzioni del compilatore anziché il <xref:System.Reflection.AssemblyKeyFileAttribute>, <xref:System.Reflection.AssemblyKeyNameAttribute>, o <xref:System.Reflection.AssemblyDelaySignAttribute>gli attributi nel codice.</xref:System.Reflection.AssemblyDelaySignAttribute> </xref:System.Reflection.AssemblyKeyNameAttribute> </xref:System.Reflection.AssemblyKeyFileAttribute>
