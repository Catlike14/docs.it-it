# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Genera &#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Throw` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30725  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Throw` dal codice di debug.
