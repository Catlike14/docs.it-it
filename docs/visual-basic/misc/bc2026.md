# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Numero di avviso '&lt;numero&gt;'per l'opzione'&lt;nomeopzione&gt;' non è configurabile o non valido
Un ID di avviso non è valido. Gli ID devono essere numeri interi maggiori di 0.  
  
 **ID errore:** BC2026  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare l'ID di avviso specificato per assicurarsi che sia valido.
