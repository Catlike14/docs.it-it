# <a name="39get39-statements-are-no-longer-supported"></a>Le istruzioni 'Get' non sono più supportate
`Get`le istruzioni non sono più supportate. La funzionalità di I/O dei file è disponibile nello spazio dei nomi `Microsoft.VisualBasic` .  
  
 `Get`non è supportata per le operazioni di file e può essere utilizzato solo l'istruzione.  
  
 **ID errore:** BC30829  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Operazioni sui file utilizzando i membri di `System.IO`, `FileSystemObject`, e [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] funzioni di runtime.
