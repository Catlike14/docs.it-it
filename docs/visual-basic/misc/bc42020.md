# <a name="variable-declaration-without-an-39as39-clause-type-of-object-assumed"></a>Dichiarazione di variabile senza un &#39; Come &#39; clausola. tipo di oggetto presuppone
Una dichiarazione di variabile non specifica una clausola `As` .  
  
 Una clausola `As` identifica un tipo di dati da associare a un elemento di programmazione. In un [Dim (istruzione)](../../visual-basic/language-reference/statements/dim-statement.md), specifica il tipo di dati della variabile o variabili. Se non si include una clausola `As` nell'istruzione `Dim` , il tipo di dati della variabile sar√† `Object`.  
  
 Per impostazione predefinita, si tratta di un messaggio di avviso. Per informazioni su come nascondere gli avvisi o considerarli come errori, vedere [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID errore:** BC42020  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Includere una clausola `As` nell'istruzione `Dim` per specificare il tipo di dati della variabile.
