# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39;AddHandler&#39; la dichiarazione deve terminare con un corrispondente &#39;End AddHandler&#39;
Una dichiarazione `AddHandler` deve terminare con un'istruzione `End AddHandler` .  
  
 **ID errore:** BC31115  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare che la dichiarazione `AddHandler` termini con un'istruzione `End AddHandler` .
