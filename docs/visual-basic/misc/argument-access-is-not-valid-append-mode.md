# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argomento &#39; Accedi al &#39; non è valido. Valori validi per l'accodamento vengono modalità &#39;OpenAccess.Write &#39; e &#39;OpenAccess.Default &#39;
Un valore access non è valido in modalità Append.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Modificare il valore in `OpenAccess.Write` o `OpenAccess.Default`.
