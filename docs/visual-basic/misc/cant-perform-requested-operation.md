# <a name="can39t-perform-requested-operation-visual-basic"></a>Può &#39; t eseguire l'operazione richiesta (Visual Basic)
Non è possibile eseguire un'operazione perché potrebbe invalidare lo stato corrente del progetto. Ad esempio, è possibile che si sia tentato di modificare codice che è in esecuzione.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Arrestare l'esecuzione del codice, apportare le modifiche necessarie e quindi ripetere l'operazione.
