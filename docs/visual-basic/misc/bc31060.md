# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39;&lt;PropertyName&gt; &#39; definisce in modo implicito &#39; &lt;membername&gt;&#39;, che è in conflitto con un membro con lo stesso nome in &lt;tipo&gt; &#39; &lt; TypeName&gt;&#39;
Il nome di un membro di tipo è in conflitto con il nome di un membro creato in modo implicito per una proprietà. Le proprietà creano in modo implicito diverse variabili implicite che possono essere in conflitto con i membri definiti.  
  
 **ID errore:** BC31060  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rinominare il membro dichiarato in modo esplicito per rimuovere il conflitto di denominazione.
