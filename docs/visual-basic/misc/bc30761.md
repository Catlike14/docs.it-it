# <a name="project-39ltprojectnamegt39-cannot-reference-project-39ltprojectnamegt39-because-39ltprojectnamegt39-directly-or-indirectly-references-39ltprojectnamegt39"></a>Progetto &#39; &lt;projectname&gt; &#39; non può fare riferimento a progetto &#39; &lt;projectname&gt; &#39; perché &#39; &lt;projectname&gt; &#39; fa riferimento direttamente o indirettamente a &#39; &lt;projectname&gt;&#39;
Un progetto include un riferimento a un secondo progetto. Il secondo progetto include a sua volta un riferimento al progetto contenente un riferimento a esso.  
  
 **ID errore:** BC30761  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere uno dei riferimenti.
