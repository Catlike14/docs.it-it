# <a name="argument-39path39-is-nothing-or-empty"></a>Argomento &#39;percorso&#39; è Nothing o vuoto
Il nome di un percorso non è valido perché viene inizializzato su `Nothing` o è vuoto. Un percorso deve avere un tipo di dati specifico e un nome valido.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere un nome di percorso valido.
