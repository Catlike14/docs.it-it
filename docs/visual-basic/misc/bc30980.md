 <span data-ttu-id="42df4-104">Ad esempio, con `Option Infer` impostato su `On`, negli esempi seguenti non vengono compilati:</span><span class="sxs-lookup"><span data-stu-id="42df4-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
-   <span data-ttu-id="42df4-105">Dichiarazioni</span><span class="sxs-lookup"><span data-stu-id="42df4-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="42df4-106">Ciclo`For` </span><span class="sxs-lookup"><span data-stu-id="42df4-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="42df4-107">Ciclo`For Each` </span><span class="sxs-lookup"><span data-stu-id="42df4-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="42df4-108">**ID errore:** BC30980</span><span class="sxs-lookup"><span data-stu-id="42df4-108">**Error ID:** BC30980</span></span>
