# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39; option Explicit &#39; può essere seguita solo da &#39; su &#39; o &#39; Off &#39;
Le istruzioni`Option Explicit` devono specificare `On` o `Off`. Tutti gli altri valori non sono validi.  
  
 **ID errore:** BC30640  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare `Option Explicit On` o `Option Explicit Off`.
