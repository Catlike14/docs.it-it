# <a name="39ltelementnamegt39-is-not-a-parameter-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>&#39; &lt;elementname&gt;&#39; non è un parametro di metodo di estensione &#39;&lt; NomeMetodo&gt;&#39; definito in &#39;&lt; TypeName&gt;&#39;
Una chiamata di routine fornisce un argomento aggiuntivo non definito dal metodo di estensione.  
  
 **ID errore:** BC36585  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'argomento superfluo dall'elenco di argomenti.
