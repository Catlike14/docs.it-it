# <a name="operator-39operator39-is-not-defined-for-types-39type139-and-39type239"></a>Operatore operator &#39; &#39; non è definito per i tipi di &#39; type1 &#39; e &#39; type2 &#39;
L'operatore 'operator' non è definito per i tipi '<tipo1>' e '<tipo2>'. È possibile utilizzare la proprietà 'Value' per ottenere il valore di stringa del primo elemento di '\<ElementoPadre >'.  
  
 È stato usato un valore letterale XML in un file binario o in un confronto `Boolean` .  
  
 **ID errore:** BC31195  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare la proprietà `Value` del valore letterale XML per il file binario o il confronto `Boolean` .
