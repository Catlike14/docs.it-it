# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>I/o file con tipo &#39; &lt;typename&gt;&#39; non è valido
Si sta tentando di scrivere un tipo non supportato in un file.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare la struttura per determinare dove si verifica il tipo non supportato e quindi modificarlo in un tipo valido.
