# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Errore &#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Error` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30724  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Error` dal codice di debug.
