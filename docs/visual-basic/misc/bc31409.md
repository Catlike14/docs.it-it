# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;metodo&gt; &#39; &lt;methodname1&gt; &#39; deve essere dichiarato come &#39;esegue l'overload&#39; perché un altro &#39; &lt;methodname2&gt; &#39; è dichiarato &#39; Esegue l'overload&#39;
Un'altra proprietà o un altro metodo con lo stesso nome è dichiarato con il modificatore `Overloads` .  
  
 **ID errore:** BC31409  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Aggiungere il modificatore `Overloads` alla proprietà o al metodo.  
  
2.  Rimuovere il modificatore `Overloads` da tutte le proprietà o da tutti i metodi con lo stesso nome.
