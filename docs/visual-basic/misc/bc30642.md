# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39; facoltativo &#39; e &#39; ParamArray &#39; non possono essere combinati
I modificatori `Optional` e `ParamArray` si escludono reciprocamente.  
  
 **ID errore:** BC30642  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare `ParamArray` o `Optional`, ma non entrambi.
