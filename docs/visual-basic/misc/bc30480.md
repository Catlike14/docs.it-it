# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>Condiviso &#39; Sub nuovo &#39; non può essere dichiarato &#39; &lt;identificatore&gt;&#39;
Si è provato a usare un identificatore in una dichiarazione `Shared Sub New` . Questa operazione non è valida.  
  
 **ID errore:** BC30480  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'identificatore.
