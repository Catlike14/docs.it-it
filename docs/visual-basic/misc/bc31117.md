# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39; RaiseEvent &#39; la dichiarazione deve terminare con un corrispondente &#39; End RaiseEvent &#39;
Una dichiarazione `RaiseEvent` deve terminare con un'istruzione `End RaiseEvent` .  
  
 **ID errore:** BC31117  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare che la dichiarazione `RaiseEvent` termini con un'istruzione `End RaiseEvent` .
