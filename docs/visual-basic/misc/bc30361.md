# <a name="39ltproperty1gt39-and-39ltproperty2gt39-cannot-overload-each-other-because-only-one-is-declared-39default39"></a>&#39; &lt;property1&gt;&#39; e &#39;&lt; Property2&gt;&#39; non è in rapporto di overload perché solo uno è dichiarato come &#39; predefinito &#39;
Se una proprietà specifica `Default`, tutte le proprietà di overload nel nome devono specificare anche `Default`.  
  
 **ID errore:** BC30361  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che tutte le proprietà di overload siano dichiarate come `Default`.
