# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-does-not-inherit-from-39systemattribute39"></a>&#39; &lt;typename&gt;&#39; non può essere utilizzato come attributo perché non eredita da &#39; System. Attribute &#39;
Si è provato a usare una classe che non è derivata da `System.Attribute`.  
  
 **ID errore:** BC31504  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Definire gli attributi personalizzati come classi che derivano da `System.Attribute` aggiungendo un'istruzione `Imports` per la prima riga di codice nella classe.
