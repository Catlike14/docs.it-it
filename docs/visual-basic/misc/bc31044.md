# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>Gli eventi dichiarati con un &#39;come&#39; clausola deve avere un tipo delegato
È stato specificato un tipo che non è un delegato.  
  
 **ID errore:** BC31044  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere la clausola `As` .  
  
2.  Specificare un delegato con la clausola `As` .
