# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'Case' non sono valide nella finestra controllo immediato
`Case`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30719  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Case` dalla finestra di controllo **immediato** .
