# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Case&#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Case` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30719  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Case` dalla finestra di controllo **immediato** .
