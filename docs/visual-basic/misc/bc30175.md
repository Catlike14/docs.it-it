# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-declared-in-39ltnamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;type1&gt; &#39;&lt; TypeName&gt;&#39; e &lt;type2&gt; &#39;&lt; TypeName&gt;&#39; dichiarati nel &#39;&lt; nome&gt;&#39; sono in conflitto &lt;dello spazio dei nomi&gt; &#39;&lt; NomeSpazioDeiNomi&gt;&#39;
Due elementi dichiarati, ad esempio una classe e un modulo, hanno lo stesso nome nello stesso spazio dei nomi.  
  
 **ID errore:** BC30175  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare la dichiarazione di uno degli elementi in uno spazio dei nomi diverso.  
  
-   Modificare il nome di uno degli elementi.
