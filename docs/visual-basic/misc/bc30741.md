# <a name="39lttypename1gt39-cannot-be-converted-to-39lttypename2gt39"></a>&#39; &lt;NomeTipo1&gt;&#39; non può essere convertito in &#39;&lt; NomeTipo2&gt;&#39;
È stata tentata una conversione del tipo tra due tipi incompatibili.  
  
 **ID errore:** BC30741  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare l'esecuzione del cast dei tipi.
