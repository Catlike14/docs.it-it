# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Campo &#39; &lt;fieldname&gt; &#39; è di tipo non supportato
Un campo viene dichiarato come un tipo non può essere rappresentato da Visual Studio.  
  
 **ID errore:** BC30656  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Definire il campo come un tipo supportato.  
  
2.  Sviluppare un nuovo tipo che descrive i dati nel campo.
