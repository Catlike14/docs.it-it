# Non &#232; possibile usare il carattere tipo in una dichiarazione di parametro di tipo
Una dichiarazione di parametro di tipo contiene almeno un carattere di tipo identificatore.  
  
 Un parametro di tipo di un tipo generico deve essere un nome di Visual Basic valido. I caratteri consentiti non includono i caratteri di tipo identificatore, ovvero `%`, `&`, `@`, `!`, `#` e `$`. Vedere [Declared Element Names](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID errore:** BC32041  
  
### Per correggere l'errore  
  
-   Rimuovere il carattere o i caratteri di tipo identificatore dalla dichiarazione di parametro di tipo.
