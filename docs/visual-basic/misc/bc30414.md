# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>Valore di tipo &#39; &lt;type1&gt;&#39; non può essere convertito in &#39;&lt; type2&gt;&#39; perché i tipi matrice hanno numeri di dimensioni diversi
Si è provato a convertire una matrice in un'altra matrice con un numero di dimensioni diverso.  
  
 **ID errore:** BC30414  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare una o entrambe le matrici in modo che il numero di dimensioni corrisponda.
