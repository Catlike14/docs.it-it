# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;Proprietà&gt; parametri non possono essere dichiarati &#39; ByRef &#39;
I parametri per le proprietà devono essere dichiarati usando il modificatore `ByVal` .  
  
 **ID errore:** BC30651  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Definire i parametri di proprietà come `ByVal`.
