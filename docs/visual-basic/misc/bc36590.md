# <a name="too-few-type-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Numero insufficiente di argomenti di tipo al metodo di estensione &#39; &lt;NomeMetodo&gt;&#39; definito in &#39;&lt; TypeName&gt;&#39;
È stato richiamato un metodo di estensione generico con meno argomenti di tipo rispetto ai parametri di tipo nella definizione del metodo.  
  
 Quando si richiama un metodo generico, è necessario fornire un argomento di tipo per ogni parametro di tipo definito dal metodo.  
  
 **ID errore:** BC36590  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere gli argomenti di tipo all'elenco di argomenti di tipo in modo che ne esista uno per ogni parametro di tipo del metodo di estensione che si sta chiamando.
