# <a name="39end-raiseevent39-must-be-the-first-statement-on-a-line"></a>&#39; End RaiseEvent &#39; deve essere la prima istruzione in una riga
Un'istruzione `End RaiseEvent` segue un separatore di istruzioni due punti (:). `End RaiseEvent` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC31120
