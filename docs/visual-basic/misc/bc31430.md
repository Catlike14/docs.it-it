# <a name="expression-of-type-39lttypename1gt39-can-never-be-of-type-39lttypename2gt39"></a><span data-ttu-id="97915-102">Espressione di tipo &#39; &lt;NomeTipo1&gt;&#39; non può mai essere di tipo &#39;&lt; NomeTipo2&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="97915-102">Expression of type &#39;&lt;typename1&gt;&#39; can never be of type &#39;&lt;typename2&gt;&#39;</span></span>
<span data-ttu-id="97915-103">Un'espressione `TypeOf`...`Is` consente di eseguire il test di una variabile di riferimento a un oggetto in un tipo di dati che non può contenere.</span><span class="sxs-lookup"><span data-stu-id="97915-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="97915-104">In alcuni casi il compilatore può determinare che un test dell'espressione `TypeOf`...`Is` possa avere solo esito negativo, ad esempio se non esiste alcuna relazione di ereditarietà tra le due classi.</span><span class="sxs-lookup"><span data-stu-id="97915-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="97915-105">Il codice seguente può generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="97915-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="97915-106">Poiché <xref:System.Windows.Forms.Form> e <xref:System.Array> sono tipi completamente non correlati, il compilatore può determinare che l'espressione `TypeOf`...`Is` restituisca `False` per qualsiasi valore di `refVar`.</span><span class="sxs-lookup"><span data-stu-id="97915-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="97915-107">**ID errore:** BC31430</span><span class="sxs-lookup"><span data-stu-id="97915-107">**Error ID:** BC31430</span></span>
