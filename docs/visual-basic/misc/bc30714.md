# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Riprendi&#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni Resume possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30714  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Resume` dal codice di debug.
