# <a name="properties-cannot-be-declared-39ltmodifiernamegt39"></a>Proprietà non possono essere dichiarate &#39; &lt;nomemodificatore&gt;&#39;
La sintassi della dichiarazione `Property` contiene un errore. Questo errore in genere si verifica quando si è specificata una parola chiave che non può essere usata in una dichiarazione `WithEvents`, ad esempio `Property` .  
  
 **ID errore:** BC30639  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare la sintassi della dichiarazione `Property` .
