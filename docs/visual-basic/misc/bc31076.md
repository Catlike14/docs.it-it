# <a name="39ltmodulenamegt39-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>&#39; &lt;modulename&gt;&#39; è un modulo e non può fare riferimento come assembly
Si è provato a usare un modulo come assembly.  
  
 **ID errore:** BC31076  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il riferimento a un assembly.
