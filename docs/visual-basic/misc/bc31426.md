# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>I nomi di tipo non intrinseco non sono consentiti nelle espressioni di compilazione condizionali
L'espressione di compilazione condizionale fa riferimento a un tipo non intrinseco, ma sono consentiti solo i tipi intrinseci.  
  
 Per altre informazioni sul tipo intrinseco in [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)], vedere [Data Types Summary](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **ID errore:** BC31426  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Sostituire il tipo non intrinseco con un tipo intrinseco.
