# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>Alias di Imports &#39; &lt;nome&gt; &#39; è in conflitto con &#39; &lt;name2&gt; &#39; dichiarato nello spazio dei nomi radice
Il nome dell'alias usato nell'istruzione `Imports` è in conflitto con un altro nome dichiarato nello spazio dei nomi radice.  
  
 **ID errore:** BC31403  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Scegliere un nome univoco per l'alias usato dall'istruzione `Imports` .
