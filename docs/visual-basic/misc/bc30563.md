# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>'&lt;nome&gt;'è ambiguo negli oggetti applicazione'&lt;elenco&gt;'
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilatore non dispone di regole di risoluzione dei conflitti, è necessario evitare ambiguità nei nomi manualmente. L'elenco fornito nei dettagli del messaggio di errore indica gli oggetti dell'applicazione in cui il nome è ambiguo.  
  
 **ID errore:** BC30563  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un nome completo.
