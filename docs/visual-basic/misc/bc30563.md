# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39;&lt;nome&gt; &#39; è ambiguo negli oggetti dell'applicazione &#39; &lt;elenco&gt;&#39;
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il compilatore Visual Basic non dispone di regole di risoluzione dei conflitti. è necessario risolvere le ambiguità nei nomi manualmente. L'elenco fornito nei dettagli del messaggio di errore indica gli oggetti dell'applicazione in cui il nome è ambiguo.  
  
 **ID errore:** BC30563  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un nome completo.
