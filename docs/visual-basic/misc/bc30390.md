# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39;&lt;metodoqualificato&gt; &#39; non è accessibile in questo contesto perché è &#39; &lt;modificatore&gt;&#39;
Si è provato ad accedere a un metodo che non è accessibile in questo contesto a causa di un modificatore che è stato applicato a tale metodo.  
  
 **ID errore:** BC30390  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il modificatore.
