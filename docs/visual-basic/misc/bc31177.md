# <a name="white-space-may-not-appear-here"></a>Non è possibile usare spazi vuoti
Nel nome di un valore letterale elemento XML sono inclusi uno o più spazi.  
  
 **ID errore:** BC31177  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere lo spazio o gli spazi dal nome del valore letterale elemento XML.
