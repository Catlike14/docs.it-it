# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni Loop non sono valide nella finestra di controllo immediato
Le istruzioni`Loop` non possono essere usate nella finestra di controllo **immediato** .  
  
 **ID errore:** BC30708  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Loop` dal codice di debug.
