# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>Il tag del commento XML &#39; &lt;tag&gt; &#39; non è consentito in un &#39; &lt;elemento&gt; &#39; elemento di linguaggio
Tag di commento XML \<tag > non è consentito in un '\<elemento >' elemento. Il commento XML verrà ignorato.  
  
 Un tag di commento XML non è supportato dall'elemento di linguaggio al quale viene applicato.  
  
 **ID errore:** BC42306  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il tag di commento XML non supportato.
