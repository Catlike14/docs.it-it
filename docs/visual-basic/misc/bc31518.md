# <a name="assembly-39ltassemblynamegt39-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Assembly &#39; &lt;assemblyname&gt; &#39; non può essere creata perché il percorso è più lungo di 259 caratteri
Il percorso per questo assembly supera la lunghezza massima di 259 caratteri.  
  
 **ID errore:** BC31518  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Abbreviare il percorso usando nomi più brevi o selezionando una gerarchia di file ridotta.
