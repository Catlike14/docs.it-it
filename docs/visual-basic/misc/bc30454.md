# L&#39;espressione non &#232; un metodo
La destinazione della chiamata di metodo non è un metodo.  
  
 **ID errore:** BC30454  
  
### Per correggere l'errore  
  
-   Verificare che il nome del metodo sia stato scritto correttamente.
