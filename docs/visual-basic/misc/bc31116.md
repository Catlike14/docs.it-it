# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39; RemoveHandler &#39; la dichiarazione deve terminare con un corrispondente &#39; End RemoveHandler &#39;
Una dichiarazione `RemoveHandler` deve terminare con un'istruzione `End RemoveHandler` .  
  
 **ID errore:** BC31116  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare che la dichiarazione `RemoveHandler` termini con un'istruzione `End RemoveHandler` .
