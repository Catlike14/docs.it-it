<span data-ttu-id="ef77e-103">Un nome di proprietà può essere definito una sola volta nella dichiarazione di un tipo anonimo.</span><span class="sxs-lookup"><span data-stu-id="ef77e-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="ef77e-104">Ad esempio, le dichiarazioni seguenti non sono valide:</span><span class="sxs-lookup"><span data-stu-id="ef77e-104">For example, the following declarations are not valid:</span></span>

'' Not valid, because the property name inferred for both properties is
