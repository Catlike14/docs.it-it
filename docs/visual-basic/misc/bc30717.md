# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'Try' non sono valide nella finestra controllo immediato
`Try`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30717  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Try` dal codice di debug.
