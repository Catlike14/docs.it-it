# <a name="39ltnamegt39-cannot-expose-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-to-ltspecifier1gt-39lttype1gt39-through-ltspecifier2gt-39lttype2gt39"></a>&#39; &lt;nome&gt;&#39; non può esporre il tipo delegato sottostante &#39;&lt; tipodelegato&gt;&#39; dell'evento che sta implementando in &lt;identificatore1&gt; &#39;&lt; type1&gt;&#39; tramite &lt;identificatore2&gt; &#39;&lt; type2&gt;&#39;
