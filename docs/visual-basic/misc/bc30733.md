# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&#39;&lt;MemberName&gt; &#39; è già dichiarato da &#39; &lt;nomeroutine&gt;&#39;, cui è stato generato per questo &lt;containername&gt;
È stato dichiarato un membro che ha lo stesso nome di un membro dichiarato da Common Language Runtime.  
  
 **ID errore:** BC30733  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il nome del membro per rimuovere il conflitto.
