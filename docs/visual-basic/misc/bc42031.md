# <a name="39catch39-block-never-reached-ltexceptiongt-handled-above-in-the-same-39try39-statement"></a>Blocco 'Catch' mai raggiunto. &lt;eccezione&gt; gestito in precedenza nella stessa istruzione "Try"
Un blocco `Catch` nel codice non è raggiungibile perché è gestito in un blocco `Try` precedente.  
  
 Per impostazione predefinita, si tratta di un messaggio di avviso. Per ulteriori informazioni su come nascondere gli avvisi o considerarli come errori, vedere [configurazione degli avvisi in Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic)  
  
 **ID errore:** BC42031  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione ridondante.
