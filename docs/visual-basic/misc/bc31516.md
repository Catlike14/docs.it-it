# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>Tipo &#39; &lt;typename&gt;&#39; non può essere utilizzato come attributo perché non è dichiarato &#39; Pubblica &#39;
Gli attributi devono essere pubblici.  
  
 **ID errore:** BC31516  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Aggiungere il modificatore `Public` alla classe Attribute.
