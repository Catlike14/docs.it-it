# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-with-these-argumentslterrorgt"></a>Non è riuscita perché la risoluzione dell'overload non accessibile &#39; &lt;metodo&gt;&#39; può essere chiamato con questi argomenti:&lt;errore&gt;
È stata effettuata una chiamata a un metodo di overload, ma il compilatore non riesce a trovare un metodo con un elenco di argomenti corrispondenti.  
  
 **ID errore:** BC30518  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare l'elenco di argomenti fornito.
