# <a name="39end-sub39-must-be-the-first-statement-on-a-line"></a>&#39; End Sub &#39; deve essere la prima istruzione in una riga
Un'istruzione `End Sub` segue un separatore di istruzioni due punti (:). `End Sub` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC32031
