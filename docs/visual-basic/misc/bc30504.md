# <a name="property-39ltpropertynamegt39-cannot-be-declared-39default39-because-it-overrides-a-property-on-the-base-lttypegt-39lttypenamegt39-that-is-not-default"></a>Proprietà &#39; &lt;propertyname&gt; &#39; non possono essere dichiarati come &#39;predefinito&#39; perché esegue l'override di una proprietà sulla base &lt;tipo&gt; &#39; &lt;typename&gt; &#39; che non è predefinita
Si è provato a dichiarare una proprietà `Default`, che non è una dichiarazione valida perché esegue l'override di una proprietà che non è stata dichiarata `Default` nella classe base.  
  
 **ID errore:** BC30504  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la dichiarazione `Default` .
