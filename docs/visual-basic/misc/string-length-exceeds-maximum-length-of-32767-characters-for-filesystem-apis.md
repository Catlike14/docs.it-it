# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-39filesystem39-apis"></a>Lunghezza della stringa supera la lunghezza massima di 32767 caratteri per &#39;FileSystem&#39; API
La lunghezza di una stringa supera la lunghezza massima di 32767 caratteri.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Abbreviare la stringa.
