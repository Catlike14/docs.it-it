# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Uscita &#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Exit` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30713  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Exit` dal codice di debug.
