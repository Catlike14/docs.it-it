# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'Exit' non sono valide nella finestra controllo immediato
`Exit`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30713  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Exit` dal codice di debug.
