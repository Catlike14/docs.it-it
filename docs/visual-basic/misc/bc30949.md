# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-3939"></a><span data-ttu-id="b521d-102">Inizializzatore di matrice non può essere specificato per una dimensione non costante; usare l'inizializzatore vuoto&#39;{}&#39;</span><span class="sxs-lookup"><span data-stu-id="b521d-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer &#39;{}&#39;</span></span>
<span data-ttu-id="b521d-103">Una matrice inizializza una dimensione che non è nota in fase di compilazione.</span><span class="sxs-lookup"><span data-stu-id="b521d-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="b521d-104">Il codice seguente genera questo errore.</span><span class="sxs-lookup"><span data-stu-id="b521d-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="b521d-105">Il codice seguente consente di evitare l'errore.</span><span class="sxs-lookup"><span data-stu-id="b521d-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="b521d-106">**ID errore:** BC30949</span><span class="sxs-lookup"><span data-stu-id="b521d-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b521d-107">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="b521d-107">To correct this error</span></span>  
  
-   <span data-ttu-id="b521d-108">Se possibile, specificare una dimensione costante nella dichiarazione di matrice.</span><span class="sxs-lookup"><span data-stu-id="b521d-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
-   <span data-ttu-id="b521d-109">Se non è possibile specificare una dimensione costante, è necessario inizializzare la matrice con un ciclo quando la dimensione non costante diventa nota.</span><span class="sxs-lookup"><span data-stu-id="b521d-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
