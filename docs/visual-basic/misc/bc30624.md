# <a name="39structure39-statement-must-end-with-a-matching-39end-structure39"></a>&#39; Struttura &#39; l'istruzione deve terminare con un corrispondente &#39; Struttura di fine &#39;
Le strutture iniziano con un'istruzione `Structure` e terminano con un costrutto `End Structure` .  
  
 **ID errore:** BC30624  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il blocco di struttura termini con un'istruzione `End Structure` .
