# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39; &lt;nomeinterfaccia1&gt;&#39; non è possibile implementare &#39;&lt; NomeMetodo&gt;&#39; perché non esiste alcuna corrispondenza &lt;metodo&gt; interfaccia &#39;&lt; nomeinterfaccia2&gt;&#39;
Si è provato a implementare un metodo in un'interfaccia, ma non esiste un metodo corrispondente nell'interfaccia da cui deriva la prima interfaccia.  
  
 **ID errore:** BC30401  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il nome del metodo sia stato digitato in modo corretto oppure fornire il metodo mancante.
