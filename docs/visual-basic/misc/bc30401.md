# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39;&lt;interfacename1&gt; &#39; non può implementare &#39; &lt;NomeMetodo&gt; &#39; perché non corrisponde ad alcuna &lt;metodo&gt; sull'interfaccia &#39; &lt; interfacename2&gt;&#39;
Si è provato a implementare un metodo in un'interfaccia, ma non esiste un metodo corrispondente nell'interfaccia da cui deriva la prima interfaccia.  
  
 **ID errore:** BC30401  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il nome del metodo sia stato digitato in modo corretto oppure fornire il metodo mancante.
