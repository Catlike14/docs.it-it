# <a name="paramarray-parameters-must-be-declared-39byval39"></a>I parametri ParamArray devono essere dichiarati &#39; ByVal &#39;
I parametri`ParamArray` non possono usare il modificatore `ByRef` .  
  
 **ID errore:** BC30667  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare i parametri `ParamArray` con il modificatore `ByVal` .
