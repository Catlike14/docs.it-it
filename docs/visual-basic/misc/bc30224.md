# Previsto &#39;Is&#39;
Un operatore `TypeOf` si verifica senza una clausola `Is`.  
  
 **ID errore:** BC30224  
  
### Per correggere l'errore  
  
-   Aggiungere una clausola `Is` dopo l'operatore `TypeOf`, ad esempio, `TypeOf MyControl Is ComboBox`.
