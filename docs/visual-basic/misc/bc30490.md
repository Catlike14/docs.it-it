# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>&#39;Predefinito&#39; non può essere combinato con &#39; &lt;identificatore&gt;&#39;
Si è provato a combinare la parola chiave `Default` con un identificatore non valido in questo contesto, ad esempio `Shared` o `Private`.  
  
 **ID errore:** BC30490  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere `Default`.  
  
     oppure  
  
-   Rimuovere l'identificatore in conflitto.
