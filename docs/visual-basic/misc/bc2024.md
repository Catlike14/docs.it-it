# <a name="the-default-assembly-reference-39ltreferencegt39-has-already-been-automatically-added-ignored"></a>Il riferimento all'assembly predefinito &#39; &lt;riferimento&gt; &#39; è già stato automaticamente aggiunto ignorate;
Si è provato ad aggiungere il riferimento all'assembly predefinito, ma l'assembly è già stato aggiunto, pertanto il tentativo viene ignorato.  
  
 **ID errore:** BC2024  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il codice superfluo.
