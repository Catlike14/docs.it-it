# <a name="file-ltfilenamegt-could-not-be-found"></a>File &lt;filename&gt; non è stato trovato
Non è stato possibile trovare il file specificato. È possibile che sia stato specificato un percorso del file errato.  
  
 **ID errore:** BC2001  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il nome e il percorso del file siano corretti.
