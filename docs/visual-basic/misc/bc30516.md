# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-accepts-this-number-of-arguments"></a>Non è riuscita perché la risoluzione dell'overload non accessibile &#39; &lt;metodo&gt;&#39; accetta questo numero di argomenti
È stata effettuata una chiamata a un metodo di overload, ma il compilatore non riesce a trovare un metodo con un elenco di argomenti corrispondenti.  
  
 **ID errore:** BC30516  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare l'elenco di argomenti fornito.
