# <a name="unable-to-include-xml-fragment-39ltelementpathgt39-of-file-39ltcommentfilegt39"></a>Non è possibile includere il frammento XML &#39; &lt;elementpath&gt;&#39; del file &#39;&lt; filecommento&gt;&#39;
Un tag `<include>` in un commento XML contiene un'espressione XPath per un elemento che non esiste oppure il file contiene codice XML in formato non corretto.  
  
 **ID errore:** BC42320  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che il codice XML nel file sia valido.  
  
2.  Verificare che il frammento XML esista nel file.
