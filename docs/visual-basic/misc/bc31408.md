# <a name="39ltspecifier1gt39-and-39ltspecifier2gt39-cannot-be-combined"></a>&#39; &lt;identificatore1&gt;&#39; e &#39;&lt; identificatore2&gt;&#39; non possono essere combinati
Due identificatori incompatibili, ad esempio `Shadows` e `Overrides` , vengono usati insieme.  
  
 **ID errore:** BC31408  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere uno degli identificatori.
