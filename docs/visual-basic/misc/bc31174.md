# <a name="xml-processing-instruction-name-39name39-is-not-valid"></a>Nome dell'istruzione &#39; nome &#39; di elaborazione XML non è valido
Il nome fornito per un'istruzione di elaborazione XML non è valido.  
  
 **ID errore:** BC31174  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Correggere gli errori nel nome dell'istruzione di elaborazione XML o rimuoverla.
