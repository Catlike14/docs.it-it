# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39;Consentire a&#39; e &#39;impostare&#39; istruzioni di assegnazione non sono più supportate
`Let` e `Set` non possono essere usati nelle assegnazioni.  
  
 **ID errore:** BC30807  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Omettere `Let` e `Set` dalle istruzioni di assegnazione, ad esempio:  
  
     `X = 2`
