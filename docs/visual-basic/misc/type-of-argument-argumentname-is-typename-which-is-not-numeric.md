# <a name="type-of-argument-39ltargumentnamegt39-is-39lttypenamegt39-which-is-not-numeric"></a>Tipo di argomento &#39; &lt;NomeArgomento&gt;&#39; è &#39;&lt; TypeName&gt;&#39; che non è numerico
Un argomento non numerico si verifica in una situazione in cui è previsto un argomento numerico.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che l'argomento sia numerico.
