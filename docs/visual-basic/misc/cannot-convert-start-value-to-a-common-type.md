# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>Impossibile convertire il valore di tipo &#39; &lt;type1&gt;&#39;, il valore limite di &#39; &lt;type2&gt; &#39; e il valore di incremento &#39; &lt;type3&gt; &#39; per un tipo comune
Il valori del controllo `For...Next` specificati sono di tipi diversi e quindi non sono validi perché non possono essere convertiti in un tipo comune.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Modificare il tipo di uno o più valori in modo che tutti i valori possano essere convertiti in un tipo comune.
