# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39; Set &#39; metodo non può avere più di un parametro
Più parametri sono stati definiti per le routine di una proprietà `Set` .  
  
 **ID errore:** BC31063  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere i parametri aggiuntivi dalle routine `Set` di una proprietà.
