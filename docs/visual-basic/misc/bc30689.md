# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>L'istruzione non può trovarsi all'esterno del corpo di un metodo o di un'espressione lambda su più righe
Le istruzioni eseguibili devono essere usate senza routine.  
  
 **ID errore:** BC30689  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare l'istruzione in una routine o subroutine.
