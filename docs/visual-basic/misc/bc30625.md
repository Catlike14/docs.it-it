# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39; Modulo &#39; l'istruzione deve terminare con un corrispondente &#39; Modulo di fine &#39;
I blocchi del modulo devono terminare con un'istruzione `End``Module` .  
  
 **ID errore:** BC30625  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere un'istruzione `End Module` alla fine del blocco `Module` .
