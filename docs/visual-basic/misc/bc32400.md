# Impossibile creare la classe &#39;&lt;nomeclasse&gt;&#39;: &lt;errore&gt;
Si è verificato un errore nell'inizializzazione di una classe. L'installazione di [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] potrebbe essere danneggiata.  
  
 **ID errore:** BC32400  
  
### Per correggere l'errore  
  
1.  Esaminare il messaggio di errore tra virgolette e intraprendere l'azione appropriata.  
  
2.  Reinstallare [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] e ricompilare il programma.  
  
3.  Se l'errore si ripresenta, reinstallare il compilatore [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Se l'errore persiste dopo la reinstallazione, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
