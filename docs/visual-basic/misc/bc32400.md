# <a name="class-39ltclassnamegt39-could-not-be-created-lterrorgt"></a>Classe&lt;classname&gt;' non è possibile creare: &lt;errore&gt;
Si è verificato un errore nell'inizializzazione di una classe. Una possibile causa è danneggiato [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] installazione.  
  
 **ID errore:** BC32400  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Esaminare il messaggio di errore tra virgolette e intraprendere l'azione appropriata.  
  
2.  Reinstallare il [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] e ricompilare il programma.  
  
3.  Se l'errore si ripresenta, reinstallare il compilatore [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
4.  Se l'errore persiste dopo la reinstallazione, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
