# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>Istruzioni 'Continue' non sono valide nella finestra controllo immediato
`Continue`istruzioni possono essere utilizzate solo nel codice sorgente.  
  
 **ID errore:** BC30780  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione `Continue` dal codice di debug.
