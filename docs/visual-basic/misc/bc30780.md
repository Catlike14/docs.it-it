# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Continuare a&#39; istruzioni non sono valide nella finestra controllo immediato
Le istruzioni`Continue` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30780  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione `Continue` dal codice di debug.
