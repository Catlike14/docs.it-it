# <a name="construct-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39lttypenamegt39"></a>Il costrutto contiene un riferimento indiretto all'assembly &#39; &lt;assemblyname&gt;&#39;, che contiene &#39; &lt;typename&gt;&#39;
Il costrutto contiene un riferimento indiretto all'assembly '\<assemblyname >', che contiene \<typename >. Aggiungere un riferimento al file \<filename > al progetto.  
  
 Un'espressione usa un tipo, ad esempio una classe, una struttura, un'interfaccia, un'enumerazione o un delegato, ma l'assembly non ha un riferimento di progetto all'assembly che definisce il tipo.  
  
 **ID errore:** BC31528  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Nelle proprietà del progetto aggiungere un riferimento al file contenente l'assembly che definisce il tipo in uso.
