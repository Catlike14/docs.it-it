# <a name="conversion-operators-cannot-convert-from-a-derived-type"></a>Gli operatori di conversione non possono convertire da un tipo derivato
Un operatore di conversione è dichiarato con un tipo di parametro derivato dal tipo restituito.  
  
 In fase di compilazione, Visual Basic considera una conversione predefinita esiste da qualsiasi tipo riferimento a qualsiasi tipo nella propria gerarchia di ereditarietà, vale a dire qualsiasi tipo da cui deriva o che deriva da esso. Questo tipo di conversione potrebbe non riuscire in fase di esecuzione, tuttavia il compilatore non può prevedere i risultati in fase di esecuzione, quindi ne consente la compilazione.  
  
 Il compilatore considera questa conversione come già definita, quindi non consente di ridefinirla.  
  
 **ID errore:** BC33031  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere completamente questa definizione dell'operatore. È già predefinita.
