# <a name="conversion-from-39date39-to-39double39-requires-calling-the-39datetooadate39-method"></a>Conversione da &#39;data&#39; a &#39;Double&#39; è necessario chiamare il &#39;' date. ToOADate&#39; (metodo)
Si è tentato di eseguire il cast un `Date` valore un `Double` valore, che non può essere eseguita senza usare il <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> metodo.  
  
 **ID errore:** BC30532  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Utilizzare il <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> metodo per convertire il valore.
