# <a name="method-39ltmethodnamegt39-has-no-parameter-named-39ltparameternamegt39"></a>Metodo &#39; &lt;NomeMetodo&gt;&#39; dispone di alcun parametro denominato &#39;&lt; ParameterName&gt;&#39;
L'argomento denominato specificato non corrisponde agli argomenti del metodo di destinazione.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che l'argomento denominato sia stato specificato correttamente.
