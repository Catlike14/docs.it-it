# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39; EndIf &#39; le istruzioni non sono supportate. Usa &#39; Se &#39; In alternativa
L'istruzione `EndIf` non è più supportata.  
  
 **ID errore:** BC30826  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare `End If` per terminare le istruzioni `If` .
