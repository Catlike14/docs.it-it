# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39;Ottenere&#39; l'istruzione deve terminare con un corrispondente &#39;End Get&#39;
Le routine della proprietà`Get` devono terminare con un'istruzione `End Get` .  
  
 **ID errore:** BC30631  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che la routine della proprietà `Get` termini con un costrutto `End Get` .
