# &#39;Inherits&#39; non &#232; valido nei moduli
Un'istruzione `Inherits` si verifica all'interno di un modulo. I moduli non possono derivare dalle classi.  
  
 **ID errore:** BC30230  
  
### Per correggere l'errore  
  
-   Rimuovere l'istruzione `Inherits` oppure digitare di nuovo il modulo come classe.
