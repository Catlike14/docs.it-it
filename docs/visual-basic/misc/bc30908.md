<span data-ttu-id="2625a-103">Una definizione di classe o interfaccia include un [eredita istruzione](../../visual-basic/language-reference/statements/inherits-statement.md) che specifica un tipo annidato all'interno di esso.</span><span class="sxs-lookup"><span data-stu-id="2625a-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="2625a-104">L'ereditarietà deve essere lineare, non circolare.</span><span class="sxs-lookup"><span data-stu-id="2625a-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="2625a-105">Un tipo non può ereditare da un tipo che eredita da esso.</span><span class="sxs-lookup"><span data-stu-id="2625a-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="2625a-106">Una restrizione correlata è una relazione in cui un tipo non può ereditare da un tipo non ancora definito.</span><span class="sxs-lookup"><span data-stu-id="2625a-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="2625a-107">L'ereditarietà presuppone la capacità di riutilizzare membri della classe base, che a sua volta richiede la definizione di questi membri.</span><span class="sxs-lookup"><span data-stu-id="2625a-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="2625a-108">Di conseguenza, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] non è possibile compilare codice come illustrato nell'esempio seguente.</span><span class="sxs-lookup"><span data-stu-id="2625a-108">Therefore, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] cannot compile code such as the following example.</span></span>  
  
```  
Public Class outerClass  
    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```
