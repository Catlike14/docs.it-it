 <span data-ttu-id="1515c-104">L'ereditarietà deve essere lineare, non circolare.</span><span class="sxs-lookup"><span data-stu-id="1515c-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="1515c-105">Un tipo non può ereditare da un tipo che eredita da esso.</span><span class="sxs-lookup"><span data-stu-id="1515c-105">A type cannot inherit from a type that inherits from it.</span></span>

```  
    Public Class innerClass  
        Public Sub doSomething()

    End Class

```
