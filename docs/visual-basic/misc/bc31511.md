# <a name="attribute-member-39ltmembernamegt39-cannot-be-the-target-of-an-assignment-because-it-is-not-declared-39public39"></a>Membro dell'attributo &#39; &lt;membername&gt;&#39; non può essere la destinazione di un'assegnazione perché non è dichiarato &#39; Pubblica &#39;
Si è provato ad assegnare un valore a un membro privato di un attributo.  
  
 **ID errore:** BC31511  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere l'assegnazione.  
  
2.  Se si usano gli attributi personalizzati che sono stati sviluppati, modificare il modificatore di accesso del membro dell'attributo in `Public`.
