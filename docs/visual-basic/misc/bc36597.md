# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>Vai a etichetta &#39; &lt;NomeEtichetta&gt;&#39; non può essere utilizzato all'interno di espressioni lambda
Il corpo di un'espressione lambda deve essere un'espressione che calcola e restituisce un singolo valore. Non sono consentite istruzioni, comprese le istruzioni `GoTo` .  
  
 **ID errore:** BC36597
