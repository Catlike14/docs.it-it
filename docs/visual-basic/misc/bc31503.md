# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>&#39;&lt;TypeName&gt; &#39; non può essere utilizzato come attributo perché non è una classe
Si è provato a usare come attributo un tipo che non è una classe.  
  
 **ID errore:** BC31503  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Definire come classi gli attributi personalizzati che derivano da `System.Attribute`.
