# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39;End SyncLock&#39; deve essere preceduto da un oggetto corrispondente &#39;SyncLock&#39;
I blocchi`SyncLock` iniziano con la parola chiave `SyncLock` e terminano con il costrutto `End SyncLock` .  
  
 **ID errore:** BC30674  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il blocco `SyncLock` inizi con un'istruzione `SyncLock` .
