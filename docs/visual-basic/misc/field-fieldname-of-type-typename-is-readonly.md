# <a name="field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-39readonly39"></a>Campo &#39; &lt;fieldname&gt;&#39; di tipo &#39;&lt; TypeName&gt;&#39; è &#39; Sola lettura &#39;
Si è provato a scrivere in un campo dichiarato `ReadOnly`.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il campo in modo che non sia più dichiarato `ReadOnly`.  
  
-   -oppure-  
  
-   Non provare a scrivere in questo campo.
