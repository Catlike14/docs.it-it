# <a name="class-not-registered-on-local-machine"></a>Classe non registrata nel computer locale
Questo errore in genere è dovuto a un file danneggiato. Potrebbe essere necessario reinstallare [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Reinstallare [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].
