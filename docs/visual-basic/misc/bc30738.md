# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39; Sub Main &#39; è dichiarato più di una volta in &#39; &lt;dello spazio dei nomi&gt;&#39;: &lt;messaggio&gt;
`Sub Main` può essere dichiarato solo all'interno di uno spazio dei nomi.  
  
 **ID errore:** BC30738  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che nel progetto sia presente una sola routine `Sub Main` .
