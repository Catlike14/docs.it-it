# <a name="unable-to-create-xml-documentation-file-39ltnamegt39-ltmessagegt"></a>Impossibile creare il file di documentazione XML &#39; &lt;nome&gt;&#39;: &lt;messaggio&gt;
Il file di documentazione XML non può essere creato a causa di un errore.  
  
 **ID errore:** BC42311  
  
 I messaggi seguenti vengono visualizzati con questo errore:  
  
-   [XML comment block cannot be associated with any language element that supports the application of XML documentation comments](../../visual-basic/misc/bc42312.md).  
  
-   [XML comment tag 'returns' is not permitted on a 'WriteOnly' Property](../../visual-basic/misc/bc42313.md).  
  
-   [Il commento XML non può essere applicato più volte in un elemento parziale \<tipo >](../../visual-basic/misc/bc42314.md).  
  
-   [XML comment tag 'returns' is not permitted on a 'declare sub' language element](../../visual-basic/misc/bc42315.md).  
  
-   [Errore di analisi della documentazione XML: tag di inizio '\<tag >' non esiste un corrispondente tag di fine](../../visual-basic/misc/bc42316.md).  
  
-   [Parametro di tipo del commento XML \<parametro > non corrisponde a un parametro di tipo corrispondente \<parola chiave > istruzione](../../visual-basic/misc/bc42317.md).  
  
-   [XML comment type parameter must have a 'name' attribute](../../visual-basic/misc/bc42318.md).  
  
-   [XML comment exception must have a 'cref' attribute](../../visual-basic/language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md).
