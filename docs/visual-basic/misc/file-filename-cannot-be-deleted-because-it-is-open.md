# <a name="file-39ltfilenamegt39-cannot-be-deleted-because-it-is-open"></a>File &#39; &lt;filename&gt; &#39; non può essere eliminato perché è aperto
Si è provato a eliminare un file aperto.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Chiudere il file prima di eliminarlo.
