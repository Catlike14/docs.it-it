# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>Interfaccia &#39; &lt;interfacename&gt;&#39; può essere implementata solo una volta da questo tipo
L'interfaccia specificata è già implementata da questo tipo e non può essere implementata nuovamente.  
  
 **ID errore:** BC31033  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Implementare l'interfaccia usando questo tipo una sola volta.
