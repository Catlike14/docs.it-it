# <a name="operator-39ltoperatorgt39-must-have-two-parameters"></a>Operatore &#39; &lt;operatore&gt;&#39; deve avere due parametri
Un operatore binario viene definito con meno di due o più di due parametri.  
  
 Un operatore binario deve avere esattamente due parametri.  
  
 **ID errore:** BC33015  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare la definizione per specificare esattamente due parametri.  
  
-   Se è necessario un solo parametro, è necessario definire un operatore unario.  
  
-   Se è necessario alcun parametro o più di due, è necessario utilizzare il [istruzione Function](../../visual-basic/language-reference/statements/function-statement.md) per definire un `Function` procedure anziché un operatore.
