# <a name="index-39ltindexnumbergt39-for-dimension-39ltdimensionnumbergt39-is-out-of-range"></a>Indice &#39; &lt;NumeroIndice&gt; &#39; per la dimensione &#39; &lt;numerodimensione&gt; &#39; è compreso nell'intervallo
Uno degli indici supera l'intervallo definito.  
  
 **ID errore:** BC30702  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare la logica della routine in modo che non vengano superati i limiti della matrice.  
  
-   Ridefinire la matrice per consentire un intervallo più ampio di valori.
