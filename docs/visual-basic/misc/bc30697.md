# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39; &lt;nomemembro1&gt;&#39; non è possibile eseguire l'override &#39;&lt; nomemembro2&gt;&#39; perché si differenziano per i tipi di parametri facoltativi
I membri di overload devono presentare tipi di dati diversi per uno o più parametri obbligatori.  
  
 **ID errore:** BC30697  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Definire almeno un parametro obbligatorio come tipo di dati diverso.
