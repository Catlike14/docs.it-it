# L&#39;istruzione &#39;Option &lt;identificatore&gt;&#39; pu&#242; essere specificata solo una volta per file
Le istruzioni `Option Compare`, `Option Explicit` e `Option Strict` possono essere usate una sola volta nello stesso file di origine.  
  
 **ID errore:** BC30225  
  
### Per correggere l'errore  
  
-   Rimuovere l'istruzione `Option` duplicata.
