# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>&#39; l'opzione &lt;identificatore&gt;&#39; istruzione può apparire una sola volta per ogni file
Le istruzioni `Option Compare`, `Option Explicit`e `Option Strict` possono essere usate una sola volta nello stesso file di origine.  
  
 **ID errore:** BC30225  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione `Option` duplicata.
