-   <span data-ttu-id="c011c-169">**Includere solo la logica di estensione.**</span><span class="sxs-lookup"><span data-stu-id="c011c-169">**Include only the extension logic.**</span></span> <span data-ttu-id="c011c-170">La logica del `My` estensione dello spazio dei nomi deve includere solo il codice necessario per esporre la funzionalità richiesta nel `My` dello spazio dei nomi.</span><span class="sxs-lookup"><span data-stu-id="c011c-170">The logic included in the `My` namespace extension should include only the code that is needed to expose the required functionality in the `My` namespace.</span></span> <span data-ttu-id="c011c-171">Poiché l'estensione risiederà nei progetti utente come codice sorgente, l'aggiornamento del componente di estensione comporta un costo di manutenzione elevata e deve essere evitato se possibile.</span><span class="sxs-lookup"><span data-stu-id="c011c-171">Because your extension will reside in user projects as source code, updating the extension component incurs a high maintenance cost and should be avoided if possible.</span></span>  
  
-   <span data-ttu-id="c011c-172">**Ridurre al minimo ipotesi di progetto.**</span><span class="sxs-lookup"><span data-stu-id="c011c-172">**Minimize project assumptions.**</span></span> <span data-ttu-id="c011c-173">Quando si creano le estensioni di `My` spazio dei nomi, non presupporre che un set di riferimenti, le importazioni a livello di progetto o le impostazioni del compilatore specifici (ad esempio, `Option Strict` off).</span><span class="sxs-lookup"><span data-stu-id="c011c-173">When you create your extensions of the `My` namespace, do not assume a set of references, project-level imports, or specific compiler settings (for example, `Option Strict` off).</span></span> <span data-ttu-id="c011c-174">Al contrario, ridurre al minimo le dipendenze e tutti i riferimenti di tipo completi utilizzando il `Global` (parola chiave).</span><span class="sxs-lookup"><span data-stu-id="c011c-174">Instead, minimize dependencies and fully qualify all type references by using the `Global` keyword.</span></span> <span data-ttu-id="c011c-175">Inoltre, assicurarsi che l'estensione venga compilata con `Option Strict` per ridurre al minimo gli errori nell'estensione.</span><span class="sxs-lookup"><span data-stu-id="c011c-175">Also, ensure that the extension compiles with `Option Strict` on to minimize errors in the extension.</span></span>  
  
-   <span data-ttu-id="c011c-176">**Isolare il codice di estensione.**</span><span class="sxs-lookup"><span data-stu-id="c011c-176">**Isolate the extension code.**</span></span> <span data-ttu-id="c011c-177">Inserire il codice in un singolo file l'estensione rende facilmente distribuibile come modello di elemento Visual Studio.</span><span class="sxs-lookup"><span data-stu-id="c011c-177">Placing the code in a single file makes your extension easily deployable as a Visual Studio item template.</span></span> <span data-ttu-id="c011c-178">Per ulteriori informazioni, vedere "Creazione di pacchetti e distribuzione delle estensioni" più avanti in questo argomento.</span><span class="sxs-lookup"><span data-stu-id="c011c-178">For more information, see "Packaging and Deploying Extensions" later in this topic.</span></span> <span data-ttu-id="c011c-179">Inserimento di tutti i `My` codice di estensione dello spazio dei nomi in un singolo file o una cartella distinta in un progetto verrà inoltre consentire agli utenti di individuare il `My` estensione dello spazio dei nomi.</span><span class="sxs-lookup"><span data-stu-id="c011c-179">Placing all the `My` namespace extension code in a single file or a separate folder in a project will also help users locate the `My` namespace extension.</span></span>  
  
##  <span data-ttu-id="c011c-180"><a name="designing"></a>Progettazione di librerie di classi per My</span><span class="sxs-lookup"><span data-stu-id="c011c-180"><a name="designing"></a> Designing Class Libraries for My</span></span>  
 <span data-ttu-id="c011c-181">Come avviene con la maggior parte dei modelli a oggetti, alcuni modelli di progettazione funzionano in modo efficiente il `My` dello spazio dei nomi e gli altri non.</span><span class="sxs-lookup"><span data-stu-id="c011c-181">As is the case with most object models, some design patterns work well in the `My` namespace and others do not.</span></span> <span data-ttu-id="c011c-182">Quando si progetta un'estensione per il `My` dello spazio dei nomi, considerare i principi seguenti:</span><span class="sxs-lookup"><span data-stu-id="c011c-182">When designing an extension to the `My` namespace, consider the following principles:</span></span>  
  
-   <span data-ttu-id="c011c-183">**Metodi senza stati.**</span><span class="sxs-lookup"><span data-stu-id="c011c-183">**Stateless methods.**</span></span> <span data-ttu-id="c011c-184">Metodi di `My` dello spazio dei nomi deve fornire una soluzione completa per un'attività specifica.</span><span class="sxs-lookup"><span data-stu-id="c011c-184">Methods in the `My` namespace should provide a complete solution to a specific task.</span></span> <span data-ttu-id="c011c-185">Assicurarsi che i valori dei parametri passati al metodo forniscano tutto l'input necessario per completare l'attività particolare.</span><span class="sxs-lookup"><span data-stu-id="c011c-185">Ensure that the parameter values that are passed to the method provide all the input required to complete the particular task.</span></span> <span data-ttu-id="c011c-186">Evitare di creare metodi che si basano sullo stato precedente, ad esempio le connessioni aperte alle risorse.</span><span class="sxs-lookup"><span data-stu-id="c011c-186">Avoid creating methods that rely on prior state, such as open connections to resources.</span></span>  
  
-   <span data-ttu-id="c011c-187">**Istanze globale.**</span><span class="sxs-lookup"><span data-stu-id="c011c-187">**Global instances.**</span></span> <span data-ttu-id="c011c-188">L'unico stato che viene mantenuto il `My` spazio dei nomi è globale per il progetto.</span><span class="sxs-lookup"><span data-stu-id="c011c-188">The only state that is maintained in the `My` namespace is global to the project.</span></span> <span data-ttu-id="c011c-189">Ad esempio, `My.Application.Info` incapsula lo stato condiviso in tutta l'applicazione.</span><span class="sxs-lookup"><span data-stu-id="c011c-189">For example, `My.Application.Info` encapsulates state that is shared throughout the application.</span></span>  
  
-   <span data-ttu-id="c011c-190">**Tipi di parametro semplici.**</span><span class="sxs-lookup"><span data-stu-id="c011c-190">**Simple parameter types.**</span></span> <span data-ttu-id="c011c-191">Semplificare le operazioni, evitando di tipi di parametri complessi.</span><span class="sxs-lookup"><span data-stu-id="c011c-191">Keep things simple by avoiding complex parameter types.</span></span> <span data-ttu-id="c011c-192">Creare invece metodi che non utilizzino alcun parametro di input o che accettano tipi di input semplici, ad esempio stringhe, tipi primitivi e così via.</span><span class="sxs-lookup"><span data-stu-id="c011c-192">Instead, create methods that either take no parameter input or that take simple input types such as strings, primitive types, and so on.</span></span>  
  
-   <span data-ttu-id="c011c-193">**Metodi factory.**</span><span class="sxs-lookup"><span data-stu-id="c011c-193">**Factory methods.**</span></span> <span data-ttu-id="c011c-194">Alcuni tipi sono necessariamente difficile creare un'istanza.</span><span class="sxs-lookup"><span data-stu-id="c011c-194">Some types are necessarily difficult to instantiate.</span></span> <span data-ttu-id="c011c-195">Fornendo i metodi factory come estensioni per il `My` dello spazio dei nomi consente di individuare più facilmente e utilizzare tipi che rientrano in questa categoria.</span><span class="sxs-lookup"><span data-stu-id="c011c-195">Providing factory methods as extensions to the `My` namespace enables you to more easily discover and consume types that fall into this category.</span></span> <span data-ttu-id="c011c-196">Un esempio di un metodo factory che funziona bene è `My.Computer.FileSystem.OpenTextFileReader`.</span><span class="sxs-lookup"><span data-stu-id="c011c-196">An example of a factory method that works well is `My.Computer.FileSystem.OpenTextFileReader`.</span></span> <span data-ttu-id="c011c-197">Sono disponibili molti tipi di flusso in .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="c011c-197">There are several stream types available in the .NET Framework.</span></span> <span data-ttu-id="c011c-198">Specificando in particolare, i file di testo di `OpenTextFileReader` consente all'utente di capire quale flusso da utilizzare.</span><span class="sxs-lookup"><span data-stu-id="c011c-198">By specifying text files specifically, the `OpenTextFileReader` helps the user understand which stream to use.</span></span>  
  
 <span data-ttu-id="c011c-199">Queste linee guida non precludono i principi di progettazione generali per le librerie di classi.</span><span class="sxs-lookup"><span data-stu-id="c011c-199">These guidelines do not preclude general design principles for class libraries.</span></span> <span data-ttu-id="c011c-200">Piuttosto, sono raccomandazioni ottimizzate per gli sviluppatori che utilizzano Visual Basic e `My` dello spazio dei nomi.</span><span class="sxs-lookup"><span data-stu-id="c011c-200">Rather, they are recommendations that are optimized for developers who are using Visual Basic and the `My` namespace.</span></span> <span data-ttu-id="c011c-201">Per i principi di progettazione generali per la creazione di librerie di classi, vedere [Framework Design Guidelines](http://msdn.microsoft.com/library/5fbcaf4f-ea2a-4d20-b0d6-e61dee202b4b).</span><span class="sxs-lookup"><span data-stu-id="c011c-201">For general design principles for creating class libraries, see [Framework Design Guidelines](http://msdn.microsoft.com/library/5fbcaf4f-ea2a-4d20-b0d6-e61dee202b4b).</span></span>
