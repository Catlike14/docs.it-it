### <a name="to-create-a-data-bound-datarepeater"></a><span data-ttu-id="d5685-109">Per creare un oggetto con associazione a dati DataRepeater</span><span class="sxs-lookup"><span data-stu-id="d5685-109">To create a data-bound DataRepeater</span></span>  
  
1.  <span data-ttu-id="d5685-110">Trascinare un <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controllare dal **Visual Basic Power Packs** scheda il **casella degli strumenti** a un form o un controllo contenitore.</span><span class="sxs-lookup"><span data-stu-id="d5685-110">Drag a <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control from the **Visual Basic PowerPacks** tab in the **Toolbox** to a form or container control.</span></span>  
  
2.  <span data-ttu-id="d5685-111">Trascinare i quadratini di ridimensionamento e la posizione per le dimensioni e posizionare il controllo.</span><span class="sxs-lookup"><span data-stu-id="d5685-111">Drag the sizing and position handles to size and position the control.</span></span>  
  
     <span data-ttu-id="d5685-112">Si noti che il controllo dispone di due aree rettangolari.</span><span class="sxs-lookup"><span data-stu-id="d5685-112">Note that the control has two rectangular regions.</span></span> <span data-ttu-id="d5685-113">L'area superiore è il *modello di elemento*; i controlli aggiunti al modello verranno ripetuti in ogni elemento nel <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controllo in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="d5685-113">The upper region is the *item template*; controls added to the template will be repeated in each item in the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control at run time.</span></span> <span data-ttu-id="d5685-114">L'area inferiore è il *viewport*, in cui verranno visualizzati gli elementi.</span><span class="sxs-lookup"><span data-stu-id="d5685-114">The lower region is the *viewport*, where the items will be displayed.</span></span>  
  
     <span data-ttu-id="d5685-115">È anche possibile ridimensionare e posizionare il controllo o il modello di elemento modificando la **dimensioni** e **posizione** proprietà nella finestra Proprietà.</span><span class="sxs-lookup"><span data-stu-id="d5685-115">You can also size and position the control or the item template by changing the **Size** and **Position** properties in the Properties window.</span></span>  
  
3.  <span data-ttu-id="d5685-116">Scegliere **Mostra origini dati** dal menu **Dati**.</span><span class="sxs-lookup"><span data-stu-id="d5685-116">On the **Data** menu, click **Show Data Sources**.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="d5685-117">Se il **Zdroje dat** finestra è vuota, aggiungere un'origine dati a esso.</span><span class="sxs-lookup"><span data-stu-id="d5685-117">If the **Data Sources** window is empty, add a data source to it.</span></span> <span data-ttu-id="d5685-118">Per altre informazioni, vedere [Add new data sources](/visualstudio/data-tools/add-new-data-sources) (Aggiungere nuove origini dati).</span><span class="sxs-lookup"><span data-stu-id="d5685-118">For more information, see [Add new data sources](/visualstudio/data-tools/add-new-data-sources).</span></span>  
  
4.  <span data-ttu-id="d5685-119">Nel **Zdroje dat** finestra, selezionare il nodo di primo livello per la tabella che contiene i dati che si desidera associare.</span><span class="sxs-lookup"><span data-stu-id="d5685-119">In the **Data Sources** window, select the top-level node for the table that contains the data that you want to bind.</span></span>  
  
5.  <span data-ttu-id="d5685-120">Modificare il tipo di rilascio della tabella `Details` facendo clic `Details` nell'elenco a discesa nel nodo della tabella.</span><span class="sxs-lookup"><span data-stu-id="d5685-120">Change the drop type of the table to `Details` by clicking `Details` in the drop-down list on the table node.</span></span>  
  
6.  <span data-ttu-id="d5685-121">Selezionare il nodo della tabella e trascinarlo nell'area modello dell'elemento di <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controllo.</span><span class="sxs-lookup"><span data-stu-id="d5685-121">Select the table node and drag it onto the item template region of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span>  
  
     <span data-ttu-id="d5685-122">È possibile specificare tipi di controlli che vengono visualizzati per ogni campo.</span><span class="sxs-lookup"><span data-stu-id="d5685-122">You can specify which types of controls are displayed for each field.</span></span> <span data-ttu-id="d5685-123">Per altre informazioni, vedere [impostare il controllo da creare durante il trascinamento dalla finestra Origini dei dati](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span><span class="sxs-lookup"><span data-stu-id="d5685-123">For more information, see [Set the control to be created when dragging from the Data Sources window](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span></span>
