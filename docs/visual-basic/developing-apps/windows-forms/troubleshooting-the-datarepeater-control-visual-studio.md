## <a name="datarepeater-keyboard-and-mouse-events-are-not-raised"></a><span data-ttu-id="55e5d-104">DataRepeater tastiera e Mouse eventi non vengono generati.</span><span class="sxs-lookup"><span data-stu-id="55e5d-104">DataRepeater Keyboard and Mouse Events Are Not Raised</span></span>  
 <span data-ttu-id="55e5d-105">Alcuni <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> eventi di controllo, ad esempio gli eventi di tastiera e mouse, non vengono generati.</span><span class="sxs-lookup"><span data-stu-id="55e5d-105">Some <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control events, such as keyboard and mouse events, are not raised.</span></span> <span data-ttu-id="55e5d-106">Si tratta di un comportamento correlato alla progettazione.</span><span class="sxs-lookup"><span data-stu-id="55e5d-106">This is by design.</span></span> <span data-ttu-id="55e5d-107">Il <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controllo stesso è un contenitore per <xref:Microsoft.VisualBasic.PowerPacks.DataRepeaterItem> oggetti e non è accessibile in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="55e5d-107">The <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control itself is a container for <xref:Microsoft.VisualBasic.PowerPacks.DataRepeaterItem> objects and cannot be accessed at run time.</span></span> <span data-ttu-id="55e5d-108">Il <xref:Microsoft.VisualBasic.PowerPacks.DataRepeaterItem> non espone eventi in fase di progettazione.</span><span class="sxs-lookup"><span data-stu-id="55e5d-108">The <xref:Microsoft.VisualBasic.PowerPacks.DataRepeaterItem> does not expose events at design time.</span></span> <span data-ttu-id="55e5d-109">Pertanto, fare clic su un elemento o un tasto quando l'elemento ha lo stato attivo non genera un evento.</span><span class="sxs-lookup"><span data-stu-id="55e5d-109">Therefore, clicking an item or pressing a key when the item has focus does not raise an event.</span></span>  
  
 <span data-ttu-id="55e5d-110">L'eccezione è quando il <xref:System.Windows.Forms.Control.Padding%2A> proprietà è impostata su un grande valore abbastanza per esporre i bordi del <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controllo.</span><span class="sxs-lookup"><span data-stu-id="55e5d-110">The exception to this is when the <xref:System.Windows.Forms.Control.Padding%2A> property is set to a large enough value to expose the edges of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span> <span data-ttu-id="55e5d-111">In questo caso, fare clic sul margine esposto genererà gli eventi del mouse.</span><span class="sxs-lookup"><span data-stu-id="55e5d-111">In this case, clicking in the exposed margin will raise mouse events.</span></span>  
  
 <span data-ttu-id="55e5d-112">Per risolvere questo problema, aggiungere un <xref:System.Windows.Forms.Panel> controllo il <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater.ItemTemplate%2A> sezione del <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controllare e quindi aggiungere il resto dei controlli per il <xref:System.Windows.Forms.Panel>.</span><span class="sxs-lookup"><span data-stu-id="55e5d-112">To resolve this issue, add a <xref:System.Windows.Forms.Panel> control to the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater.ItemTemplate%2A> section of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control, and then add the rest of the controls to the <xref:System.Windows.Forms.Panel>.</span></span> <span data-ttu-id="55e5d-113">È quindi possibile aggiungere codice per il <xref:System.Windows.Forms.Panel> gestori eventi per gli eventi di tastiera e mouse.</span><span class="sxs-lookup"><span data-stu-id="55e5d-113">You can then add code to the <xref:System.Windows.Forms.Panel> control's event handlers for keyboard and mouse events.</span></span>

 <span data-ttu-id="55e5d-117">È possibile spostare il <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> inferiore di <xref:System.Windows.Forms.BindingNavigator> controllo in fase di progettazione o aggiungere codice simile a quanto segue nel `Load` gestore dell'evento.</span><span class="sxs-lookup"><span data-stu-id="55e5d-117">You can either move the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> lower than the <xref:System.Windows.Forms.BindingNavigator> control at design time, or add code resembling the following in the `Load` event handler.</span></span>
