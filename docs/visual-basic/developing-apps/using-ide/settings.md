# <a name="visual-basic-settings"></a>Impostazioni di Visual Basic
Impostazioni di Visual Basic sono progettate per ottimizzare l'esperienza utente e la produttività degli sviluppatori Visual Basic. È parte di [personalizzazione delle impostazioni di sviluppo in Visual Studio](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3) funzionalità che consente di personalizzare e salvare le impostazioni nell'ambiente di sviluppo integrato (IDE) di Visual Studio.  
  
 È inoltre possibile porta o ricaricare le impostazioni IDE in altri computer. Per ulteriori informazioni, vedere [procedura: condivisione di impostazioni tra computer](http://msdn.microsoft.com/en-us/1131fb10-35c1-42da-9cd8-91aa3235b882).

## <a name="dialog-boxes"></a>Finestre di dialogo  
  
|Funzionalità|Comportamento|
|---|---|  
|**Nuovo progetto** la finestra di dialogo|Se nel computer sono installate altre piattaforme, l'IDE consente la casella combinata che consente di specificare la piattaforma di destinazione.|  
|[Finestra di dialogo Opzioni](https://docs.microsoft.com/visualstudio/ide/reference/options-dialog-box-visual-studio)|Consente di Visual Basic semplificati **opzioni** pagine. Fare clic su di **Mostra tutte le impostazioni** casella di controllo per visualizzare l'elenco completo delle opzioni.|  
  
## <a name="keyboard"></a>Tastiera  
  
|Funzionalità|Comportamento|  
|---|---|  
|Tasti di scelta rapida predefiniti di Visual Basic 6.0|Mappatura della tastiera di supporta tipico Visual Basic 6.|
