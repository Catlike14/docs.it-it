# <a name="visual-basic-intellisense-code-snippets"></a>Frammenti di codice IntelliSense di Visual Basic
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]include una libreria contenente diverse centinaia di parti di codice, denominate frammenti di codice IntelliSense, che sono pronti per essere inserite nell'applicazione. Ogni frammento esegue un'attività di programmazione completezza, ad esempio la creazione di un'eccezione personalizzata, inviare un messaggio di posta elettronica o disegnare un cerchio. È possibile inserire un frammento di codice sorgente con pochi clic del mouse. È inoltre possibile creare frammenti personalizzati che le esigenze aziendali, aggiungerli alla libreria e quindi utilizzano quando necessario. Per altre informazioni, vedere [Code Snippets](https://docs.microsoft.com/visualstudio/ide/code-snippets) (Frammenti di codice).  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Uno strumento di produttività per gli utenti di Visual Basic  
 Questi frammenti di aumentare la produttività riducendo la quantità di tempo dedicata alla ricerca di esempi di codice, riducendo il tempo necessario per imparare a usare una funzionalità poco e supportando il riutilizzo del codice.  
  
 La libreria di codice supporta le seguenti funzionalità:  
  
-   Inserimento di frammenti di codice in un il **Editor di codice**.  
  
-   Creare nuove attività che è possibile riutilizzare nei progetti.  
  
-   Creare nuove attività da condividere con il gruppo di lavoro e colleghi.  
  
-   Modifica delle attività.  
  
-   Download di altre attività di terze parti.  
  
 Questi blocchi di codice sono disponibili in [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   È possibile aggiungerli tramite il menu di scelta rapida di **Editor di codice**.  
  
-   È possibile trascinare i file di codice XML da **Esplora File** nel file di codice sorgente.
