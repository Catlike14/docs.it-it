## <a name="increase-the-maximum-open-file-limit"></a><span data-ttu-id="c2871-128">Aumentare il limite massimo di file aperti</span><span class="sxs-lookup"><span data-stu-id="c2871-128">Increase the maximum open file limit</span></span>

<span data-ttu-id="c2871-129">Il limite di file aperti predefinito per macOS potrebbe non essere sufficiente per alcuni carichi di lavoro di .NET Core, ad esempio il ripristino di progetti o l'esecuzione di unit test.</span><span class="sxs-lookup"><span data-stu-id="c2871-129">The default open file limit on macOS may not be sufficient for some .NET Core workloads, such as restoring projects or running unit tests.</span></span>

<span data-ttu-id="c2871-130">È possibile aumentare questo limite seguendo questa procedura:</span><span class="sxs-lookup"><span data-stu-id="c2871-130">You can increase this limit by following these steps:</span></span>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN"
        "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
  <dict>
    <key>Label</key>
    <string>limit.maxfiles</string>
    <key>ProgramArguments</key>
    <array>
      <string>launchctl</string>
      <string>limit</string>
      <string>maxfiles</string>
      <string>2048</string>
      <string>4096</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
    <key>ServiceIPC</key>

  </dict>

```
