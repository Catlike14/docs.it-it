# <a name="itypelibresolver-interface"></a>Interfaccia ITypeLibResolver
Fornisce il [metodo ResolveTypeLib](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md), che consente di risolvere il percorso del file di una libreria dei tipi.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo ResolveTypeLib](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md)|Risolve il nome semplice di una libreria dei tipi restituendo il percorso completo.|

 **Libreria:** TlbRef. lib  
  
 **Versione di .NET framework:** 4, 3.5, 3.0, 2.0
