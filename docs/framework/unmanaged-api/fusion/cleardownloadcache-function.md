# <a name="cleardownloadcache-function"></a>Funzione ClearDownloadCache
Cancella la cache di assembly globale degli assembly scaricato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT ClearDownloadCache ();  
```
