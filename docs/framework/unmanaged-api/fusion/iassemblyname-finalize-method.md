# <a name="iassemblynamefinalize-method"></a>Metodo IAssemblyName::Finalize
Questo consente [IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) oggetto per rilasciare risorse ed eseguire altre operazioni di pulizia prima che venga chiamato il distruttore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT Finalize ();  
```
