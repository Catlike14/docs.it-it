# <a name="couninitializecor-function"></a>Funzione CoUninitializeCor
`CoUninitializeCor` è obsoleto.  
  
## <a name="syntax"></a>Sintassi  
  
```  
STDAPI_(void) CoUninitializeCor(void);  
```  
  
## <a name="remarks"></a>Note  
 Common language runtime non può essere scaricato da un processo. Per rimuovere completamente il runtime da un processo in esecuzione, è necessario arrestare il processo.
