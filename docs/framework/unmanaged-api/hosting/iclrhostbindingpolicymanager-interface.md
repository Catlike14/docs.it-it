# <a name="iclrhostbindingpolicymanager-interface"></a>Interfaccia ICLRHostBindingPolicyManager
Fornisce metodi per l'host per la valutazione dei criteri di associazione corrente e comunicare le modifiche dei criteri per un assembly specificato.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[EvaluatePolicy (metodo)](../../../../docs/framework/unmanaged-api/hosting/iclrhostbindingpolicymanager-evaluatepolicy-method.md)|Valuta i criteri di associazione per conto dell'host.|  
|[ModifyApplicationPolicy (metodo)](../../../../docs/framework/unmanaged-api/hosting/iclrhostbindingpolicymanager-modifyapplicationpolicy-method.md)|Modifica i criteri di associazione per l'assembly specificato e crea una nuova versione del criterio.|
