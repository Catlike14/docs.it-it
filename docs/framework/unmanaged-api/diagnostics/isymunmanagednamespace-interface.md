# <a name="isymunmanagednamespace-interface"></a>Interfaccia ISymUnmanagedNamespace
Rappresenta uno spazio dei nomi.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[GetName (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagednamespace-getname-method.md)|Ottiene il nome di questo spazio dei nomi.|  
|[GetNamespaces (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagednamespace-getnamespaces-method.md)|Ottiene gli elementi figlio di questo spazio dei nomi.|  
|[GetVariables (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagednamespace-getvariables-method.md)|Restituisce tutte le variabili definite in ambito globale all'interno di questo spazio dei nomi.|  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym.idl, CorSym.h
