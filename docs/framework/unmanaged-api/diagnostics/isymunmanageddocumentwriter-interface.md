# <a name="isymunmanageddocumentwriter-interface"></a>Interfaccia ISymUnmanagedDocumentWriter
Fornisce i metodi per la scrittura di un documento cui viene fatto riferimento in un archivio di simboli.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[SetCheckSum (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddocumentwriter-setchecksum-method.md)|Imposta le informazioni di checksum.|  
|[SetSource (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddocumentwriter-setsource-method.md)|Imposta l'origine per un documento che viene scritto incorporata.|  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym.idl, CorSym.h
