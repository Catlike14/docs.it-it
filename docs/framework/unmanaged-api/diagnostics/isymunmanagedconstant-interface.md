# <a name="isymunmanagedconstant-interface"></a>Interfaccia ISymUnmanagedConstant
Fornisce l'accesso alle costanti non gestite.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[GetName (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedconstant-getname-method.md)|Ottiene il nome della costante.|  
|[GetSignature (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedconstant-getsignature-method.md)|Ottiene la firma della costante.|  
|[GetValue (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedconstant-getvalue-method.md)|Ottiene il valore della costante.|  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym.idl, CorSym.h
