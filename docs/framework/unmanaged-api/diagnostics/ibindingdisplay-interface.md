# <a name="ibindingdisplay-interface"></a>Interfaccia IBindingDisplay
Fornisce metodi che consentono di visualizzare informazioni correnti di associazione dell'applicazione in esecuzione.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[GetCurrentDisplay (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/ibindingdisplay-getcurrentdisplay-method.md)|Restituisce le informazioni di visualizzazione di associazione corrente.|  
|[InitializeForProcess (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/ibindingdisplay-initializeforprocess-method.md)|Inizializza il `IBindingDisplay` oggetto.|
