# <a name="inotifysource2-interface"></a>Interfaccia INotifySource2
Dichiara un metodo per impostare i filtri di notifica.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[SetNotifyFilter (metodo)](../../../../docs/framework/unmanaged-api/diagnostics/inotifysource2-setnotifyfilter-method.md)|Assegna un filtro di notifica per l'uso con questa origine.|
