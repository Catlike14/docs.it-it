# <a name="isymunmanagedreadergetsymbolstorefilename-method"></a>Metodo ISymUnmanagedReader::GetSymbolStoreFileName
Fornisce il nome di file su disco dell'archivio di simboli.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetSymbolStoreFileName (  
    [in]  ULONG32 cchName,  
    [out] ULONG32 *pcchName,  
    [out, size_is (cchName),  
        length_is (*pcchName)] WCHAR szName[]);  
```  
  
#### <a name="parameters"></a>Parametri  
 `cchName`  
 [in] Le dimensioni del `szName` buffer.  
  
 `pcchName`  
 [out] Un puntatore a una variabile che riceve la lunghezza del nome restituito `szName`, inclusa la terminazione null.  
  
 `szName`  
 [out] Puntatore a una variabile che riceve il nome del file dell'archivio di simboli.  
  
## <a name="return-value"></a>Valore restituito  
 S_OK se il metodo ha esito positivo. in caso contrario, E_FAIL o un altro codice di errore.  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym. idl, CorSym.h
