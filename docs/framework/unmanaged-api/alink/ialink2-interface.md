# <a name="ialink2-interface"></a>Interfaccia IALink2
Integra il [interfaccia IALink](../../../../docs/framework/unmanaged-api/alink/ialink-interface.md) dichiarando i metodi che operano sulle interfacce di metadati più recenti, ad esempio [interfaccia IMetaDataEmit2](../../../../docs/framework/unmanaged-api/metadata/imetadataemit2-interface.md). IALink2 dichiara inoltre i metodi che offrono maggiore flessibilità di utilizzo su interfacce IALink originali.
