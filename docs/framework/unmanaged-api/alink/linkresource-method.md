# <a name="linkresource-method"></a>Metodo LinkResource
Collegamenti in una risorsa.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT LinkResource(  
    mdAssembly  AssemblyID,  
    LPCWSTR     pszFileName,  
    LPCWSTR     pszNewLocation,  
    LPCWSTR     pszResourceName,  
    DWORD       dwFlags  
) PURE;  
```  
  
#### <a name="parameters"></a>Parametri  
 `AssemblyID`  
 ID dell'assembly.  
  
 `pszFileName`  
 Nome del file.  
  
 `pszNewLocation`  
 Nuovo nome file facoltativo. Se diverso da NULL, `pszFileName` pszNewLocation verranno copiati.  
  
 `pszResourceName`  
 Nome della risorsa.  
  
 `dwFlags`  
 Flag di accessibilità, ad esempio `mrPublic` e `mrPrivate`. Questo parametro può essere passato a [DefineManifestResource (metodo)](../../../../docs/framework/unmanaged-api/metadata/imetadataassemblyemit-definemanifestresource-method.md).  
  
## <a name="return-value"></a>Valore restituito  
 Se il metodo ha esito positivo, restituisce S_OK.  
  
## <a name="requirements"></a>Requisiti  
 Richiede alink.h.
