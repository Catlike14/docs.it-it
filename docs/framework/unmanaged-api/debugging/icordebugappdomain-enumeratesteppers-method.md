# <a name="icordebugappdomainenumeratesteppers-method"></a>Metodo ICorDebugAppDomain::EnumerateSteppers

## <a name="syntax"></a>Sintassi  
  
```  
HRESULT EnumerateSteppers (  
    [out] ICorDebugStepperEnum   **ppSteppers

```
