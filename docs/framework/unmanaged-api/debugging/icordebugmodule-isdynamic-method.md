# <a name="icordebugmoduleisdynamic-method"></a>Metodo ICorDebugModule::IsDynamic
Ottiene un valore che indica se questo modulo è dinamico.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT IsDynamic(  
    [out] BOOL *pDynamic  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `pDynamic`  
 [out] `true` se questo modulo è dinamico; in caso contrario, `false`.  
  
## <a name="remarks"></a>Note  
 Un modulo dinamico può aggiungere nuove classi ed eliminare classi esistenti, anche dopo che è stato caricato il modulo. Il [ICorDebugManagedCallback:: LoadClass](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback-loadclass-method.md) e [ICorDebugManagedCallback](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback-unloadclass-method.md) callback informano il debugger quando una classe è stato aggiunta o eliminata.
