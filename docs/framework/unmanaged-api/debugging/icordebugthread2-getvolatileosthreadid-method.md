# <a name="icordebugthread2getvolatileosthreadid-method"></a>Metodo ICorDebugThread2::GetVolatileOSThreadID
Ottiene l'identificatore del thread del sistema operativo per ICorDebugThread2.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetVolatileOSThreadID (  
    [out] DWORD    *pdwTid  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `pdwTid`  
 [out] L'identificatore del thread del sistema operativo per il thread.
