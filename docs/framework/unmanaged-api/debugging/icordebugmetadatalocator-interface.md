# <a name="icordebugmetadatalocator-interface"></a>Interfaccia ICorDebugMetaDataLocator
Fornisce informazioni sui metadati al debugger.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[GetMetaData (metodo)](../../../../docs/framework/unmanaged-api/debugging/icordebugmetadatalocator-getmetadata-method.md)|Chiede al debugger di restituire il percorso completo di un modulo i cui metadati sono necessari per completare un'operazione richiesta dal debugger.|
