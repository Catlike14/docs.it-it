# <a name="icordebugcontrollerisrunning-method"></a>Metodo ICorDebugController::IsRunning
Ottiene un valore che indica se il thread del processo attualmente in esecuzione liberamente.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```
