# <a name="icordebugcontrollerisrunning-method"></a>Metodo ICorDebugController::IsRunning

## <a name="syntax"></a>Sintassi  
  
```  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```
