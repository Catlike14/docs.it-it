# <a name="icordebughandlevaluegethandletype-method"></a>Metodo ICorDebugHandleValue::GetHandleType
Ottiene un valore che indica il tipo di handle a cui fa riferimento l'oggetto ICorDebugHandleValue.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetHandleType (  
    [out] CorDebugHandleType  *pType  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `pType`  
 [out] Un puntatore a un valore di enumerazione CorDebugHandleType che indica il tipo di questo handle.
