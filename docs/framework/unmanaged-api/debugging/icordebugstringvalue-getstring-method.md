# <a name="icordebugstringvaluegetstring-method"></a>Metodo ICorDebugStringValue::GetString
Ottiene la stringa a cui fa riferimento questo oggetto ICorDebugStringValue.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetString (  
    [in] ULONG32    cchString,  
    [out] ULONG32   *pcchString,  
    [out, size_is(cchString), length_is(*pcchString)]   
        WCHAR       szString[]  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `cchString`  
 [in] Dimensione della matrice `szString`.  
  
 `pcchString`  
 [out] Un puntatore al numero di caratteri restituiti nel `szString` matrice.  
  
 `szString`  
 [out] Matrice che archivia la stringa recuperata.
