# <a name="imetadatatablesgetstringheapsize-method"></a>Metodo IMetaDataTables::GetStringHeapSize
Ottiene le dimensioni, in byte, dell'heap delle stringhe.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetStringHeapSize (  
    [out] ULONG   *pcbStrings  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `pcbStrings`  
 [out] Puntatore alla dimensione, in byte, dell'heap delle stringhe.
