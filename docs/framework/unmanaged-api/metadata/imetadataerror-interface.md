# <a name="imetadataerror-interface"></a>Interfaccia IMetaDataError
Fornisce un meccanismo di callback per la segnalazione di errori durante il merge dei metadati.  
  
> [!NOTE]
>  Il `IMetaDataError` interfaccia deve essere implementata dal client.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[OnError (metodo)](../../../../docs/framework/unmanaged-api/metadata/imetadataerror-onerror-method.md)|Fornisce la notifica degli errori che si verificano durante l'unione dei metadati.|
