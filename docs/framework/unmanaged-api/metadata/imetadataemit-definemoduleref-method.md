# <a name="imetadataemitdefinemoduleref-method"></a>Metodo IMetaDataEmit::DefineModuleRef

## <a name="syntax"></a>Sintassi  
  
```  
HRESULT DefineModuleRef (     
    [in]  LPCWSTR           szName,   
    [out] mdModuleRef       *pmur   
);  
```
