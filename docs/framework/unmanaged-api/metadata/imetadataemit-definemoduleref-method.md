# <a name="imetadataemitdefinemoduleref-method"></a>Metodo IMetaDataEmit::DefineModuleRef
Crea la firma dei metadati per un modulo con il nome specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT DefineModuleRef (     
    [in]  LPCWSTR           szName,   
    [out] mdModuleRef       *pmur   
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `szName`  
 [in] Il nome di altri file di metadati, in genere una DLL. Questo Ã¨ solo il nome del file. Non utilizzare un nome e percorso completo.  
  
 `pmur`  
 [out] L'oggetto assegnato `mdModuleRef` token.
