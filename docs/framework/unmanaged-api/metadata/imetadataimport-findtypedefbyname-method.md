# <a name="imetadataimportfindtypedefbyname-method"></a>Metodo IMetaDataImport::FindTypeDefByName
Ottiene un puntatore ai metadati TypeDef token per il <xref:System.Type> con il nome specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT FindTypeDefByName  
   [in]  LPCWSTR       szTypeDef,  
   [in]  mdToken       tkEnclosingClass,  
   [out] mdTypeDef     *ptd  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `szTypeDef`  
 [in] Il nome del tipo per il quale ottenere il token TypeDef.  
  
 `tkEnclosingClass`  
 [in] Token TypeDef o TypeRef che rappresenta la classe che lo contiene. Se il tipo da cercare non Ã¨ una classe annidata, impostare questo valore su NULL.  
  
 `ptd`  
 [out] Puntatore al token TypeDef corrispondente.
