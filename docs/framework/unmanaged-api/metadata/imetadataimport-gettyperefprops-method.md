# <a name="imetadataimportgettyperefprops-method"></a>Metodo IMetaDataImport::GetTypeRefProps
Ottiene i metadati associati di <xref:System.Type> a cui fa riferimento il token TypeRef specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetTypeRefProps (  
   [in]  mdTypeRef   tr,  
   [out] mdToken     *ptkResolutionScope,  
   [out] LPWSTR      szName,  
   [in]  ULONG       cchName,  
   [out] ULONG       *pchName  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `tr`  
 [in] Il token TypeRef che rappresenta il tipo da restituire i metadati.  
  
 `ptkResolutionScope`  
 [out] Un puntatore all'ambito in cui viene fatto riferimento. Questo valore Ã¨ un token AssemblyRef o ModuleRef.  
  
 `szName`  
 [out] Un buffer contenente il nome del tipo.  
  
 `cchName`  
 [in] La dimensione in caratteri wide della richiesta `szName`.  
  
 `pchName`  
 [out] Dimensione restituita in caratteri "wide" di `szName`.
