# <a name="icorprofilerfunctionenumreset-method"></a>Metodo ICorProfilerFunctionEnum::Reset
Sposta il cursore dell'enumeratore nella posizione iniziale della sequenza.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT Reset();  
```
