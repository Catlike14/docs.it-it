# Risorse (WPF)
Una risorsa è un oggetto che può essere riutilizzato in posizioni diverse nell'applicazione.  WPF supporta i diversi tipi di risorse.  Queste risorse sono illustrati due tipi di risorse: Risorse XAML e file di dati della risorsa.  Esempi di risorse XAML sono inclusi pennelli e stili.  I file di dati delle risorse vengono file di dati non eseguibili di che un'applicazione necessita.
