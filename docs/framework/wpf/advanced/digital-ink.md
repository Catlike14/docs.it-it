# Input penna
In questa sezione viene descritto l'utilizzo dell'input penna in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].  In passato disponibile solo nel'SDK per Tablet PC, l'input digitale è ora disponibile nella versione principale di Windows Presentation Foundation.  Questo significa che è ora possibile sviluppare applicazioni complete per Tablet PC utilizzando la potenza di Windows Presentation Foundation.
