# Gestione avanzata dell&#39;input penna
[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] viene fornito con l'oggetto <xref:System.Windows.Controls.InkCanvas>, un elemento che è possibile inserire nell'applicazione per iniziare immediatamente a raccogliere e visualizzare input penna.  Se tuttavia il livello di controllo offerto dall'oggetto <xref:System.Windows.Controls.InkCanvas> non è sufficiente, è possibile aumentarlo personalizzando le classi di raccolta e rendering dell'input penna tramite <xref:System.Windows.Input.StylusPlugIns>.  
  
 Le classi <xref:System.Windows.Input.StylusPlugIns> forniscono un meccanismo per l'implementazione del controllo di basso livello sull'input <xref:System.Windows.Input.Stylus> e per il rendering dinamico dell'input penna.  La classe <xref:System.Windows.Input.StylusPlugIns.StylusPlugIn> fornisce un meccanismo per implementare un comportamento personalizzato e applicarlo al flusso di dati provenienti dal dispositivo stilo per prestazioni ottimali.  L'oggetto <xref:System.Windows.Input.StylusPlugIns.DynamicRenderer>, un controllo <xref:System.Windows.Input.StylusPlugIns.StylusPlugIn> specializzato, consente di personalizzare in tempo reale i dati dell'input penna con rendering dinamico, il che significa che l'oggetto <xref:System.Windows.Input.StylusPlugIns.DynamicRenderer> traccia immediatamente l'input penna quando vengono generati i dati <xref:System.Windows.Input.StylusPoint>, in modo che sembrino "fluire" dal dispositivo stilo.
