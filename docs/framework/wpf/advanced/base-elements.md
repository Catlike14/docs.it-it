# Elementi di base
Quattro classi principali \(<xref:System.Windows.UIElement>, <xref:System.Windows.ContentElement>, <xref:System.Windows.FrameworkElement> e <xref:System.Windows.FrameworkContentElement>\) implementano una percentuale sostanziale delle funzionalità degli elementi comuni disponibili nella programmazione WPF.  Nel presente SDK, queste quattro classi vengono indicate come classi degli elementi di base.
