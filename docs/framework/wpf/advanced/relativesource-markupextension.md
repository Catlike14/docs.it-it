# <a name="relativesource-markupextension"></a><span data-ttu-id="02792-102">Estensione del markup RelativeSource</span><span class="sxs-lookup"><span data-stu-id="02792-102">RelativeSource MarkupExtension</span></span>
<span data-ttu-id="02792-103">Specifica le proprietà di un <xref:System.Windows.Data.RelativeSource> origine dell'associazione da utilizzare all'interno di un [estensione di Markup di associazione](../../../../docs/framework/wpf/advanced/binding-markup-extension.md), o quando si imposta la <xref:System.Windows.Data.Binding.RelativeSource%2A> proprietà di un <xref:System.Windows.Data.Binding> elemento in XAML.</span><span class="sxs-lookup"><span data-stu-id="02792-103">Specifies properties of a <xref:System.Windows.Data.RelativeSource> binding source, to be used within a [Binding Markup Extension](../../../../docs/framework/wpf/advanced/binding-markup-extension.md), or when setting the <xref:System.Windows.Data.Binding.RelativeSource%2A> property of a <xref:System.Windows.Data.Binding> element established in XAML.</span></span>  
  
## <a name="xaml-attribute-usage"></a><span data-ttu-id="02792-104">Uso della sintassi XAML per gli attributi</span><span class="sxs-lookup"><span data-stu-id="02792-104">XAML Attribute Usage</span></span>  
  
```xml  
<Binding RelativeSource="{RelativeSource modeEnumValue}" .../>  
```  
  
## <a name="xaml-attribute-usage-nested-within-binding-extension"></a><span data-ttu-id="02792-105">Utilizzo della sintassi XAML per gli attributi (annidati nell'estensione Binding)</span><span class="sxs-lookup"><span data-stu-id="02792-105">XAML Attribute Usage (nested within Binding extension)</span></span>  
  
```xml

```  
  
## <a name="xaml-object-element-usage"></a><span data-ttu-id="02792-106">Utilizzo della sintassi XAML per gli elementi oggetto</span><span class="sxs-lookup"><span data-stu-id="02792-106">XAML Object Element Usage</span></span>  
  
```xml  
<Binding>  
  <Binding.RelativeSource>  
    <RelativeSource Mode="modeEnumValue"/>  
  </Binding.RelativeSource>  
</Binding>  
- or   
<Binding>  
  <Binding.RelativeSource>  
    <RelativeSource  
      Mode="FindAncestor"  
      AncestorType="{x:Type typeName}"  
      AncestorLevel="intLevel"  
    />  
  </Binding.RelativeSource>  
</Binding>  
```
