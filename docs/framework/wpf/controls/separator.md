# Separatore
Un controllo <xref:System.Windows.Controls.Separator> disegna una riga, orizzontale o verticale, tra gli elementi nei controlli, ad esempio <xref:System.Windows.Controls.ListBox>, <xref:System.Windows.Controls.Menu> e <xref:System.Windows.Controls.ToolBar>.
