# TextBlock
Il controllo <xref:System.Windows.Controls.TextBlock> fornisce un supporto di testo flessibile per le applicazioni [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].  L'elemento è destinato principalmente a scenari di [!INCLUDE[TLA2#tla_ui](../../../../includes/tla2sharptla-ui-md.md)] di base che non richiedono più paragrafi di testo.
