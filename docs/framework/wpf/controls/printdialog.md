# PrintDialog
Il controllo <xref:System.Windows.Controls.PrintDialog> viene utilizzato per creare un'istanza di una finestra di dialogo di stampa standard per la configurazione automatica di oggetti <xref:System.Printing.PrintTicket> e <xref:System.Printing.PrintQueue> secondo l'input dell'utente.
