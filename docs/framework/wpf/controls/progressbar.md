# ProgressBar
<xref:System.Windows.Controls.ProgressBar> indica lo stato di avanzamento di un'operazione.  Il controllo <xref:System.Windows.Controls.ProgressBar> è costituito da una finestra che viene riempita con il colore evidenziatore di sistema nel corso dell’avanzamento di un'operazione.
