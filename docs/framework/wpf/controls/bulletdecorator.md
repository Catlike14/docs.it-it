# <a name="bulletdecorator"></a>BulletDecorator
<xref:System.Windows.Controls.Primitives.BulletDecorator>ha due proprietà di contenuto: <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> e <xref:System.Windows.Controls.Decorator.Child%2A>. Il <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> proprietà definisce il <xref:System.Windows.UIElement> da utilizzare come punto elenco. Il <xref:System.Windows.Controls.Decorator.Child%2A> proprietà definisce un <xref:System.Windows.UIElement> che viene allineato visivamente con il punto elenco.
