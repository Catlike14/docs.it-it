# Stili e modelli di Control
Windows Presentation Foundation \(WPF\) dispone di un oggetto <xref:System.Windows.Controls.ControlTemplate> che contiene la [struttura ad albero](GTMT) di quel determinato controllo.  È possibile modificare la struttura e l'aspetto di un controllo modificando il relativo oggetto <xref:System.Windows.Controls.ControlTemplate>.  Non è possibile sostituire solo una parte della struttura ad albero di un controllo. Per modificare la struttura ad albero di un controllo, è necessario impostare la proprietà <xref:System.Windows.Controls.Control.Template%2A> di tale controllo sul relativo oggetto <xref:System.Windows.Controls.ControlTemplate> nuovo e completo.  
  
 I temi del desktop determinano quale dizionario risorse viene utilizzato.  I temi del desktop determinano quale dizionario risorse viene utilizzato.  Per ottenere i dizionari risorse per i temi del desktop, vedere [Temi WPF predefiniti](http://go.microsoft.com/fwlink/?LinkID=158252).  
  
 Nella tabella seguente vengono descritti i nomi file dei dizionari risorse e i temi del desktop corrispondenti.  
  
|File di tema|Tema del desktop|  
|------------------|----------------------|  
|Classic.xaml|Aspetto di Windows classico \(Windows 95, Windows 98 e Windows 2000\) nel sistema operativo Windows XP.|  
|Luna.NormalColor.xaml|Tema blu predefinito in Windows XP.|  
|Luna.Homestead.xaml|Tema verde oliva in Windows XP.|  
|Luna.Metallic.xaml|Tema argento in Windows XP.|  
|Royale.NormalColor.xaml|Tema predefinito nel sistema operativo Windows XP Media Center Edition.|  
|Aero.NormalColor.xaml|Tema predefinito nel sistema operativo Windows Vista.|
