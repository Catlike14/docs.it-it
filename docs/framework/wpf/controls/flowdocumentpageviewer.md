# FlowDocumentPageViewer
Il controllo <xref:System.Windows.Controls.FlowDocumentPageViewer> viene utilizzato per visualizzare il contenuto di <xref:System.Windows.Documents.FlowDocument> pagina per pagina.  Si differenzia da <xref:System.Windows.Controls.FlowDocumentScrollViewer> che presenta il contenuto di <xref:System.Windows.Documents.FlowDocument> in un visualizzatore a scorrimento.
