# <a name="flowdocumentscrollviewer"></a>FlowDocumentScrollViewer
Il <xref:System.Windows.Controls.FlowDocumentScrollViewer> controllo consente di visualizzare <xref:System.Windows.Documents.FlowDocument> contenuto in un contenitore a scorrimento. Si differenzia <xref:System.Windows.Controls.FlowDocumentPageViewer>, che visualizza il contenuto per ogni pagina.
