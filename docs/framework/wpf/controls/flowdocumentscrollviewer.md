# FlowDocumentScrollViewer
Il controllo <xref:System.Windows.Controls.FlowDocumentScrollViewer> viene utilizzato per visualizzare il contenuto di <xref:System.Windows.Documents.FlowDocument> in un contenitore a scorrimento.  Si differenzia da <xref:System.Windows.Controls.FlowDocumentPageViewer>, in cui il contenuto viene visualizzato per pagina.
