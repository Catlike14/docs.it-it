# Hosting di applicazioni WPF
Le applicazioni browser XAML di WPF \(XBAP\) sono applicazioni client avanzate che è possibile distribuire in un server Web e avviate in un browser.  L'host WPF \(PresentationHost.exe\) è registrato come shell e gestore MIME per i file XAML e XBAP.  Pertanto, in Internet Explorer viene automaticamente avviato l'host WPF quando viene avviata un'applicazione XBAP.  Gli utenti di Firefox possono installare componenti aggiuntivi consentono anche a Firefox di ospitare applicazioni XBAP.  Un'applicazione XBAP può essere ospitata in altri browser o applicazioni autonome tramite le API di hosting native del browser fornite da WPF.
