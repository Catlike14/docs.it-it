# Oggetti Drawing
Gli oggetti <xref:System.Windows.Media.Drawing> vengono utilizzati per disegnare forme, immagini o testo in modo efficace.  Gli oggetti Drawing vengono utilizzati quando si eseguono disegni con un oggetto <xref:System.Windows.Media.DrawingBrush> o si utilizzano oggetti <xref:System.Windows.Media.Visual> nella programmazione.
