# Forme
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] fornisce supporto per il disegno di forme di base utilizzando la classe <xref:System.Windows.Shapes.Shape>.  Gli oggetti Shape derivano da <xref:System.Windows.FrameworkElement> e, in quanto tali, fanno parte del sistema di layout e possono essere disegnati utilizzando semplicemente [!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)].
