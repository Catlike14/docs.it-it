# Procedura: specificare campi di archiviazione privati
Usare la proprietà <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> di [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] sull'attributo <xref:System.Data.Linq.Mapping.DataAttribute> per definire il nome di un campo di archiviazione sottostante.  
  
 Per esempi di codice, vedere <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.  
  
### Per specificare il nome di un campo di archiviazione sottostante  
  
1.  Aggiungere la proprietà <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> all'attributo <xref:System.Data.Linq.Mapping.ColumnAttribute>.  
  
2.  Assegnare il nome del campo come valore della proprietà <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.
