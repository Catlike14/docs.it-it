|Argomento|Descrizione|  
|-----------|-----------------|  
|[Tipi SQL-CLR non corrispondenti](../../../../../../docs/framework/data/adonet/sql/linq/sql-clr-type-mismatches.md)|La sezione "Semantica Null" di questo argomento include una discussione relativa al tipo booleano SQL a tre stati rispetto a due stati common language runtime (CLR) <xref:System.Boolean>, il valore letterale `Nothing` (Visual Basic) e `null` (c#) e altri problemi simili.|
