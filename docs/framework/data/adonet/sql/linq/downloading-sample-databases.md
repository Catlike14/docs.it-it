# Download dei database di esempio
In numerosi esempi e procedure dettagliate nella documentazione di [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] vengono usati il database di esempio Northwind e SQL Server Express Edition.  È possibile scaricare questi prodotti gratuitamente dal sito di download Microsoft.  
  
## Download del database Northwind  
  
#### Per scaricare e installare il database di esempio Northwind per SQL Server  
  
1.  Avviare Internet Explorer.  
  
2.  Passare al sito Web relativo ai [database di esempio Northwind e Pubs](http://go.microsoft.com/fwlink?linkid=64296).  
  
3.  Scegliere **Download**.  
  
4.  Nella finestra di dialogo **Download file** scegliere **Salva**.  
  
5.  Dopo aver eseguito il download del file, fare doppio clic sul file **Nwind.exe** per installare il database.  
  
     Per impostazione predefinita, il database viene installato in *unità*:\\SQL Server 2000 Sample Databases.  
  
## Download di SQL Server Express Edition  
 SQL Server Express Edition è disponibile gratuitamente e può essere ridistribuito con le applicazioni.  Se si usa [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)], SQL Server Express Edition è incluso nelle versioni Pro e successive.  
  
#### Per scaricare e installare SQL Server Express Edition  
  
1.  Avviare Internet Explorer.  
  
2.  Passare al sito Web [Area download Microsoft](http://go.microsoft.com/fwlink?linkid=74602).  
  
3.  Digitare SQL Server Express nella casella delle parole chiave.  
  
4.  Fare clic su **Vai**.  
  
5.  Nella pagina dei risultati fare clic sul collegamento alla pagina di download **Microsoft SQL Server 2005 Express Edition**.  
  
6.  Attenersi alle istruzioni di installazione visualizzate nel sito Web.  
  
## Download di Management Studio Express  
 Se si desidera modificare un database scaricato, è possibile accedervi da **Esplora server** nell'ambiente di sviluppo integrato \(IDE\) di [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] o usare Microsoft SQL Server Management Studio Express \(SSMSE\).  
  
#### Per scaricare Management Studio Express  
  
-   Seguire le istruzioni disponibili nel [sito SSMSE](http://go.microsoft.com/fwlink/?LinkId=95933).
