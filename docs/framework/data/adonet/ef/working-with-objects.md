# Utilizzo di oggetti
[!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] consente di eseguire query sui dati e di inserire, aggiornare ed eliminare dati, espressi come oggetti CLR \(Common Language Runtime\) tipizzati che sono istanze dei tipi di entità. I tipi di entità rappresentano le entità definite nel modello concettuale.  [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] esegue il mapping di entità e relazioni definite in un modello concettuale a un'origine dati.  In [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] sono disponibili funzionalità che consentono di materializzare i dati restituiti dall'origine dati come oggetti, rilevare le modifiche apportate agli oggetti, gestire la concorrenza, propagare le modifiche agli oggetti nell'origine dati e associare oggetti a controlli.  
  
 Per altre informazioni sull'utilizzo di oggetti nella versione più recente di Entity Framework, vedere la pagina relativa all'[utilizzo di oggetti](http://go.microsoft.com/fwlink/?LinkId=235289).
