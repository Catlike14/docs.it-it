Specifica i tipi di attestazioni da rimuovere nella credenziale federativa.  
  
 \<System. ServiceModel >  
\<associazioni >  
\<wsFederatedBinding >  
\<associazione >  
\<sicurezza >  
\<messaggio >  
\<claimTypeRequirements >  
  
## <a name="syntax"></a>Sintassi  
  
```xml  
<claimTypeRequirements>  
      <remove claimType="URI" />  
</claimTypeRequirements>  
```
