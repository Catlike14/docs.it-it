# Trasporti
Il livello di trasporto è al livello più basso dello stack dei canali.  I trasporti principali usati in [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] sono HTTP, HTTPS, TCP e named pipe.  Contenuto della sezione sono elencati tutti gli elementi di configurazione che consentono di configurare tipi di trasporti diversi.
