# <a name="lttokenrequestparametersgt"></a>&lt;tokenRequestParameters&gt;
Questo elemento di configurazione contiene elementi figlio che possono essere usati per specificare elementi XML aggiuntivi da inviare nel corpo del messaggio al servizio token di sicurezza quando viene richiesto un token.
