# <a name="ltservicetimeoutsgt"></a>&lt;serviceTimeouts&gt;
Specifica il timeout per un servizio.  
  
 \<system.ServiceModel>  
\<i comportamenti >  
\<serviceBehaviors>  
\<comportamento >  
\<serviceTimeouts >  
  
## <a name="syntax"></a>Sintassi  
  
```xml  
<serviceTimeouts transactionTimeout="TimeSpan" />  
```
