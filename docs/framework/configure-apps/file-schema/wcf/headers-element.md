# <a name="ltheadersgt-element"></a>Elemento &lt;headers&gt;
Contiene una raccolta di intestazioni di indirizzo richiesta per indirizzare correttamente l'autorità emittente locale. È possibile usare la parola chiave `add` per aggiungere un'intestazione a questa raccolta.
