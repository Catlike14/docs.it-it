# Sintassi delle direttive WCF
Specifica le impostazioni usate dalle direttive di [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] nei file con estensione svc per gestire l'esecuzione dei compilatori.  Ciascuna direttiva può contenere uno o più attributi associati al relativo valore, specifici per tale direttiva.  [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] include solo la direttiva [@ServiceHost](../../../../../docs/framework/configure-apps/file-schema/wcf-directive/servicehost.md).  
  
## In questa sezione  
 [@ServiceHost](../../../../../docs/framework/configure-apps/file-schema/wcf-directive/servicehost.md)  
 Definisce gli attributi specifici della pagina usati dal compilatore dei file con estensione svc.  Può essere usata solo nei file con estensione svc.
