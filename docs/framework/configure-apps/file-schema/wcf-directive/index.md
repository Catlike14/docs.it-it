# <a name="wcf-directive-syntax"></a>Sintassi delle direttive WCF
Specifica le impostazioni utilizzate dalle direttive di Windows Communication Foundation (WCF) nei file con estensione svc per indirizzare i compilatori. Ciascuna direttiva può contenere uno o più attributi associati al relativo valore, specifici per tale direttiva. [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] include solo la direttiva [@ServiceHost](../../../../../docs/framework/configure-apps/file-schema/wcf-directive/servicehost.md).  
  
## <a name="in-this-section"></a>In questa sezione  
 [@ServiceHost](../../../../../docs/framework/configure-apps/file-schema/wcf-directive/servicehost.md)  
 Definisce gli attributi specifici della pagina usati dal compilatore dei file con estensione svc. Può essere usata solo nei file con estensione svc.
