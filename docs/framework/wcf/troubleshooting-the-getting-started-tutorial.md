<span data-ttu-id="2ec34-123">**La compilazione dell'applicazione client: il nome di tipo o spazio dei nomi 'CalculatorClient' non è stato trovato (probabilmente manca un utilizzo della direttiva o un riferimento all'assembly?)**</span><span class="sxs-lookup"><span data-stu-id="2ec34-123">**Compiling the client application: The type or namespace name 'CalculatorClient' could not be found (are you missing a using directive or an assembly reference?)**</span></span>
