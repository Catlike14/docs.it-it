# <a name="wcf-system-requirements"></a>Requisiti di sistema per WCF
[!INCLUDE[indigo1](../../../includes/indigo1-md.md)] è un'infrastruttura di comunicazione usata per creare applicazioni distribuite. Di seguito vengono elencati i requisiti che consentono di eseguire [!INCLUDE[indigo2](../../../includes/indigo2-md.md)].  
  
## <a name="system-requirements"></a>Requisiti di sistema  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] viene installato per impostazione predefinita su [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] può essere installato anche su [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)] o [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Nota** funzionalità di Accodamento messaggi (MSMQ) di [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] è supportato solo in [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], e [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
