# <a name="wcf-system-requirements"></a>Requisiti di sistema per WCF
Windows Communication Foundation (WCF) è un'infrastruttura di comunicazione che consente di creare applicazioni distribuite. Di seguito sono elencati i requisiti che consentono di WCF per l'esecuzione.  
  
## <a name="system-requirements"></a>Requisiti di sistema  
 WCF è installato per impostazione predefinita nel [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 WCF può anche essere installato in [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], o [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Nota** funzionalità di Accodamento messaggi (MSMQ) di WCF è supportata solo sugli [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], e [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
