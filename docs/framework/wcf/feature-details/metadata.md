# <a name="metadata"></a>Metadati
Windows Communication Foundation (WCF) offre un'infrastruttura per l'esportazione, la pubblicazione, il recupero e l'importazione di metadati del servizio. Servizi WCF utilizzano i metadati per descrivere come interagire con gli endpoint del servizio affinché strumenti, ad esempio Svcutil.exe, possono generare automaticamente il codice client per accedere al servizio.
