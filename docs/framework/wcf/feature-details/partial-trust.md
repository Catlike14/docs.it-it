# <a name="partial-trust"></a>Attendibilità parziale
A partire da [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)], i chiamanti parzialmente attendibili possono accedere a tipi e metodi pubblici implementati in <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>e <xref:System.ServiceModel.Web>. Contenuto della sezione vengono illustrati gli scenari supportati per utilizzare [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] all'interno di un'applicazione parzialmente attendibile, così come il sottoinsieme limitato di funzionalità di [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] disponibile per le applicazioni che sono in esecuzione con autorizzazioni ridotte di sicurezza dall'accesso di codice (CAS, Code Access Security).
