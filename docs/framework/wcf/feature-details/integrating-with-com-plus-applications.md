# <a name="integrating-with-com-applications"></a>Integrazione con applicazioni COM+
Windows Communication Foundation (WCF) fornisce un ambiente completo per la creazione di applicazioni distribuite. Se si dispone di una grande quantità di logica dell'applicazione basata su componenti ospitata in COM+, è possibile utilizzare WCF per estendere la logica esistente anziché riscriverla. Negli argomenti di questa sezione viene descritto come utilizzare COM+ con WCF.
