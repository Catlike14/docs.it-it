9. <span data-ttu-id="1d26d-125">Copiare i caratteri esadecimali dalla casella.</span><span class="sxs-lookup"><span data-stu-id="1d26d-125">Copy the hexadecimal characters from the box.</span></span> <span data-ttu-id="1d26d-126">Se questa identificazione personale viene usata nel codice per `X509FindType`, rimuovere gli spazi tra i numeri esadecimali.</span><span class="sxs-lookup"><span data-stu-id="1d26d-126">If this thumbprint is used in code for the `X509FindType`, remove the spaces between the hexadecimal numbers.</span></span> <span data-ttu-id="1d26d-127">Ad esempio, l'identificazione personale "a9 09 50 2d d8 2a e4 14 33 e6 f8 38 86 b0 0d che 42 77 a3 2a 7b" nel codice deve essere specificata come "a909502dd82ae41433e6f83886b00d4277a32a7b".</span><span class="sxs-lookup"><span data-stu-id="1d26d-127">For example, the thumbprint "a9 09 50 2d d8 2a e4 14 33 e6 f8 38 86 b0 0d 42 77 a3 2a 7b" should be specified as "a909502dd82ae41433e6f83886b00d4277a32a7b" in code.</span></span>
