# <a name="hosting"></a>Hosting
Negli argomenti di questa sezione viene descritto l'hosting dei servizi. Un servizio può essere ospitato da Internet Information Services (IIS), servizio Attivazione processo Windows (WAS), Windows Server AppFabric, un servizio Windows o da un'applicazione gestita, questa opzione è noto anche come *self-hosting*.  
  
 È importante notare che l'esecuzione di un servizio o di una qualsiasi estensione da un host non attendibile compromette la sicurezza.
