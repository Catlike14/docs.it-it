# <a name="interoperability-with-web-services-enhancements-30"></a>Interoperabilità con Web Services Enhancements 3.0
Client Windows Communication Foundation (WCF) e servizi sono compatibili a livello di transito con Web Services Enhancements 3.0 per Microsoft .NET (WSE) client e servizi quando i client WCF e servizi sono configurati per usare la versione dell'agosto 2004 di Specifica WS-Addressing.

 Viene descritto come configurare un servizio WCF, in modo che un Web Services Enhancements 3.0 per Microsoft .NET (WSE) client possa comunicare con esso.  
  
 [Procedura: Configurare un client WCF per l'interoperabilità con servizi WSE 3.0](../../../../docs/framework/wcf/feature-details/how-to-configure-a-wcf-client-to-interoperate-with-wse3-0-services.md)  
 Illustra in dettaglio come configurare un client WCF, in modo che comunica con Web Services Enhancements 3.0 per il servizio di Microsoft .NET (WSE).
