# Integrazione AJAX e supporto JSON
Il supporto [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] per AJAX \(ASP.NET Asynchronous JavaScript e XML\) e il formato dati JSON \(JavaScript Object Notation\) consentono ai servizi di [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] di esporre operazioni ai client AJAX.I client AJAX sono pagine Web che eseguono codice JavaScript e accedono a questi servizi di [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] utilizzando richieste HTTP.Negli argomenti di questa sezione vengono fornite informazioni su questo supporto e su come implementarlo.
