# <a name="core-communications-internal-duplex-transport-channels"></a>Comunicazioni principali: canali di trasporto duplex interni
In questo argomento vengono elencate tutte le eccezioni generate dai canali di trasporto Duplex interni di Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Elenco delle eccezioni  
  
|Codice risorsa|Stringa di risorsa|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|Per inviare un messaggio su canali duplex compositi del server, è necessario che sia impostata la proprietà "Via" o l'intestazione "A" del messaggio.|
