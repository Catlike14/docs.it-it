# Utilit&#224; di comunicazione principali
In questo argomento vengono elencate tutte le eccezioni generate dalle utilità di comunicazione principali.  
  
## Elenco delle eccezioni  
  
|Codice risorsa|Stringa di risorsa|  
|--------------------|------------------------|  
|AsyncObjectAlreadyEnded|Il metodo End è già stato chiamato su questo oggetto del risultato asincrono.|
