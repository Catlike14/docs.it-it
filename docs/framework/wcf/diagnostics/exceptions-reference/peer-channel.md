# <a name="peer-channel"></a>Canale del peer
In questo argomento vengono elencate tutte le eccezioni generate dal canale peer [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
## <a name="exception-list"></a>Elenco delle eccezioni  
  
|Codice risorsa|Stringa di risorsa|  
|-------------------|---------------------|  
|InsufficientCredentials|Le credenziali specificate non sono sufficienti a eseguire l'operazione richiesta. Specificare un valore valido per l'operazione specificata|  
|InvalidResolverMode|Il valore PeerResolverMode specificato non è valido. Specificare PeerResolveMode.Auto, Predefinito o PNRP.|  
|PeerNodeAborted|PeerNode non può essere aperto perché è stato terminato.|  
|PeerNullRegistrationInfo|Le informazioni di registrazione non possono essere null. Assicurarsi che l'operazione di registrazione venga richiamata con un oggetto RegistrationInfo valido.|  
|PeerNullResolveInfo|Le informazioni di risoluzione non possono essere `null`. Assicurarsi che l'operazione di risoluzione venga richiamata con un oggetto ResolveInfo valido.|
