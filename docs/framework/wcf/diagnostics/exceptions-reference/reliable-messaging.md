# <a name="reliable-messaging"></a>Messaggistica affidabile
In questo argomento vengono elencate tutte le eccezioni generate dalla messaggistica affidabile.  
  
## <a name="exception-list"></a>Elenco delle eccezioni  
  
|Codice risorsa|Stringa di risorsa|  
|-------------------|---------------------|  
|InvalidBufferRemaining|È stato ricevuto un messaggio dal buffer con la dimensione rimanente specificata. Il buffer rimanente non può essere minore di zero.|
