# Comunicazioni principali: framework dei canali
In questo argomento vengono elencate tutte le eccezioni generate dal framework dei canali di [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
## Elenco delle eccezioni  
  
|Codice sorgente|Stringa di risorsa|  
|---------------------|------------------------|  
|ReceiveRequestTimedOut|La richiesta ricevuta all'indirizzo locale specificato è scaduta dopo il tempo indicato.È possibile che la durata consentita per l'operazione fosse una porzione di un timeout più lungo.|  
|ReceiveTimedOutNoLocalAddress|L'operazione di ricezione è scaduta dopo il tempo specificato.È possibile che la durata consentita per l'operazione fosse una porzione di un timeout più lungo.|  
|UnableToDemuxChannel|Non è disponibile alcun canale per accettare il messaggio con l'azione specificata.|
