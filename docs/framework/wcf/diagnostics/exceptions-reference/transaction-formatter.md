# <a name="transaction-formatter"></a>Formattatore delle transazioni
In questo argomento vengono elencate tutte le eccezioni generate dal formattatore delle transazioni.  
  
## <a name="exception"></a>Eccezione  
  
|Codice risorsa|Stringa di risorsa|  
|-------------------|---------------------|  
|TMCommunicationError|Si è verificato un errore durante la comunicazione con il gestore di transazioni distribuite.|  
|WhereaboutsNoHostName|Nelle informazioni di percorso del gestore MSDTC non è specificato alcun nome host.|  
|WhereaboutsReadFailed|Impossibile deserializzare le informazioni di percorso del gestore MSDTC.|  
|WhereaboutsSignatureMissing|Nelle informazioni di percorso del gestore MSDTC non è specificata alcuna firma del percorso standard.|  
|WsatUriCreationFailed|Impossibile creare un indirizzo del servizio di registrazione a partire dalle informazioni di percorso del gestore MSDTC.|
