# TransactionBridgeRecoveryFailure
Id: 146  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## Descrizione  
 Questo evento indica che si sono verificati errori di MSDTC TransactionBridge durante il ripristino.È una condizione irreversibile, con conseguente chiusura del servizio MSDTC.Nell'evento vengono specificati l'eccezione e il nome e l'ID del processo.
