# FailedToInitializeTraceSource
Id: 101  
  
 Gravità: errore  
  
 Categoria: traccia  
  
## Descrizione  
 L'origine della traccia non è stata inizializzata.  La traccia è disattivata.  Nell'evento vengono indicati l'eccezione, il nome del processo e l'ID del processo.
