# WmiExecMethodFailed
Id: 125  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che la richiesta WMI ExecMethod non è stata elaborata.Vengono indicati l'eccezione nonché il nome e l'ID del processo.
