# MessageLoggingOff
Id: 117  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che la registrazione dei messaggi è stata disattivata.Vengono indicati il nome e l'ID del processo.
