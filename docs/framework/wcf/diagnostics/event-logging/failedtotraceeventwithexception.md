# <a name="failedtotraceeventwithexception"></a>FailedToTraceEventWithException
Id: 105  
  
 Gravità: errore  
  
 Categoria: traccia  
  
## <a name="description"></a>Descrizione  
 Uno o più eventi non sono tracciati.  
  
 Vengono indicati la stringa dell'evento originale, l'eccezione, il nome e l'ID del processo.
