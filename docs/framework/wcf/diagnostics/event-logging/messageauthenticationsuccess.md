# MessageAuthenticationSuccess
Id: 169  
  
 Gravità: informativo  
  
 Categoria: SecurityAudit  
  
## Descrizione  
 Questo evento indica che l'autenticazione a livello di messaggio ha avuto esito positivo.Nell'evento vengono indicati il servizio, l'azione, l'identità client e l'ID attività.
