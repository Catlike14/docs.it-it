# <a name="piiloggingon"></a>PiiLoggingOn
ID: 107  
  
 Gravità: errore  
  
 Categoria: traccia  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che la registrazione delle informazioni personali è stata attivata. Le informazioni riservate verranno registrate. Vengono indicati il nome e l'ID del processo.
