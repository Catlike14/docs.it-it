# <a name="bindingerror"></a>BindingError
ID: 159  
  
 Gravità: errore  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificato un errore durante l'analisi dell'associazione specifica del sito specifico, pertanto il protocollo è temporaneamente disattivato per il sito. Vengono indicati l'associazione, l'origine, l'eccezione, il nome e l'ID del processo.
