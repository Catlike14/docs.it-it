# ProtocolRecoveryBeginningFailure
Id: 144  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## Descrizione  
 Questo evento indica un errore del protocollo WS\-Atomic Transaction MSDTC all'inizio del ripristino.La funzionalità WS\-Atomic Transaction viene pertanto disattivata.Vengono indicati l'ID e il nome del protocollo, l'eccezione, il nome e l'ID del processo.
