# FailedToLogMessage
Id: 113  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che un messaggio non è stato registrato.Vengono indicati l'eccezione nonché il nome e l'ID del processo.
