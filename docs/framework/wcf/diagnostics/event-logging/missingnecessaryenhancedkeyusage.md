# <a name="missingnecessaryenhancedkeyusage"></a>MissingNecessaryEnhancedKeyUsage
ID: 157  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che un certificato di identità con il nome soggetto e l'identificazione personale specificati non fornisce i dati specifici tra i valori EnhancedKeyUsages. Vengono indicati il nome e l'ID del processo.
