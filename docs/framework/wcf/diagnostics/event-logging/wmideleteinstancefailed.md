# WmiDeleteInstanceFailed
Id: 122  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che la richiesta WMI DeleteInstance non è stata elaborata.Nell'evento vengono specificati l'eccezione, il nome del processo e l'ID del processo.
