# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated
Id: 153  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che non è stato possibile convalidare un certificato di identità con l'identificazione personale specificata. Vengono indicati il nome e l'ID del processo.
