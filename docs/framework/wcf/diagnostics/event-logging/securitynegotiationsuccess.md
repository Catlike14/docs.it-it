# SecurityNegotiationSuccess
Id: 171  
  
 Gravità: informativo  
  
 Categoria: SecurityAudit  
  
## Descrizione  
 Questo evento indica che la negoziazione di sicurezza è andata a buon fine, elencando nome del servizio, azione, ClientIdentity, ActivityId e negoziazione.
