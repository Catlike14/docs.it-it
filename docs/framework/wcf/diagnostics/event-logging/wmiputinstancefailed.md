# WmiPutInstanceFailed
ID: 121  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che la richiesta WMI PutInstance non è stata elaborata.Vengono indicati l'oggetto WMI, l'eccezione, il nome e l'ID del processo.
