# <a name="impersonationfailure"></a>ImpersonationFailure
ID: 176  
  
 Gravità: errore  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che la rappresentazione non è riuscita ed elenca il nome del metodo, l'identità del client e l'ID attività.
