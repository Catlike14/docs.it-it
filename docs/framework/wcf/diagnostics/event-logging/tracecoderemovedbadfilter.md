# TraceCodeRemovedBadFilter
ID: 114  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che è stato rimosso un filtro di registrazione dei messaggi non valido che stava accedendo al corpo del messaggio.L'elenco comprende XPath, nome e ID del processo.
