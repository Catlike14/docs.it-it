# WmiPropertyMissing
ID: 129  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che si è verificata un'incongruenza di tipo.Per la classe CIM specificata non è disponibile la proprietà specificata del tipo specificato.Vengono indicati il nome e l'ID del processo.
