# WmiGetObjectFailed
Id: 120  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che la richiesta WMI GetObject non è stata elaborata.Vengono indicati l'oggetto WMI, l'eccezione, il nome e l'ID del processo.
