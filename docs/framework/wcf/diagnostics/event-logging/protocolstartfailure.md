# ProtocolStartFailure
Id: 143  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## Descrizione  
 Questo evento indica che è stato impossibile avviare il servizio del protocollo WS\-Atomic Transaction.  La funzionalità WS\-Atomic Transaction viene pertanto disattivata.  Vengono indicati l'ID e il nome del protocollo, l'eccezione, il nome e l'ID del processo.
