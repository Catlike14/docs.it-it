# <a name="webhostunhandledexception"></a>WebHostUnhandledException
ID: 109  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificata un'eccezione WebHost non gestita. Vengono indicati informazioni sul mittente, eccezione, nome e ID del processo.
