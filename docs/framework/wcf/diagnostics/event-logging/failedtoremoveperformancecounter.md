# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
Id: 119  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che un contatore delle prestazioni non è stato rimosso. L'evento elenca il nome della categoria, il nome del contatore, il nome dell'istanza, l'eccezione, il nome e l'ID del processo.
