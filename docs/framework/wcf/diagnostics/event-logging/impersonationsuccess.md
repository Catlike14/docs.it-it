# ImpersonationSuccess
ID: 175  
  
 Gravità: informativo  
  
 Categoria: registro di controllo  
  
## Descrizione  
 Questo evento indica che la rappresentazione è riuscita ed elenca il nome del metodo, l'identità del client e l'ID attività.
