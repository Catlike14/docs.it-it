# ThumbPrintNotFound
ID: 152  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## Descrizione  
 Questo evento indica che non è stato trovato un certificato di identità con l'identificazione personale specificata.Vengono indicati il nome e l'ID del processo.
