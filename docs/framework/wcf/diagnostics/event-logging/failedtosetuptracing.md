# FailedToSetupTracing
Id: 100  
  
 Gravità: errore  
  
 Categoria: traccia  
  
## Descrizione  
 Tentativo di configurare la traccia è riuscito.  La traccia è disattivata.  
  
 Nell'evento vengono indicati l'eccezione, il nome del processo e l'ID del processo.
