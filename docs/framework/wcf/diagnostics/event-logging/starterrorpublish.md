# StartErrorPublish
Id: 158  
  
 Gravità: errore  
  
 Categoria: SMSvcHost  
  
## Descrizione  
 Questo evento indica che l'avvio del servizio NT non è riuscito.  Impossibile pubblicare l'endpoint di attesa.  Vengono indicati il nome e l'ID del processo.
