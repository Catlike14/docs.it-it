# WmiUnregistrationFailed
Id: 127  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che la registrazione del provider WMI non è stata annullata.Vengono indicati l'oggetto WMI, l'errore, il nome e l'ID del processo.
