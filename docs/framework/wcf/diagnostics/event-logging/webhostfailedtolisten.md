# WebHostFailedToListen
Id: 112  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che si è verificato un errore durante il tentativo di ascolto dell'URL specifico.Questo processo di lavoro viene interrotto.Nell'evento vengono indicati l'eccezione, il nome e l'ID del processo.
