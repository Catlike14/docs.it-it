# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
ID: 176  
  
 Gravità: informativo  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che l'autorizzazione del servizio è riuscita. Vengono indicati il servizio, l'azione, l'identità client, il contesto di autorizzazione, l'ID attività e ServiceAuthorizationManager.
