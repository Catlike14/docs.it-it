# <a name="wasconnectiontimedout"></a>WasConnectionTimedout
Id: 163  
  
 Gravità: errore  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che una connessione richiesta dal servizio di attivazione dei processi di Windows è scaduta, elencando il nome e l'ID del processo.
