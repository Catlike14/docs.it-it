# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
ID: 137  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica un tentativo di elaborazione di un evento imprevisto da parte di una macchina a stati. L'evento è stato considerato irreversibile. L'evento elenca l'ID transazione, il nome della macchina a stati, lo stato corrente, la cronologia, il nome e i dettagli dell'evento nonché il nome processo e l'ID processo.
