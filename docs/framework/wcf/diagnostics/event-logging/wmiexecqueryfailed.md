# <a name="wmiexecqueryfailed"></a>WmiExecQueryFailed
ID: 124  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che la richiesta WMI ExecQuery non è stata elaborata. Vengono indicati l'eccezione nonché il nome e l'ID del processo.
