# <a name="protocolstopped"></a>ProtocolStopped
Id: 151  
  
 Gravità: errore  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che il servizio del protocollo WS-Atomic Transaction è stato arrestato. Vengono indicati l'ID e il nome del protocollo e del processo. .
