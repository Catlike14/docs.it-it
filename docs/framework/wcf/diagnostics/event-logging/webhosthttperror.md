# WebHostHttpError
Id: 110  
  
 Gravità: errore  
  
 Categoria: ServiceModel  
  
## Descrizione  
 Questo evento indica che si è verificata un'eccezione HTTP WebHost.  Vengono indicati informazioni sul mittente, HtmlErrorMessage, eccezione, nome e ID del processo.
