# TransportAuthenticationSuccess
Id: 173  
  
 Gravità: informativo  
  
 Categoria: SecurityAudit  
  
## Descrizione  
 Questo evento indica che l'autenticazione a livello di trasporto ha avuto esito positivo.Nell'evento vengono indicati il servizio, l'identità client e l'ID attività.
