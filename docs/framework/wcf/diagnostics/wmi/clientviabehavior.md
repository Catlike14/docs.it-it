# ClientViaBehavior
ClientViaBehavior  
  
## Sintassi  
  
```  
class ClientViaBehavior : Behavior  
{  
  string Uri;  
};  
```  
  
## Metodi  
 La classe ClientViaBehavior non definisce metodi.  
  
## Proprietà  
 La classe ClientViaBehavior dispone della proprietà seguente:  
  
### Uri  
 Tipo di dati: stringa  
  
 Tipo di accesso: in sola lettura  
  
 ViaUri.  
  
## Requisiti  
  
|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-------------------------------------|  
|Spazio dei nomi|Definito in root\\ServiceModel|
