# <a name="transactionflowattribute"></a>TransactionFlowAttribute
TransactionFlowAttribute  
  
## <a name="syntax"></a>Sintassi  
  
```  
class TransactionFlowAttribute : Behavior  
{  
  string TransactionFlowOption;  
};  
```  
  
## <a name="methods"></a>Metodi  
 La classe TransctionFlowAttribute non definisce metodi.  
  
## <a name="properties"></a>Proprietà  
 La classe TransactionFlowAttribute dispone della proprietà seguente:  
  
### <a name="transactionflowoption"></a>TransactionFlowOption  
 Tipo di dati: stringa  
  
 Tipo di accesso: sola lettura  
  
 Indica se viene utilizzato o meno il flusso delle transazioni.  
  
## <a name="requirements"></a>Requisiti  
  
|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-----------------------------------|  
|Spazio dei nomi|Definito in root\ServiceModel|
