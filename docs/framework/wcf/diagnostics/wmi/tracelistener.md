# <a name="tracelistener"></a>TraceListener
TraceListener.  
  
## <a name="syntax"></a>Sintassi  
  
```  
class TraceListener  
{  
  string Name;  
  TraceListenerArgument TraceListenerArguments[];  
};  
```  
  
## <a name="methods"></a>Metodi  
 La classe TraceListener non definisce metodi.  
  
## <a name="properties"></a>Proprietà  
 La classe TraceListener dispone delle proprietà seguenti:  
  
### <a name="name"></a>Nome  
 Tipo di dati: stringa  
  
 Tipo di accesso: sola lettura  
  
 Nome del listener di traccia.  
  
### <a name="tracelistenerarguments"></a>TraceListenerArguments  
 Tipo di dati: matrice di TraceListenerArgument  
  
 Tipo di accesso: sola lettura  
  
 Argomenti del listener di traccia.

|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-----------------------------------|  
|Spazio dei nomi|Definito in root\ServiceModel|
