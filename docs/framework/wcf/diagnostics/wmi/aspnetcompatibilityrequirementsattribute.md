# AspNetCompatibilityRequirementsAttribute
AspNetCompatibilityRequirementsAttribute  
  
## Sintassi  
  
```  
class AspNetCompatibilityRequirementsAttribute : Behavior  
{  
  string RequirementsMode;  
};  
```  
  
## Metodi  
 La classe AspNetCompatibilityRequirementsAttribute non definisce metodi.  
  
## Proprietà  
 La classe AspNetCompatibilityRequirementsAttribute dispone della proprietà seguente.  
  
### RequirementsMode  
 Tipo di dati: stringa  
  
 Tipo di accesso: in sola lettura.  
  
 Indica se è attiva la modalità di compatibilità ASP.NET.  
  
## Requisiti  
  
|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-------------------------------------|  
|Spazio dei nomi|Definito in root\\ServiceModel|
