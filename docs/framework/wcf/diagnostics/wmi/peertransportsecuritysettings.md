# PeerTransportSecuritySettings
PeerTransportSecuritySettings  
  
## Sintassi  
  
```  
class PeerTransportSecuritySettings  
{  
  string CredentialType;  
};  
```  
  
## Metodi  
 La classe PeerTransportSecuritySettings non definisce alcun metodo.  
  
## Proprietà  
 La classe PeerTransportSecuritySettings ha la proprietà seguente:  
  
### CredentialType  
 Tipo di dati: stringa  
  
 Tipo di accesso: in sola lettura.  
  
 Tipo di credenziale di trasporto dell'elemento di sicurezza del peer.  
  
## Requisiti  
  
|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-------------------------------------|  
|Spazio dei nomi|Definito in root\\ServiceModel|
