# <a name="msmqintegrationbindingelement"></a>MsmqIntegrationBindingElement
MsmqIntegrationBindingElement  
  
## <a name="syntax"></a>Sintassi  
  
```  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## <a name="methods"></a>Metodi  
 La classe MsmqIntegrationBindingElement non definisce metodi.  
  
## <a name="properties"></a>Proprietà  
 La classe MsmqIntegrationBindingElement dispone della proprietà seguente:  
  
### <a name="serializationformat"></a>SerializationFormat  
 Tipo di dati: stringa  
  
 Tipo di accesso: sola lettura  
  
 Formato usato dall'associazione per serializzare messaggi.

|Spazio dei nomi|Definito in root\ServiceModel|
