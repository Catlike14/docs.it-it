# <a name="privacynoticebindingelement"></a>PrivacyNoticeBindingElement
PrivacyNoticeBindingElement  
  
## <a name="syntax"></a>Sintassi  
  
```  
class PrivacyNoticeBindingElement : BindingElement  
{  
  sint32 PrivacyNoticeVersion;  
  string Url;  
};  
```  
  
## <a name="methods"></a>Metodi  
 La classe PrivacyNoticeBindingElement non definisce metodi.  
  
## <a name="properties"></a>Proprietà  
 La classe PrivacyNoticeBindingElement dispone delle proprietà seguenti:  
  
### <a name="privacynoticeversion"></a>PrivacyNoticeVersion  
 Tipo di dati: sint32  
  
 Tipo di accesso: sola lettura  
  
 Versione dell'informativa sulla privacy.  
  
### <a name="url"></a>Url  
 Tipo di dati: stringa  
  
 Tipo di accesso: sola lettura  
  
 URL dell'informativa sulla privacy.

|---------|-----------------------------------|  
|Spazio dei nomi|Definito in root\ServiceModel|
