# <a name="getoperationcounterinstancename"></a>GetOperationCounterInstanceName
Recupera il nome dell'istanza del contatore delle prestazioni per l'operazione.  
  
## <a name="syntax"></a>Sintassi  
  
```  
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## <a name="parameters"></a>Parametri  
  
### <a name="operation"></a>Operazione  
 [in] stringa  
  
## <a name="requirements"></a>Requisiti  
  
|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-----------------------------------|  
|Spazio dei nomi|Definito in root\ServiceModel|
