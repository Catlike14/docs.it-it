# PeerResolverBindingElement
PeerResolverBindingElement  
  
## Sintassi  
  
```  
class PeerResolverBindingElement : BindingElement  
{  
  string ReferralPolicy;  
};  
```  
  
## Metodi  
 La classe PeerResolverBindingElement non definisce metodi.  
  
## Proprietà  
 La classe PeerResolverBindingElement dispone della proprietà seguente:  
  
### ReferralPolicy  
 Tipo di dati: stringa  
  
 Tipo di accesso: in sola lettura.  
  
 Determina come vengono condivisi i riferimenti tra i peer.

|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-------------------------------------|  
|Spazio dei nomi|Definito in root\\ServiceModel|
