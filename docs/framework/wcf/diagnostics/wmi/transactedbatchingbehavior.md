# TransactedBatchingBehavior
TransactedBatchingBehavior  
  
## Sintassi  
  
```  
class TransactedBatchingBehavior : Behavior  
{  
  sint32 MaxBatchSize;  
};  
```  
  
## Metodi  
 La classe TransactedBatchingBehavior non definisce metodi.  
  
## Proprietà  
 La classe TransactedBatchingBehavior dispone della proprietà seguente:  
  
### MaxBatchSize  
 Tipo di dati: sint32  
  
 Tipo di accesso: in sola lettura.  
  
 Dimensione massima del batch transazionale.  
  
## Requisiti  
  
|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-------------------------------------|  
|Spazio dei nomi|Definito in root\\ServiceModel|
