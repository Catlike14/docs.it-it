# Classe Behavior
## Sintassi  
  
```  
class Behavior  
{  
  string Type;  
};  
```  
  
## Metodi  
 La classe Behavior non definisce metodi.  
  
## Proprietà  
 La classe Behavior dispone della proprietà seguente:  
  
### Tipo  
 Tipo di dati: stringa  
  
 Tipo di accesso: in sola lettura.  
  
 Tipo del comportamento.  
  
## Requisiti  
  
|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-------------------------------------|  
|Spazio dei nomi|Definito in root\\ServiceModel|
