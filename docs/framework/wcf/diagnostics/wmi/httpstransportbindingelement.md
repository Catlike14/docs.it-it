# <a name="httpstransportbindingelement"></a>HttpsTransportBindingElement
HttpsTransportBindingElement  
  
## <a name="syntax"></a>Sintassi  
  
```  
class HttpsTransportBindingElement : HttpTransportBindingElement  
{  
  boolean RequireClientCertificate;  
};  
```  
  
## <a name="methods"></a>Metodi  
 La classe HttpsTransportBindingElement non definisce metodi.  
  
## <a name="properties"></a>Proprietà  
 La classe HttpsTransportBindingElement dispone della proprietà seguente:  
  
### <a name="requireclientcertificate"></a>RequireClientCertificate  
 Tipo di dati: booleano  
  
 Tipo di accesso: sola lettura  
  
 Valore che indica se l'autenticazione client SSL è necessaria.

|MOF|Dichiarato in Servicemodel.mof.|  
|---------|-----------------------------------|  
|Spazio dei nomi|Definito in root\ServiceModel|
