# <a name="service-calls-duration"></a>Del servizio: Durata delle chiamate
Nome contatore: Durata delle chiamate  
  
## <a name="description"></a>Descrizione  
 Durata media delle chiamate a questo servizio. La durata media viene calcolata in base a questa equazione: (N1-N0)/(D1-D0).
