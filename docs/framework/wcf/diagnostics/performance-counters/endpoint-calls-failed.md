# <a name="endpoint-calls-failed"></a>Endpoint: chiamate non riuscite
Nome contatore: chiamate non riuscite  
  
## <a name="description"></a>Descrizione  
 Numero di chiamate all'endpoint con eccezioni non gestite. Questo contatore viene incrementato ogni volta che si verifica un'eccezione non gestita all'endpoint.
