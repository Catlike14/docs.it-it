# Messaggi in coda rifiutati
Nome contatore: messaggi in coda rifiutati.  
  
## Descrizione  
 Numero dei messaggi per il servizio rifiutati dal trasporto in coda.  
  
 Per ulteriori informazioni sulle circostanze che determinano il rifiuto di un messaggio, vedere l'articolo sulla [gestione dei messaggi non elaborabili](http://go.microsoft.com/fwlink/?LinkID=96233) \(la pagina potrebbe essere in inglese\).
