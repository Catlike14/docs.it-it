# <a name="systemservicemodelchannelsmessagesent"></a>System.ServiceModel.Channels.MessageSent
System.ServiceModel.Channels.MessageSent  
  
## <a name="description"></a>Descrizione  
 Inviato un messaggio tramite canale.  
  
 Questa traccia viene emessa all'invio di un messaggio [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] se l'origine di traccia `System.ServiceModel` è attivata a livello di informazioni o di dettaglio. Questa traccia è necessaria per la visualizzazione del tasto di direzione della correlazione tra i messaggi nella visualizzazione del grafico delle attività.
