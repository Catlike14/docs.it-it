# Microsoft.Transactions.TransactionBridge.RegisterParticipant
Il servizio del protocollo WS\-Atomic Transaction ha registrato correttamente un partecipante per un protocollo di controllo.  
  
## Descrizione  
 Tracciato quando il gestore transazioni locale riconosce correttamente una richiesta Register da un partecipante per un protocollo 2pc.
