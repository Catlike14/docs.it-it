# <a name="systemservicemodelchannelspeerneighbormessagereceived"></a>System.ServiceModel.Channels.PeerNeighborMessageReceived
Messaggio ricevuto da un router adiacente peer.  
  
## <a name="description"></a>Descrizione  
 Questa traccia si verifica durante l'elaborazione di un messaggio in arrivo da un nodo nell'elenco dei router adiacenti.
