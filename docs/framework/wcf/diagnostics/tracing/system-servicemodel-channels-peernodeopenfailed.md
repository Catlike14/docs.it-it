# System.ServiceModel.Channels.PeerNodeOpenFailed
Tentativo di apertura di una connessione con un router adiacente non riuscito.  
  
## Descrizione  
 Questa traccia si verifica dopo il tentativo di stabilire una connessione con un router adiacente.  Le possibili cause possono includere un resolver che non risponde, un'impostazione di un'associazione non corretta o errori di rete temporanei.  
  
## Risoluzione dei problemi  
 Verificare che entrambi nodi abbiano indirizzi validi e che siano configurati in modo appropriato.
