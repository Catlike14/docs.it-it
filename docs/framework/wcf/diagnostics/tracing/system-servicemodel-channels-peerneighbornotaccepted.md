# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Un elemento adiacente peer non è stato accettato.  
  
## <a name="description"></a>Descrizione  
 Questa traccia si verifica durante l'elaborazione di una connessione a un elemento adiacente. Tra le possibili cause vi sono le impostazioni di sicurezza non corrispondenti, il timeout scaduto, un elemento adiacente non in grado di accettare una connessione oppure problemi di rete temporanei.  
  
## <a name="troubleshooting"></a>Risoluzione dei problemi  
 Verificare se le impostazioni di sicurezza non sono corrispondenti, il timeout è scaduto, un elemento adiacente non è in grado di accettare una connessione o vi sono problemi di rete temporanei.
