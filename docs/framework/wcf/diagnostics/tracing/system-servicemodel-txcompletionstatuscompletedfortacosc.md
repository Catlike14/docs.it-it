# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC
La transazione specifica è stata completata al momento della chiusura della sessione.  
  
## <a name="description"></a>Descrizione  
 Traccia eseguita quando la sessione attiva corrente viene chiusa e l'elemento `TransactionAutoCompleteOnSessionClose` è impostato su `true`.
