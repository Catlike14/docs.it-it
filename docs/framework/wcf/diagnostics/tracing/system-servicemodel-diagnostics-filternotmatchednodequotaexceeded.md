# System.ServiceModel.Diagnostics.FilterNotMatchedNodeQuotaExceeded
System.ServiceModel.Diagnostics.FilterNotMatchedNodeQuotaExceeded  
  
## Descrizione  
 La valutazione del filtro di registrazione dei messaggi a fronte del messaggio ha superato la quota del nodo impostata nel filtro.
