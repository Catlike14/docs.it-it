System.ServiceModel.Activation.WebHostCompilation  
  
## Descrizione  
 Si verifica quando un servizio ospitato da Internet Information Service \(IIS\) o da Windows Process Activation Service \(WAS\) viene compilato da codice inline presente nel file con estensione svc.
