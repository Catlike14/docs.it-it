# System.ServiceModel.CommunicationObjectOpenFailed
Apertura non riuscita di un oggetto di comunicazione.  
  
## Descrizione  
 L'apertura di un oggetto di comunicazione non è riuscita perché durante l'esecuzione del relativo metodo Open è stata generata un'eccezione.Un'altra traccia vicina potrebbe contenere dettagli sull'eccezione.
