# System.ServiceModel.Channels.PeerFloodedMessageNotMatched
Un PeerNode ha ricevuto un messaggio non corrispondente a nessun canale locale.  
  
## Descrizione  
 Si verifica durante l'elaborazione di un messaggio flood in arrivo.
