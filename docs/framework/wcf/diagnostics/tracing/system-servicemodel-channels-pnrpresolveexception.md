# System.ServiceModel.Channels.PnrpResolveException
Rilevata un'eccezione durante i tentativi di PnrpResolverService di eseguire un'operazione di risoluzione.  
  
## Descrizione  
 Questa traccia si verifica durante un'operazione di risoluzione da parte del resolver PNRP.
