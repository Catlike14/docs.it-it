# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Descrizione  
 È stato ricevuto un messaggio di risposta, ma non ci sono richieste in attesa del messaggio. Il messaggio verrà eliminato dalla fase di esecuzione di [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
 Questa traccia può essere utilizzata per il debug di messaggi persi.
