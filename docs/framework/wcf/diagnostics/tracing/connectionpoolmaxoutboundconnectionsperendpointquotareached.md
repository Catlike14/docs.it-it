# System.ServiceModel.Channels.ConnectionPoolMaxOutboundConnectionsPerEndpointQuotaReached
System.ServiceModel.Channels.ConnectionPoolMaxOutboundConnectionsPerEndpointQuotaReached  
  
## Descrizione  
 È stata raggiunta la quota di MaxOutboundConnectionsPerEndpoint, pertanto la connessione è stata chiusa e non è stata memorizzata in questo pool di connessioni.
