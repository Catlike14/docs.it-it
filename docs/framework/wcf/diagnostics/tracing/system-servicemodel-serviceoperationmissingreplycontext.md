# System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## Descrizione  
 ReplyContext è Null per un'operazione bidirezionale.Nei dati della traccia viene visualizzato un messaggio che indica che per l'operazione di risposta non è disponibile alcun IRequestContext da utilizzare per la risposta.  
  
 Questa traccia è utile per l'esecuzione del debug durante lo sviluppo.
