# Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFailed
Servizio del protocollo WS\-AT: impossibile inviare un messaggio Register al coordinatore  
  
## Descrizione  
 Viene tracciato se il gestore transazioni locale non è in grado di eseguire la registrazione con il gestore transazioni superiore a causa dell'impossibilità di inviare un messaggio.  
  
## Risoluzione dei problemi  
 Cercare nel messaggio di traccia l'eccezione che provoca l'errore di invio del messaggio.
