# System.Runtime.Serialization.ObjectWithLargeDepth
System.Runtime.Serialization.ObjectWithLargeDepth  
  
## Descrizione  
 La scrittura di un oggetto con struttura ricorsiva e notevole profondità presenta alcune limitazioni.Provare a ridurre la profondità dell'oggetto.
