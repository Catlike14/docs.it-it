# <a name="debugging-on-the-client"></a>Esecuzione del debug sul client
Per rendere più semplice per gli utenti di scrivere applicazioni client per il servizio WCF, è possibile aggiungere la [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) comportamento al file di configurazione del servizio del servizio. Questo comportamento può essere utilizzato per pubblicare pagine della Guida e per includere fra i dettagli relativi agli errori SOAP restituiti al client le informazioni sulle eccezioni gestite.
