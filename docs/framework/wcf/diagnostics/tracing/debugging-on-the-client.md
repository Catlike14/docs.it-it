# Esecuzione del debug sul client
Per semplificare la scrittura di applicazioni client per un determinato servizio [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] è possibile aggiungere il comportamento di servizio [\<debugServizio\>](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) al file di configurazione di tale servizio.Questo comportamento può essere utilizzato per pubblicare pagine della Guida e per includere fra i dettagli relativi agli errori SOAP restituiti al client le informazioni sulle eccezioni gestite.
