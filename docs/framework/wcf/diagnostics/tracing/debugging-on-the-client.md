# <a name="debugging-on-the-client"></a>Esecuzione del debug sul client
Per semplificare agli utenti di scrivere applicazioni client per il [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] servizio, è possibile aggiungere il [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) comportamento al file di configurazione del servizio del servizio. Questo comportamento può essere utilizzato per pubblicare pagine della Guida e per includere fra i dettagli relativi agli errori SOAP restituiti al client le informazioni sulle eccezioni gestite.
