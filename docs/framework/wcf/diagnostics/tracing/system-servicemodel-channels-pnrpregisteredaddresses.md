# System.ServiceModel.Channels.PnrpRegisteredAddresses
Il revolver Pnrp ha registrato gli indirizzi del nodo locale.  
  
## Descrizione  
 Questa traccia si verifica dopo la registrazione corretta degli indirizzi del nodo locale da parte del servizio PNRP.
