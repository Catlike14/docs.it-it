# System.ServiceModel.TxSourceTxScopeRequiredIsTransactedTransport
La transazione specificata è stata ricevuta per l'operazione specificata da un trasporto transazionale, ad esempio MSMQ.  
  
## Descrizione  
 Viene tracciato se la transazione resa disponibile per il codice utente è stata creata dal trasporto stesso.
