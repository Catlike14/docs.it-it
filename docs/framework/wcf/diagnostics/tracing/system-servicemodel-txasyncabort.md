# System.ServiceModel.TxAsyncAbort
La transazione specificata è stata interrotta in modo asincrono.  
  
## Descrizione  
 La transazione corrente è stata interrotta a causa di un altro partecipante che vota per l'interruzione, al verificarsi del timeout o di un altro errore all'interno del partecipante a una transazione.  
  
## Risoluzione dei problemi  
 Se si tratta di un'interruzione imprevista, esaminare tutti i registri di sistema per determinare il motivo effettivo dell'interruzione.
