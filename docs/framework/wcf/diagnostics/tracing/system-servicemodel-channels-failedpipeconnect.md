# <a name="systemservicemodelchannelsfailedpipeconnect"></a>System.ServiceModel.Channels.FailedPipeConnect
Tentativo di connettere all'endpoint della named pipe non riuscito. Verrà effettuato un altro tentativo entro il periodo di timeout specificato.  
  
## <a name="description"></a>Descrizione  
 Questa traccia informativa indica un errore di connessione all'endpoint della named pipe. Ciò può verificarsi se l'endpoint della named pipe non viene trovato o è occupato. Verranno eseguiti ulteriori tentativi a brevi intervalli di tempo fino all'esito positivo di un tentativo oppure fino alla scadenza di OpenTimeout.
