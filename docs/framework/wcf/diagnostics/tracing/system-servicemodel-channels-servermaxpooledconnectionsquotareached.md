# System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached
System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached  
  
## Descrizione  
 È stata raggiunta la quota di MaxOutboundConnectionsPerEndpoint. Pertanto, la connessione è stata chiusa e non riutilizzata dal listener.
