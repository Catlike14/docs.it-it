# <a name="end-to-end-tracing-scenarios"></a>Scenari di analisi end-to-end
Questa sezione contiene argomenti che descrivono scenari diversi per l'uso della funzione di analisi.  
  
 L'analisi delle attività viene implementata per scenari Reply/Response, OneWay e Duplex usando gli elementi seguenti:  
  
-   Modalità sincrona o asincrona  
  
-   BasicHttpBinding, netNamedPipeBinding, netTcpBinding, wsHttpBinding o netMsmqBinding  
  
-   Servizio COM  
  
-   Propagazione dell'ID attività dell'utente
