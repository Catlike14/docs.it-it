# System.ServiceModel.PerformanceCountersFailedForService
System.ServiceModel.PerformanceCountersFailedForService  
  
## Descrizione  
 Impossibile caricare i contatori delle prestazioni per il servizio.  Non sono disponibili contatori delle prestazioni per il servizio.
