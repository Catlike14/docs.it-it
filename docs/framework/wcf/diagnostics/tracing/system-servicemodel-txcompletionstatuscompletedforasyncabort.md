# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
La transazione specificata per l'operazione indicata è stata completata a causa di un'interruzione asincrona.  
  
## <a name="description"></a>Descrizione  
 La transazione corrente è stata interrotta a causa di un altro partecipante che vota per l'interruzione, al verificarsi del timeout o di un altro errore all'interno del partecipante a una transazione.  
  
## <a name="troubleshooting"></a>Risoluzione dei problemi  
 Se si tratta di un'interruzione imprevista, esaminare tutti i registri eventi di sistema per determinare il motivo effettivo dell'interruzione.
