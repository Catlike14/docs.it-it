# System.ServiceModel.TxSourceTxScopeRequiredIsCreateNewTransaction
La transazione specificata per l'operazione specificata è stata appena creata.  
  
## Descrizione  
 Viene tracciato se la transazione disponibile per il codice utente è stata appena creata dall'infrastruttura.
