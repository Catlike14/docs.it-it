# <a name="systemruntimeserializationfactorytypenotfound"></a>System.Runtime.Serialization.FactoryTypeNotFound
System.Runtime.Serialization.FactoryTypeNotFound  
  
## <a name="description"></a>Descrizione  
 Il deserializzatore non ha alcuna conoscenza del tipo da usare durante la deserializzazione. Considerare l'ipotesi di aggiungere il tipo specificato come tipo factory nel codice XML all'elenco dei tipi noti, usando ad esempio l'attributo KnownTypeAttribute.
