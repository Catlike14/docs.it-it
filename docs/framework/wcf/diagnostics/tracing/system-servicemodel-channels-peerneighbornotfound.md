# System.ServiceModel.Channels.PeerNeighborNotFound
Non è stato possibile eseguire un'operazione su un router adiacente perché non è stato trovato nell'elenco dei router adiacenti.  
  
## Descrizione  
 Questo errore si verifica durante la connessione o la chiusura di un router adiacente.
