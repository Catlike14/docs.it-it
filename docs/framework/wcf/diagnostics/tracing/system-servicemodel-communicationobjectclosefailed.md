# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Chiusura non riuscita di un oggetto di comunicazione.  
  
## <a name="description"></a>Descrizione  
 La chiusura di un oggetto di comunicazione non è riuscita perché durante l'esecuzione del suo metodo Close è stata generata un'eccezione. Un'altra traccia vicina potrebbe contenere dettagli sull'eccezione.
