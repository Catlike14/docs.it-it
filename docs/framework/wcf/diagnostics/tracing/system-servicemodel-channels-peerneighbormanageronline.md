# System.ServiceModel.Channels.PeerNeighborManagerOnline
PeerNode è in linea e indica la presenza di uno o più nodi adiacenti connessi.  
  
## Descrizione  
 Questa traccia si verifica dopo un tentativo riuscito di connessione con un router adiacente.
