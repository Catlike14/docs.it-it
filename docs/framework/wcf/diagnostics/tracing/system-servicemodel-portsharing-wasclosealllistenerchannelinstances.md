# System.ServiceModel.PortSharing.WasCloseAllListenerChannelInstances
System.ServiceModel.PortSharing.WasCloseAllListenerChannelInstances  
  
## Descrizione  
 Chiusura di tutte le istanze di canali del listener utilizzando il servizio di attivazione dei processi di Windows \(WAS, Windows Process Activation Service\).
