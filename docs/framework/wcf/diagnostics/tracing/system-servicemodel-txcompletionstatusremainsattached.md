# System.ServiceModel.TxCompletionStatusRemainsAttached
La transazione specificata dell'operazione specificata resta allegata all'elemento InstanceContext.  
  
## Descrizione  
 Traccia eseguita al termine dell'elaborazione dell'elemento ServiceOperation quando transazione non viene impostata come completata.
