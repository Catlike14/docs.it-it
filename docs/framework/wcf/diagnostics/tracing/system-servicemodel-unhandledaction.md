# System.ServiceModel.UnhandledAction
System.ServiceModel.UnhandledAction  
  
## Descrizione  
 Nessun servizio disponibile per gestire questa azione.  
  
 Tutte le tracce del contatore delle prestazioni possono essere utilizzate per eseguire il debug dei relativi problemi.
