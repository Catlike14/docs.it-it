# System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## Descrizione  
 È stata raggiunta una delle limitazioni MaxConcurrentCalls\/Instances\/Sessions e un messaggio verrà accodato.  
  
 È possibile usare questa traccia per regolare i valori relativi alla limitazione delle richieste in base al carico di produzione previsto ed effettivo.
