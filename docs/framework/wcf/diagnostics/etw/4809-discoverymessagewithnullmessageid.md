# 4809 - DiscoveryMessageWithNullMessageId
## Proprietà  
  
|||  
|-|-|  
|ID|4809|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando il messaggio di individuazione viene eliminato poiché l'intestazione del messaggio non conteneva la proprietà MessageId richiesta.  
  
## Messaggio  
 Un messaggio %1 è stato eliminato poiché l'intestazione del messaggio non conteneva la proprietà MessageId richiesta.
