|||

|Parole chiave|ServiceModel|  
|Livello|Informazioni|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato per segnalare il conteggio di channel factory memorizzate nella cache di base client.  
  
## <a name="message"></a>Messaggio  
 Il numero di channel factory memorizzato nella cache è: '%1'.  È possibile memorizzare nella cache al massimo '%2' channel factory.
