# <a name="4810---discoverymessagewithnullmessagesequence"></a>4810 - DiscoveryMessageWithNullMessageSequence
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4810|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando un messaggio di individuazione viene eliminato da DiscoveryClient poiché privo della proprietà DiscoveryMessageSequence.  
  
## <a name="message"></a>Messaggio  
 Un messaggio %1 con messageId='%2' è stato eliminato da DiscoveryClient poiché non conteneva la proprietà DiscoveryMessageSequence.
