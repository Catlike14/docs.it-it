# <a name="4812---discoverymessagewithnullreplyto"></a>4812 - DiscoveryMessageWithNullReplyTo
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4812|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando un messaggio di individuazione viene eliminato da DiscoveryClient poiché privo di indirizzo ReplyTo.  
  
## <a name="message"></a>Messaggio  
 Un messaggio di richiesta di individuazione con messageId='%1' è stato eliminato poiché non conteneva un indirizzo ReplyTo.
