# 3396 - BufferedAsyncWriteStart

## Descrizione  
 Questo evento viene generato quando la scrittura asincrona memorizzata nel buffer è stata completata.  
  
## Messaggio  
 BufferId:%1 ha completato la scrittura di '%2' byte per il flusso sottostante.
