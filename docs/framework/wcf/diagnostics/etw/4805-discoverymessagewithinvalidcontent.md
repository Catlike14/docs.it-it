# <a name="4805---discoverymessagewithinvalidcontent"></a>4805 - DiscoveryMessageWithInvalidContent
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4805|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando il messaggio di individuazione include il contenuto non valido.  
  
## <a name="message"></a>Messaggio  
 Un messaggio %1 con messageId='%2' è stato eliminato poiché conteneva contenuto non valido.
