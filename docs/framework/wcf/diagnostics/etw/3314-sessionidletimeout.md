# 3314 - SessionIdleTimeout
## Proprietà  
  
|||  
|-|-|  
|ID|3314|  
|Parole chiave|ServiceModel|  
|Livello|Errore|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## Descrizione  
 Questo evento viene generato quando la sessione si interrompe per il canale di inattività.  
  
## Messaggio  
 Interruzione sessione per canale di inattività nell'URI:'%1'.
