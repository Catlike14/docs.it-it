# 5406 - GetIssuerNameFailure
## Proprietà  
  
|||  
|-|-|  
|ID|5406|  
|Parole chiave|Sicurezza|  
|Livello|Errore|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato in caso di esito negativo del recupero del nome dell'autorità emittente.  
  
## Messaggio  
 Recupero nome autorità emittente da tokenId %1 non riuscito.
