# 4815 - EndpointDiscoverabilityEnabled
## Proprietà  
  
|||  
|-|-|  
|ID|4815|  
|Parole chiave|Individuazione|  
|Livello|Informazioni|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando l'individuabilità dell'endpoint è stata abilitata.  
  
## Messaggio  
 L'individuabilità dell'endpoint con EndpointAddress\='%1' e ListenUri\='%2' è stata abilitata.
