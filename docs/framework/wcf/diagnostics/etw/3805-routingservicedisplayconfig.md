# 3805 - RoutingServiceDisplayConfig
## Proprietà  
  
|||  
|-|-|  
|ID|3805|  
|Parole chiave|RoutingServices|  
|Livello|Dettagliato|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando il servizio di routing è configurato con RouteOnHeadersOnly.  
  
## Messaggio  
 Servizio di routing configurato con RouteOnHeadersOnly: %1, SoapProcessingEnabled: %2, EnsureOrderedDispatch: %3.
