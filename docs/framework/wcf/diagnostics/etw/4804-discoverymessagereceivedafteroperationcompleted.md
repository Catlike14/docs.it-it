# <a name="4804---discoverymessagereceivedafteroperationcompleted"></a>4804 - DiscoveryMessageReceivedAfterOperationCompleted
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4804|  
|Parole chiave|Individuazione|  
|Livello|Informazioni|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando il messaggio di individuazione viene ricevuto dopo il completamento dell'operazione.  
  
## <a name="message"></a>Messaggio  
 Un messaggio %1 con messageId='%2' è stato eliminato da DiscoveryClient poiché è stata completata l'operazione %3 corrispondente.
