# <a name="4600---messagelogeventsizeexceeded"></a>4600 - MessageLogEventSizeExceeded
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4600|  
|Parole chiave|WCFMessageLogging|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando il messaggio non è stato registrato perché supera la dimensione degli eventi ETW.  
  
## <a name="message"></a>Messaggio  
 Impossibile registrare il messaggio in quanto supera la dimensione degli eventi ETW
