# 3815 - RoutingServiceProcessingMessage
## Proprietà  
  
|||  
|-|-|  
|ID|3815|  
|Parole chiave|RoutingServices|  
|Livello|Informazioni|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando il servizio di routing elabora un messaggio.  
  
## Messaggio  
 Servizio di routing: è in corso l'elaborazione di un messaggio con ID '%1', azione '%2', URL in ingresso '%3' ricevuto nella transazione %4.
