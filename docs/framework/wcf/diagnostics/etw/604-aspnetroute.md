# 604 - AspNetRoute
## Proprietà  
  
|||  
|-|-|

|Parole chiave|RoutingServices|  
|Livello|Dettagliato|

## Descrizione  
 Questo evento viene generato quando viene aggiunta una nuova route ASP.NET '%1' con serviceType '%2' e serviceHostFactoryType '%3'.  
  
## Messaggio  
 È stata aggiunta una nuova route ASP.NET '%1' con serviceType '%2' e serviceHostFactoryType '%3'.
