# 3312 - MessageReadByEncoder

## Descrizione  
 Questo evento viene generato quando il messaggio trasmesso è stato letto dal codificatore.  
  
## Messaggio  
 Un messaggio di dimensioni '%1' byte è stato letto dal codificatore.
