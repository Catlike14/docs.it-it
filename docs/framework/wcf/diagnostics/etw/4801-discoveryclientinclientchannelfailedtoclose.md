# 4801 - DiscoveryClientInClientChannelFailedToClose
## Proprietà  
  
|||  
|-|-|  
|ID|4801|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando è impossibile chiudere DiscoveryClient creato all'interno di DiscoveryClientChannel che pertanto è stato interrotto.  
  
## Messaggio  
 Impossibile chiudere DiscoveryClient creato all'interno di DiscoveryClientChannel e pertanto è stato interrotto.
