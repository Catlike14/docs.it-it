# <a name="4806---discoverymessagewithinvalidrelatestooroperationcompleted"></a>4806 - DiscoveryMessageWithInvalidRelatesToOrOperationCompleted
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4806|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando il messaggio di individuazione viene eliminato da DiscoveryClient poiché è stata completata l'operazione corrispondente o il valore relatesTo non è valido.  
  
## <a name="message"></a>Messaggio  
 Un messaggio %1 con messageId='%2' e relatesTo='%3' è stato eliminato da DiscoveryClient poiché è stata completata l'operazione %4 corrispondente o il valore relatesTo non è valido.
