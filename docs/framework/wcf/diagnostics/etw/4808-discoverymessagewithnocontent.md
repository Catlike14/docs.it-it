# 4808 - DiscoveryMessageWithNoContent
## Proprietà  
  
|||  
|-|-|  
|ID|4808|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando un messaggio di individuazione viene eliminato poiché contenente un indirizzo ReplyTo non valido.  
  
## Messaggio  
 Un messaggio %1 è stato eliminato poiché non conteneva alcun contenuto.
