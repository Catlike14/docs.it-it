# <a name="4814---endpointdiscoverabilitydisabled"></a>4814 - EndpointDiscoverabilityDisabled
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4814|  
|Parole chiave|Individuazione|  
|Livello|Informazioni|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando l'individuabilità dell'endpoint è stata disabilitata.  
  
## <a name="message"></a>Messaggio  
 L'individuabilità dell'endpoint con EndpointAddress='%1' e ListenUri='%2' è stata disabilitata.
