# 4817 - InnerChannelCreationFailed
## Proprietà  
  
|||  
|-|-|  
|ID|4817|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando DiscoveryClientChannel non è riuscito a creare il canale con un endpoint individuato.  DiscoveryClientChannel tenterà ora di usare l'endpoint individuato disponibile successivo.  
  
## Messaggio  
 DiscoveryClientChannel non è riuscito a creare il canale con un endpoint individuato con EndpointAddress\='%1' e Via\='%2'.  DiscoveryClientChannel tenterà ora di usare l'endpoint individuato disponibile successivo.
