# 57395 - ShipAssertExceptionMessage
## Proprietà  
  
|||  
|-|-|  
|ID|57395|  
|Parole chiave|Infrastruttura|  
|Livello|Errore|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## Descrizione  
 Questo evento viene generato quando si è verificato un errore imprevisto.  
  
## Messaggio  
 Errore imprevisto.  Le applicazioni non devono tentare di gestire questo errore.  Per motivi di diagnostica, all'errore è associato questo messaggio in inglese: %1.
