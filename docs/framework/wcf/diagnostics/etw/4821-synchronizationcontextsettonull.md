## <a name="description"></a>Descrizione  
 Questo evento viene generato quando SynchronizationContext è stato impostato su null da DiscoveryClientChannel prima dell'avvio dell'operazione di ricerca.  
  
## <a name="message"></a>Messaggio  
 SynchronizationContext è stato impostato su null da DiscoveryClientChannel prima dell'avvio dell'operazione di ricerca.
