## <a name="description"></a>Descrizione  
 Questo evento viene generato quando un messaggio di risposta viene reinviato dal servizio di routing.  
  
## <a name="message"></a>Messaggio  
 Servizio di routing: è in corso il reinvio di un messaggio di risposta con azione '%1'.
