# 3826 - RoutingServiceUsingExistingTransaction
## Proprietà  
  
|||  
|-|-|

|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Questo evento viene generato quando il servizio di routing invia messaggi tramite la transazione esistente.  
  
## Messaggio  
 Il servizio di routing invierà messaggi tramite la transazione esistente '%1'.
