# <a name="4010---pendingsessionqueuefull"></a>4010 - PendingSessionQueueFull
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4010|  
|Parole chiave|ActivationServices|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando l'invio della sessione non è riuscito perché la coda delle sessioni in sospeso è piena.  
  
## <a name="message"></a>Messaggio  
 Invio di sessione per '%1' non riuscito. La coda delle sessioni in sospeso è piena con '%2' elementi in sospeso.
