# Attivazione dei processi di Windows
In questa sezione sono inclusi esempi che illustrano l'attivazione del servizio tramite protocolli di rete.  
  
## In questa sezione  
 [Attivazione di NamedPipe](../../../../docs/framework/wcf/samples/namedpipe-activation.md)  
 Viene descritto l'hosting di un servizio che utilizza i servizi di attivazione dei processi Windows \(WAS\) per attivare un servizio che comunica su named pipe.  
  
 [Attivazione TCP](../../../../docs/framework/wcf/samples/tcp-activation.md)  
 Viene descritto come ospitare un servizio che utilizza i servizi di attivazione dei processi Windows \(WAS\) per attivare un servizio che comunica mediante il protocollo net.tcp.
