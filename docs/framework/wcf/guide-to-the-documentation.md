# <a name="guide-to-the-documentation"></a>Guida alla documentazione
Contenuto della sezione vengono fornite istruzioni sulla documentazione di Windows Communication Foundation (WCF). I documenti collegati sono punti di partenza consigliati, raggruppati secondo specifici interessi e livelli di esperienza.  
  
 Per installare una versione autonoma di documentazione e un visualizzatore della Guida, scaricare il [Microsoft Windows SDK v 7.1](http://go.microsoft.com/fwlink/?LinkID=194146&clcid=0x409).  
  
## <a name="new-to-windows-communication-foundation-programming"></a>Novità della programmazione in Windows Communication Foundation  
  
-   Se si ha familiarità con la programmazione con WCF e si desidera vedere applicazioni di esempio che di lavoro, vedere gli argomenti elencati [degli esempi di Windows Communication Foundation](../../../docs/framework/wcf/samples/index.md).  
  
-   Per un'esercitazione che illustra in dettaglio i passaggi di base della creazione di un servizio WCF e un client, vedere [esercitazione introduttiva](../../../docs/framework/wcf/getting-started-tutorial.md).  
  
-   Se è interessati a concetti di WCF, vedere gli argomenti in di [panoramica dei concetti](../../../docs/framework/wcf/conceptual-overview.md) sezione.  
  
-   Per esempi di grafici delle configurazioni di sicurezza client/server, vedere [comuni scenari di sicurezza](../../../docs/framework/wcf/feature-details/common-security-scenarios.md).  
  
## <a name="programming-in-depth"></a>Informazioni dettagliate sulla programmazione  
  
-   Se si è pronti a iniziare lo sviluppo di un'applicazione, vedere [programmazione WCF di base](../../../docs/framework/wcf/basic-wcf-programming.md).  
  
-   Se si cercano informazioni aggiuntive su una particolare caratteristica o funzionalità di WCF, vedere gli argomenti in [dettagli delle funzionalità di WCF](../../../docs/framework/wcf/feature-details/index.md).  
  
-   Se si desidera estendere o personalizzare WCF per adattarli ai propri requisiti, vedere [estensione di WCF](../../../docs/framework/wcf/extending/index.md).  
  
-   Per informazioni sugli strumenti che consentono di creare ed eseguire il debug di applicazioni WCF, vedere [strumenti di Windows Communication Foundation](../../../docs/framework/wcf/tools.md).  
  
-   Utilizzo dei file XML di configurazione è dei modi principali di programmazione client e servizi WCF. Per la documentazione di riferimento per gli elementi XML utilizzati nei file di configurazione, vedere [Schema di configurazione WCF](../../../docs/framework/configure-apps/file-schema/wcf/index.md).

-   Integrazione con [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] remoting è illustrato in [la migrazione delle applicazioni .NET Remoting a WCF](../../../docs/framework/wcf/feature-details/migrating-net-remoting-applications-to-wcf.md).  
  
-   Per integrare un'applicazione COM+ esistente con un servizio WCF o un client, vedere [l'integrazione con panoramica delle applicazioni COM+](../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md).  
  
-   Per integrare un'applicazione COM esistente con un servizio WCF o un client, vedere [l'integrazione con applicazioni COM](../../../docs/framework/wcf/feature-details/integrating-with-com-applications.md).  
  
-   Per integrare un'applicazione MSMQ esistente con un servizio WCF o un client, vedere [procedura: i messaggi in coda di Exchange con endpoint WCF](../../../docs/framework/wcf/feature-details/how-to-exchange-queued-messages-with-wcf-endpoints.md) e [come: i messaggi di Exchange con endpoint WCF e applicazioni di Accodamento messaggi](../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).
