## <a name="remarks"></a><span data-ttu-id="3eacb-178">Note</span><span class="sxs-lookup"><span data-stu-id="3eacb-178">Remarks</span></span>  
 <span data-ttu-id="3eacb-179">L'elemento `<Assembly>` definisce i criteri di runtime per tutti i tipi in un assembly.</span><span class="sxs-lookup"><span data-stu-id="3eacb-179">The `<Assembly>` element defines runtime policy for all the types in an assembly.</span></span> <span data-ttu-id="3eacb-180">È diverso dall'elemento [\<Library>](../../../docs/framework/net-native/library-element-net-native.md), che specifica una libreria, ma dipende dagli elementi figlio per definire i criteri di reflection di runtime.</span><span class="sxs-lookup"><span data-stu-id="3eacb-180">It differs from the [\<Library>](../../../docs/framework/net-native/library-element-net-native.md) element, which specifies a library but depends on its child elements to define runtime reflection policy.</span></span> <span data-ttu-id="3eacb-181">L'elemento `<Assembly>` si applica a tutti i tipi in un assembly a meno che questi non siano sottoposti a override da un elemento figlio.</span><span class="sxs-lookup"><span data-stu-id="3eacb-181">The `<Assembly>` element applies to all the types in an assembly unless they are overridden by a child element.</span></span>
