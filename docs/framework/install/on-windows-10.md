# <a name="install-the-net-framework-on-windows-10-and-windows-server-2016"></a>Installare .NET Framework in Windows 10 e Windows Server 2016

È necessario .NET Framework per l'esecuzione di molte applicazioni in Windows. Le istruzioni riportate in questo articolo consentono di installare le versioni di .NET Framework che è necessario. Il [.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47) è la versione più recente.

Si potrebbe è arrivati a questa pagina dopo il tentativo di eseguire un'applicazione e visualizzare una finestra di dialogo nel computer è simile a quello riportato di seguito:

![Impossibile avviare l'applicazione](./media/this-application-could-not-be-started.png)

## <a name="net-framework-471"></a>.NET framework 4.7.1

.NET Framework 4.7.1 è incluso in:

* [Windows 10 rientrano creatori di aggiornamento (versione 1709)](https://www.microsoft.com/software-download/windows10)
* [Windows Server 2016 (versione 1709)](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709)

> [!div class="button"]
[Scaricare .NET Framework 4.7.1](https://www.microsoft.com/net/download/thank-you/net471?utm_source=ms-docs&utm_medium=referral)

Il [.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47) può essere utilizzato per eseguire le applicazioni compilate per .NET Framework 4.0 tramite 4.7.1.

È possibile installare il [.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47) su:

* Aggiornamento di creatori di Windows 10 (versione 1703)
* Aggiornamento dell'anniversario di Windows 10 (versione 1607)
* Windows Server 2016

.NET Framework 4.7.1 non è supportato in:

* Windows 10 1507
* Windows 10 1511

Se si usa Windows 10 1507 o 1511 e si desidera installare .NET Framework 4.7.1, è innanzitutto necessario eseguire l'aggiornamento a una versione successiva di Windows 10.

## <a name="net-framework-461"></a>.NET Framework 4.6.1

Il [.NET Framework 4.6.1](https://www.microsoft.com/download/details.aspx?id=49981) è la versione più recente supportata .NET Framework in Windows 10 1507 e 1511.

.NET Framework 4.6.1 supporta le app compilate per .NET Framework 4.0 tramite 4.6.1.

## <a name="net-framework-35"></a>.NET Framework 3.5

Seguire le istruzioni per l'installazione di [.NET Framework 3.5 in Windows 10](dotnet-35-windows-10.md).

.NET Framework 3.5 supporta le app compilate per .NET Framework 1.0 tramite 3.5.

## <a name="additional-information"></a>Informazioni aggiuntive

Versioni di .NET framework 4. x sono aggiornamenti sul posto da versioni precedenti. Ciò significa che le operazioni seguenti:

- È possibile avere solo una versione di .NET Framework 4. x installati nel computer.

- Se è già installata una versione successiva, è possibile installare una versione precedente di .NET Framework nel computer.

- versioni 4. x di .NET Framework possono essere utilizzate per eseguire le applicazioni compilate per .NET Framework 4.0 tramite tale versione. Ad esempio, .NET Framework 4.7 consente di eseguire le applicazioni compilate per .NET Framework 4.0 tramite 4.7. La versione più recente (.NET Framework 4.7.1) può essere utilizzata per eseguire verrà applicazioni compilate tutte le versioni di .NET Framework a partire da 4.0.

Per un elenco di tutte le versioni di .NET Framework disponibili per il download, vedere il [Scarica .NET](https://www.microsoft.com/net/download?utm_source=ms-docs&utm_medium=referral) pagina.
