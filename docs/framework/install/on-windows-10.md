# <a name="install-the-net-framework-on-windows-10"></a>Installare .NET Framework in Windows 10

Può essere necessario avere .NET Framework installato per eseguire determinate app in Windows 10. Le istruzioni seguenti possono essere usate come supporto e variano a seconda della versione di .NET Framework richiesta.

## <a name="net-framework-47"></a>.NET Framework 4.7

.NET Framework 4.7 è incluso in [Windows 10 Creators Update](https://www.microsoft.com/software-download/windows10).

Se si usa una versione precedente di Windows 10, è necessario eseguire l'aggiornamento a [Windows 10 Creators Update](https://www.microsoft.com/software-download/windows10) per ottenere .NET Framework 4.7.

.NET Framework 4.7 supporta le app create per .NET Framework 4.0 o versioni successive. Le altre versioni di .NET Framework are sono disponibili nella pagina Web [Download .NET Framework](https://www.microsoft.com/net/download/framework) (Scarica .NET Framework).

## <a name="net-framework-35"></a>.NET Framework 3.5

Seguire le istruzioni per l'installazione di [.NET Framework 3.5 in Windows 10](dotnet-35-windows-10.md).

.NET Framework 3.5 supporta le app create per .NET Framework da 1.0 a 3.5.

## <a name="help"></a>?

È possibile [contattare Microsoft per assistenza](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework) se non si riesce a installare la versione corretta di .NET Framework.
