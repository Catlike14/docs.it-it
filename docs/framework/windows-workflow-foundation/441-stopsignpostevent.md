# <a name="441--stopsignpostevent1"></a>441- StopSignpostEvent1
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|441|  
|Parole chiave|Risoluzione dei problemi|  
|Livello|Informazioni|  
|Canale|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>Descrizione  
 Quando si traccia di attività, indica che un messaggio ha terminato di attraversare il limite di un'attività nell'invio o nella ricezione.  
  
## <a name="message"></a>Messaggio  
 Limite dell'attività.  
  
## <a name="details"></a>Dettagli  
  
|Nome elemento dati|Tipo elemento dati|Descrizione|  
|--------------------|--------------------|-----------------|  
|Dati estesi|`xs:string`|Nome dell'attività.|  
|AppDomain|`xs:string`|Stringa restituita da AppDomain.CurrentDomain.FriendlyName.|
