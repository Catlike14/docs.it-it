# 39457 - TrackingRecordRaised
## Proprietà  
  
|||  
|-|-|  
|ID|39457|  
|Parole chiave|WFRuntime|  
|Livello|Informazioni|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Indica che TrackingRecord è aumentato a TrackingParticipant.  
  
## Messaggio  
 Record di rilevamento %1 aumentato a %2.  
  
## Dettagli  
  
|Nome elemento dati|Tipo elemento dati|Descrizione|  
|------------------------|------------------------|-----------------|  
|RecordNumber|xs:string|Numero del record di rilevamento.|  
|ParticipantId|xs:string|Partecipante del rilevamento.|  
|AppDomain|xs:string|Stringa restituita da AppDomain.CurrentDomain.FriendlyName.|
