# <a name="connection-string-and-connection-string-name"></a>Connection String e Connection String Name
**Stringa di connessione** proprietà specifica la stringa di connessione che l'archivio di istanze del flusso di lavoro SQL deve utilizzare per connettersi a un database di persistenza. Questo parametro è facoltativo. **Nome della stringa di connessione** proprietà specifica il nome della stringa di connessione denominata che deve usare l'archivio di istanze del flusso di lavoro SQL per connettersi al database di persistenza. Questo parametro è facoltativo. È necessario specificare un valore per la proprietà Connection String Name o la proprietà Connection String se non si desidera l'archivio di istanze del flusso di lavoro SQL per utilizzare la stringa di connessione predefinita denominata **DefaultSqlWorkflowInstanceStoreConnectionString** .
