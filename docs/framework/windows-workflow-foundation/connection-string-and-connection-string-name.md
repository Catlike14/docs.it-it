# Connection String e Connection String Name
La proprietà **Connection String** specifica la stringa di connessione che deve essere utilizzata dall'archivio di istanze del flusso di lavoro SQL per la connessione a un database di persistenza.Questo parametro è facoltativo.La proprietà **Connection String Name** specifica il nome della stringa di connessione denominata che deve essere utilizzata dall'archivio di istanze del flusso di lavoro SQL per la connessione al database di persistenza.Questo parametro è facoltativo.È necessario specificare un valore per la proprietà Connection String Name o per la proprietà Connection String se non si desidera che l'archivio di istanze del flusso di lavoro SQL utilizzi la stringa di connessione predefinita denominata **DefaultSqlWorkflowInstanceStoreConnectionString**.
