# <a name="rules-samples"></a>Esempi di regole
Contenuto della sezione sono contenuti esempi che illustrano l'uso del motore per le regole del flusso di lavoro da [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)].  
  
## <a name="in-this-section"></a>In questa sezione  
 [Criteri avanzati](../../../../docs/framework/windows-workflow-foundation/samples/advanced-policy.md)  
 Viene illustrato come estendere l'esempio di criterio semplice. Oltre alle regole di sconto residenziale e sconto aziendale dell'esempio di criterio semplice, sono state aggiunte molte nuove regole.  
  
 [Criteri semplici](../../../../docs/framework/windows-workflow-foundation/samples/simple-policy.md)  
 Viene illustrato come usare un'attività <xref:System.Workflow.Activities.PolicyActivity> in un flusso di lavoro.  
  
 [IfElse con regole](../../../../docs/framework/windows-workflow-foundation/samples/ifelse-with-rules.md)  
 Viene illustrato l'uso di una condizione della regola con un'attività <xref:System.Workflow.Activities.IfElseActivity>.  
  
 [ConditionedActivityGroup](../../../../docs/framework/windows-workflow-foundation/samples/conditioned-activity-group.md)  
 Viene illustrata un'applicazione di prenotazione di viaggio.  
  
 [Elaborazione di ordini con criteri](../../../../docs/framework/windows-workflow-foundation/samples/order-processing-with-policy.md)  
 Vengono illustrate alcune delle funzionalità chiave introdotte nel [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)] di Windows Workflow Foundation (WF).
