# <a name="validation"></a>Convalida
Contenuto della sezione sono contenuti esempi che illustrano la convalida di attività.  
  
## <a name="in-this-section"></a>In questa sezione  
 [Convalida di attività esterna](../../../../docs/framework/windows-workflow-foundation/samples/external-activity-validation.md)  
 In questo esempio viene illustrato come aggiungere la logica di convalida a un'attività incorporata di cui l'utente non è l'autore.  
  
 [Convalida di base](../../../../docs/framework/windows-workflow-foundation/samples/basic-validation.md)  
 Viene illustrata un'attività `CreateProduct` che verifica che l'argomento `Cost` sia minore o uguale al relativo argomento `Price`.  
  
 [OverloadGroups](../../../../docs/framework/windows-workflow-foundation/samples/overloadgroups.md)  
 Viene illustrata un'attività `CreateLocation` che dispone di argomenti obbligatori e di altri facoltativi e che consente all'utente di scegliere di fornire uno dei due diversi set di argomenti.  
  
 [Convalida di relazioni tra attività](../../../../docs/framework/windows-workflow-foundation/samples/activity-relationships-validation.md)  
 Viene illustrato come convalidare le relazioni tra le attività.
