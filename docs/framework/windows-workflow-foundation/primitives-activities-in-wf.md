# <a name="primitives-activities-in-wf"></a>Attività primitive in WF
In [!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] sono disponibili diverse attività fornite dal sistema che semplificano l'esecuzione di attività comuni.  
  
|Attività|Descrizione|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Assign>|Assegna un valore a una variabile nell'ambito corrente.|  
|<xref:System.Activities.Statements.Delay>|Applica uno stato inattivo a un percorso di esecuzione, consentendo possibilmente di scaricare il flusso di lavoro.|  
|<xref:System.Activities.Statements.InvokeDelegate>|Esegue un delegato che deriva da <xref:System.Activities.ActivityDelegate> ed è esposto come proprietà.|  
|<xref:System.Activities.Statements.InvokeMethod>|Esegue un metodo pubblico di un oggetto CLR.|  
|<xref:System.Activities.Statements.WriteLine>|Scrive una stringa specifica nella console o in un oggetto <xref:System.IO.TextWriter> specificato.|
