# 39459 - TrackingDataExtracted
## Proprietà  
  
|||  
|-|-|  
|ID|39459|  
|Parole chiave|WFRuntime|  
|Livello|Dettagliato|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Indica che i dati di rilevamento sono stati estratti in un'attività.  
  
## Messaggio  
 Rilevamento dati %1 estratti nell'attività %2.  
  
## Dettagli  
  
|Nome elemento dati|Tipo elemento dati|Descrizione|  
|------------------------|------------------------|-----------------|  
|Dati|xs:string|Nome dei dati estratti.|  
|Attività|xs:string|Nome dell'attività.|  
|AppDomain|xs:string|Stringa restituita da AppDomain.CurrentDomain.FriendlyName.|
