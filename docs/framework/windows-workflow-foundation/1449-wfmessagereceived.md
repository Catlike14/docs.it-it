# 1449 - WfMessageReceived
## Proprietà  
  
|||  
|-|-|  
|ID|1449|  
|Parole chiave|WFServices|  
|Livello|Informazioni|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Indica che un messaggio è stato ricevuto da un flusso di lavoro.

 Messaggio ricevuto dal flusso di lavoro
