# <a name="guide-to-the-windows-workflow-documentation"></a>Guida alla documentazione dei flussi di lavoro di Windows
In questo argomento sono contenute informazioni sull'utilizzo della documentazione di Windows Workflow Foundation. I documenti collegati sono punti di partenza consigliati, raggruppati secondo specifici interessi e livelli di esperienza.  
  
## <a name="new-to-windows-workflow-foundation-programming"></a>Novità relative alla programmazione di Windows Workflow Foundation  
  
-   Se ha esperienza di programmazione con Windows Workflow Foundation e si desidera semplicemente visualizzare alcune applicazioni di esempio, vedere gli argomenti elencati in [degli esempi di flusso di lavoro di Windows](../../../docs/framework/windows-workflow-foundation/samples/index.md).  
  
-   Per una descrizione dei vari modelli di controllo del flusso di Windows Workflow Foundation, vedere [Progettazione flussi di lavoro](../../../docs/framework/windows-workflow-foundation/designing-workflows.md).  
  
-   Per un'esercitazione che illustra i passaggi di base della creazione di un'applicazione Windows Workflow Foundation, vedere [esercitazione introduttiva](../../../docs/framework/windows-workflow-foundation/getting-started-tutorial.md).  
  
-   Se è interessati a concetti di Windows Workflow Foundation, vedere gli argomenti in di [panoramica concettuale del flusso di lavoro di Windows](../../../docs/framework/windows-workflow-foundation/conceptual-overview.md) sezione.  
  
-   Per un elenco dei termini utilizzati nella documentazione di Windows Workflow Foundation, vedere [glossario di Windows Workflow Foundation per .NET Framework 4.5](../../../docs/framework/windows-workflow-foundation/glossary.md).  
  
-   Per un elenco di nuovi concetti e funzionalità, vedere [novità di Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/whats-new.md).  
  
## <a name="programming-in-depth"></a>Informazioni dettagliate sulla programmazione  
  
-   Per un'analisi approfondita del modello a oggetti Windows Workflow Foundation, vedere [architettura del flusso di lavoro di Windows](../../../docs/framework/windows-workflow-foundation/architecture.md).  
  
-   Se si è pronti a iniziare lo sviluppo di un'applicazione, vedere [programmazione di Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/programming.md).  
  
-   Se si desidera estendere o personalizzare Windows Workflow Foundation in base alle proprie esigenze, vedere [estensione Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/extend.md).
