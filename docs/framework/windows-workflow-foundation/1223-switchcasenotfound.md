# 1223 - SwitchCaseNotFound
## Proprietà  
  
|||  
|-|-|  
|ID|1223|  
|Parole chiave|WFActivities|  
|Livello|Informazioni|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Indica che un'attività di case con un risultato corrispondente di espressione non è stata trovata in un'attività di opzione.  
  
## Messaggio  
 L'attività Switch '%1' non è in grado di trovare un'attività Case corrispondente al risultato dell'espressione.
