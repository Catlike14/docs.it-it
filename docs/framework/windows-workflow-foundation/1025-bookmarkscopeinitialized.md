# 1025 - BookmarkScopeInitialized
## Proprietà  
  
|||  
|-|-|  
|ID|1025|  
|Parole chiave|WFRuntime|  
|Livello|Dettagliato|  
|Canale|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descrizione  
 Indica che BookmarkScope è stato inizializzato.  
  
## Messaggio  
 BookmarkScope con TemporaryId: '%1' è stato inizializzato con ID: '%2'.  
  
## Dettagli  
  
|Nome elemento dati|Tipo elemento dati|Descrizione|  
|------------------------|------------------------|-----------------|  
|TemporaryId|xs:string|ID temporaneo del segnalibro.|  
|InitializedId|xs:string|ID inizializzato del segnalibro.|  
|AppDomain|xs:string|Stringa restituita da AppDomain.CurrentDomain.FriendlyName.|
