# <a name="migration-activity-in-wf"></a>Attività Migration in WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] fornisce la <xref:System.Activities.Statements.Interop> attività per l'esecuzione di attività che derivano da attività all'interno di un flusso di lavoro basato su <xref:System.Activities.Activity>. Per altre informazioni, vedere la [materiale sussidiario sulla migrazione](../../../docs/framework/windows-workflow-foundation/migration-guidance.md) sezione.  
  
> [!NOTE]
>  Il <xref:System.Activities.Statements.Interop> attività non viene visualizzato nella casella degli strumenti della finestra di progettazione del flusso di lavoro a meno che non dispone di progetto del flusso di lavoro relativo **Framework di destinazione** impostazione **.Net Framework 4** o versione successiva.
