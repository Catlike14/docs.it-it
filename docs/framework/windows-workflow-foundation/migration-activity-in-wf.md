# Attivit&#224; Migration in WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] fornisce la <xref:> System.Activities.Statements.Interop?qualifyHint=False&autoUpgrade=True attività per l'esecuzione di attività che derivano da <xref:System.WorkflowModel.Activities.Activity> all'interno di un flusso di lavoro basato su <xref:System.Activities.Activity>. [!INCLUDE[crdefault](../../../includes/crdefault-md.md)] il [Guida alla migrazione](../../../docs/framework/windows-workflow-foundation//migration-guidance.md) sezione.  
  
> [!NOTE]
>  Il <xref:System.Activities.Statements.Interop> attività non viene visualizzato nella casella degli strumenti della finestra di progettazione del flusso di lavoro a meno che non dispone di progetto del flusso di lavoro relativo **Framework di destinazione** impostazione **.Net Framework 4** o versione successiva.
