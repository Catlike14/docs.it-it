# <a name="4207---maximumretriesexceededforsqlcommand"></a>4207 - MaximumRetriesExceededForSqlCommand
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|ID|4207|  
|Parole chiave|Quota, WFInstanceStore|  
|Livello|Informazioni|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Indica che il provider SQL interrompe i tentativi di eseguire un comando SQL in quanto il numero massimo di tentativi è stato raggiunto.  
  
## <a name="message"></a>Messaggio  
 Interruzione dei tentativi di eseguire un comando SQL. Numero massimo di tentativi raggiunto.
