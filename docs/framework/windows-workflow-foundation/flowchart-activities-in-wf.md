# Attivit&#224; del diagramma di flusso in WF
In [!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] sono disponibili diverse attività fornite dal sistema per il controllo dell'esecuzione e per la creazione di un ramo all'interno di un diagramma di flusso.  
  
|||  
|-|-|  
|<xref:System.Activities.Statements.Flowchart>|Esegue attività contenute usando il paradigma del diagramma di flusso comune.|  
|<xref:System.Activities.Statements.FlowDecision>|Oggetto <xref:System.Activities.Statements.FlowNode> specifico che fornisce la possibilità di modellare un nodo condizionale con due risultati.|  
|<xref:System.Activities.Statements.FlowSwitch%601>|Oggetto <xref:System.Activities.Statements.FlowNode> specifico che consente la modellazione di un costrutto di passaggio, con un'espressione di un tipo definito nell'identificatore di tipo dell'attività e un singolo risultato per ogni corrispondenza.|
