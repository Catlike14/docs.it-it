# Utilizzo di trasformazioni nel codice gestito GDI+
Esempi di trasformazioni affini comprendono rotazione, adattamento, creazione di un riflesso, taglio e traslazione.  In [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)], la classe <xref:System.Drawing.Drawing2D.Matrix> fornisce la base per l'esecuzione di trasformazioni affini di disegni vettoriali, immagini e testo.  
  
## In questa sezione  
 [Utilizzo della trasformazione di tipo World](../../../../docs/framework/winforms/advanced/using-the-world-transformation.md)  
 Descrive la procedura per adattare e ruotare immagini utilizzando una matrice di trasformazione di tipo World.  
  
 [Importanza dell'ordine delle trasformazioni](../../../../docs/framework/winforms/advanced/why-transformation-order-is-significant.md)  
 Illustra l'importanza dell'ordine delle operazioni di trasformazione.
