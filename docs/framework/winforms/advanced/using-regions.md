# Utilizzo delle regioni
La classe <xref:System.Drawing.Region> di [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] consente di definire una forma personalizzata.  La forma può essere composta da linee, poligoni e curve.  
  
 Le regioni sono in genere utilizzate a due scopi: ridimensionamento e verifica passaggio.  La verifica passaggio consiste nel determinare se è stato fatto clic con il mouse in una determinata area dello schermo.  Il ridimensionamento consiste nel limitare il disegno a una determinata area.
