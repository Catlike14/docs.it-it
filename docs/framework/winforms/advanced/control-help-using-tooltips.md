# Visualizzazione della Guida relativa a un controllo tramite le descrizioni comandi
È possibile utilizzare il componente <xref:System.Windows.Forms.ToolTip> per visualizzare una breve descrizione specifica dei singoli controlli nei Windows Form.  Il componente <xref:System.Windows.Forms.ToolTip> fornisce una proprietà che specifica il testo visualizzato per ciascun controllo nel form.  Per ulteriori informazioni sull'utilizzo del componente <xref:System.Windows.Forms.ToolTip> in tal modo, vedere [Procedura: impostare le descrizioni comandi per i controlli in un Windows Form in fase di progettazione](../../../../docs/framework/winforms/controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md).  Il componente <xref:System.Windows.Forms.ToolTip> può essere inoltre configurato in modo che venga visualizzato dopo un determinato periodo di tempo.  Per ulteriori informazioni, vedere [Procedura: modificare il ritardo del componente ToolTip di Windows Form](../../../../docs/framework/winforms/controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md).
