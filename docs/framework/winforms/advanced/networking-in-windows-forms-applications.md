# Networking in Windows Forms Applications
.NET Framework fornisce classi per la visualizzazione delle pagine Web, il download di contenuti Web, l'interazione con i siti FTP \(File Transfer Protocol\) e l'utilizzo dei servizi Web, rendendo più agevole l'inserimento di funzionalità di rete nell'applicazione.  Le risorse descritte di seguito semplificano la comprensione delle tecnologie di rete di .NET Framework e delle modalità di integrazione di tali tecnologie in Windows Form.  
  
## Riferimenti  
 <xref:System.Net>  
 Spazio dei nomi radice per le classi in .NET Framework che gestiscono la connettività di rete.  
  
 <xref:System.Net.WebClient>  
 Classe utile per il recupero a livello di codice del contenuto basato sul Web o su HTTP.  
  
 <xref:System.Net.FtpWebRequest>  
 Classe per il recupero e l'invio di file mediante FTP.  
  
 <xref:System.Windows.Forms.WebBrowser>  
 Classe wrapper gestita per il controllo `WebBrowser` incluso in Windows.
