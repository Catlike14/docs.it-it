# <a name="networking-in-windows-forms-applications"></a>Operazioni di rete nelle applicazioni Windows Forms
.NET Framework fornisce classi per la visualizzazione delle pagine Web, scaricare il contenuto Web, l'interazione con i siti di file transfer protocol (FTP) e servizi Web, semplificando la funzionalità di rete nell'applicazione. Le risorse seguenti consentono di comprendere le tecnologie di rete di .NET Framework e come è possibile integrare in Windows Form.  
  
## <a name="reference"></a>Riferimenti  
 <xref:System.Net>  
 Lo spazio dei nomi radice per le classi in .NET Framework che gestiscono la connettività di rete.  
  
 <xref:System.Net.WebClient>  
 Classe utile per il recupero a livello di programmazione di Web o contenuto basato su HTTP.  
  
 <xref:System.Net.FtpWebRequest>  
 Una classe per il recupero e l'invio di file con FTP.  
  
 <xref:System.Windows.Forms.WebBrowser>  
 Una classe wrapper gestito per il `WebBrowser` controllo che è incluso in Windows.
