# Controlli da utilizzare in Windows Form
Di seguito è riportato un elenco alfabetico di controlli e componenti che possono essere utilizzati in Windows Form. Oltre ai controlli Windows Form descritti in questa sezione, è possibile aggiungere controlli ActiveX e controlli personalizzati a un Windows Form. Se non si trova il controllo che necessario elencati di seguito, è possibile crearne di nuovi. Per informazioni dettagliate, vedere [lo sviluppo di controlli Windows Form in fase di progettazione](../../../../docs/framework/winforms/controls/developing-windows-forms-controls-at-design-time.md). Per ulteriori informazioni sulla scelta del controllo è necessario, vedere [controlli Windows Form per funzione](../../../../docs/framework/winforms/controls/windows-forms-controls-by-function.md).  
  
> [!NOTE]
>  [!INCLUDE[vbprvb](../../../../includes/vbprvb-md.md)]controlli si basano su classi fornite dal [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)].
