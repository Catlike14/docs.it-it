# Componente FolderBrowserDialog (Windows Form)
Il componente `FolderBrowserDialog` Windows Form consente di visualizzare un'interfaccia utilizzabile per sfogliare e selezionare una cartella oppure crearne una nuova.  Si tratta di un complemento del componente [Componente OpenFileDialog](../../../../docs/framework/winforms/controls/openfiledialog-component-windows-forms.md), utilizzato per la ricerca e la selezione dei file.
