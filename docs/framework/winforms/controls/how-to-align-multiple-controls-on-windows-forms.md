# Procedura: allineare pi&#249; controlli in Windows Form
Per uniformare il layout dell'interfaccia utente \(UI, User Interface\) di un'applicazione basata su Windows, è possibile posizionare gruppi di controlli con un singolo comando.  
  
> [!NOTE]
>  È possibile che le finestre di dialogo e i comandi di menu visualizzati siano diversi da quelli descritti nella Guida a seconda delle impostazioni attive o dell'edizione del programma.  Per modificare le impostazioni, scegliere **Importa\/esporta impostazioni** dal menu **Strumenti**.  Per ulteriori informazioni, vedere [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/it-it/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### Per allineare più controlli in un form  
  
1.  Aprire il form che contiene i controlli da posizionare in **Progettazione Windows Form**.  
  
2.  Selezionare i controlli da allineare in modo che il primo controllo selezionato sia il controllo primario a cui gli altri devono allinearsi.  
  
3.  Scegliere **Allinea** dal menu **Formato**, quindi selezionare una delle sette opzioni disponibili.
