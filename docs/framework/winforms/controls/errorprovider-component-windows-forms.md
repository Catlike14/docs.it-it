# Componente ErrorProvider (Windows Form)
Il componente `ErrorProvider` Windows Form viene utilizzato per visualizzare all'utente, in maniera non invadente, che si è verificato un errore.  Viene utilizzato in genere in combinazione con la convalida dell'input dell'utente in un form, o con la visualizzazione di errori in un dataset.
