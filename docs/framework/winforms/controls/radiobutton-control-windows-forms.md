# Controllo RadioButton (Windows Form)
I controlli `RadioButton` Windows Form presentano all'utente un insieme di due o più opzioni in cui è possibile selezionare una sola opzione.  Nonostante possa apparire simile, il funzionamento dei pulsanti di opzione e delle caselle di controllo è fondamentalmente diverso. Se si seleziona un pulsante di opzione, non sarà infatti possibile selezionarne anche un altro all'interno dello stesso gruppo.
