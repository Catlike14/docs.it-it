# Controlli per Windows Form
Procedendo nella progettazione e nella modifica dell'interfaccia utente delle applicazioni Windows Form, si rende necessario aggiungere, allineare e posizionare i controlli.  I controlli sono oggetti contenuti all'interno di oggetti form.  Ogni tipo di controllo presenta un insieme di proprietà, metodi ed eventi che lo rendono adatto a un particolare scopo.  È possibile modificare i controlli all'interno della finestra di progettazione e scrivere il codice per l'aggiunta dinamica dei controlli in fase di esecuzione.
