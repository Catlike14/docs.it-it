# Controllo GroupBox (Windows Form)
I controlli <xref:System.Windows.Forms.GroupBox> Windows Form vengono utilizzati per organizzare altri controlli in gruppi identificabili.  Le caselle di gruppo vengono in genere utilizzate per suddividere un form in base alle funzioni.  Si supponga ad esempio di disporre di un form per gli ordini in cui sono specificate le opzioni di spedizione, quale il corriere notturno da utilizzare.  Il raggruppamento in una casella di gruppo di tutte le opzioni fornisce all'utente una visione d'insieme logica.  Il controllo <xref:System.Windows.Forms.GroupBox> ha caratteristiche simili a quelle del controllo <xref:System.Windows.Forms.Panel> ma solo il controllo <xref:System.Windows.Forms.GroupBox> visualizza una didascalia mentre solo il controllo <xref:System.Windows.Forms.Panel> è dotato di barre di scorrimento.
