# Ordinamento dati nel controllo DataGridView Windows Form
Per impostazione predefinita, gli utenti possono ordinare i dati in un controllo `DataGridView` facendo clic sull'intestazione di una colonna di caselle di testo.  Se necessario, è possibile modificare la proprietà `SortMode` di specifiche colonne per consentire agli utenti di effettuare l'ordinamento in base ad altri tipi di colonne.  È inoltre possibile ordinare i dati a livello di codice in base a qualsiasi colonna o in base a più colonne.
