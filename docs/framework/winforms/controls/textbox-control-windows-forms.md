# Controllo TextBox (Windows Form)
Le caselle di testo di Windows Form vengono utilizzate per ricevere l'input dall'utente o per la visualizzazione di testo.  Il controllo `TextBox` viene in genere utilizzato per il testo modificabile, anche se è possibile definire il testo in modo che risulti in sola lettura.  Le caselle di testo possono visualizzare più righe, mandare il testo a capo in base alle dimensioni del controllo e aggiungere formattazione di base.  Il controllo `TextBox` consente un unico formato per il testo visualizzato o inserito nel controllo.
