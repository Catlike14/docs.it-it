 <span data-ttu-id="b73e9-125">Inoltre, per impostazione predefinita, un <xref:System.Windows.Forms.GroupBox> controllo dispone di una propria <xref:System.Windows.Forms.Control.TabIndex%2A> valore, <span data-ttu-id="b73e9-127">Pertanto, ogni controllo all'interno di un <xref:System.Windows.Forms.GroupBox> ha un proprio decimale <xref:System.Windows.Forms.Control.TabIndex%2A> valore, che inizia
