# Utilizzare il Document Object Model HTML gestito
Il Document Object Model HTML gestito fornisce un wrapper basato su [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] per le classi HTML esposte da Internet Explorer.  Utilizzare tali classi per modificare pagine HTML incluse nel controllo <xref:System.Windows.Forms.WebBrowser> o per compilare pagine del tutto nuove.  
  
## In questa sezione  
 [Procedura: accedere al Document Object Model HTML gestito](../../../../docs/framework/winforms/controls/how-to-access-the-managed-html-document-object-model.md)  
 Viene illustrato come ottenere un'istanza valida di un oggetto <xref:System.Windows.Forms.HtmlDocument> da un'applicazione Windows Form o da un oggetto <xref:System.Windows.Forms.UserControl> incluso in Internet Explorer.  
  
 [Procedura: accedere all'origine HTML nel Document Object Model HTML gestito](../../../../docs/framework/winforms/controls/how-to-access-the-html-source-in-the-managed-html-document-object-model.md)  
 Viene illustrato come ottenere l'origine HTML originale non modificata e come ottenere l'origine attiva che riflette lo stato corrente del DOM.  
  
 [Procedura: modificare gli stili di un elemento nel Document Object Model HTML gestito](../../../../docs/framework/winforms/controls/how-to-change-styles-on-an-element-in-the-managed-html-document-object-model.md)  
 Viene descritto come modificare gli stili, utilizzati per controllare l'aspetto visivo degli elementi.  
  
 [Accesso a frame nel Document Object Model HTML gestito](../../../../docs/framework/winforms/controls/accessing-frames-in-the-managed-html-document-object-model.md)  
 Vengono descritti frame e pagine contenenti frame e illustrato come accedere al DOM di un frame.  
  
 [Accesso ai membri non esposti del Document Object Model HTML gestito](../../../../docs/framework/winforms/controls/accessing-unexposed-members-on-the-managed-html-document-object-model.md)  
 Viene illustrato come accedere a membri del DOM sottostante che non dispongono di un equivalente gestito.
