# Controllo LinkLabel (Windows Form)
Il controllo `LinkLabel` Windows Form consente di aggiungere collegamenti ipertestuali alle applicazioni Windows Form.  Il controllo `LinkLabel` può essere utilizzato per gli stessi scopi per cui si utilizza il controllo <xref:System.Windows.Forms.Label> e anche per impostare parte del testo come collegamento a un oggetto o a una pagina Web.
