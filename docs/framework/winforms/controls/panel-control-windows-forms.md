# Controllo Panel (Windows Form)
I controlli `Panel` Windows Form vengono utilizzati per organizzare altri controlli in gruppi identificabili.  I pannelli vengono in genere utilizzati per suddividere un form in base alla funzione.  Il controllo `Panel` è simile al controllo <xref:System.Windows.Forms.GroupBox>, con la differenza che il controllo `Panel` contiene barre di scorrimento mentre il controllo <xref:System.Windows.Forms.GroupBox> visualizza una didascalia.
