# Cenni preliminari sul controllo ContextMenuStrip
> [!NOTE]
>  Benché il controllo <xref:System.Windows.Forms.ContextMenuStrip> sostituisca il controllo <xref:System.Windows.Forms.ContextMenu> delle versioni precedenti aggiungendo funzionalità, il controllo <xref:System.Windows.Forms.ContextMenu> viene mantenuto per compatibilità con le versioni precedenti e per utilizzo futuro se lo si desidera.  
  
 I menu di scelta rapida vengono visualizzati in corrispondenza della posizione del mouse quando l'utente fa clic con il pulsante destro del mouse.  I *menu di scelta rapida* forniscono opzioni per l'area client o per il controllo presente in corrispondenza del puntatore del mouse.  
  
 Sebbene il controllo <xref:System.Windows.Forms.ContextMenuStrip> sia stato progettato per supportare il nuovo controllo <xref:System.Windows.Forms.ToolStrip> e i controlli correlati, è possibile associarlo altrettanto facilmente ad altri controlli.  
  
 Nella tabella riportata di seguito sono elencate classi importanti correlate al controllo <xref:System.Windows.Forms.ContextMenuStrip>.  
  
|Classe|Descrizione|  
|------------|-----------------|  
|<xref:System.Windows.Forms.ToolStripMenuItem>|Rappresenta un'opzione selezionabile visualizzata in un oggetto <xref:System.Windows.Forms.MenuStrip> o <xref:System.Windows.Forms.ContextMenuStrip>.|  
|<xref:System.Windows.Forms.ToolStripDropDown>|Rappresenta un controllo che consente all'utente di selezionare un singolo elemento da un elenco visualizzato quando fa clic su un oggetto <xref:System.Windows.Forms.ToolStripDropDownButton> o una voce di menu di livello superiore.|  
|<xref:System.Windows.Forms.ToolStripDropDownItem>|Fornisce funzionalità di base per i controlli derivati dalla classe <xref:System.Windows.Forms.ToolStripItem> che, quando selezionati, visualizzano elementi a discesa.|
