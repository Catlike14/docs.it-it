# Utilizzo della finestra di progettazione con il controllo DataGridView di Windows Form
In Visual Studio è possibile utilizzare la finestra di progettazione con il controllo `DataGridView` per svolgere numerose attività di impostazione senza scrivere codice.  Tali attività includono l'associazione del controllo a un'origine dati, la modifica delle colonne utilizzate per la visualizzazione di dati e la regolazione dell'aspetto e del comportamento di base del controllo.
