## <a name="virtual"></a><span data-ttu-id="69bed-129">Virtuale</span><span class="sxs-lookup"><span data-stu-id="69bed-129">Virtual</span></span>  
 <span data-ttu-id="69bed-130">La modalità virtuale, è possibile implementare operazioni di gestione dati.</span><span class="sxs-lookup"><span data-stu-id="69bed-130">With virtual mode, you can implement your own data management operations.</span></span> <span data-ttu-id="69bed-131">Ciò è necessario per mantenere i valori delle colonne in modalità di associazione quando il controllo viene ordinato in base a colonne associate.</span><span class="sxs-lookup"><span data-stu-id="69bed-131">This is necessary to maintain the values of unbound columns in bound mode when the control is sorted by bound columns.</span></span> <span data-ttu-id="69bed-132">L'utilizzo principale di modalità virtuale, è tuttavia, per ottimizzare le prestazioni durante l'interazione con grandi quantità di dati.</span><span class="sxs-lookup"><span data-stu-id="69bed-132">The primary use of virtual mode, however, is to optimize performance when interacting with large amounts of data.</span></span>  
  
 <span data-ttu-id="69bed-133">Si collega il <xref:System.Windows.Forms.DataGridView> controllo a una cache gestiti e i controlli di codice quando le righe di dati vengono effettuato il push e pull.</span><span class="sxs-lookup"><span data-stu-id="69bed-133">You attach the <xref:System.Windows.Forms.DataGridView> control to a cache that you manage, and your code controls when data rows are pushed and pulled.</span></span> <span data-ttu-id="69bed-134">Per ridurre il footprint di memoria, la cache deve essere simile al numero di righe attualmente visualizzate.</span><span class="sxs-lookup"><span data-stu-id="69bed-134">To keep the memory footprint small, the cache should be similar in size to the number of rows currently displayed.</span></span> <span data-ttu-id="69bed-135">Quando l'utente scorre le nuove righe nella visualizzazione, il codice richiede nuovi dati dalla cache e, facoltativamente, Elimina i dati obsoleti dalla memoria.</span><span class="sxs-lookup"><span data-stu-id="69bed-135">When the user scrolls new rows into view, your code requests new data from the cache and optionally flushes old data from memory.</span></span>  
  
 <span data-ttu-id="69bed-136">Quando si implementa la modalità virtuale, è necessario tenere traccia di una nuova riga è necessaria nel modello di dati e quando eseguire il rollback, l'aggiunta della nuova riga.</span><span class="sxs-lookup"><span data-stu-id="69bed-136">When you are implementing virtual mode, you will need to track when a new row is needed in the data model and when to rollback the addition of the new row.</span></span> <span data-ttu-id="69bed-137">L'implementazione esatta di questa funzionalità varia in base l'implementazione del modello di dati e la semantica delle transazioni del modello di dati; indica se l'ambito di commit è a livello di cella o riga.</span><span class="sxs-lookup"><span data-stu-id="69bed-137">The exact implementation of this functionality will depend on the implementation of the data model and the transaction semantics of the data model; whether commit scope is at the cell or row level.</span></span>
