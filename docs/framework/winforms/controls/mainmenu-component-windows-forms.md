# Componente MainMenu (Windows Form)
> [!NOTE]
>  Benché `MenuStrip` e `ContextMenuStrip` sostituiscano i controlli `MainMenu` e <xref:System.Windows.Forms.ContextMenu> delle versioni precedenti aggiungendovi funzionalità, `MainMenu` e <xref:System.Windows.Forms.ContextMenu> vengono mantenuti per compatibilità con le versioni precedenti e per un eventuale utilizzo futuro.  
  
 Il componente `MainMenu` Windows Form consente la visualizzazione di un menu in fase di esecuzione.
