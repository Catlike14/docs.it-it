# <a name="how-to-resize-controls-on-windows-forms"></a><span data-ttu-id="13b06-102">Procedura: ridimensionare i controlli di un Windows Form</span><span class="sxs-lookup"><span data-stu-id="13b06-102">How to: Resize Controls on Windows Forms</span></span>
<span data-ttu-id="13b06-103">È possibile ridimensionare singoli controlli e per ridimensionare più controlli del tipo uguale o diverso, ad esempio <xref:System.Windows.Forms.Button> e <xref:System.Windows.Forms.GroupBox> controlli.</span><span class="sxs-lookup"><span data-stu-id="13b06-103">You can resize individual controls, and you can resize multiple controls of the same or different kind, such as <xref:System.Windows.Forms.Button> and <xref:System.Windows.Forms.GroupBox> controls.</span></span>  
  
### <a name="to-resize-a-control"></a><span data-ttu-id="13b06-104">Per ridimensionare un controllo</span><span class="sxs-lookup"><span data-stu-id="13b06-104">To resize a control</span></span>  
  
1.  <span data-ttu-id="13b06-105">Fare clic sul controllo per ridimensionare e trascinare uno dei quadratini di otto ridimensionamento.</span><span class="sxs-lookup"><span data-stu-id="13b06-105">Click the control to be resized and drag one of the eight sizing handles.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="13b06-106">Selezionare il controllo e premere i tasti di direzione tenendo premuto il tasto MAIUSC per ridimensionare il controllo di un pixel alla volta.</span><span class="sxs-lookup"><span data-stu-id="13b06-106">Select the control and press the ARROW keys while holding down the SHIFT key to resize the control one pixel at a time.</span></span> <span data-ttu-id="13b06-107">Premere i tasti di direzione verso il basso o a destra tenendo premuto il tasto MAIUSC e CTRL per ridimensionare il controllo in incrementi di grandi dimensioni.</span><span class="sxs-lookup"><span data-stu-id="13b06-107">Press the DOWN or RIGHT arrow keys while holding down the SHIFT and CTRL keys to resize the control in large increments.</span></span>  
  
### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="13b06-108">Per ridimensionare più controlli in un form</span><span class="sxs-lookup"><span data-stu-id="13b06-108">To resize multiple controls on a form</span></span>  
  
1.  <span data-ttu-id="13b06-109">Tenere premuto il tasto CTRL o MAIUSC e selezionare i controlli da ridimensionare.</span><span class="sxs-lookup"><span data-stu-id="13b06-109">Hold down the CTRL or SHIFT key and select the controls you want to resize.</span></span> <span data-ttu-id="13b06-110">La dimensione del primo controllo selezionato viene utilizzato per gli altri controlli.</span><span class="sxs-lookup"><span data-stu-id="13b06-110">The size of the first control you select is used for the other controls.</span></span>  
  
2.  <span data-ttu-id="13b06-111">Nel **formato** menu, scegliere **Rendi uguali**e selezionare una delle quattro opzioni.</span><span class="sxs-lookup"><span data-stu-id="13b06-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="13b06-112">I primi tre comandi modificano le dimensioni dei controlli in modo che corrisponda il primo controllo selezionato.</span><span class="sxs-lookup"><span data-stu-id="13b06-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
