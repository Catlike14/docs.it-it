# <a name="how-to-copy-controls-between-windows-forms"></a>Procedura: copiare i controlli tra Windows Form
Un controllo deve essere copiato nel modulo stesso, in un altro form all'interno del progetto o negli Appunti per l'uso in altre soluzioni.  
  
### <a name="to-copy-a-control"></a>Per copiare un controllo  
  
1.  Selezionare il controllo e dal **modifica** dal menu **copia**.  
  
 Questo controllo può essere incollato ora a tutti i moduli che accetta il tipo di controllo. Inoltre, il controllo è stato aggiunto negli Appunti.
