# Procedura: copiare i controlli tra Windows Form
È possibile copiare un controllo nello stesso form, in un altro form del progetto o negli Appunti, per poterlo utilizzare in un contesto differente.  
  
### Per copiare un controllo  
  
1.  Selezionare il controllo e scegliere **Copia** dal menu **Modifica**.  
  
 È possibile a questo punto copiare il controllo in qualsiasi form che accetti quel tipo di controllo.  Il controllo è stato, inoltre, aggiunto agli Appunti.
