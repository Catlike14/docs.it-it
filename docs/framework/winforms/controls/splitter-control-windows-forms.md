# Controllo Splitter (Windows Form)
I controlli `Splitter` di Windows Form vengono utilizzati in fase di esecuzione per ridimensionare i controlli ancorati.  Il controllo `Splitter` viene spesso utilizzato in form con controlli che devono presentare dati di lunghezza variabile, ad esempio in Esplora risorse, i cui riquadri contengono informazioni la cui lunghezza può cambiare in momenti diversi.  
  
> [!NOTE]
>  Benché il controllo <xref:System.Windows.Forms.SplitContainer> sostituisca il controllo `Splitter` delle versioni precedenti aggiungendo funzionalità, il controllo `Splitter` viene mantenuto per compatibilità con le versioni precedenti e per utilizzo futuro se lo si desidera.
