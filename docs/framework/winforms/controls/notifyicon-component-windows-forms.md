# Componente NotifyIcon (Windows Form)
Il componente `NotifyIcon` di Windows Form consente di visualizzare icone nell'area di notifica dello stato della barra delle applicazioni relative ai processi eseguiti in background e che non sono dotati di altre interfacce utente.  Un esempio può essere rappresentato da un programma di protezione dai virus, al quale è possibile accedere facendo clic su un'icona riportata nell'area di notifica dello stato della barra delle applicazioni.
