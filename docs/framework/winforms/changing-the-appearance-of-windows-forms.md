# Changing the Appearance of Windows Forms
È possibile personalizzare l'aspetto delle applicazioni Windows Form in molti modi diversi, ad esempio modificando il bordo, l'opacità, la forma, lo stile o l'impostazione di un'immagine di sfondo per l'applicazione Windows Form.
