# Esempi di codice socket
Gli esempi di codice seguenti viene illustrato come utilizzare la classe <xref:System.Net.Sockets.Socket> come client per connettersi ai servizi di rete remota e come server per ascoltare le connessioni dai client remoti.  
  
## In questa sezione  
 [Esempio di socket client sincrono](../../../docs/framework/network-programming/synchronous-client-socket-example.md)  
 Viene illustrato come implementare un client sincrono <xref:System.Net.Sockets.Socket> che si connette a un server e visualizzare i dati restituiti dal server.  
  
 [Esempio di socket server sincrono](../../../docs/framework/network-programming/synchronous-server-socket-example.md)  
 Viene illustrato come implementare un server sincrono <xref:System.Net.Sockets.Socket> che accetta connessioni da un client e da echi che supportano i dati ricevuti dal client.  
  
 [Esempio di socket client asincrono](../../../docs/framework/network-programming/asynchronous-client-socket-example.md)  
 Viene illustrato come implementare un client asincrono <xref:System.Net.Sockets.Socket> che si connette a un server e visualizzare i dati restituiti dal server.  
  
 [Esempio di socket server asincrono](../../../docs/framework/network-programming/asynchronous-server-socket-example.md)  
 Viene illustrato come implementare un server <xref:System.Net.Sockets.Socket> asincrono che accetta connessioni da un client e da echi che supportano i dati ricevuti dal client.
