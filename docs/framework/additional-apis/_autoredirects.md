# <a name="httpwebrequestautoredirects-field"></a>HttpWebRequest. \_AutoRedirects campo

`HttpWebRequest._AutoRedirects` è un <xref:System.Int32> che riflette il numero di reindirizzamenti apportate per questo <xref:System.Net.HttpWebRequest>.

## <a name="syntax"></a>Sintassi  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> Il `HttpWebRequest._AutoRedirects` campo è interno e non ha significato essere utilizzato direttamente nel codice.
> 
> Microsoft non supporta l'utilizzo di questo campo in un'applicazione di produzione in qualsiasi circostanza.

## <a name="requirements"></a>Requisiti

**Namespace:** <xref:System.Net>

**Assembly:** System (System. dll)

**Versioni di .NET framework:** disponibile dalla 2.0.
