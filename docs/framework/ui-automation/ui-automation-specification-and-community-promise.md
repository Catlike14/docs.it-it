# <a name="ui-automation-specification-and-community-promise"></a>UI Automation Specification e Community Promise
Automazione interfaccia utente fornisce l'accesso a livello di codice alla maggior parte degli elementi dell'interfaccia utente sul desktop, l'abilitazione di prodotti di assistive technology, ad esempio gli screen reader per fornire informazioni sull'interfaccia utente per gli utenti finali e per modificare l'interfaccia utente con mezzi diversi dall'input standard .
