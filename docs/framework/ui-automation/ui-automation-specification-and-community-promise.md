# <a name="ui-automation-specification-and-community-promise"></a>UI Automation Specification e Community Promise
Automazione interfaccia utente fornisce accesso programmatico al maggior parte degli elementi di interfaccia utente sul desktop, consentendo a prodotti di assistive technology, ad esempio gli screen reader per fornire informazioni sull'interfaccia utente per gli utenti finali e per modificare l'interfaccia utente con mezzi diversi dall'input standard .
