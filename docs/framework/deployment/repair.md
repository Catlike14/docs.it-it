# <a name="repair-the-net-framework"></a>Ripristinare .NET Framework

In alcune situazioni l'installazione di .NET Framework può risultare danneggiata ed è necessario il ripristino. Questo potrebbe verificarsi se l'app si blocca subito dopo l'avvio o se non è possibile installare versioni più recenti di .NET Framework.

È possibile ripristinare l'installazione di .NET Framework usando lo [strumento di ripristino di .NET Framework](http://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe). 

Se l'app non si avvia ancora dopo il ripristino di .NET Framework, il problema potrebbe riguardare l'applicazione. In tal caso, contattare l'autore dell'applicazione.
