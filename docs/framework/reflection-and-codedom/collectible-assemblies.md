# <a name="collectible-assemblies-for-dynamic-type-generation"></a><span data-ttu-id="b2b6c-102">Assembly ritirabili per la generazione di tipo dinamico</span><span class="sxs-lookup"><span data-stu-id="b2b6c-102">Collectible assemblies for dynamic type generation</span></span>

<span data-ttu-id="b2b6c-103">*Assembly ritirabili* sono gli assembly dinamici che possono essere scaricati senza scaricare il dominio applicazione in cui sono stati creati.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-103">*Collectible assemblies* are dynamic assemblies that can be unloaded without unloading the application domain in which they were created.</span></span> <span data-ttu-id="b2b6c-104">È possibile recuperare tutti i memoria gestita e utilizzato da un assembly ritirabile e i tipi che contiene.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-104">All managed and unmanaged memory used by a collectible assembly and the types it contains can be reclaimed.</span></span> <span data-ttu-id="b2b6c-105">Informazioni quali il nome dell'assembly viene rimosso dalle tabelle interne.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-105">Information such as the assembly name is removed from internal tables.</span></span>

<span data-ttu-id="b2b6c-106">Per abilitare lo scaricamento, utilizzare il <xref:System.Reflection.Emit.AssemblyBuilderAccess.RunAndCollect?displayProperty=nameWithType> flag quando si crea un assembly dinamico.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-106">To enable unloading, use the <xref:System.Reflection.Emit.AssemblyBuilderAccess.RunAndCollect?displayProperty=nameWithType> flag when you create a dynamic assembly.</span></span> <span data-ttu-id="b2b6c-107">L'assembly è temporaneo (ovvero, non è possibile salvarlo) ed è soggetto alle limitazioni descritte nella [limitazioni degli assembly ritirabili](#restrictions-on-collectible-assemblies) sezione.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-107">The assembly is transient (that is, it cannot be saved) and is subject to limitations described in the [Restrictions on Collectible Assemblies](#restrictions-on-collectible-assemblies) section.</span></span> <span data-ttu-id="b2b6c-108">Quando si rilasciano tutti gli oggetti associati all'assembly, common language runtime (CLR) Scarica automaticamente un assembly ritirabile.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-108">The common language runtime (CLR) unloads a collectible assembly automatically when you release all objects associated with the assembly.</span></span> <span data-ttu-id="b2b6c-109">Tutti gli altri aspetti degli assembly ritirabili vengono creati e utilizzati nello stesso modo degli altri assembly dinamici.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-109">In all other respects, collectible assemblies are created and used in the same way as other dynamic assemblies.</span></span>

## <a name="lifetime-of-collectible-assemblies"></a><span data-ttu-id="b2b6c-110">Durata degli assembly ritirabili</span><span class="sxs-lookup"><span data-stu-id="b2b6c-110">Lifetime of collectible assemblies</span></span>

<span data-ttu-id="b2b6c-111">La durata di un assembly ritirabile è controllata dall'esistenza di riferimenti a tipi che contiene e gli oggetti creati da tali tipi.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-111">The lifetime of a collectible assembly is controlled by the existence of references to the types it contains and the objects that are created from those types.</span></span> <span data-ttu-id="b2b6c-112">Common language runtime non viene scaricato un assembly come uno o più dei seguenti esistono (`T` è qualsiasi tipo che viene definito nell'assembly):</span><span class="sxs-lookup"><span data-stu-id="b2b6c-112">The common language runtime does not unload an assembly as long as one or more of the following exist (`T` is any type that is defined in the assembly):</span></span> 

- <span data-ttu-id="b2b6c-113">Istanza di `T`.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-113">An instance of `T`.</span></span>

- <span data-ttu-id="b2b6c-114">Un'istanza di una matrice di `T`.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-114">An instance of an array of `T`.</span></span>
 
- <span data-ttu-id="b2b6c-115">Un'istanza di un tipo generico che ha `T` come uno dei relativi argomenti di tipo.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-115">An instance of a generic type that has `T` as one of its type arguments.</span></span> <span data-ttu-id="b2b6c-116">Ciò include le raccolte generiche di `T`, anche se tale raccolta è vuota.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-116">This includes generic collections of `T`, even if that collection is empty.</span></span>

- <span data-ttu-id="b2b6c-117">Un'istanza di <xref:System.Type> o <xref:System.Reflection.Emit.TypeBuilder> che rappresenta `T`.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-117">An instance of <xref:System.Type> or <xref:System.Reflection.Emit.TypeBuilder> that represents `T`.</span></span> 

   > [!IMPORTANT]
   > <span data-ttu-id="b2b6c-118">È necessario rilasciare tutti gli oggetti che rappresentano parti dell'assembly.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-118">You must release all objects that represent parts of the assembly.</span></span> <span data-ttu-id="b2b6c-119">Il <xref:System.Reflection.Emit.ModuleBuilder> che definisce `T` mantiene un riferimento al <xref:System.Reflection.Emit.TypeBuilder>e <xref:System.Reflection.Emit.AssemblyBuilder> oggetto mantiene un riferimento al <xref:System.Reflection.Emit.ModuleBuilder>, pertanto è necessario rilasciare i riferimenti a questi oggetti.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-119">The <xref:System.Reflection.Emit.ModuleBuilder> that defines `T` keeps a reference to the <xref:System.Reflection.Emit.TypeBuilder>, and the <xref:System.Reflection.Emit.AssemblyBuilder> object keeps a reference to the <xref:System.Reflection.Emit.ModuleBuilder>, so references to these objects must be released.</span></span> <span data-ttu-id="b2b6c-120">Anche l'esistenza di un <xref:System.Reflection.Emit.LocalBuilder> o <xref:System.Reflection.Emit.ILGenerator> utilizzata nella costruzione di `T` impedisce lo scaricamento.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-120">Even the existence of a <xref:System.Reflection.Emit.LocalBuilder> or an <xref:System.Reflection.Emit.ILGenerator> used in the construction of `T` prevents unloading.</span></span>

- <span data-ttu-id="b2b6c-121">Un riferimento statico a `T` da un altro tipo definito in modo dinamico `T1` che è ancora raggiungibile mediante l'esecuzione del codice.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-121">A static reference to `T` by another dynamically defined type `T1` that is still reachable by executing code.</span></span> <span data-ttu-id="b2b6c-122">Ad esempio, `T1` possono derivare da `T`, o `T` potrebbe essere il tipo di un parametro in un metodo di `T1`.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-122">For example, `T1` might derive from `T`, or `T` might be the type of a parameter in a method of `T1`.</span></span>
 
- <span data-ttu-id="b2b6c-123">Oggetto **ByRef** a un campo statico a cui appartiene `T`.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-123">A **ByRef** to a static field that belongs to `T`.</span></span>

- <span data-ttu-id="b2b6c-124">Oggetto <xref:System.RuntimeTypeHandle>, <xref:System.RuntimeFieldHandle>, o <xref:System.RuntimeMethodHandle> che fa riferimento a `T` o a un componente di `T`.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-124">A <xref:System.RuntimeTypeHandle>, <xref:System.RuntimeFieldHandle>, or <xref:System.RuntimeMethodHandle> that refers to `T` or to a component of `T`.</span></span>

- <span data-ttu-id="b2b6c-125">Un'istanza di qualsiasi oggetto di reflection che può essere utilizzata indirettamente o direttamente per accedere il <xref:System.Type> oggetto che rappresenta `T`.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-125">An instance of any reflection object that could be used indirectly or directly to access the <xref:System.Type> object that represents `T`.</span></span> <span data-ttu-id="b2b6c-126">Ad esempio, il <xref:System.Type> oggetto `T` può essere ottenuto da un tipo di matrice il cui tipo di elemento è `T`, o da un tipo generico che ha `T` come argomento di tipo.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-126">For example, the <xref:System.Type> object for `T` can be obtained from an array type whose element type is `T`, or from a generic type that has `T` as a type argument.</span></span> 

- <span data-ttu-id="b2b6c-127">Un metodo `M` nello stack di chiamate di un thread qualsiasi, in cui `M` è un metodo di `T` o un metodo a livello di modulo che viene definito nell'assembly.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-127">A method `M` on the call stack of any thread, where `M` is a method of `T` or a module-level method that is defined in the assembly.</span></span>

- <span data-ttu-id="b2b6c-128">Un delegato a un metodo statico è definito in un modulo dell'assembly.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-128">A delegate to a static method that is defined in a module of the assembly.</span></span>

<span data-ttu-id="b2b6c-129">Se un elemento dall'elenco esiste solo per un solo tipo o di un metodo nell'assembly, il runtime non è possibile scaricare l'assembly.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-129">If only one item from this list exists for only one type or one method in the assembly, the runtime cannot unload the assembly.</span></span>

> [!NOTE]
> <span data-ttu-id="b2b6c-130">Il runtime non scarica l'assembly fino a quando non sono eseguiti i finalizzatori per tutti gli elementi nell'elenco.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-130">The runtime does not actually unload the assembly until finalizers have run for all items in the list.</span></span>

<span data-ttu-id="b2b6c-131">Per tener traccia della durata, un tipo generico costruito, ad esempio `List<int>` (in c#) o `List(Of Integer)` (in Visual Basic) che viene creata e utilizzata viene considerata la generazione di un assembly ritirabile sono stati definiti nell'assembly che contiene il tipo generico definizione del tipo o in un assembly che contiene la definizione di uno dei relativi argomenti di tipo.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-131">For purposes of tracking lifetime, a constructed generic type such as `List<int>` (in C#) or `List(Of Integer)` (in Visual Basic) that is created and used in the generation of a collectible assembly is considered to have been defined either in the assembly that contains the generic type definition or in an assembly that contains the definition of one of its type arguments.</span></span> <span data-ttu-id="b2b6c-132">L'assembly esatto utilizzato è un dettaglio di implementazione e soggette a modifiche.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-132">The exact assembly that is used is an implementation detail and subject to change.</span></span>
 
## <a name="restrictions-on-collectible-assemblies"></a><span data-ttu-id="b2b6c-133">Limitazioni degli assembly ritirabili</span><span class="sxs-lookup"><span data-stu-id="b2b6c-133">Restrictions on collectible assemblies</span></span>

<span data-ttu-id="b2b6c-134">Le restrizioni seguenti si applicano agli assembly ritirabile:</span><span class="sxs-lookup"><span data-stu-id="b2b6c-134">The following restrictions apply to collectible assemblies:</span></span> 

- <span data-ttu-id="b2b6c-135">**Riferimenti statici** </span><span class="sxs-lookup"><span data-stu-id="b2b6c-135">**Static references** </span></span>  
  <span data-ttu-id="b2b6c-136">Tipi in un assembly dinamico comune non possono avere riferimenti statici a tipi definiti in un assembly ritirabile.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-136">Types in an ordinary dynamic assembly cannot have static references to types that are defined in a collectible assembly.</span></span> <span data-ttu-id="b2b6c-137">Se si definisce un tipo comune che eredita da un tipo in un assembly ritirabile, ad esempio un <xref:System.NotSupportedException> viene generata un'eccezione.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-137">For example, if you define an ordinary type that inherits from a type in a collectible assembly, a <xref:System.NotSupportedException> exception is thrown.</span></span> <span data-ttu-id="b2b6c-138">Un tipo in un assembly ritirabile può avere riferimenti statici a un tipo in un altro assembly ritirabile, ma estende la durata dell'assembly di riferimento per la durata dell'assembly di riferimento.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-138">A type in a collectible assembly can have static references to a type in another collectible assembly, but this extends the lifetime of the referenced assembly to the lifetime of the referencing assembly.</span></span>

- <span data-ttu-id="b2b6c-139">**Interoperabilità COM** </span><span class="sxs-lookup"><span data-stu-id="b2b6c-139">**COM interop** </span></span>  
   <span data-ttu-id="b2b6c-140">Nessuna interfaccia COM può essere definita all'interno di un assembly ritirabile e non le istanze di tipi all'interno di un assembly ritirabile possono essere convertite in oggetti COM.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-140">No COM interfaces can be defined within a collectible assembly, and no instances of types within a collectible assembly can be converted into COM objects.</span></span> <span data-ttu-id="b2b6c-141">Un tipo in un assembly ritirabile non può fungere da un COM callable wrapper (CCW) o di un runtime callable wrapper (RCW).</span><span class="sxs-lookup"><span data-stu-id="b2b6c-141">A type in a collectible assembly cannot serve as a COM callable wrapper (CCW) or runtime callable wrapper (RCW).</span></span> <span data-ttu-id="b2b6c-142">Tuttavia, tipi negli assembly ritirabili possono utilizzare gli oggetti che implementano le interfacce COM.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-142">However, types in collectible assemblies can use objects that implement COM interfaces.</span></span>

- <span data-ttu-id="b2b6c-143">**PInvoke** </span><span class="sxs-lookup"><span data-stu-id="b2b6c-143">**Platform invoke** </span></span>  
   <span data-ttu-id="b2b6c-144">Metodi che presentano il <xref:System.Runtime.InteropServices.DllImportAttribute> attributo non verrà compilato quando vengono dichiarati in un assembly ritirabile.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-144">Methods that have the <xref:System.Runtime.InteropServices.DllImportAttribute> attribute will not compile when they are declared in a collectible assembly.</span></span> <span data-ttu-id="b2b6c-145">Il <xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> istruzione non può essere utilizzata nell'implementazione di un tipo in un assembly ritirabile e tali tipi non è possibile effettuare il marshalling a codice non gestito.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-145">The <xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> instruction cannot be used in the implementation of a type in a collectible assembly, and such types cannot be marshaled to unmanaged code.</span></span> <span data-ttu-id="b2b6c-146">Tuttavia, è possibile chiamare codice nativo usando un punto di ingresso che è dichiarato in un assembly non ritirabile.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-146">However, you can call into native code by using an entry point that is declared in a non-collectible assembly.</span></span>
 
- <span data-ttu-id="b2b6c-147">**Marshalling** </span><span class="sxs-lookup"><span data-stu-id="b2b6c-147">**Marshaling** </span></span>  
   <span data-ttu-id="b2b6c-148">Impossibile effettuare il marshalling di oggetti (in particolare, delegati) che vengono definiti negli assembly ritirabile.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-148">Objects (in particular, delegates) that are defined in collectible assemblies cannot be marshaled.</span></span> <span data-ttu-id="b2b6c-149">Si tratta di una restrizione per tutti i tipi generati temporanei.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-149">This is a restriction on all transient emitted types.</span></span>

- <span data-ttu-id="b2b6c-150">**Caricamento di assembly** </span><span class="sxs-lookup"><span data-stu-id="b2b6c-150">**Assembly loading** </span></span>  
   <span data-ttu-id="b2b6c-151">La reflection emit è l'unico meccanismo supportato per il caricamento degli assembly ritirabili.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-151">Reflection emit is the only mechanism that is supported for loading collectible assemblies.</span></span> <span data-ttu-id="b2b6c-152">Gli assembly caricati con qualsiasi altra forma di caricamenti di assembly non possono essere scaricati.</span><span class="sxs-lookup"><span data-stu-id="b2b6c-152">Assemblies that are loaded by using any other form of assembly loading cannot be unloaded.</span></span>
 
- <span data-ttu-id="b2b6c-153">**Oggetti associati al contesto**  </span><span class="sxs-lookup"><span data-stu-id="b2b6c-153">**Context-bound objects**  </span></span>
