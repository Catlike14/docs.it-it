# <a name="type-providers"></a><span data-ttu-id="5a127-103">Provider di tipi</span><span class="sxs-lookup"><span data-stu-id="5a127-103">Type Providers</span></span>

<span data-ttu-id="5a127-104">Un provider di tipi F# è un componente che fornisce tipi, proprietà e metodi da utilizzare in un programma.</span><span class="sxs-lookup"><span data-stu-id="5a127-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="5a127-105">Provider di tipi generano i cosiddetti **forniti tipi**, che vengono generati dal compilatore F #, basati su un'origine dati esterna.</span><span class="sxs-lookup"><span data-stu-id="5a127-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="5a127-108">Forniti che tipi dipendono dai parametri di input per un Provider di tipi.</span><span class="sxs-lookup"><span data-stu-id="5a127-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="5a127-109">Input di questo tipo può essere un'origine dati di esempio (ad esempio un file di schema JSON), un URL che punta direttamente a un servizio esterno o una stringa di connessione a un'origine dati.</span><span class="sxs-lookup"><span data-stu-id="5a127-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="5a127-110">Un Provider di tipi può inoltre garantire che i gruppi di tipi siano espandibili solamente su richiesta; ovvero, vengono espansi se i tipi vengono effettivamente fatto riferimento dal programma.</span><span class="sxs-lookup"><span data-stu-id="5a127-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="5a127-111">Questo consente di utilizzare l'integrazione diretta su richiesta di spazi di informazioni su larga scala fortemente tipizzati, come i mercati di dati online.</span><span class="sxs-lookup"><span data-stu-id="5a127-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

<span data-ttu-id="5a127-117">Cancellazione provider di tipi di generare tipi che possono essere utilizzati solo nell'assembly o nel progetto da che vengono generati.</span><span class="sxs-lookup"><span data-stu-id="5a127-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="5a127-118">I tipi sono effimero; vale a dire, essi non vengono scritti in un assembly e non può essere utilizzati dal codice in altri assembly.</span><span class="sxs-lookup"><span data-stu-id="5a127-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="5a127-119">Possono contenere *ritardato* membri, consentendo ai tipi utilizza fornita da uno spazio di informazioni potenzialmente infinito.</span><span class="sxs-lookup"><span data-stu-id="5a127-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="5a127-120">Sono utili per l'utilizzo di un piccolo subset di un'origine dati di grandi dimensioni e interconnessi.</span><span class="sxs-lookup"><span data-stu-id="5a127-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
